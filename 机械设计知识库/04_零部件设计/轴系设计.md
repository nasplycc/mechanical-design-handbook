## 1. 轴的设计
<!-- 来源: 第7篇轴及其连接 第44页 -->

### 1.1 轴的分类
<!-- 来源: 第7篇轴及其连接 第4页 -->
| 类型 | 受力特点 | 典型应用 |
|------|----------|----------|
| 心轴 | 只受弯矩、不传递扭矩 | 滑轮轴、车轮轴 |
| 传动轴 | 主要受扭矩、弯矩小 | 汽车传动轴、万向节 |
| 转轴 | 同时受弯矩和扭矩 | 齿轮轴、电机轴 |

### 1.2 轴的材料
<!-- 来源: 第7篇轴及其连接 第36页 -->
| 材料 | σ_b(MPa) | σ_s(MPa) | 特点 | 应用 |
|------|----------|----------|------|------|
| Q235 | 375~500 | 235 | 便宜、焊接好 | 不重要轴 |
| 45钢 | 600 | 355 | 综合好、最常用 | 一般轴 |
| 40Cr | 980 | 785 | 高强度 | 重要轴 |
| 20CrMnTi | 1080 | 835 | 渗碳表面硬 | 重载轴 |
| 38CrMoAlA | 980 | 835 | 氮化变形小 | 精密轴 |

### 1.3 轴的设计步骤
<!-- 来源: 第8篇轴承 第13页 -->
**第一步：按扭矩估算最小直径**
```
d_min ≥ C·∛(P/n)

C值：
Q235: C=160
45钢: C=126
40Cr: C=112
P(kW), n(rpm), d(mm)

计算后：有键槽时
单键：d×1.03
双键：d×1.07
```

**第二步：结构设计**
- 确定各段直径和长度
- 轴肩定位：定位轴肩高度 h = (0.07~0.1)d
- 非定位轴肩高度 h = 1.5~3mm
- 轴端倒角：1×45° 或 1.5×45°
- 砂轮越程槽、退刀槽
- 配合面过盈配合处加倒角

**第三步：强度校核**
- 弯扭合成强度校核（第三强度理论）
- 疲劳强度安全系数校核
- 静强度校核（过载工况）

### 1.4 弯扭合成强度校核
<!-- 来源: 第7篇轴及其连接 第20页 -->
```
σ_ca = √(σ² + 4τ²) = √((M/W)² + 4(T/Wp)²) = √(M² + (αT)²)/W

其中：
M - 合成弯矩 M = √(M_H² + M_V²)
T - 扭矩
α - 扭矩折合系数：
  静载荷 α=0.3
  脉动循环 α=0.6
  对称循环 α=1.0
W - 抗弯截面系数 W = πd³/32

校核条件：σ_ca ≤ [σ₋₁]_b
[σ₋₁]_b：碳钢 55~60 MPa，合金钢 70~80 MPa
```

### 1.5 轴的刚度校核
<!-- 来源: 第7篇轴及其连接 第28页 -->
**弯曲刚度：**
```
f_max ≤ [f]  （挠度）
θ_max ≤ [θ]  （转角）

简支梁中点集中力：f = FL³/(48EI)
许用挠度 [f]：一般轴 0.0003~0.0005L
             精密轴 0.0002L
许用转角 [θ]：0.001~0.005 rad
```

**扭转刚度：**
```
φ = (TL)/(GIp) × 180/π ≤ [φ]  (°/m)

许用扭转角 [φ]：
一般传动轴 0.5~1.0°/m
精密轴 0.25~0.5°/m
```

### 1.6 轴的结构设计要点
<!-- 来源: 第6篇连接与紧固 第296页 -->
- 轴上零件定位可靠（轴肩、套筒、圆螺母）
- 便于加工（同轴轴段直径相近、键槽在同一母线上）
- 减小应力集中（圆角过渡、避免截面突变）
- 良好的工艺性（标准件配合处用标准公差）
- 考虑装配（导向倒角、螺纹退刀槽）

## 2. 联轴器
<!-- 来源: 第6篇连接与紧固 第3页 -->

### 2.1 联轴器分类
<!-- 来源: 第8篇轴承 第13页 -->
| 类型 | 特点 | 补偿能力 | 代表 |
|------|------|----------|------|
| 刚性联轴器 | 无补偿、结构简单 | 无 | 凸缘联轴器、套筒联轴器 |
| 无弹性元件挠性联轴器 | 可补偿位移、无缓冲 | 有 | 齿式、万向、滑块、链条 |
| 有弹性元件挠性联轴器 | 缓冲减振、补偿位移 | 有 | 弹性套柱销、弹性柱销、梅花形 |

### 2.2 常用联轴器选型
<!-- 来源: 第8篇轴承 第13页 -->
| 联轴器 | 补偿量 | 特点 | 应用 |
|--------|--------|------|------|
| 凸缘联轴器 | 无补偿 | 简单可靠、对中要求高 | 低速、对中好 |
| 弹性套柱销联轴器(GB/T 4323) | 轴向±2~4mm, 角向≤30' | 缓冲减振 | 一般传动 |
| 弹性柱销联轴器(GB/T 5014) | 轴向±2~4mm, 角向≤30' | 耐冲击 | 正反转、启动频繁 |
| 梅花形弹性联轴器(GB/T 5272) | 轴向±2~6mm, 角向≤1° | 结构简单 | 中小功率 |
| 齿式联轴器(GB/T 3852) | 轴向±3~8mm, 角向≤30' | 承载大、寿命长 | 重载 |
| 万向联轴器 | 角向≤10°~45° | 大角度传动 | 汽车、机床 |
| 滑块联轴器 | 径向≤0.5~2mm | 结构简单 | 低速 |

### 2.3 联轴器选型计算
<!-- 来源: 第8篇轴承 第13页 -->
```
T_ca = K_A·T ≤ [T_n]
n ≤ [n_max]

其中：
T_ca - 计算扭矩(N·m)
K_A - 工况系数（见表）
T - 名义扭矩 T = 9550·P/n (N·m)
[T_n] - 联轴器公称扭矩
[n_max] - 联轴器许用最高转速
[n] - 实际转速

工况系数 K_A：
工作机 | K_A
轻载（发电机、风机） | 1.3~1.5
中载（机床、压缩机） | 1.5~2.0
重载（破碎机、起重机） | 2.0~3.0
```

## 3. 离合器
<!-- 来源: 第6篇连接与紧固 第3页 -->

### 3.1 离合器分类
<!-- 来源: 第8篇轴承 第13页 -->
| 类型 | 特点 | 代表 |
|------|------|------|
| 嵌合式离合器 | 无相对滑动、传递精确 | 牙嵌式 |
| 摩擦式离合器 | 可平稳接合、过载打滑 | 圆盘摩擦、圆锥摩擦 |
| 自动离合器 | 自动接合/分离 | 离心、安全、定向 |

### 3.2 牙嵌式离合器
<!-- 来源: 第8篇轴承 第13页 -->
```
齿面挤压应力：σ_p = F/(z·b·h/2) ≤ [σ_p]
齿根弯曲应力：σ = F·h/(z·b²/6) ≤ [σ]

[σ_p]：20~40 MPa（静载）
[σ]：80~120 MPa
```

### 3.3 圆盘摩擦离合器
<!-- 来源: 第8篇轴承 第13页 -->
```
T = f·F·z·(D+d)/(4)

其中：
f - 摩擦系数（钢-钢 0.06~0.15，钢-青铜 0.05~0.12）
F - 轴向压紧力(N)
z - 摩擦面数
D,d - 摩擦面外径/内径(mm)

摩擦面平均压强：p = 4F/(π·(D²-d²)) ≤ [p]
[p]：钢-钢 0.15~0.25 MPa（接合状态）
     钢-青铜 0.3~0.5 MPa
```

### 3.4 常用安全离合器
<!-- 来源: 第8篇轴承 第13页 -->
| 类型 | 特点 | 应用 |
|------|------|------|
| 滚珠式安全离合器 | 可调扭矩、复位方便 | 过载保护 |
| 摩擦片式安全离合器 | 连续打滑、不损坏 | 张力控制 |
| 剪切销安全离合器 | 简单、需更换销 | 一次性保护 |
