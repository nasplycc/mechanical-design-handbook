### 1.1 按受力形式分类
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 类型 | 结构简图特征 | 承载方式 | 特点 | 典型应用 |
|:---:|:---:|:---:|:---|:---|

| **拉伸弹簧** | 密卷圆钩环 | 轴向拉伸力 | 初始有预紧力，钩环处应力集中 | 拉力计、农机具、玩具、离合器 |

### 3.2 压缩弹簧基本设计公式
<!-- 来源: 第2卷 第6篇连接与紧固 第67页 -->

| 项目 | 公式 | 说明 |
|:---|:---|:---|
| **刚度** | $$k = \frac{F}{f} = \frac{Gd^4}{8nD^3}$$ | G—切变模量(MPa); n—有效圈数 |
| **变形量** | $$f = \frac{8FD^3n}{Gd^4}$$ | F—轴向载荷(N) |
| **最大工作载荷** | $$F_{\max} = \frac{\pi d^3 [\tau]}{8KD}$$ | [τ]—许用切应力(MPa) |
| **最大变形量** | $$f_{\max} = \frac{8F_{\max}D^3n}{Gd^4}$$ | 对应最大载荷的变形 |
| **剪应力** | $$\tau = \frac{8KFD}{\pi d^3}$$ | K—曲度系数 |
| **曲度系数** | $$K = \frac{4C - 1}{4C - 4} + \frac{0.615}{C}$$ | 考虑曲率和切应力分布不均 |
| **弹簧指数** | $$C = \frac{D}{d}$$ | 推荐范围 4~16，常用 5~10 |
| **焊点校核** | $$\tau_{\max} \leq [\tau]$$ | 最大剪应力不超过许用值 |
| **临界载荷** | $$F_{cr} = k \cdot H_0 \cdot \frac{1}{2} \left[ 1 - \sqrt{1 - \frac{4\pi^2\mu}{b^2}} \right]$$ | μ—端点系数; b—高径比 |

### 3.3 压缩弹簧几何尺寸计算
<!-- 来源: 第2卷 第6篇连接与紧固 第195页 -->

| 项目 | 公式 | 说明 |
|:---|:---|:---|
| 弹簧指数(旋绕比) | C = D/d | 推荐 4~16 |
| 节距 | $$p = d + \frac{f_{\max}}{n} + \delta_{\min}$$ | δ_min—最小间隙(0.1d) |
| 自由高度YI型(并紧磨平) | H₀ = np + 1.5d | 两端各并紧 0.75d |
| 自由高度YII型(并紧不磨) | H₀ = np + 2d | 两端各并紧 1d |
| 总圈数YI型 | n₁ = n + 2 | 考虑两端支撑圈 |
| 总圈数YII型 | n₁ = n + 2.5 | 考虑两端支撑圈 |
| 压并高度 | H_b = n₁d | YI型 |
| 压并高度 | H_b = n₁d | YII型 |
| 螺旋角 | $$\alpha = \arctan\left(\frac{p}{\pi D}\right)$$ | 推荐 5°~9° |
| 钢丝展开长度 | $$L = \frac{\pi D n_1}{\cos\alpha}$$ | 近似 L = 3.2n₁D |
| 质量 | m = ρ · L · πd²/4 | ρ = 7.85 × 10⁻⁶ kg/mm³ (钢) |

### 3.4 拉伸弹簧基本设计公式
<!-- 来源: 第2卷 第6篇连接与紧固 第67页 -->

| 项目 | 公式 | 说明 |
|:---|:---|:---|
| **刚度** | $$k = \frac{F - F_0}{f} = \frac{Gd^4}{8nD^3}$$ | F₀—初拉力 |
| **变形量** | $$f = \frac{8(F - F_0)D^3n}{Gd^4}$$ | 需扣除初拉力段 |
| **初拉力** | $$\tau_0 = \frac{8KF_0 D}{\pi d^3}$$ | 卷绕成型时产生产生 |
| **初应力范围** | $$\tau_0 = G \cdot \frac{\delta}{D} \cdot \frac{0.5}{C^2 - 1}$$ | δ—初始间隙 |
| **经验初拉应力** | $$\tau_0' = (0.20\sim0.35) \frac{\sigma_b}{S}$$ | S—安全系数 |
| **最大工作载荷** | $$F_{\max} = \frac{\pi d^3[\tau]}{8KD} + F_0$$ | — |
| **剪应力** | $$\tau = \frac{8KFD}{\pi d^3}$$ | 与压缩弹簧相同形式 |

### 算例2：拉伸弹簧设计
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**设计要求**：设计一个圆柱螺旋拉伸弹簧，工作载荷范围 F_min = 60N → F_max = 140N，工作行程 f_work = 25mm，载荷变化频率不高。常温工作环境。

**步骤一：选择材料**

| 项目 | 取值 | 依据 |
|:---|:---|:---:|
| 材料 | 碳素弹簧钢丝 C级 | 成本低，性价比高 |
| 簧丝直径 | d = 3.0 mm | 初选 |
| 抗拉强度 σ_b | 1620 MPa | 查C级表(d=3.0)，取中值 |
| 许用切应力 [τ] | 0.32σ_b = 518 MPa | 脉动载荷 |
| 切变模量 G | 79000 MPa | — |

**步骤二：选择弹簧指数并计算曲度系数**

| 项目 | 取值 | 计算 |
|:---|:---|:---:|
| 弹簧指数 | C = 8 | 拉伸弹簧常用8~10 |
| 曲度系数 | K = 1.184 | K = (4×8-1)/(4×8-4) + 0.615/8 |
| 弹簧中径 | D = 24 mm | D = C·d = 8×3 |
| 弹簧外径 | D₂ = 27 mm | — |
| 弹簧内径 | D₁ = 21 mm | — |

**步骤三：计算有效圈数**

所需刚度：

$$k = \frac{F_{\max} - F_{\min}}{f_{\text{work}}} = \frac{140 - 60}{25} = 3.2 \text{ N/mm}$$

有效圈数：

$$n = \frac{Gd^4}{8kD^3} = \frac{79000 \times 3^4}{8 \times 3.2 \times 24^3} = \frac{79000 \times 81}{8 \times 3.2 \times 13824} = \frac{6399000}{353894.4} \approx 18.08$$

取 n = 18 圈。

**步骤四：初拉力计算**

查初应力表（C=8时τ₀' = 35~55 MPa），取τ₀' = 42 MPa：

$$F_0 = \frac{\pi d^3 \tau_0'}{8KD} = \frac{\pi \times 27 \times 42}{8 \times 1.184 \times 24} = \frac{3562.6}{227.3} \approx 15.7 \text{ N}$$

取 F₀ = 16 N。

**步骤五：校核最大工作载荷应力**

最大工作载荷 F_max = 140 N：

$$\tau_{\max} = \frac{8K(F_{\max} - F_0)D}{\pi d^3} = \frac{8 \times 1.184 \times (140 - 16) \times 24}{\pi \times 27} = \frac{8 \times 1.184 \times 124 \times 24}{\pi \times 27}$$

$$\tau_{\max} = \frac{28197.1}{84.82} \approx 332.5 \text{ MPa}$$

校核：τ_max = 332.5 < [τ] = 518 MPa，满足！

**步骤六：校核最小工作载荷应力**

F_min = 60 N：

$$\tau_{\min} = \frac{8K(F_{\min} - F_0)D}{\pi d^3} = \frac{8 \times 1.184 \times (60 - 16) \times 24}{\pi \times 27} = \frac{10000.7}{84.82} \approx 117.9 \text{ MPa}$$

**步骤七：校核实际变形量**

对应 F_max 的总变形：

$$f_{\max} = \frac{8(F_{\max} - F_0)D^3n}{Gd^4} = \frac{8 \times 124 \times 13824 \times 18}{79000 \times 81} = \frac{247463936}{6399000} \approx 38.67 \text{ mm}$$

对应 F_min 的变形：

$$f_{\min} = \frac{8(F_{\min} - F_0)D^3n}{Gd^4} = \frac{8 \times 44 \times 13824 \times 18}{79000 \times 81} = \frac{87638016}{6399000} \approx 13.70 \text{ mm}$$

工作行程：f_work = 38.67 - 13.70 = 24.97 ≈ 25 mm ✓

**步骤八：钩环强度校核**

钩环弯曲半径 R = 2d = 6mm：

$$\sigma = \frac{32F_{\max}R}{\pi d^3} = \frac{32 \times 140 \times 6}{\pi \times 27} = \frac{26880}{84.82} \approx 316.9 \text{ MPa}$$

许用弯曲应力 [σ_b] = 0.45σ_b = 729 MPa，σ < [σ_b]，满足。

| 材料 | — | 碳素弹簧钢丝 C级 | — |

### 算例3：扭转弹簧设计（补充算例）
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**设计要求**：用于某机构复位，安装扭矩 M = 500 N·mm，最大扭转角 φ_max = 90°，材料为 60Si2Mn。

**步骤一：初选参数**

| 项目 | 取值 | 计算 |
|:---|:---|:---:|
| 选材 | 60Si2Mn | — |
| 弹性模量 E | 206000 MPa | — |
| 许用弯曲应力 [σ_b] | 800 MPa | 脉动载荷取 0.5σ_b |
| 簧丝直径 | d = 2.5 mm | 初选 |
| 弹簧指数 | C = 8 | — |
| 弹簧中径 | D = 20 mm | D = C·d |
| 弯曲曲度系数 | K_b = 1.097 | K_b = (4×64-8-1)/(4×8×7) |

**步骤二：有效圈数计算**

$$n = \frac{Ed^4\varphi}{3660 \cdot M \cdot D} = \frac{206000 \times 39.0625 \times 90}{3660 \times 500 \times 20} = \frac{724218750}{36600000} \approx 19.79$$

取 n = 20 圈。

**步骤三：应力校核**

实际扭矩按 M = 500 N·mm：

$$\sigma = \frac{32M}{\pi d^3}K_b = \frac{32 \times 500}{\pi \times 15.625} \times 1.097 = \frac{16000}{49.087} \times 1.097 \approx 357.4 \text{ MPa}$$

σ = 357.4 < [σ_b] = 800 MPa，满足要求。

|:---|:---:|

$$\sigma_{OM} = -\frac{4E}{1-\mu^2} \cdot \frac{t^2}{K_1 D^2} \cdot K_2 \cdot \frac{f}{t}$$

实际载荷由刚度分配：F_总=k_总×f
先确定内簧刚度
k_内=79000×20736/(8×n×262144)=1638144000/(8n×262144)=1638144000/2097152n=781.25n N/mm
取n_内=10，k_内=78.1N/mm
k_总=k_外+k_内=225.7+78.1=303.8N/mm
在F=18000N时，f=18000/303.8=59.2mm>50mm需调整
调整n_外=8，n_内=8：k_外=197.5N/mm，k_内=97.7N/mm，k_总=295.2N/mm
f_max=18000/295.2=61mm
f_外=9000/197.5=45.6mm，f_内=9000/97.7=92.1mm
弹簧并联变形相等，f=61mm，F_外=197.5×61=12047N，F_内=97.7×61=5959N

**步骤6：间隙校核**
外簧内径D₁_外=80mm，内簧外径D₂_内=76mm
径向间隙=(80-76)/2=2mm ✓ 满足同心条件
旋向：外簧右旋，内簧左旋（防止卡滞）

**步骤7：应力校核**
外簧：τ=8×1.311×12047×100/(π×8000)=12636390/25133=502.8MPa<550MPa ✓
内簧：τ=8×1.292×5959×64/(π×1728)=3941750/5428.7=726MPa>550MPa ✗ 超应力
需再调整，可增大d_内至14mm...
（实际工程中多次迭代，此处略）

### 16.2 弹簧常见失效模式与对策
| 失效模式 | 外观特征 | 失效原因 | 改进措施 |
|:--------:|:--------|:--------|:--------|
| 疲劳断裂 | 断口有贝壳纹（疲劳辉纹） | 交变应力过大/表面缺陷 | 喷丸处理/加大d/改善表面 |
| 应力松弛 | 自由高度减小/永久变形 | 工作应力过高/温度超限 | 提高回火温度/立定处理 |
| 腐蚀疲劳 | 断口有腐蚀产物 | 腐蚀介质+交变应力 | 换不锈钢/加镀层/润滑保 |
| 氢脆断裂 | 断口齐平，无塑性变形 | 电镀后未充分去氢 | 镀后200°C回火2h |
| 磨损失效 | 接触面表面磨损 | 导杆/导套间隙不当 | 改善配合/加润滑 |
| 脆性断裂 | 断口晶粒状 | 回火不足/材料缺陷 | 控制热处理工艺 |
| 热松驰 | 高温下逐步缩短 | 温度超过材料耐热极限 | 改用耐热材料 |
| 接触疲劳 | 表面麻点/剥落 | 接触应力过大 | 减小间隙/改善表面硬度 |

### 8.7 弹簧设计常见问题与对策
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 常见问题 | 原因分析 | 对策 |
|:---|:---|:---|
| 弹簧过早断裂 | 应力过大或应力集中 | 增大d或减小C；钩环加大R；喷丸处理 |

### 8.8 弹簧自振频率计算
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

| 项目 | 公式 | 说明 |
|:---|:---:|:---|
| 压缩弹簧自振频率 | $$f_n = \frac{d}{2\pi n D^2} \sqrt{\frac{G}{2\rho}}$$ | ρ—材料密度 (kg/mm³) |
| 简化公式（钢制） | $$f_n \approx \frac{356d}{nD^2}$$ | 钢制弹簧近似值，Hz |
| 避免共振条件 | f < 0.85f_n 或 f > 1.15f_n | 避开共振区 |
