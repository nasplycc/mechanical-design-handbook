### 1.2 轴承代号规则（GB/T 272-2017）
<!-- 来源: 第8篇轴承 第13页 -->
```
类型代号 + 尺寸系列代号 + 内径代号

例：6208
6 - 深沟球轴承
2 - 尺寸系列（宽度系列0 + 直径系列2）
08 - 内径 d = 8×5 = 40mm

例：30210
3 - 圆锥滚子轴承
02 - 尺寸系列（宽度系列0 + 直径系列2）
10 - 内径 d = 10×5 = 50mm

内径代号规则：
00 → d=10mm
01 → d=12mm
02 → d=15mm
03 → d=17mm
04及以上 → d=代号×5mm
```

### 1.3 轴承选型步骤
<!-- 来源: 第8篇轴承 第13页 -->
**第一步：确定载荷类型**
- 纯径向 → 深沟球/圆柱滚子
- 径向+轴向 → 深沟球/角接触/圆锥滚子
- 纯轴向 → 推力球
- 冲击载荷 → 滚子轴承
- 高速 → 球轴承

**第二步：初选型号**
```
C ≥ P·(n·L_h/10⁶)^(1/ε)

其中：
C - 基本额定动载荷(N)
P - 当量动载荷(N)
n - 转速(rpm)
L_h - 预期寿命(h)
ε - 指数：球轴承ε=3，滚子轴承ε=10/3

预期寿命 L_h：
一般机械 5000~10000h
机床/电机 8000~12000h
汽车 4000~8000h
重型机械 10000~20000h
连续工作机械 20000~50000h
```

**第三步：计算当量动载荷**
```
P = X·F_r + Y·F_a

其中：
F_r - 径向载荷(N)
F_a - 轴向载荷(N)
X,Y - 径向/轴向载荷系数

深沟球轴承(X,Y)：
F_a/F_r ≤ e → X=1, Y=0
F_a/F_r > e → X=0.56, Y=查表

圆锥滚子轴承：
P = F_r/X + Y·F_a
```

**第四步：静强度校核**
```
P₀ = X₀·F_r + Y₀·F_a
C₀ ≥ S₀·P₀

S₀ - 静安全系数：
旋转：1~2
不旋转/缓慢摆动：0.5~1
重要场合：2~4
```

### 1.6 轴承润滑
<!-- 来源: 第8篇轴承 第32页 -->
| 润滑方式 | 适用条件 | 特点 |
|----------|----------|------|
| 脂润滑 | dn值<200000 | 简单、密封好 |
| 油润滑 | dn值>200000 | 散热好、可过滤 |
| 油雾润滑 | 高速精密 | 润滑充分 |
| 固体润滑 | 高温/真空 | 特殊场合 |

**脂润滑填充量：** 1/3~1/2 轴承空间（高速取1/3，低速取1/2）
**润滑油粘度：** 一般运动粘度 ν = 20~30 mm²/s（40°C）

### 1.7 轴承安装与拆卸
<!-- 来源: 第8篇轴承 第13页 -->
- 安装：热装（油浴80~100°C）或冷压（专用套筒）
- 严禁直接敲击套圈
- 游隙调整：圆锥滚子/角接触轴承需调整游隙/预紧

## 2. 滑动轴承
<!-- 来源: 第8篇轴承 第13页 -->

### 2.1 滑动轴承分类
<!-- 来源: 第8篇轴承 第4页 -->
| 类型 | 特点 | 应用 |
|------|------|------|
| 整体式径向滑动轴承 | 简单、磨损不可调 | 低速轻载 |
| 剖分式径向滑动轴承 | 可调间隙、装拆方便 | 一般机械 |
| 推力滑动轴承 | 承受轴向载荷 | 水轮机、立式电机 |

### 2.2 滑动轴承设计（非液体摩擦）
<!-- 来源: 第8篇轴承 第13页 -->
**压强校核：**
```
p = F/(d·B) ≤ [p]

其中：
F - 径向载荷(N)
d - 轴颈直径(mm)
B - 轴承宽度(mm)
```

**pv值校核：**
```
p·v ≤ [pv]

其中：
v = π·d·n/(60×1000) - 轴颈圆周速度(m/s)
[pv] - 许用pv值(MPa·m/s)
```

**速度校核：**
```
v ≤ [v]
```
