## 1. 概述
<!-- 来源: 第2卷 第8篇轴承 第105页 -->

### 1.1 润滑的作用
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 作用 | 说明 |
|------|------|
| **减少摩擦** | 在摩擦表面间形成润滑油膜，降低摩擦系数 |
| **降低磨损** | 减少表面直接接触，减轻磨粒磨损和粘着磨损 |
| **冷却散热** | 带走摩擦热，防止零件过热 |
| **密封防护** | 填充间隙，防止杂质侵入，防止泄漏 |
| **减振降噪** | 油膜的阻尼作用可吸收振动能量 |
| **清洁冲洗** | 循环润滑可带走磨屑和污物 |
| **防锈防腐** | 润滑油膜隔绝水气和腐蚀介质，防止表面锈蚀 |

### 1.2 密封的作用
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 作用 | 说明 |
|------|------|
| **防止泄漏** | 防止流体（气体、液体）从工作腔向外泄漏 |
| **防止污染** | 防止外部杂质（灰尘、水、磨屑）进入系统 |
| **保持压力** | 维持系统工作压力，保证系统正常运行 |
| **节约能源** | 减少泄漏损失，提高系统效率 |
| **环境保护** | 防止润滑剂泄漏污染环境 |

### 1.3 摩擦状态
<!-- 来源: 第2卷 第8篇轴承 第4页 -->

| 润滑状态 | 膜厚参数 λ | 摩擦系数 f | 特点 |
|----------|-----------|-----------|------|
| **干摩擦** | — | 0.1~0.5 | 无润滑，磨损严重 |
| **边界摩擦** | λ < 1 | 0.05~0.2 | 油膜不完整，部分表面接触 |
| **混合摩擦** | 1 ≤ λ ≤ 3 | 0.01~0.08 | 部分流体润滑，部分边界润滑 |
| **流体摩擦** | λ > 3 | 0.001~0.01 | 摩擦表面完全被油膜隔开 |
| **弹流润滑(EHL)** | — | 0.001~0.01 | 接触变形+流体动压效应，用于滚动轴承、齿轮 |

> **注**: 膜厚参数 λ = h_min / (R_q₁² + R_q₂²)¹/²，其中 h_min 为最小油膜厚度，R_q₁、R_q₂ 为两表面均方根粗糙度。

### 4.1 润滑脂的组成
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

润滑脂由**基础油** + **稠化剂** + **添加剂**三部分组成：

| 组成 | 占比 | 作用 |
|------|------|------|
| 基础油（矿物油/合成油） | 70~95% | 提供润滑性能主体 |
| 稠化剂（皂基/非皂基） | 5~30% | 形成凝胶结构，保持油不流失 |
| 添加剂 | 0~10% | 改善性能（抗氧、极压、防锈等） |

### 4.3 润滑脂的NLGI稠度等级
<!-- 来源: 第2卷 第8篇轴承 第33页 -->

| NLGI等级 | 针入度 (0.1mm) | 外观状态 | 适用场合 |

| 00 | 400~430 | 半流体 | 集中润滑、减速器 |
| 0 | 355~385 | 半流体 | 集中润滑、高流动性场合 |
| **1** | 310~340 | 较软 | 中低温、集中供脂 |
| **2** | 265~295 | 中等 | 通用、最常用 |
| **3** | 220~250 | 较硬 | 高温、重载 |

### 6.2 各润滑方式详细说明
<!-- 来源: 第2卷 第6篇连接与紧固 第83页 -->

#### 6.2.1 脂润滑
<!-- 来源: 第2卷 第8篇轴承 第4页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 将润滑脂填充在轴承或摩擦副周围，利用脂的附着力保持润滑 |
| **适用转速** | 低速~中速（dn值 < 2×10⁵ mm·r/min） |
| **适用温度** | -20~+120°C（根据脂的类型） |
| **填充量** | 轴承空间填充1/3~1/2，过多会导致温升 |
| **补充周期** | 依工况而定，通常每3~6个月或运行周期内补充 |
| **优点** | 密封简单、维护方便、不易泄漏 |
| **缺点** | 转速受限、散热差、更换困难 |

**脂润滑的 dn 值限制**（dn = 轴承内径 mm × 转速 r/min）：

| 润滑脂类型 | 极限 dn 值 |
|-----------|-----------|
| 钙基润滑脂 | 1.5×10⁵ |
| 钠基润滑脂 | 2.0×10⁵ |
| 锂基润滑脂 | 2.5~3.0×10⁵ |
| 复合锂基脂 | 4.0×10⁵ |
| 特种高速脂 | 5.0~8.0×10⁵ |

#### 6.2.2 油浴润滑
<!-- 来源: 第2卷 第8篇轴承 第237页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 轴承或齿轮部分浸入油池中，旋转带起油液 |
| **适用** | 低速~中速，水平轴 |
| **油位高度** | 轴承最下方滚动体中心位置（约1/3） |
| **优点** | 结构简单、散热好（相对脂润滑） |
| **缺点** | 转速高时搅油损失大、温升快 |

**油浴润滑的油位控制**：
- 水平轴：油位到达轴承最下方滚珠中心
- 立轴：油位到轴承下部滚珠的一半
- 齿轮：大齿轮浸入油中 1~2 个齿高

#### 6.2.3 飞溅润滑
<!-- 来源: 第2卷 第8篇轴承 第14页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 利用齿轮或甩油盘将油甩溅到轴承和其他摩擦面上 |
| **适用** | 齿轮箱、减速器 |
| **优点** | 部件不需单独供油、结构简单 |
| **缺点** | 油量控制困难，高速时搅油严重 |
| **油位** | 保证齿轮浸入深度约2~3倍齿高 |

#### 6.2.4 循环润滑
<!-- 来源: 第2卷 第8篇轴承 第264页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 通过油泵将油强制输送到各润滑点，再回流到油箱 |
| **适用** | 高速、重载、连续运行设备 |
| **供油压力** | 0.1~0.5 MPa |
| **供油量** | 根据发热量计算 |
| **过滤精度** | 20~40 μm |
| **优点** | 供油量可控、散热好、可过滤清洁 |
| **缺点** | 系统复杂、成本高、需要维护 |

**循环润滑系统组成**：
```
油箱 → 粗滤器 → 油泵 → 精滤器 → 冷却器 → 分配阀 → 润滑点
    ↑                                                │
    └────────────────── 回流管 ──────────────────────┘
```

#### 6.2.5 喷油润滑
<!-- 来源: 第2卷 第7篇轴及其连接 第257页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 用喷嘴将高压润滑油喷射到摩擦副上（尤用于高速滚动轴承） |
| **适用** | 高速轴承（dn > 10⁶）、齿轮啮合区 |
| **供油压力** | 0.2~0.4 MPa |
| **喷嘴位置** | 距轴承10~30mm，对准保持架与内圈间隙 |
| **优点** | 穿透性强、冷却效果好 |
| **缺点** | 需要高压油泵，成本高 |
| **高速应用** | 机床主轴、航空发动机主轴承（dn 可达 2×10⁶~3×10⁶） |

#### 6.2.6 油雾润滑
<!-- 来源: 第2卷 第8篇轴承 第238页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 压缩空气将润滑油雾化成 1~5μm 颗粒，随气流送入润滑点 |
| **适用** | 高速滚动轴承、链传动 |
| **供油量** | 极小，可精确控制 |
| **优点** | 耗油少、散热效果好 |
| **缺点** | 排出的油雾污染环境，需收集处理 |
| **压缩空气压力** | 0.3~0.6 MPa |

#### 6.2.7 油气润滑
<!-- 来源: 第2卷 第8篇轴承 第265页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 间歇式气液两相流，微小油滴被气流输送 |
| **适用** | 超高速轴承、精密主轴 |
| **优点** | 精确供油、润滑充分、环境友好 |
| **速度范围** | dn 值可达 3×10⁶ 以上 |
| **应用** | 高速磨头、电主轴、精密机床 |

#### 6.2.8 滴油润滑
<!-- 来源: 第2卷 第8篇轴承 第4页 -->

| 项目 | 说明 |
|------|------|
| **原理** | 通过滴油器（油杯）将油逐滴导入摩擦副 |
| **适用** | 开式齿轮、链条、导轨等 |
| **滴油量** | 每分钟数滴到数十滴 |
| **优点** | 结构简单、成本低 |
| **缺点** | 供油不均匀，需人工调节 |

## 8. 润滑油的选择与应用
<!-- 来源: 第2卷 第8篇轴承 第105页 -->

### 8.1 润滑油选择原则
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 考虑因素 | 选择依据 |
|----------|----------|
| **运动速度** | 速度越高，选粘度越低的油；高速时需低粘度以减少内摩擦 |
| **载荷大小** | 载荷越大，选粘度越高的油；重载需高粘度以维持油膜 |
| **工作温度** | 高温选高温油或高VI油；低温选低凝点油或合成油 |
| **环境条件** | 湿气、粉尘多时需防锈、密封；有酸碱性需耐腐 |
| **设备类型** | 不同设备需满足特定标准（如液压油需抗磨、汽轮机油需抗乳化） |
| **接触材料** | 与铜合金接触时需避免硫化物活性、与橡胶接触时需考虑相容性 |
| **经济性** | 在满足性能前提下选性价比高的油品 |

### 8.2 设备类型与推荐粘度选择
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

|------------------|---------------|-------------|

| 0.5~1.0 | 68~100 | 中 |
| 1.0~5.0 | 46~68 | 中 |
| 5.0~10.0 | 32~46 | 低~中 |
| > 10.0 | 15~32 | 低 |

#### 滚动轴承
<!-- 来源: 第2卷 第8篇轴承 第6页 -->

|---------------|-----------------|----------------|

## 9. 润滑脂的选择与应用
<!-- 来源: 第2卷 第8篇轴承 第193页 -->

### 9.1 润滑脂选择原则
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 考虑因素 | 选择要点 |
|----------|----------|
| **转速** | 高速时选低针入度（较软）脂，低速重载时选高针入度（较硬）脂 |
| **载荷** | 重载时需极压(EP)润滑脂 |
| **温度** | 高温选高滴点、高抗氧化脂；低温选基础油粘度低的脂 |
| **环境介质** | 潮湿选抗水性好的钙基/锂基/复合钙基脂；粉尘多选稠度较高的脂 |
| **摩擦副类型** | 滚动轴承用锂基脂，滑动面用含固体润滑剂脂 |
| **供脂方式** | 集中供脂系统选 NLGI 0~1号，手工加脂选2~3号 |

### 9.2 润滑脂补充周期
<!-- 来源: 第2卷 第8篇轴承 第268页 -->

| 工作条件 | 轴承类型 | 补充周期 |
|----------|----------|----------|
| 良好（常温、低速、清洁） | 小型电机轴承 | 6~12个月 |
| 一般（中等转速、有粉尘） | 一般机械轴承 | 3~6个月 |
| 苛刻（高温、高速、湿气） | 风机、泵轴承 | 1~3个月 |
| 恶劣（高温重载、多水） | 轧机、矿山设备 | 1~4周 |
| 密封轴承 | — | 免补充（一次性寿命润滑） |

**补充量估算**：

$$G = 0.005DB$$

其中 G — 补充量 (g)，D — 轴承外径 (mm)，B — 轴承宽度 (mm)。

## 14. 机械密封
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

### 14.1 机械密封工作原理
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

机械密封由一对垂直于旋转轴的端面——动环（与轴旋转）和静环（固定在壳体上）——在弹簧力和流体压力的作用下紧密贴合，形成密封端面。端面间维持一层极薄的液膜，实现密封与润滑的平衡。

### 14.2 机械密封基本结构
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

```
密封端面       辅助密封
┌────────┐    ┌────────┐
│ 动环   │━━━━│O型圈   │
│ 静环   │━━━━│O型圈   │
└────────┘    └────────┘
   ↑               ↑
 弹簧          传动件/紧定
```

**机械密封组成部分**：

| 组件 | 功能 |
|------|------|
| 动环（旋转环） | 与轴一起旋转，形成密封端面之一 |
| 静环（固定环） | 固定在壳体中，形成密封端面之二 |
| 弹簧（单个或多个） | 提供初始端面贴合力 |
| 辅助密封（O型圈/垫片） | 轴与动环间、壳与静环间的次密封 |
| 传动件 | 传递扭矩 |
| 防转件 | 防止静环旋转 |

### 16.1 密封类型选用流程
<!-- 来源: 第2卷 第8篇轴承 第184页 -->

```
确定密封部位特征（静/动、轴径、压差、温度、介质）
              ↓
         ┌────┴────┐
         │ 静密封？ │
         └────┬────┘
              ↓
         ┌────┴────┐
         │ 是：     │ ─→ 法兰/螺纹/平面 ─→ 垫片/O型圈/密封胶
         │ 否：     │
         └────┬────┘
              ↓
         ┌────┴────┐
         │ 旋转轴？ │
         └────┬────┘
              ↓
         ┌────┴────────────────────────────┐
         │ 是 ─→ 速度/压力/温度落实类型     │
         │     ├─ 低速(<5m/s)：毡圈/间隙封   │
         │     ├─ 中速(<15m/s)：唇形油封     │
         │     ├─ 高速：机械密封/迷宫密封    │
         │     └─ 极高pV：机械密封           │
```

### 16.2 按使用条件选用密封类型
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 使用条件 | 推荐密封类型 |
|----------|--------------|
| 低速(<3m/s)、低压、普通油 | 毡圈、唇形油封 |
| 中速(3~15m/s)、低压 | 唇形油封（骨架油封） |
| 高速(>15m/s)、低压 | 机械密封、迷宫密封 |
| 低压(<0.5MPa)、油 | 唇形密封 |
| 高压(>0.5MPa)、油 | 机械密封、填料密封 |
| 高压(>5MPa)、水 | 机械密封 |
| 高温(>200°C) | 金属垫片、迷宫、机械密封（高温型） |
| 低温(<−40°C) | PTFE、硅橡胶密封件 |
| 腐蚀性介质 | PTFE密封、氟橡胶密封件、机械密封（SiC） |
| 含颗粒介质 | 耐磨机械密封（WC-WC）、带冲洗 |

| 无泄漏要求 | 机械密封、双端面机械密封+隔离液系统 |
| 易燃易爆介质 | 双端面机械密封+隔离液/API Plan 52/53 |

### 16.3 按压力选用密封
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

| 压力范围 (MPa) | 静密封 | 动密封（旋转） | 动密封（往复） |
|---------------|--------|--------------|--------------|

## 17. 润滑与密封的常见故障及对策
<!-- 来源: 第2卷 第8篇轴承 第13页 -->

### 17.1 润滑系统常见故障
<!-- 来源: 第2卷 第8篇轴承 第105页 -->

| 故障现象 | 可能原因 | 对策 |
|----------|----------|------|
| 油温过高 | 油位过高/过低、粘度不当、负载过大 | 调整油位、更换合适粘度油品 |
| 油压偏低 | 油泵故障、滤网堵塞、管路泄漏 | 检查油泵、清洗过滤器、修补泄漏 |
| 油品乳化 | 水分混入、油品抗乳化性差 | 换油、检查密封、改进油箱结构 |
| 油品起泡 | 油位不当、回油管位置不当、添加剂耗尽 | 调整油位、改进回油管路、换油 |
| 轴承温升快 | 润滑脂过多/过少、脂型号不当 | 控制加脂量（1/3~1/2）、换脂 |
| 润滑脂泄漏 | 过量填充、密封失效 | 控制填充量、更换或维修密封 |
| 供油量不足 | 油泵磨损、管路堵塞 | 维修油泵、清洗管路 |
| 油液变质 | 氧化、杂质、使用时间过长 | 定期换油、加强过滤 |
| 集中润滑系统堵塞 | 润滑脂过稠、管路过长 | 选合适稠度脂、调整管路设计 |

### 17.2 密封系统常见故障
<!-- 来源: 第2卷 第8篇轴承 第105页 -->

| 故障现象 | 可能原因 | 对策 |
|----------|----------|------|
| 油封漏油 | 唇口磨损、轴表面划伤、安装偏斜 | 更换油封、修复轴面、正确安装 |
| 油封干磨 | 润滑不足 | 检查油位、确保唇口有油 |
| 机械密封泄漏量大 | 端面磨损、安装不当、弹簧失效 | 更换密封、重新安装、检查组件 |
| 机械密封寿命短 | pV值过高、冲洗不足、介质含颗粒 | 降低pV、改善冲洗、加过滤器 |
| O型圈密封失稳 | 压缩率不适当、沟槽尺寸不合、材料不当 | 调整压缩率、修正沟槽、换材料 |
| 垫片泄漏 | 压缩量不足、法兰变形、垫片老化 | 按规定力矩拧紧、换垫片 |
| 迷宫密封泄漏大 | 间隙过大、齿数不足 | 调整间隙、增加齿数 |
| 填料函渗漏 | 填料磨损、压盖偏松 | 调整压盖、更换填料 |
| 活塞密封内漏 | 密封件磨损失效 | 更换活塞密封件 |
