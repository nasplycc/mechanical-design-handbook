## 1 润滑基础理论
<!-- 来源: 第8篇轴承 第13页 -->

### 1.1 润滑状态分类
<!-- 来源: 第8篇轴承 第13页 -->

| 润滑状态 | 油膜厚度(h)与粗糙度(σ)之比λ | 摩擦系数f | 典型应用 |
|---------|--------------------------|----------|---------|
| 边界润滑 | λ < 1 | 0.08~0.20 | 低速重载、启动/停止阶段 |
| 混合润滑 | 1 ≤ λ ≤ 3 | 0.02~0.08 | 一般齿轮、轴承 |
| 弹性流体动力润滑(EHL) | 接触区局部 | 0.001~0.05 | 滚动轴承、齿轮啮合 |
| 流体动力润滑(HD) | h >> σ | 0.001~0.01 | 滑动轴承(径向/推力) |
| 静压润滑 | 外部供压 | 近0 | 精密主轴、重型转台 |

### 1.2 Stribeck曲线要点
<!-- 来源: 第8篇轴承 第13页 -->

- **纵轴**：摩擦系数 f
- **横轴**：轴承特性数 ηN/P（粘度×转速/载荷）
- **三个区段**：
  - I区（边界润滑）：高摩擦，表面直接接触
  - II区（混合润滑）：摩擦系数急剧下降
  - III区（流体润滑）：摩擦系数最低且平稳

### 1.3 润滑油膜形成的必要条件
<!-- 来源: 第8篇轴承 第13页 -->

1. 相对运动速度足够
2. 足够的油液供给
3. 润滑油粘度适当
4. 表面粗糙度匹配
5. 合适的工作温度范围

### 1.4 润滑剂的主要功能
<!-- 来源: 第7篇轴及其连接 第295页 -->

| 功能 | 说明 | 量化指标 |
|------|------|---------|
| 减摩 | 降低摩擦系数 | f 值、磨损率 |
| 冷却 | 带走摩擦热 | 比热容、导热系数 |
| 防锈防蚀 | 在金属表面形成保护膜 | 防锈性试验 |
| 密封 | 辅助密封间隙 | 粘度、粘附性 |
| 清洁 | 冲走磨屑和污染物 | 清净分散性 |
| 减振 | 油膜阻尼作用 | 阻尼系数 |

### 3.9 润滑脂选用原则
<!-- 来源: 第8篇轴承 第13页 -->

| 因素 | 推荐方向 | 依据 |
|------|---------|------|
| 高速 | 低NLGI等级(1#~2#)、低粘度基础油 | 降低搅动阻力、温升 |
| 低速重载 | 高NLGI等级(2#~3#)、高粘度基础油、极压脂 | 提高承载、油膜强度 |
| 高温 | 高滴点脂(复合锂、聚脲)、抗氧化 | 防止流失、氧化失效 |
| 低温 | 低凝点合成基础油、低NLGI | 保持流动性、启动扭矩小 |
| 潮湿环境 | 耐水性好的脂(钙基、钡基、复合铝基) | 防止乳化流失 |
| 振动场所 | 高NLGI等级(3#~4#)、良好机械安定性 | 防止被振出 |
| 集中润滑 | 低NLGI(0#~1#)、良好泵送性 | 保证管路输送 |
| 长寿命要求 | 合成油+聚脲/复合锂 | 高温下寿命延长3~5倍 |
| 食品机械 | 白色无毒脂(USDA H1级) | 食品安全 |
| 低噪音 | 优质基础油、经过滤、不含杂质 | 轴承噪音等级要求 |

### 4.4 粘度-温度关系 (VI影响)
<!-- 来源: 第8篇轴承 第86页 -->

| 粘度指数VI | 40°C vs 100°C粘度变化 | 低温性能 | 适用工况温度范围 |
|-----------|---------------------|---------|--------------|
| 0~50 (低VI) | 变化大、粘温曲线陡 | 差 | 窄温范围(±10°C) |
| 50~80 (中VI) | 变化中等 | 一般 | ±20°C |
| 80~95 (中高VI) | 变化较小 | 较好 | ±30°C |
| 95~120 (高VI) | 变化小 | 良好 | ±40°C |
| 120~160 (超高VI) | 变化极小 | 优异 | ±50°C |

> **粘度指数计算**：VI = (L - U) / (L - H) × 100
> L = VI=0时40°C粘度，H = VI=100时40°C粘度，U = 被测油100°C粘度对应的40°C粘度

| 工作压力 | 低压0.5~5MPa / 中压5~20MPa / 高压20~40MPa |
| 润滑点数 | 几~几百个点 |
| 控制方式 | 手动/机械/电动/程控PLC |
| 优点 | 自动化程度高、润滑可靠、节省人力 |
| 缺点 | 系统投资大、维护复杂、管路泄漏风险 |
| 典型应用 | 轧钢机、造纸机、注塑机、工程机械 |

### 5.10 润滑方式选择流程
<!-- 来源: 第8篇轴承 第184页 -->

```
输入条件：
  ┌─ 设备类型（轴承/齿轮/导轨/链条）
  ├─ 工作速度（低速/中速/高速/超高速）
  ├─ 载荷（轻/中/重/冲击）
  ├─ 温度（常温/高温/低温）
  ├─ 环境（清洁/粉尘/潮湿/腐蚀）
  └─ 自动化程度（手动的/自动的）

   ↓

判断：
  少量润滑点 → 手工/滴油/油绳
  多润滑点 → 集中润滑
  速度低 → 手工/油浴/飞溅
  速度高 → 循环/油雾/油气
  散热要求高 → 循环/喷射
  精度/自动化要求高 → 集中/循环/油气
  
   ↓

```

### 6.2 密封选型基本原则
<!-- 来源: 第6篇连接与紧固 第3页 -->

| 序号 | 原则 | 说明 |
|------|------|------|
| 1 | 密封部位状态 | 静止接口→静密封；运动接口→动密封 |
| 2 | 密封介质特性 | 油/水/气/化学介质/高温/腐蚀性 |
| 3 | 工作压力范围 | 低压(<0.5MPa)→简单密封；高压(>10MPa)→特殊密封 |
| 4 | 工作温度范围 | 高温→金属垫片/柔性石墨；低温→弹性体 |
| 5 | 运动形式 | 旋转体→油封/机械密封；往复→V形圈/格莱圈 |
| 6 | 速度范围 | 高速→非接触/机械密封；低速→接触式 |
| 7 | 寿命要求 | 易更换→价格低/简单；难更换→高质量/长寿命 |
| 8 | 安装空间 | 空间限制→O型圈/小尺寸密封 |
| 9 | 成本控制 | 一般用途→标准密封；关键场合→高性能密封 |
| 10 | 维护方便性 | 需常维护→模块化/易拆装 |

### 8.4 机械密封

#### 8.4.1 机械密封工作原理
<!-- 来源: 第6篇连接与紧固 第3页 -->

| 部件 | 功能 |
|------|------|
| 动环（旋转环） | 随轴旋转，与静环形成摩擦副 |
| 静环（静止环） | 固定在壳体上 |
| 弹簧（推环） | 提供初始端面比压 |
| 辅助密封（O型圈/波纹管） | 动环与轴、静环与壳体之间的静密封 |
| 弹簧座 | 定位弹簧 |
| 紧定螺钉 | 固定弹簧座在轴上 |
| 密封端面 | 动环和静环的高精度研磨面 |

> **工作原理**：依靠弹簧力和介质压力使动环与静环端面紧密贴合，形成极薄液膜（0.5~3μm）实现密封

#### 8.4.2 机械密封端面比压计算
<!-- 来源: 第8篇轴承 第13页 -->

端面比压是机械密封设计的核心参数：

**基本公式**：
```
pc = ps + pp - pm

其中：
pc = 端面比压(MPa)
ps = 弹簧比压(MPa)
pp = 介质压力产生的端面压力(MPa)
pm = 液膜压力(MPa, 一般取0.05~0.15MPa)
```

**弹簧比压计算**：
```
ps = Fs / A
Fs = 弹簧总力(N)
A  = 密封端面面积(mm²) = (π/4)(D2² - D1²)
D2 = 密封端面外径(mm)
D1 = 密封端面内径(mm)
```

**介质压力产生的端面压力**：
```
pp = K × p
K  = 平衡系数(载荷系数)
p  = 介质压力(MPa)

对于非平衡型：K = (D2² - d²) / (D2² - D1²)
对于平衡型：K = (D2² - db²) / (D2² - D1²)
其中 d = 轴径(mm), db = 平衡直径(mm)
```

#### 8.4.3 端面比压推荐值
<!-- 来源: 第8篇轴承 第13页 -->

| 工况 | 推荐端面比压pc(MPa) | 说明 |
|------|------------------|------|
| 一般油类介质 | 0.3~0.5 | 平衡型取小值 |
| 水类介质 | 0.3~0.6 | 润滑性差，取大值 |
| 高粘度介质 | 0.2~0.4 | 液膜易形成 |
| 低粘度介质 | 0.4~0.7 | 液膜薄，需更大比压 |
| 高速工况(v>25m/s) | 0.15~0.3 | 高速下液膜压力大 |
| 低压(<0.5MPa) | 0.3~0.5 | 主要由弹簧提供 |
| 中压(0.5~3MPa) | 0.4~0.6 | 介质压力+弹簧力 |
| 高压(>3MPa) | 0.5~0.8 | 采用平衡型 |

#### 8.4.4 弹簧比压推荐值
<!-- 来源: 第8篇轴承 第13页 -->

| 轴径mm | 弹簧比压ps(MPa) |
|-------|---------------|
| ≤20 | 0.08~0.15 |
| 20~50 | 0.10~0.18 |
| 50~100 | 0.12~0.20 |
| >100 | 0.15~0.25 |

> **弹簧比压选择原则**：
> 1. 高压时取小值（介质压力起主要作用）
> 2. 低压时取大值（弹簧力为主）
> 3. 水类介质比油类介质取大10~20%
> 4. 高速取小值，低速取大值

#### 8.4.5 PV值参数
<!-- 来源: 第8篇轴承 第13页 -->

**PV值定义**：密封端面单位面积上承受的压力与滑动速度的乘积

```
PV = pc × v

其中：
v = 端面平均滑动速度(m/s) = π × dm × n / 60000
dm = (D2 + D1)/2 = 端面平均直径(mm)
n  = 转速(r/min)
```

**PV值许用极限**

| 材料配对 | 许用PV值(MPa·m/s) | 说明 |
|---------|-----------------|------|
| 硬质合金-石墨 | pvc ≤ 60 | 最常用配对 |
| 硬质合金-硬质合金 | pvc ≤ 80 | 需润滑良好 |
| 碳化硅-石墨 | pvc ≤ 120 | 耐磨损性能优异 |
| 碳化硅-碳化硅 | pvc ≤ 160 | 极高耐磨 |
| 氧化铝陶瓷-石墨 | pvc ≤ 40 | 耐腐蚀好 |
| 不锈钢-石墨 | pvc ≤ 15 | 低端应用 |
| 碳化硅-硬质合金 | pvc ≤ 100 | 高刚性配对 |

> **PV值限制**：实际PV值需小于许用PV值，并留安全系数(S=1.2~2.0)
> **热PV值**：还需检查(PV)ᵀ = PV × T 不超过极限值(T为端面温度°C)

### 9.1 间隙密封
<!-- 来源: 第8篇轴承 第141页 -->

#### 9.1.1 圆柱间隙密封
<!-- 来源: 第8篇轴承 第13页 -->

| 项目 | 说明 |
|------|------|
| 原理 | 利用微小间隙产生流动阻力（节流效应） |
| 间隙值 | 0.01~0.1mm（视直径和压力而定） |
| 适用压力 | 低压 < 1MPa（间隙小可至5MPa） |
| 适用速度 | 任意速度 |
| 适用介质 | 油、水、气体 |
| 优点 | 无接触、无磨损、无限寿命 |
| 缺点 | 泄漏量较大、对间隙敏感 |

| 轴径mm | 推荐间隙mm(油) | 推荐间隙mm(水) | 推荐间隙mm(气) |
|-------|-------------|-------------|-------------|
| ≤25 | 0.015~0.030 | 0.010~0.020 | 0.005~0.010 |
| 25~50 | 0.020~0.040 | 0.015~0.025 | 0.008~0.015 |
| 50~80 | 0.030~0.050 | 0.020~0.035 | 0.010~0.020 |
| 80~120 | 0.040~0.060 | 0.025~0.040 | 0.015~0.025 |
| >120 | 0.050~0.100 | 0.030~0.050 | 0.020~0.030 |

**泄漏量计算公式**：
```
Q = (π × d × Δp × h³) / (12 × η × L)

Q  = 泄漏量(m³/s)
d  = 轴径(m)
Δp = 压差(Pa)
h  = 单边间隙(m)
η  = 动力粘度(Pa·s)
L  = 密封长度(m)
```

> **关键结论**：泄漏量与h³成正比，间隙增大一倍→泄漏量增大8倍

#### 9.1.2 迷宫密封
<!-- 来源: 第8篇轴承 第269页 -->

| 项目 | 说明 |
|------|------|
| 原理 | 流体通过曲折通道逐级降压，最终能量耗散 |
| 结构形式 | 直通式/交错式/阶梯式/径向式 |
| 材料 | 铝合金、铜合金、不锈钢、PTFE |
| 适用速度 | 高速(v>10m/s)效果更好 |
| 适用温度 | -200~800°C（取决于材料） |
| 适用压力 | 一般≤2MPa（多级可达10MPa） |
| 间隙 | 0.2~0.8mm（一般0.3~0.5mm） |

**迷宫密封结构对比**

| 形式 | 泄漏系数 | 结构复杂程度 | 适用线速度 | 典型应用 |
|------|---------|------------|----------|---------|
| 直通式 | 最高 | 简单 | 任意 | 汽轮机、燃气轮机 |
| 交错式 | 中等 | 中等 | ≤50m/s | 压缩机、鼓风机 |
| 阶梯式 | 较低 | 复杂 | 任意 | 汽轮机轴向密封 |
| 径向式 | 低 | 较复杂 | ≤30m/s | 离心压缩机 |
| 蜂窝式 | 最低 | 极复杂 | 任意 | 航空发动机 |

> **设计要点**：
> 1. 迷宫齿数推荐4~20个，齿越多密封效果越好
> 2. 齿顶厚度0.2~0.5mm
> 3. 齿距3~10mm
> 4. 齿高3~15mm
> 5. 密封间隙越小效果越好，但需考虑热膨胀

### 9.2 离心密封
<!-- 来源: 第8篇轴承 第13页 -->

| 项目 | 说明 |
|------|------|
| 原理 | 利用离心力将液体甩回，阻止泄漏 |
| 结构 | 甩油盘、甩油环、挡油环、离心叶轮 |
| 适用介质 | 油、水（有一定粘度） |
| 适用速度 | n>1000rpm（转速越高效果越好） |
| 间隙 | 0.3~1.0mm |

| 类型 | 适用线速度m/s | 密封压力差 | 特点 |
|------|------------|-----------|------|
| 甩油盘（平盘） | ≥5 | 无压差 | 防止油沿轴爬行 |
| 甩油环（带齿） | ≥8 | 微弱压差 | 防油能力强 |
| 离心叶轮 | ≥15 | ≤0.5MPa | 兼有泵送作用 |
| 径向叶片 | ≥10 | ≤0.3MPa | 制造简单 |

> **设计注意事项**：
> 1. 甩油盘直径一般比轴径大2~4倍
> 2. 甩油盘与壳体间隙越小离心效果越好
> 3. 离心密封通常作为辅助密封（与骨架油封配合使用）
> 4. 停机时离心密封失效，需配合其他密封

### 10.4 机械密封摩擦热与冷却
<!-- 来源: 第8篇轴承 第13页 -->

**端面发热量计算**：
```
Q = f × pc × v × A
Q = 摩擦热(W)
f = 摩擦系数(石墨-碳化硅正常工况0.05~0.12)
pc = 端面比压(Pa)
v = 滑动速度(m/s)
A = 端面面积(m²)
```

**冲洗量计算**：
```
Qw = Q / (cp × ρ × Δt)
Qw = 冲洗流量(m³/s)
cp = 介质比热容(J/kg·K)
ρ = 介质密度(kg/m³)
Δt = 温升(K)，一般取3~10K
```

| 密封工作条件 | 散热方式 | 推荐冲洗量L/min |
|-------------|---------|--------------|
| 一般清水泵(p≤2MPa, t≤80°C) | 自冲洗 | 3~10 |
| 热水泵(t>100°C) | 自冲洗+冷却器 | 5~15 |
| 高温油泵(t>200°C) | 外冲洗+冷却器 | 8~20 |
| 高速泵(v>25m/s) | 强制冲洗+冷却套 | 10~30 |
| 含颗粒介质 | 外冲洗(清洁液) | 5~25 |

### 10.7 机械密封常见故障
<!-- 来源: 第8篇轴承 第105页 -->

| 故障现象 | 可能原因 | 处理措施 |
|---------|---------|---------|

### A.4 常用设计计算公式汇总
<!-- 来源: 第8篇轴承 第13页 -->

| 参数 | 公式 | 符号说明 |
|------|------|---------|
| 滑动轴承摩擦系数 | f = (2πηnv/p)(1 - exp(-p/ηn)) | η=粘度, n=转速, p=比压 |
| 油膜厚度(径向轴承) | h₀ = 0.1D × (ηn/pS)^(2/3) | D=轴径, S=松驰比 |
| 间隙密封泄漏量 | Q = πdΔph³/(12ηL) | d=轴径, h=间隙 |
| 端面比压 | p_c = p_s + K·p - p_m | K=平衡系数 |
| PV值 | PV = p_c × v | v=滑动速度 |
| O型圈压缩率 | ε = (d₂ - t)/d₂ × 100% | d₂=截面直径 |
| 弹簧比压 | p_s = 2Ft × π(D₂+D₁) / (2A) | — |
| 垫片密封力 | F = A₀·y + 2πD·b·m·p | y=垫片比压 |
