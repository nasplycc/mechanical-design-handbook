### 1.4 螺栓强度计算
<!-- 来源: 第2卷  第847页 -->
**受拉螺栓：**
```
σ = F/(π·d₁²/4) ≤ [σ]
d₁ ≥ √(4F/(π·[σ]))

其中：
F - 轴向载荷(N)
d₁ - 螺纹小径(mm)
[σ] = σ_s / S - 许用应力
S - 安全系数(静载: 1.2~2.5, 变载: 2.5~5)
```

**受剪螺栓（铰制孔）：**
```
τ = F/(n·π·d₀²/4) ≤ [τ]  （剪切）
σ_p = F/(d₀·δ_min) ≤ [σ_p]  （挤压）

其中：
d₀ - 螺栓杆直径(mm)
n - 剪切面数
δ_min - 最小被连接件厚度(mm)
```

### 1.5 螺栓组设计要点
<!-- 来源: 第2卷  第847页 -->
- 螺栓布置应使受力均匀
- 同一组螺栓材料、直径、长度应一致
- 螺栓间距：t₀ ≤ 5d（高压容器）或 t₀ ≤ 10d（一般）
- 边距：e ≥ d + (3~6)mm
- 考虑扳手空间（见扳手空间表）

### 1.6 螺纹防松方法
<!-- 来源: 第2卷  第847页 -->
| 方法 | 原理 | 适用 |
|------|------|------|
| 弹簧垫圈 | 弹力+切口 | 一般场合 |
| 双螺母 | 对顶螺母产生拉力 | 一般场合 |
| 尼龙自锁螺母 | 尼龙环变形锁紧 | 振动场合 |
| 开口销+槽形螺母 | 机械锁紧 | 重要场合 |
| 止动垫片 | 折弯锁紧 | 重要场合 |
| 螺纹锁固胶 | 化学锁紧 | 不可拆场合 |
| 串联钢丝 | 钢丝牵制 | 成组螺栓 |

### 1.7 螺纹连接预紧力
<!-- 来源: 第2卷  第208页 -->
```
T = K·F·d

其中：
T - 拧紧扭矩(N·m)
K - 扭矩系数(一般取0.15~0.2)
F - 预紧力(N)
d - 公称直径(mm)

预紧力推荐值：
F = (0.5~0.7)·σ_s·A_s  (一般连接)
F = (0.7~0.9)·σ_s·A_s  (重要连接)
```

### 2.3 平键强度校核
<!-- 来源: 第2卷  第847页 -->
```
挤压应力：σ_p = 2T/(d·k·L) ≤ [σ_p]
剪切应力：τ = 2T/(d·b·L) ≤ [τ]

其中：
T - 传递扭矩(N·mm)
d - 轴径(mm)
k - 键与轮毂的接触高度 ≈ h/2
L - 键的工作长度 = 公称长度 - b (圆头) 或 = 公称长度 (平头)

许用值：
[σ_p]：静载 钢-钢 100~120 MPa, 钢-铸铁 50~60 MPa
       轻微冲击 钢-钢 70~90 MPa, 钢-铸铁 30~45 MPa
       冲击 钢-钢 40~60 MPa, 钢-铸铁 15~30 MPa
[τ]：60~90 MPa
```

### 3.3 花键强度校核
<!-- 来源: 第2卷  第847页 -->
```
σ_p = 2T/(ψ·z·h·l·D_m) ≤ [σ_p]

其中：
ψ - 载荷分布不均匀系数(一般取0.7~0.8)
z - 齿数
h - 齿的工作高度
l - 齿的工作长度
D_m - 平均直径 = (D+d)/2

许用挤压应力 [σ_p]：
静载：100~200 MPa
轻微冲击：50~100 MPa
冲击：20~50 MPa
```

## 5. 焊接
<!-- 来源: 第2卷  第19页 -->

### 5.1 常用焊接方法
<!-- 来源: 第2卷  第847页 -->
| 方法 | 特点 | 适用 |
|------|------|------|
| 手工电弧焊 | 灵活、质量依赖操作 | 一般结构 |
| CO₂气体保护焊 | 效率高、成本低 | 碳钢、低合金钢 |
| 氩弧焊(TIG/MIG) | 质量高、变形小 | 不锈钢、铝合金 |
| 埋弧焊 | 效率高、质量稳定 | 长直焊缝 |
| 电阻焊 | 快速、自动化 | 薄板 |
| 激光焊 | 精密、深宽比大 | 精密零件 |

### 5.2 焊缝类型
<!-- 来源: 第2卷  第847页 -->
| 类型 | 符号 | 特点 |
|------|------|------|
| 对接焊缝 | ── | 受力好、需开坡口 |
| 角焊缝 | ⊿ | 施工简单、受力较差 |
| T形焊缝 | ┴ | T形接头 |
| 搭接焊缝 | ⊃ | 搭接接头 |

### 5.3 角焊缝强度计算
<!-- 来源: 第2卷  第847页 -->
```
τ = F/(0.7·K·L_w) ≤ [τ']

其中：
K - 焊脚尺寸
L_w - 焊缝计算长度 = 实际长度 - 2K
[τ'] - 角焊缝许用剪应力 ≈ 0.6~0.8·[σ]

角焊缝焊脚尺寸K推荐：
最小板厚t | 最小焊脚K
t≤4 | K=t
4<t≤6 | K=3~4
6<t≤12 | K=5~6
t>12 | K=0.7t(最小)
```

### 5.4 焊接结构设计要点
<!-- 来源: 第2卷  第10页 -->
- 焊缝位置应便于施焊
- 避免焊缝交叉和密集
- 对接焊缝优先，角焊缝次之
- 考虑焊接变形和残余应力
- 重要结构需焊后热处理（去应力退火500~650°C）
