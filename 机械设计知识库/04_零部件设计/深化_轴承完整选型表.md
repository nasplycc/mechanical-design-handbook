### 算例1：深沟球轴承选型与寿命计算
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**已知条件：** 一转轴直径d=45mm，转速n=1200r/min，径向载荷Fr=2800N，轴向载荷Fa=850N，要求轴承寿命≥20000h，载荷平稳，工作温度<100°C。

**解：**

**步骤1：初选轴承型号**
根据d=45mm，初选6209深沟球轴承
查表：6209，d=45mm，D=85mm，B=19mm，Cr=31.5kN，C0r=20.4kN

**步骤2：判断轴向载荷与额定静载荷之比**
Fa/C0r = 850 / 20400 = 0.0417
查e值和Y值（插值）：
Fa/C0r=0.028→e=0.22，Y=1.99
Fa/C0r=0.056→e=0.26，Y=1.71
Fa/C0r=0.0417时：
e = 0.22 + (0.0417-0.028)/(0.056-0.028) × (0.26-0.22) = 0.22 + 0.0137/0.028×0.04 = 0.240
Y = 1.99 - (0.0417-0.028)/(0.056-0.028) × (1.99-1.71) = 1.99 - 0.489×0.28 = 1.853

**步骤3：判断Fa/Fr与e关系**
Fa/Fr = 850/2800 = 0.304 > e=0.240
∴ 采用P = XFr + YFa 公式

**步骤4：计算当量动载荷P**
X=0.56，Y=1.853
P = 0.56×2800 + 1.853×850 = 1568 + 1575 = 3143N ≈ 3.14kN

**步骤5：计算额定寿命**
球轴承ε=3
L10h = 10⁶/(60×1200) × (31.5/3.14)³
= 13.89 × (10.03)³
= 13.89 × 1009
= 14015 h

**步骤6：验证**
14015h < 20000h，不满足要求。

**步骤7：重新选型**
改选6309轴承（d=45mm，Cr=52.0kN，C0r=30.5kN）
L10h = 13.89 × (52.0/3.14)³ = 13.89 × (16.56)³ = 13.89 × 4541 = 63075h

> 20000h，满足要求。

**步骤8：校核极限转速**
润滑脂润滑，6309极限转速6700r/min > 1200r/min，满足。

**步骤9：结果**

|:----:|:--:|
| 选用型号 | 6309 |
| 内径d | 45mm |
| 外径D | 100mm |
| 宽度B | 25mm |
| 基本额定寿命L10h | 63075h |
| 轴承质量 | 0.837kg |
| 轴肩最小直径 | 54mm |
| 外壳肩最大直径 | 91mm |

### 算例2：圆锥滚子轴承配对计算
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**已知条件：** 一轴两端各安装一个30208圆锥滚子轴承，面对面安装(DF)。径向载荷Fr₁=3500N，Fr₂=2500N，外加轴向力Ka=800N（方向从轴承1指向轴承2），转速n=800r/min，要求寿命≥15000h。

**解：**

**步骤1：查轴承参数**
30208：d=40mm，D=80mm，T=19.75mm，Cr=44.0kN，C0r=36.0kN
e=0.35，Y=1.7

**步骤2：计算派生轴向力**
Fs₁ = Fr₁ / (2Y) = 3500 / (2×1.7) = 3500/3.4 = 1029N
Fs₂ = Fr₂ / (2Y) = 2500 / (2×1.7) = 2500/3.4 = 735N

**步骤3：受力分析（面对面DF安装）**
面对面安装时，派生力方向：从外圈大端指向小端
Fs₁方向→，Fs₂方向←
外加轴向力Ka：←（从轴承1指向轴承2）

计算轴承2受到的总推力：
Fs₁ + Ka = 1029 + 800 = 1829N > Fs₂ = 735N

⇒ 轴向力平衡结果：
Fa₁ = Fs₁ = 1029N（轴承1被压紧？不，再看）

正确分析方法：取轴为受力体
若 Fs₁ + Ka ≥ Fs₂，则轴有向右移动趋势，轴承2被压紧
Fa₁ = Fs₁ = 1029N（放松端）
Fa₂ = Fs₁ + Ka = 1029 + 800 = 1829N（压紧端）

验算：Fs₂ + Fa₂ ≠ Fa₁ + Ka ✓

**步骤4：计算各轴承当量动载荷**

**轴承1（放松端）：**
Fa₁/Fr₁ = 1029/3500 = 0.294 < e=0.35
∴ P₁ = Fr₁ = 3500N = 3.50kN

**轴承2（压紧端）：**
Fa₂/Fr₂ = 1829/2500 = 0.732 > e=0.35
∴ P₂ = 0.4Fr₂ + Y×Fa₂ = 0.4×2500 + 1.7×1829 = 1000 + 3109 = 4109N = 4.11kN

取 P = max(P₁, P₂) = 4.11kN

**步骤5：计算寿命**
滚子轴承ε=10/3
L10h = 10⁶/(60×800) × (44.0/4.11)^(10/3)
= 20.83 × (10.71)^(3.333)
lg(10.71) = 1.0298
(10/3)×1.0298 = 3.433
10^3.433 = 2709

L10h = 20.83 × 2709 = 56428h

> 15000h，满足要求。

**步骤6：结果汇总**

| 项目 | 轴承1 | 轴承2 |
|:----:|:-----:|:-----:|
| 型号 | 30208 | 30208 |
| 径向载荷Fr | 3500N | 2500N |
| 派生轴向力Fs | 1029N | 735N |
| 实际轴向力Fa | 1029N(放松) | 1829N(压紧) |
| 当量动载荷P | 3.50kN | 4.11kN |
| 轴承寿命 | 56428h（由轴承2决定） | |
| 预期寿命 | ≥15000h ✓ | |
