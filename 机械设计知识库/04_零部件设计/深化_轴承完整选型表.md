### 算例1：深沟球轴承选型与寿命计算
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**已知条件：** 一转轴直径d=45mm，转速n=1200r/min，径向载荷Fr=2800N，轴向载荷Fa=850N，要求轴承寿命≥20000h，载荷平稳，工作温度<100°C。

**解：**

**步骤1：初选轴承型号**
根据d=45mm，初选6209深沟球轴承
查表：6209，d=45mm，D=85mm，B=19mm，Cr=31.5kN，C0r=20.4kN

**步骤2：判断轴向载荷与额定静载荷之比**
Fa/C0r = 850 / 20400 = 0.0417
查e值和Y值（插值）：
Fa/C0r=0.028→e=0.22，Y=1.99
Fa/C0r=0.056→e=0.26，Y=1.71
Fa/C0r=0.0417时：
e = 0.22 + (0.0417-0.028)/(0.056-0.028) × (0.26-0.22) = 0.22 + 0.0137/0.028×0.04 = 0.240
Y = 1.99 - (0.0417-0.028)/(0.056-0.028) × (1.99-1.71) = 1.99 - 0.489×0.28 = 1.853

**步骤3：判断Fa/Fr与e关系**
Fa/Fr = 850/2800 = 0.304 > e=0.240
∴ 采用P = XFr + YFa 公式

**步骤4：计算当量动载荷P**
X=0.56，Y=1.853
P = 0.56×2800 + 1.853×850 = 1568 + 1575 = 3143N ≈ 3.14kN

**步骤5：计算额定寿命**
球轴承ε=3
L10h = 10⁶/(60×1200) × (31.5/3.14)³
= 13.89 × (10.03)³
= 13.89 × 1009
= 14015 h

**步骤6：验证**
14015h < 20000h，不满足要求。

**步骤7：重新选型**
改选6309轴承（d=45mm，Cr=52.0kN，C0r=30.5kN）
L10h = 13.89 × (52.0/3.14)³ = 13.89 × (16.56)³ = 13.89 × 4541 = 63075h

> 20000h，满足要求。

**步骤8：校核极限转速**
润滑脂润滑，6309极限转速6700r/min > 1200r/min，满足。

### 算例2：圆锥滚子轴承配对计算
<!-- 来源: 第2卷 第6篇连接与紧固 第3页 -->

**已知条件：** 一轴两端各安装一个30208圆锥滚子轴承，面对面安装(DF)。径向载荷Fr₁=3500N，Fr₂=2500N，外加轴向力Ka=800N（方向从轴承1指向轴承2），转速n=800r/min，要求寿命≥15000h。

**解：**

**步骤1：查轴承参数**
30208：d=40mm，D=80mm，T=19.75mm，Cr=44.0kN，C0r=36.0kN
e=0.35，Y=1.7

**步骤2：计算派生轴向力**
Fs₁ = Fr₁ / (2Y) = 3500 / (2×1.7) = 3500/3.4 = 1029N
Fs₂ = Fr₂ / (2Y) = 2500 / (2×1.7) = 2500/3.4 = 735N

**步骤3：受力分析（面对面DF安装）**
面对面安装时，派生力方向：从外圈大端指向小端
Fs₁方向→，Fs₂方向←
外加轴向力Ka：←（从轴承1指向轴承2）

计算轴承2受到的总推力：
Fs₁ + Ka = 1029 + 800 = 1829N > Fs₂ = 735N

⇒ 轴向力平衡结果：
Fa₁ = Fs₁ = 1029N（轴承1被压紧？不，再看）

正确分析方法：取轴为受力体
若 Fs₁ + Ka ≥ Fs₂，则轴有向右移动趋势，轴承2被压紧
Fa₁ = Fs₁ = 1029N（放松端）
Fa₂ = Fs₁ + Ka = 1029 + 800 = 1829N（压紧端）

验算：Fs₂ + Fa₂ ≠ Fa₁ + Ka ✓

**步骤4：计算各轴承当量动载荷**

**轴承1（放松端）：**
Fa₁/Fr₁ = 1029/3500 = 0.294 < e=0.35
∴ P₁ = Fr₁ = 3500N = 3.50kN

**轴承2（压紧端）：**
Fa₂/Fr₂ = 1829/2500 = 0.732 > e=0.35
∴ P₂ = 0.4Fr₂ + Y×Fa₂ = 0.4×2500 + 1.7×1829 = 1000 + 3109 = 4109N = 4.11kN

取 P = max(P₁, P₂) = 4.11kN

**步骤5：计算寿命**
滚子轴承ε=10/3
L10h = 10⁶/(60×800) × (44.0/4.11)^(10/3)
= 20.83 × (10.71)^(3.333)
lg(10.71) = 1.0298
(10/3)×1.0298 = 3.433
10^3.433 = 2709

L10h = 20.83 × 2709 = 56428h

> 15000h，满足要求。

**步骤6：结果汇总**

## 14. 补充算例

### 算例3：角接触球轴承配对选型与寿命计算
**已知条件：** 某高速主轴，d=50mm，n=8000r/min，径向载荷Fr=1200N，轴向载荷Fa=600N，要求寿命≥10000h，选用背对背(DB)配对。

**解：**

**步骤1：初选轴承**
根据d=50mm和n=8000r/min，初选7010C DB配对（单个7010C：Cr=20.0kN，C0r=16.0kN，e=0.38）

**步骤2：成对安装参数**
DB配对时，Cr(成对) = 1.625×Cr(单列) = 1.625×20.0 = 32.5kN
C0r(成对) = 2×C0r(单列) = 32.0kN

**步骤3：计算当量动载荷**
Fa/Fr = 600/1200 = 0.50 > e=0.38
DB配对联轴承：X=0.61, Y=0.99（双列α=15°）
P = 0.61×1200 + 0.99×600 = 732 + 594 = 1326N = 1.326kN

**步骤4：寿命计算**
球轴承ε=3
L10h = 10⁶/(60×8000) × (32.5/1.326)³ = 2.083 × (24.51)³ = 2.083 × 14722 = 30666h > 10000h ✓

**步骤5：极限转速校核**
7010C脂润滑极限转速8000r/min，等于n=8000r/min，仍需控制温升
建议采用油雾润滑以提高极限转速余量

**步骤6：结果汇总**
| 项目 | 数值 |
|:----:|:----:|
| 选用型号 | 7010C DB(背对背配对) |
| 成对额定动载荷 | 32.5kN |
| 当量动载荷P | 1.326kN |
| 计算寿命L10h | 30666h |
| 脂润滑极限转速 | 8000r/min |

### 算例4：调心滚子轴承选型（重载低速工况）
**已知条件：** 某振动筛支承，d=100mm，n=400r/min，径向载荷Fr=85000N，轴向载荷Fa=15000N，冲击较大，要求寿命≥20000h。

**解：**

**步骤1：初选轴承**
d=100mm，重载工况，初选22220调心滚子轴承
查表：Cr=300kN，C0r=340kN，e=0.38，Y1=1.8，Y2=2.6

**步骤2：计算当量动载荷**
Fa/Fr = 15000/85000 = 0.176
Fa/Fr = 0.176 < e=0.38
∴ P = Fr + Y1·Fa = 85000 + 1.8×15000 = 85000 + 27000 = 112000N = 112kN
（注：调心滚子轴承Fa/Fr≤e时取P=XFr+Y1Fa，X=1）

**步骤3：考虑冲击载荷**
振动筛冲击系数 fd = 1.5~2.0，取1.8
P_eq = 1.8 × 112 = 201.6kN

**步骤4：寿命计算**
滚子轴承ε=10/3
L10h = 10⁶/(60×400) × (300/201.6)^(10/3)
= 41.67 × (1.488)^(3.333)
lg(1.488)=0.1726, (10/3)×0.1726=0.575
10^0.575=3.76
L10h = 41.67 × 3.76 = 15668h

**步骤5：结果判断**
15668h < 20000h，略不足，改选22320（Cr=510kN，C0r=565kN）
L10h = 41.67 × (510/201.6)^(3.333) = 41.67 × (2.530)^(3.333)
lg(2.530)=0.403, (10/3)×0.403=1.343, 10^1.343=22.02
L10h = 41.67 × 22.02 = 91757h > 20000h ✓

**步骤6：最终结果**

|:----:|:----:|
| 选用型号 | 22320（调心滚子轴承） |
| 基本额定动载荷Cr | 510kN |
| 当量动载荷Peq | 201.6kN |
| 计算寿命L10h | 91757h |
| 安装配合 | 轴k6/m6，外壳K7/M7 |

### 算例5：圆柱滚子轴承（自由端配置）
**已知条件：** 大型齿轮减速器中间轴，d=80mm，n=1500r/min，径向载荷Fr=18000N，轴向载荷忽略，温度70°C，要求寿命≥30000h，一端固定、一端浮动。

**解：**

**步骤1：轴承配置方案**
浮动端：选用NU216圆柱滚子轴承（允许轴向位移）
固定端：选用NJ216或配对角接触球轴承
查NU216：d=80mm，D=140mm，B=26mm，Cr=100kN，C0r=92kN

**步骤2：当量动载荷**
纯径向载荷且无轴向力，Fa=0
P = Fr = 18000N = 18kN

**步骤3：温度修正**
70°C < 120°C，温度系数kt=1.0

**步骤4：寿命计算**
滚子轴承ε=10/3
L10h = 10⁶/(60×1500) × (100/18)^(10/3)
= 11.11 × (5.556)^(3.333)
lg(5.556)=0.7447, (10/3)×0.7447=2.482, 10^2.482=303.5
L10h = 11.11 × 303.5 = 3371h

**步骤5：重新选型**
3371h远小于30000h，改选NU220（d=100mm偏大，但需增大）
实际需重新设计轴径或选用更大的NU系列
试选NU224（d=120mm，Cr=212kN）:
L10h = 11.11 × (212/18)^(3.333) = 11.11 × (11.78)^(3.333)
lg(11.78)=1.071, ×3.333=3.570, 10^3.570=3715
L10h = 11.11 × 3715 = 41270h > 30000h ✓
