## 1. 概述
<!-- 来源: 第8篇轴承 第105页 -->

弹簧是一种利用弹性变形来储存和释放能量的机械零件，广泛应用于各类机械、仪器、车辆及日常设备中。其主要功能包括：

- **减振缓冲** — 吸收振动和冲击能量
- **储能释能** — 储存能量并在需要时释放
- **控制运动** — 控制机构的运动或位置
- **测量载荷** — 弹簧力与变形量成比例，可用于测力

弹簧设计的基本要求：在规定的变形范围内具有稳定的弹性特性；具有足够的疲劳寿命和可靠性；结构紧凑，便于制造和安装。

## 2. 弹簧分类
<!-- 来源: 第8篇轴承 第13页 -->

### 2.1 按受力性质分类
<!-- 来源: 第8篇轴承 第13页 -->

| 分类 | 类型 | 特点 | 典型应用 |
|------|------|------|----------|
| **压缩弹簧** | 圆柱螺旋压缩弹簧 | 承受轴向压缩力，圈与圈之间留有间隙 | 减振器、阀门、缓冲装置 |
| **拉伸弹簧** | 圆柱螺旋拉伸弹簧 | 承受轴向拉伸力，初始状态各圈紧密贴合 | 复位机构、拉簧秤、发动机气门 |
| **扭转弹簧** | 圆柱螺旋扭转弹簧 | 承受绕弹簧轴线的扭矩 | 扭力杆、夹紧装置、发条机构 |
| **板簧** | 钢板弹簧/碟形弹簧 | 承受弯曲载荷 | 汽车悬架、重型机械缓冲 |
| **其他** | 环形弹簧、膜片弹簧、气弹簧 | 特殊受力形式 | 离合器、精密仪器 |

### 2.2 按弹簧形状分类
<!-- 来源: 第8篇轴承 第13页 -->

| 形状类型 | 说明 | 特点 |
|----------|------|------|
| **圆柱螺旋弹簧** | 螺旋线为圆柱面 | 制造简单，应用最广 |
| **圆锥螺旋弹簧** | 螺旋线为圆锥面 | 自振频率可变，减振效果好 |
| **碟形弹簧** | 圆锥形碟状薄片 | 刚度大，短行程大载荷 |
| **环形弹簧** | 由内环和外环交替组成 | 吸能能力强，用于重型缓冲 |
| **板弹簧** | 由若干长度不等的钢板叠合 | 刚度可变，用于车辆悬架 |
| **平面涡卷弹簧** | 平面螺旋形状 | 储能密度高，用于发条机构 |
| **扭杆弹簧** | 直杆承受扭转 | 结构简单，刚度大 |
| **空气弹簧** | 利用气体压缩弹性 | 刚度可调，舒适性好 |
| **橡胶弹簧** | 利用橡胶弹性变形 | 减振隔音效果好 |
| **膜片弹簧** | 碟形薄片 | 用于离合器、制动器 |

### 2.3 按载荷性质分类
<!-- 来源: 第8篇轴承 第4页 -->

| 分类 | 说明 |
|------|------|
| **静载荷弹簧** | 承受恒定或缓慢变化的载荷，主要考虑强度和变形 |
| **动载荷弹簧** | 承受交变或冲击载荷，需考虑疲劳强度和自振频率 |
| **冲击载荷弹簧** | 承受瞬时大冲击力，需考虑缓冲能力和吸能特性 |

### 2.4 按工作条件分类
<!-- 来源: 第8篇轴承 第13页 -->

| 分类 | 温度范围 | 环境要求 |
|------|----------|----------|
| **普通弹簧** | -40°C ~ +120°C | 一般环境 |
| **高温弹簧** | +120°C ~ +400°C | 耐热合金材料 |
| **低温弹簧** | -250°C ~ -40°C | 低温韧性要求 |
| **耐腐蚀弹簧** | 按环境定 | 不锈钢或特殊涂层 |
| **导电弹簧** | 按环境定 | 铍青铜等导电材料 |

### 4.3 弹簧指数（旋绕比）C 的推荐值
<!-- 来源: 第8篇轴承 第87页 -->

| d (mm) | 推荐 C 值范围 | 常用 C 值 |
|--------|---------------|-----------|
| 0.2~0.4 | 7~14 | 8~12 |
| 0.5~1.0 | 5~12 | 6~10 |
| 1.1~2.5 | 5~10 | 5~8 |
| 2.6~6.0 | 4~9 | 5~7 |
| 7.0~16 | 4~8 | 5~6 |

> **弹簧指数 C 的通用推荐范围: 4~16**。
> - C 值过小（C < 4）：弹簧卷制困难，内侧应力集中严重
> - C 值过大（C > 16）：弹簧稳定性差，易产生侧向弯曲
> - 一般推荐: C = 5~8

### 4.4 曲度系数 K
<!-- 来源: 第8篇轴承 第13页 -->

曲度系数（应力修正系数）K 用于修正螺旋弹簧因曲率引起的应力集中：

$$K = \frac{4C - 1}{4C - 4} + \frac{0.615}{C}$$

| C | K | C | K | C | K | C | K |
|---|----|---|----|---|----|---|----|
| 4 | 1.404 | 6 | 1.253 | 8 | 1.184 | 12 | 1.129 |
| 4.5 | 1.352 | 6.5 | 1.235 | 9 | 1.162 | 14 | 1.113 |
| 5 | 1.311 | 7 | 1.213 | 10 | 1.145 | 16 | 1.102 |
| 5.5 | 1.276 | 7.5 | 1.197 | 11 | 1.135 | 18 | 1.092 |

### 4.5 支撑圈数与总圈数
<!-- 来源: 第6篇连接与紧固 第39页 -->

| 弹簧端部形式 | 支撑圈数 n₂ | 总圈数 n₁ |
|--------------|-------------|-----------|
| 两端并紧磨平 | 1~1.5（推荐1.25） | n₁ = n + 1.25~2.5 |
| 两端并紧不磨 | 1~1.5（推荐1.5） | n₁ = n + 2~3 |
| 两端不并紧 | — | n₁ = n |

> **注**: 重要的弹簧总圈数 n₁ 宜为 0.5 或整数倍，以便于制造。

### 4.6 弹簧长细比
<!-- 来源: 第8篇轴承 第13页 -->

弹簧的自由高度与中径之比称为长细比（细长比）：

$$b = \frac{H_0}{D}$$

| 弹簧类型 | 推荐 b 值 | 注意 |
|----------|-----------|------|
| 两端固定压缩弹簧 | b ≤ 5.3 | 稳定性好 |
| 一端固定一端回转 | b ≤ 3.7 | 需注意稳定性 |
| 两端回转 | b ≤ 2.6 | 易失稳 |
| 拉伸弹簧 | 不限 | 不受稳定性限制 |

## 5. 圆柱螺旋压缩弹簧设计
<!-- 来源: 第8篇轴承 第13页 -->

### 5.1 基本设计公式
<!-- 来源: 第8篇轴承 第13页 -->

#### 弹簧刚度 k
<!-- 来源: 第7篇轴及其连接 第437页 -->

$$k = \frac{F}{f} = \frac{Gd^4}{8D^3n} = \frac{Gd}{8C^3n}$$

其中：
- k — 弹簧刚度 (N/mm)
- F — 弹簧承受的载荷 (N)
- f — 变形量 (mm)
- G — 切变模量 (MPa)
- d — 弹簧线径 (mm)
- D — 弹簧中径 (mm)
- n — 有效圈数
- C — 弹簧指数 (C = D/d)

#### 剪应力 τ
<!-- 来源: 第7篇轴及其连接 第44页 -->

工作时弹簧圈内侧应力最大：

$$τ = \frac{8FD}{πd^3}·K = \frac{8FC}{πd^2}·K$$

其中 K 为曲度系数。

**强度条件**：

$$τ_{\text{max}} ≤ [τ]$$

#### 变形量 f
<!-- 来源: 第8篇轴承 第153页 -->

$$f = \frac{8FD^3n}{Gd^4} = \frac{8FC^3n}{Gd}$$

### 5.2 各参数关系汇总表
<!-- 来源: 第6篇连接与紧固 第85页 -->

| 参数关系 | 公式 | 适用说明 |
|----------|------|----------|
| 弹簧刚度 k | k = F/f = Gd⁴/(8D³n) | 设计基本关系 |
| 线径 d | d = √(8F_max C K / (π[τ])) | 由强度条件确定 |
| 有效圈数 n | n = Gd⁴f/(8FD³) | 由变形条件确定 |
| 应力 τ | τ = 8FDK/(πd³) | 强度校核 |
| 变形 f | f = 8FD³n/(Gd⁴) | 变形校核 |
| 刚度 k | k = Gd/(8C³n) | 用 C 表达 |
| 中径 D | D = Cd | 由 C 确定 |
| 外径 D₂ | D₂ = D + d | 安装空间尺寸 |
| 内径 D₁ | D₁ = D - d | 安装空间尺寸 |

| 端部形式 | 代号 | 特点 | 应用 |
|----------|------|------|------|
| 两端圈并紧磨平 | YI | 端面平整，受力均匀 | 重要弹簧，高精度 |
| 两端圈并紧不磨 | YII | 端面较平，制造简单 | 一般用途 |
| 两端圈不并紧 | YIII | 制造最简单 | 小载荷、低精度 |

### 5.5 压缩弹簧的稳定性
<!-- 来源: 第8篇轴承 第105页 -->

当弹簧的长细比 b = H₀/D 过大时，会失去稳定性而发生侧向弯曲。

**临界失稳载荷**：

$$F_c = C_B k H_0$$

其中 C_B 为稳定系数（取决于端部固定形式）：

| 端部形式 | C_B |
|----------|------|
| 两端固定 | 0.5~0.6 |
| 一端固定一端回转 | 0.25~0.35 |
| 两端回转 | 0.16~0.25 |

**防止失稳的措施**：
- 设置导杆或导套（导杆直径 ≈ D₁ - 1~2mm，导套内径 ≈ D₂ + 1~2mm）
- 减小长细比，使 b ≤ 推荐值
- 采用组合弹簧（内、外弹簧套装）

## 6. 圆柱螺旋拉伸弹簧设计
<!-- 来源: 第8篇轴承 第13页 -->

### 6.1 拉伸弹簧的特殊性
<!-- 来源: 第6篇连接与紧固 第77页 -->

拉伸弹簧与压缩弹簧的设计原理基本相同，但有以下特殊点：

1. **初始拉力**：密卷拉伸弹簧在无载荷状态下圈与圈之间已存在压紧力，由初始拉力 F₀ 表征
2. **无支撑圈**：端部钩环直接参与工作
3. **钩环强度**：端部钩环是薄弱环节，需专门校核

### 6.2 设计公式
<!-- 来源: 第6篇连接与紧固 第67页 -->

拉伸弹簧的剪应力计算与压缩弹簧相同：

$$τ = \frac{8FDK}{πd³}$$

当存在初始拉力时，工作载荷与变形关系为：

$$F = F₀ + kf$$

其中 F₀ 为初始拉力，通常取：

$$F₀ ≈ \frac{τ₀πd³}{8D}$$

初始切应力 τ₀ 推荐值：

| 材料 | τ₀ (MPa) |
|------|----------|
| 碳素弹簧钢丝 | 0.15σ_b ~ 0.30σ_b |
| 合金弹簧钢 | 0.20σ_b ~ 0.35σ_b |
| 不锈钢 | 0.10σ_b ~ 0.25σ_b |

### 6.3 端部钩环形式
<!-- 来源: 第8篇轴承 第13页 -->

| 钩环类型 | 代号 | 特点 |
|----------|------|------|
| 半圆钩环 | LI | 制造简单，用于小尺寸 |
| 长臂半圆钩环 | LII | 安装方便，受力较好 |
| 圆钩环 | LIII | 常用形式 |
| 可转钩环 | LIV | 可调节方向 |
| 偏心圆钩环 | LV | 安装空间受限 |
| 螺纹端 | LVI | 高强度连接 |
| 锥形端 | LVII | 用于大尺寸弹簧 |

### 6.4 钩环弯曲应力校核
<!-- 来源: 第7篇轴及其连接 第20页 -->

钩环弯曲处的应力：

$$σ_b = \frac{32F}{\pi d³}·\frac{r²}{r² - d²/4}$$

其中 r 为钩环弯曲半径。

**强度条件**：σ_b ≤ [σ_b]

## 7. 圆柱螺旋扭转弹簧设计
<!-- 来源: 第8篇轴承 第13页 -->

### 7.1 基本特点
<!-- 来源: 第8篇轴承 第13页 -->

扭转弹簧承受绕弹簧轴线方向的扭矩，端部绕轴线产生角位移。

### 7.2 设计公式
<!-- 来源: 第6篇连接与紧固 第67页 -->

#### 弯曲应力 σ
<!-- 来源: 第7篇轴及其连接 第371页 -->

$$σ = \frac{K_b·T}{Z} = \frac{32K_b·T}{\pi d³}$$

其中：
- T — 扭矩 (N·mm)
- K_b — 弯曲曲度系数
- Z — 抗弯截面模量 (mm³)

**弯曲曲度系数 K_b**：

$$K_b = \frac{4C - 1}{4C - 4}$$

#### 扭转角 φ
<!-- 来源: 第7篇轴及其连接 第19页 -->

$$φ = \frac{64TDn}{Ed⁴} = \frac{3670TDn}{Ed⁴} (°)$$

或：

$$φ = \frac{\pi TDn}{Ed⁴} × 180 \ (\text{度})$$

#### 扭转刚度 k_t
<!-- 来源: 第7篇轴及其连接 第165页 -->

$$k_t = \frac{T}{φ} = \frac{Ed^4}{64Dn} \quad (\text{N·mm/rad})$$

### 7.3 扭转弹簧设计参数
<!-- 来源: 第6篇连接与紧固 第3页 -->

| 参数 | 符号 | 说明 |
|------|------|------|
| 扭矩 | T | 工作扭矩 (N·mm) |
| 弯曲应力 | σ | 弹簧圈内侧应力 (MPa) |
| 扭转角 | φ | 工作扭转角 (度) |
| 自由角度 | φ₀ | 未受载时的旋转角位置 |
| 圈间间隙 | δ | 最大载荷时圈间应留≥0.5mm间隙 |
| 扭臂长度 | a | 端部扭臂长度 (mm) |

## 8. 板簧设计
<!-- 来源: 第8篇轴承 第13页 -->

### 8.1 板簧分类
<!-- 来源: 第8篇轴承 第13页 -->

| 类型 | 结构特点 | 适用场合 |
|------|----------|----------|
| 多片钢板弹簧 | 若干长度递减的钢板叠合 | 卡车、客车悬架 |
| 少片变截面钢板弹簧 | 2~4片，截面沿长度变化 | 轿车、轻型车 |
| 单片钢板弹簧 | 单片抛物线形 | 部分轿车后悬架 |
| 碟形弹簧 | 圆锥形薄片，可叠合使用 | 大载荷小行程 |

### 8.2 多片钢板弹簧设计公式
<!-- 来源: 第6篇连接与紧固 第67页 -->

#### 刚度

$$\frac{1}{k} = \frac{L³}{6E}·\sum_{i=1}^{n} \frac{l_i}{I_i}$$

其中 L 为有效长度，l_i 和 I_i 分别为第 i 片的长度和截面惯性矩。

#### 应力

$$\sigma_{\text{max}} = \frac{6FL}{nbh²}$$

其中 b 为板宽，h 为板厚，n 为片数。

### 8.3 碟形弹簧设计公式
<!-- 来源: 第6篇连接与紧固 第3页 -->

| 参数 | 公式 | 说明 |
|------|------|------|
| 载荷 F | F = (4Eδ³/(K₁D²(1-μ²))) · [ (h₀/δ - f/δ)(h₀/δ - f/(2δ)) + 1 ] | 单个碟簧 |
| 应力 σ_OM | σ_OM = (4Eδ²/(K₁D²(1-μ²))) · K₂ · (h₀/δ - f/δ) | 关键点应力 |
| 刚度 k | k = dF/df | 非线性特性 |
| 组合刚度（叠合） | k_total = n·k | n 个相同碟簧叠合 |
| 组合刚度（对合） | 1/k_total = 1/k₁ + 1/k₂ + ... | 串联组合 |

其中：
- D — 碟簧外径
- δ — 碟簧厚度
- h₀ — 碟簧内锥高度
- f — 变形量
- K₁, K₂ — 碟簧几何系数

## 9. 弹簧设计步骤
<!-- 来源: 第8篇轴承 第13页 -->

### 9.1 压缩/拉伸弹簧设计完整步骤
<!-- 来源: 第6篇连接与紧固 第250页 -->

#### 步骤1：明确设计条件
<!-- 来源: 第6篇连接与紧固 第296页 -->

| 输入参数 | 符号 | 单位 | 说明 |
|----------|------|------|------|
| 最大工作载荷 | F₂ | N | 工作时的最大载荷 |
| 最小工作载荷 | F₁ | N | 预紧载荷 |
| 工作行程 | h | mm | h = f₂ - f₁ |
| 弹簧外径/内径/中径约束 | D₂/D₁/D | mm | 安装空间限制 |
| 载荷性质 | Ⅰ/Ⅱ/Ⅲ类 | — | 交变频率及重要性 |
| 工作环境 | 温度/介质 | — | 材料选择依据 |
| 其它约束 | — | — | 成本、寿命、制造条件 |

#### 步骤2：选择材料和许用应力
<!-- 来源: 第6篇连接与紧固 第69页 -->

1. 根据工作条件选择材料（参见第3节材料选择对照表）
2. 确定许用切应力 [τ]（参见3.7节许用应力表）
3. 确定切变模量 G

#### 步骤3：初选弹簧指数 C
<!-- 来源: 第6篇连接与紧固 第296页 -->

1. 根据线径 d 初选 C（参见4.3节）
2. 若线径未知，先预设 C = 5~8
3. 计算曲度系数 K（详见4.4节）

#### 步骤4：计算弹簧线径 d
<!-- 来源: 第6篇连接与紧固 第296页 -->

由强度条件：

$$d ≥ \sqrt{\frac{8F₂CK}{\pi[τ]}}$$

**计算流程**：
1. 假设 d 初值，查对应 C 推荐范围
2. 选 C → 算 K → 算 d
3. 若计算 d 与假设不符，重新迭代
4. 圆整到标准钢丝直径

**标准钢丝直径系列 (mm)**：

0.2, 0.25, 0.3, 0.35, 0.4, 0.45, 0.5, 0.6, 0.7, 0.8, 0.9, 1.0, 1.2, 1.6, 2.0, 2.5, 3.0, 3.5, 4.0, 4.5, 5.0, 6.0, 7.0, 8.0, 9.0, 10, 12, 14, 16, 18, 20, 22, 25, 28, 30, 32, 35, 40, 45, 50

#### 步骤5：计算弹簧中径 D 和尺寸
<!-- 来源: 第6篇连接与紧固 第296页 -->

$$D = Cd$$
$$D₂ = D + d$$
$$D₁ = D - d$$

校核安装空间。

#### 步骤6：确定有效圈数 n
<!-- 来源: 第6篇连接与紧固 第296页 -->

由变形条件：

$$n = \frac{Gd⁴f₂}{8F₂D³} = \frac{Gd}{8C³k}$$

圆整到 0.5 的整数倍。压缩弹簧 n ≥ 2 圈，推荐 n ≥ 3~4 圈。

#### 步骤7：计算总圈数 n₁
<!-- 来源: 第6篇连接与紧固 第296页 -->

| 端部形式 | 总圈数 n₁ |
|----------|-----------|
| 两端磨平 | n₁ = n + 1.5~2.5 |
| 两端不磨 | n₁ = n + 2~3 |
| 拉伸弹簧 | n₁ = n |

#### 步骤8：计算弹簧刚度 k
<!-- 来源: 第6篇连接与紧固 第296页 -->

$$k = \frac{Gd⁴}{8D³n}$$

校核刚度是否满足要求。

#### 步骤9：确定自由高度 H₀
<!-- 来源: 第6篇连接与紧固 第296页 -->

**压缩弹簧**（两端磨平）：
- n₁ = n + 1.5 时：H₀ = nt + 1.5d
- n₁ = n + 2 时：H₀ = nt + 2d

其中节距 t 由下式确定：

$$t = d + \frac{f₂}{n} + δ_{\text{min}}$$

δ_min 为余隙（最大载荷时圈间最小间隙），通常取 δ_min ≥ 0.1d。

**并紧高度**：H_b = n₁·d（端部磨平时）或 H_b = (n₁ - 0.5)d

**限界检查**：H₀ ≥ H_b + f₂

#### 步骤10：校核稳定性（压缩弹簧）
<!-- 来源: 第8篇轴承 第181页 -->

- 长细比 b = H₀/D
- 如 b 超出许用范围，需加导杆/导套或重新设计

#### 步骤11：校核疲劳强度（动载荷弹簧）
<!-- 来源: 第6篇连接与紧固 第296页 -->

参见第12节疲劳强度部分。

#### 步骤12：绘制弹簧工作图
<!-- 来源: 第6篇连接与紧固 第296页 -->

标注弹簧特性曲线（F-f 曲线），图上需标明：
- 自由高度 H₀
- 预压高度 H₁（预压载荷 F₁）
- 工作高度 H₂（最大载荷 F₂）
- 极限高度 H_j（极限载荷 F_j）
- 技术要求（材料、热处理、表面处理等）

## 11. 选用原则
<!-- 来源: 第8篇轴承 第13页 -->

### 11.1 通用选用原则
<!-- 来源: 第8篇轴承 第13页 -->

| 序号 | 原则 | 说明 |
|------|------|------|
| 1 | **优先选用标准件** | 应优先选用国标/行标中的标准弹簧，降低成本、缩短周期 |
| 2 | **载荷特性匹配** | 载荷大小、性质（静/动/冲击）决定弹簧类型和材料等级 |
| 3 | **安装空间约束** | D₂ 和 H₀ 必须满足安装空间要求，导杆/导套间隙要合理 |
| 4 | **变形量匹配** | 确保工作行程在弹簧线性变形范围内，避免进入非线性区 |
| 5 | **疲劳寿命匹配** | 交变载荷次数 ≥ 10⁶ 时选择Ⅰ类弹簧，采用高强度材料 |
| 6 | **环境适应性** | 温度、腐蚀、磁场等环境条件决定材料选择和表面处理 |
| 7 | **经济性** | 在满足性能前提下选用最经济的材料及制造工艺 |
| 8 | **互换性** | 同一设备中同规格弹簧应可互换 |

### 11.2 按载荷大小选用
<!-- 来源: 第8篇轴承 第13页 -->

| 载荷范围 | 推荐弹簧类型 | 推荐材料 | 说明 |
|----------|--------------|----------|------|
| 微载荷 (<10N) | 小螺旋弹簧、平面涡卷弹簧 | 不锈钢丝、磷青铜 | 仪表、精密机构 |
| 小载荷 (10~100N) | 圆柱螺旋弹簧 | 65Mn、碳素钢丝 | 一般机电器具 |
| 中载荷 (100~5000N) | 圆柱螺旋弹簧 | 60Si2Mn、55Si2MnB | 一般机械 |
| 大载荷 (5~50kN) | 圆柱螺旋弹簧、碟形弹簧 | 60Si2CrA、50CrVA | 工程机械、重型设备 |
| 超大载荷 (>50kN) | 碟形弹簧组合、环形弹簧 | 60Si2Mn | 重型缓冲器、大型锻压设备 |

### 11.3 按使用环境选用
<!-- 来源: 第8篇轴承 第13页 -->

| 环境条件 | 推荐措施 |
|----------|----------|
| 高温 (>150°C) | 使用耐热钢（50CrVA、30W4Cr2VA），降低许用应力 |
| 低温 (<-40°C) | 使用奥氏体不锈钢，避免低温脆性 |
| 腐蚀介质 | 不锈钢（304、316）、铜合金、涂覆防腐层 |
| 潮湿环境 | 镀锌、镀镉、镀铬或达克罗涂层 |
| 磁场环境 | 无磁性材料（铍青铜、奥氏体不锈钢） |
| 真空环境 | 选用低挥发性材料，特殊润滑处理 |
| 食品设备 | 304/316不锈钢，符合卫生标准 |
| 医疗设备 | 符合生物相容性要求 |

### 11.4 组合弹簧选用
<!-- 来源: 第8篇轴承 第13页 -->

当单个弹簧无法满足空间或载荷要求时，可采用组合弹簧。

| 组合方式 | 形式 | 特点 |
|----------|------|------|
| **串联组合** | 两个或以上弹簧首尾相接 | 刚度减小（1/k = 1/k₁ + 1/k₂ + ...） |
| **并联组合** | 多个弹簧并列安装 | 刚度增大（k = k₁ + k₂ + ...） |
| **内外组合** | 不同直径弹簧套装（内外反向旋） | 空间利用率高，避免卡死 |
| **叠合组合** | 碟形弹簧相同方向叠合 | 刚度成倍增加 |

**内外组合弹簧的设计要点**：
- 内、外弹簧旋向相反（一左一右）
- 弹簧间隙 ≥ d/4
- 均分载荷：F₂_total = F₂_out + F₂_in
- 并紧高度一致：H_b_out = H_b_in
- 刚度配合：k_out / k_in = F_out / F_in

### 11.5 按弹簧指数 C 选用指南
<!-- 来源: 第7篇轴及其连接 第453页 -->

| C 值 | 适用场景 |
|------|----------|
| 4~5 | 大载荷、小空间，制造稍困难 |
| 5~7 | 一般用途，工艺性好，推荐 |
| 7~10 | 中小载荷，要求变形大 |
| 10~16 | 微小载荷，要求灵敏度高，需防失稳 |

## 12. 疲劳强度与稳定性
<!-- 来源: 第8篇轴承 第105页 -->

### 12.1 疲劳强度校核
<!-- 来源: 第7篇轴及其连接 第45页 -->

对于承受交变载荷的重要弹簧（Ⅰ类、Ⅱ类），需进行疲劳强度校核。

#### 疲劳应力计算
<!-- 来源: 第8篇轴承 第13页 -->

| 应力参数 | 公式 | 说明 |
|----------|------|------|
| 最小工作应力 | τ_min = 8F₁DK/(πd³) | 预紧应力 |
| 最大工作应力 | τ_max = 8F₂DK/(πd³) | 最大工作应力 |
| 应力幅 | τ_a = (τ_max - τ_min)/2 | 交变应力幅值 |
| 平均应力 | τ_m = (τ_max + τ_min)/2 | 应力平均值 |
| 应力循环特征 | r = τ_min/τ_max | r ≥ -1 |

#### 疲劳安全系数
<!-- 来源: 第7篇轴及其连接 第22页 -->

$$S = \frac{τ₀ + ψτ·τ_m}{τ_a} ≥ [S]$$

其中：
- τ₀ — 材料的脉冲疲劳极限
- ψτ — 材料对应力比的敏感系数
- [S] — 许用安全系数（Ⅰ类 [S] = 1.5~2.0，Ⅱ类 [S] = 1.3~1.7）

### 12.2 提高疲劳寿命的措施
<!-- 来源: 第7篇轴及其连接 第20页 -->

| 措施 | 作用 |
|------|------|
| **喷丸强化** | 表面形成压应力，显著提高疲劳寿命 |
| **减少应力集中** | 降低端部曲率突变，大圆弧过渡 |
| **表面处理** | 镀层、渗碳、渗氮提高表面强度 |
| **避免缺口** | 制造过程中避免划伤、压痕 |
| **尺寸因素** | 大直径钢丝的尺寸效应更明显 |
| **预变形处理** | 产生残余有利应力 |
| **改进材料** | 选用纯净度高的材料 |

### 12.3 弹簧共振校核
<!-- 来源: 第8篇轴承 第13页 -->

弹簧的自振频率应避开工作频率，以避免共振。

**压缩弹簧的自振频率**：

$$f = \frac{d}{2\pi D²n}·\sqrt{\frac{G}{2ρ}} \ (\text{Hz})$$

其中 ρ 为材料密度（钢的 ρ = 7.85×10⁻⁶ kg/mm³）。

**共振条件**：工作频率 ≤ f/2 或 ≥ 2f 时安全。

### 13.1 制造流程
<!-- 来源: 第8篇轴承 第13页 -->

#### 冷卷弹簧制造流程
<!-- 来源: 第8篇轴承 第184页 -->

```
盘料检验 → 冷卷成型 → 切断 → 去应力退火 → 端部加工（磨平/钩环） 
    → 淬火+回火（如需要）→ 喷丸 → 立定处理 → 强压处理（如需要）
    → 表面处理（镀锌/磷化/涂装）→ 检验 → 包装
```

#### 热卷弹簧制造流程
<!-- 来源: 第8篇轴承 第184页 -->

```
棒料检验 → 加热 → 热卷成型 → 切断 → 淬火+回火 
    → 喷丸强化 → 立定处理 → 端面磨平 → 检验 → 表面处理 → 包装
```

### 13.2 各工艺环节要点
<!-- 来源: 第8篇轴承 第13页 -->

| 工序 | 要点说明 |
|------|----------|
| **冷卷** | 适用于 d ≤ 10mm，自动卷簧机连续卷制 |
| **热卷** | 适用于 d > 10mm，加热至 800~1000°C 卷制 |
| **去应力退火** | 冷卷后 250~350°C，消除加工应力 |
| **淬火** | 加热至淬火温度后油冷或水冷（合金钢油冷） |
| **回火** | 按要求硬度选择合适的回火温度 |
| **喷丸强化** | 钢丸直径 0.3~2.0mm，喷射速度 50~100m/s |
| **立定处理** | 反复压缩至极限载荷多次，稳定尺寸 |
| **强压处理** | 超过弹簧材料的屈服点，产生残余应力 |
| **表面处理** | 镀锌、磷化、发蓝、达克罗等 |

### 14.3 弹簧特性曲线（F-f 曲线）

   - 预压点（F₁, f₁）

   - 自由状态点（0, 0）

5. **刚度标注** — k = ΔF/Δf
6. **公差范围** — 上下偏差线
