## 5. 镁合金
<!-- 来源: 第1卷  第7页 -->

### 5.1 镁合金特点
<!-- 来源: 第1卷  第1009页 -->
- 密度最低的结构金属（1.74~1.85 g/cm³）
- 比强度/比刚度高
- 减振性好
- 易切削加工
- 耐蚀性较差（需表面处理）

## 7. 有色金属选用速查
<!-- 来源: 第1卷  第1009页 -->

| 工况 | 推荐材料 | 理由 |
|------|----------|------|
| 导电 | T2/T3 纯铜 | 导电率最高 |
| 导热 | T2 纯铜 / 6063 铝 | 导热好 |
| 耐蚀（一般） | 5052 铝 / TA1 钛 | 耐大气腐蚀 |
| 耐蚀（海洋） | QAl9-4 / TA1 钛 | 耐氯离子 |
| 高强度 | 7075 铝 / TC4 钛 | 比强度高 |
| 轻量化 | 镁合金 / 铝 | 密度最低 |
| 耐磨轴承 | 锡青铜 QSn6.5 / 巴氏合金 | 减摩性好 |
| 弹簧 | QBe2 / QSn6.5 | 弹性极限高 |
| 易切削 | HPb59-1 黄铜 | 含铅易切 |
| 深冲压 | H68 黄铜 / 5052 铝 | 塑性好 |
