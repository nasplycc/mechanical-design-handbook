### 1.3 人体尺寸的年龄变化
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

人体尺寸随年龄变化，设计时应考虑：

| 年龄段 | 变化特点 | 影响设计参数 |
|--------|----------|-------------|
| 18~25岁 | 身体发育完成，尺寸稳定 | - |
| 26~35岁 | 尺寸达到最大，力量最充足 | 操作力设计参考 |
| 36~45岁 | 身高轻微降低（约5~10mm），腰围增加 | 座椅宽度 |
| 46~55岁 | 身高降低约10~20mm，力量下降 | 操作力安全余量 |
| 55岁以上 | 身高降低约20~40mm，视力下降 | 显示字符尺寸、操作力 |
| 儿童（6~12岁） | 身高约为成年人60~80% | 儿童安全防护间距 |

## 2. 人体尺寸百分位数选择
<!-- 来源: 第1卷 第1篇一般设计资料 第625页 -->

### 2.1 百分位数的选用原则
<!-- 来源: 第1卷 第1篇一般设计资料 第625页 -->

在设计中，百分位选择取决于设计参数对使用者的影响程度：

| 设计原则 | 采用百分位 | 适用范围 | 示例 |
|----------|-----------|----------|------|
| 满足大尺寸人群 | P95~P99（上限值） | 尺寸需要"通过" | 门高、通道宽度、座位间距 |
| 满足小尺寸人群 | P1~P5（下限值） | 尺寸需要"够到" | 操作手柄高度、搁板高度 |
| 满足中等人群 | P50 | 可调节范围的中心值 | 可调座椅的调节范围中值 |
| 可调节设计 | P5~P95 | 用户可自行调节 | 座椅高度、方向盘位置 |
| 极限设计 | P1~P99 | 安全要求极高 | 紧急出口、安全带 |

### 2.2 百分位数选择表
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 使用场景 | 设计参数 | 推荐百分位 | 说明 |
|----------|----------|-----------|------|
| 肢体活动范围 | 最大可达距离 | P5（小身材） | 确保小身材也能触及 |
| 肢体活动范围 | 最小空间尺寸 | P95（大身材） | 确保大身材也能容纳 |
| 操作力 | 最大操作力 | P5（小力量） | 确保小力量者也能操作 |
| 眼高 | 视区上限 | P5（低眼高） | 确保矮个子能看到 |
| 眼高 | 视区下限 | P95（高眼高） | 确保高个子也能看到 |
| 坐姿膝高 | 桌下净空 | P95~P99 | 确保腿部空间 |
| 坐姿臀宽 | 座椅宽度 | P95~P99 | 确保臀部容纳 |
| 手操作区 | 水平面工作区 | P5~P95 | 可调范围 |
| 通道宽度 | 最小宽度 | P95（或P99） | 确保通行 |
| 重心高度 | 稳定性计算 | P95 | 最不稳定状态 |
| 指尖操作 | 按钮间距 | P95（手指宽度） | 防止误操作 |

### 2.3 功能修正量
<!-- 来源: 第1卷 第1篇一般设计资料 第704页 -->

设计中应在人体尺寸基础上增加功能修正量：

| 修正类型 | 修正量/mm | 说明 |
|----------|-----------|------|
| 着装修正量 | 立姿身高+25，坐姿+10~15 | 衣物引起 |
| 心理修正量 | 通道+100~200 | 避免压抑感 |
| 姿势修正量 | 立姿-10（自然姿势） | 放松立姿高度降低 |
| 操作动作修正 | 上肢前伸-50~80 | 实际可达距离小于测量值 |
| 鞋底修正 | +20~30（身高方向） | 鞋底厚度 |

### 2.4 人体尺寸的设计应用公式
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 设计项目 | 计算公式 | 说明 |
|----------|----------|------|
| 门/通道高度 | H = h_max(P95/P99) + 鞋厚 + 心理修正 + 安全帽 | 确保95%以上人群通过 |
| 搁板/操作面高度 | H = h_min(P5) - 鞋厚 - 姿势修正 | 确保小身材可触及 |
| 座椅高度 | H = (P5小腿+足高) ~ (P95小腿+足高) 可调 | 可调范围覆盖90%人群 |
| 显示器高度 | H = P5眼高 ~ P95眼高，推荐偏向低百分位 | 兼顾坐姿和立姿 |
| 工作台宽度 | W = P95臀宽 + 活动间隙（200~300mm） | 确保工作空间 |
| 按钮最小间距 | D = P95指尖宽 + 间隙（约20mm） | 防止误触 |
| 通道最小宽度 | W = P95最大肩宽 + 运动余量（200~300mm） | 单人通行 |
| 双人通道宽 | W = 2 × P95肩宽 + 间距（约500mm） | 双向通行 |
| 通道门宽度 | W = P95最大肩宽 + 衣着量 + 心理余量 | 一般≥800mm |

### 4.1 工作岗位尺寸设计原则（GB/T 14776-1993）
<!-- 来源: 第1卷 第5篇机械产品结构设计 第53页 -->

GB/T 14776-1993《人类工效学 工作岗位尺寸设计原则》规定了工作岗位尺寸设计的基本原则和要求。

| 设计原则 | 内容 | 说明 |
|----------|------|------|
| 适应性原则 | 适应人体尺寸 | 以人体尺寸数据为基础 |
| 可调性原则 | 优先考虑可调设计 | 适应不同体型人群 |
| 功能原则 | 满足工作功能要求 | 操作方便、高效 |
| 安全原则 | 避免不安全因素 | 防止碰撞、挤压、跌落 |
| 舒适原则 | 保持自然姿势 | 减少疲劳和职业损伤 |
| 可视性原则 | 保证视线通畅 | 观察方便、不眩光 |

### 4.2 工作岗位分类与尺寸
<!-- 来源: 第1卷 第1篇一般设计资料 第134页 -->

| 岗位类型 | 坐姿/立姿 | 台面高度/mm | 活动空间/m² | 典型应用 |
|----------|-----------|-------------|-------------|----------|
| 精细工作 | 坐姿 | 800~950 | ≥1.5 | 装配、检验、绘图 |

## 12. 人机系统设计原则
<!-- 来源: 第1篇 第3页 [前言] -->

### 12.1 人机系统设计基本原则
<!-- 来源: 第1卷 第1篇一般设计资料 第296页 -->

| 原则 | 说明 | 应用示例 |
|------|------|----------|
| 以人为中心 | 人机系统核心是人 | 操作界面按人认知设计 |
| 功能分配合理 | 人擅长的功能给人 | 机器-重复/高精度/大力量 |
| | 机器擅长的功能给机器 | 人-判断/决策/适应 |
| 反馈原则 | 系统状态及时反馈 | 声音/灯光/振动/屏幕 |
| 一致性原则 | 操作方式一致 | 左上启动，右下停止 |
| 容错原则 | 允许操作失误 | 防止误触、撤销功能 |
| 可视性原则 | 系统状态一目了然 | 仪表、指示灯 |
| 文化适应性 | 符合使用习惯 | 文字、符号、方向 |

### 12.2 人机功能分配
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 功能特性 | 人的优势 | 机器的优势 |
|----------|----------|-------------|
| 感知能力 | 模式识别、非规则信号识别 | 高精度测量、定量感知 |
| 信息处理 | 模糊推理、经验判断、创造 | 快速计算、大容量存储 |
| 运动能力 | 精细控制、灵活适应 | 精确重复、大力量输出 |
| 记忆能力 | 长期经验、联想能力 | 精确记忆、快速检索 |
| 决策能力 | 不确定情况下判断、适应新情况 | 规则确定时快速决策 |
| 耐力 | 易疲劳、有情绪变化 | 持续工作、不受情绪影响 |
| 多通道 | 听觉/视觉/触觉可切换 | 单通道但精确 |
| 抗干扰 | 有噪声环境可辨别重要信号 | 受干扰影响小 |
