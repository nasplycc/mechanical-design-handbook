### 2.1 功能准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 要点 | 设计方法 | 实例 |
|:---|:---|:---|
| 明确功能分解 | 功能树分析法（FAST） | 整机→子系统→零部件功能逐层分解 |
| 运动功能实现 | 选型匹配：转动→转动（齿轮）、转动→直线（丝杠） | 机床主轴驱动 |
| 承载功能 | 传力路径最简原则 | 压机机架→直接传力→工件 |
| 定位功能 | 6点定位原则（完全约束） | 工件在夹具中定位 |
| 导向功能 | 导向面长度≥行程的2/3 | 直线导轨设计 |
| 密封功能 | 静密封（O圈）→动密封（油封/机械密封） | 液压缸活塞密封 |
| 锁紧功能 | 自锁（螺纹/斜面）或额外锁紧（弹簧垫圈） | 千斤顶自锁 |
| 连接功能 | 可拆（螺纹连接）vs 不可拆（焊接/铆接） | 按照维修要求选型 |
| 功能冗余 | 关键功能双通道 | 飞机刹车系统 |

**功能设计步骤：**

```
需求输入 → 功能分解（总功能→子功能） → 功能元求解 → 方案组合 → 评价筛选 → 详细设计
```

### 2.2 强度准则
<!-- 来源: 第5篇机械产品结构设计 第25页 -->

| 强度类型 | 判断依据 | 设计方法 |
|:---|:---|:---|
| 静强度 | σ_max ≤ [σ] | 安全系数法 n=n₁×n₂×n₃ = 1.5~3.5 |
| 疲劳强度 | σ_a ≤ σ_{-1}/n_f | S-N曲线，疲劳极限n_f=1.3~3.0 |
| 接触强度 | σ_H ≤ [σ_H] | Hertz接触公式 |
| 冲击强度 | σ_d = K_d·σ_st | 动载系数K_d=1~3 |
| 挤压强度 | σ_bs ≤ [σ_bs] | 接触面积校核 |
| 剪切强度 | τ ≤ [τ] | 剪切面积校核 |
| 扭转强度 | τ_T ≤ [τ_T] | T/W_t校核 |
| 复合强度 | σ_r ≤ [σ] | 第四强度理论（Mises） |
| 局部强度 | 应力集中K_σ≤许用 | 圆角/过渡结构 |

**安全系数选取：**

| 工况 | 安全系数 n | 说明 |
|:---:|:---:|:---|
| 静载、材料可靠 | 1.2~1.5 | 塑性材料 |
| 一般动载 | 1.5~2.5 | 通用机械 |
| 较大冲击 | 2.5~4.0 | 冲击设备 |
| 疲劳（无限寿命） | 1.3~2.5 | 考虑存活率 |
| 安全关键件 | 3.0~5.0 | 电梯吊索、航空件 |

**强度设计三步法：**

1. **受力分析**：分清外力类别（静/动/冲击/交变）
2. **危险截面确定**：应力最大点、应力集中处
3. **强度校核**：许用应力法或极限状态法

### 2.3 刚度准则
<!-- 来源: 第5篇机械产品结构设计 第39页 -->

| 变形类型 | 控制指标 | 校核公式 |
|:---|:---|:---|
| 弯曲变形 | y_max ≤ [y] | $y = \frac{FL^3}{3EI}$（悬臂梁） |
| 扭转变形 | θ_max ≤ [θ] | $\theta = \frac{TL}{GJ}$ |
| 接触变形 | δ ≤ [δ] | Hertz公式 |
| 拉伸/压缩变形 | ΔL ≤ [ΔL] | $\Delta L = \frac{NL}{EA}$ |
| 稳定性 | P ≤ P_cr/n_st | Euler公式 $P_{cr}=\frac{\pi^2EI}{(\mu L)^2}$ |

**提高刚度的方法（按效果排序）：**

| 方法 | 效果 | 说明 |
|:---|:---|:---:|
| 增大截面惯性矩 I | ★★★★★ | 采用空心截面、工字钢、加强筋 |
| 缩短跨度 L | ★★★★★ | 减小梁的长度 |
| 合理布置支撑 | ★★★★☆ | 两端支撑→多点支撑 |
| 选用高模量材料 | ★★★☆☆ | 钢(E=206GPa)>铝(E=70GPa) |
| 减小载荷 | ★★☆☆☆ | 设计轻量化 |
| 预变形补偿 | ★★☆☆☆ | 反向加工/预压 |

**截面惯性矩对比：**

| 截面形状 | 尺寸（mm） | I（cm⁴） | 重量比 | 刚度/重量比 |
|:---|:---:|:---:|:---:|:---:|
| 实心圆 | φ50 | 30.7 | 1.00 | 1.00 |
| 空心圆 | φ60/φ30 | 94.5 | 1.08 | **4.67** |
| 实心方 | 50×50 | 52.1 | 1.27 | 1.33 |
| 工字钢 | 100×68 | 245 | 1.18 | **6.77** |
| 槽钢 | 100×48 | 192 | 1.02 | **6.13** |
| 矩形(竖放) | 30×100 | 250 | 0.95 | **8.55** |

> **结论：** 同样重量下，工字钢和矩形竖放截面的刚度远优于实心圆，是轻量化首选。

### 2.4 工艺准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 工艺类型 | 核心要求 | 设计要点 |
|:---|:---|:---|
| 铸造工艺 | 拔模斜度1°~3°、避免尖角 | 壁厚均匀R≥3~5mm，过渡圆角 |
| 锻造工艺 | 分模面平直、纤维流向 | 锻压比≥2，脱模角≥3°~7° |
| 冲压工艺 | 最小弯曲半径r≥(0.5~1)×t | 孔距≥2t，避免窄槽 |
| 焊接工艺 | 焊缝布置对称、减少应力集中 | 焊缝对接尽量避开应力最大区 |
| 切削工艺 | 避免深孔、窄槽、内尖角 | 加工面应连续平直 |
| 热处理工艺 | 截面均匀、避免锐边 | 留磨削余量（渗碳淬火后） |
| 装配工艺 | 先内后外、先下后上、定位基准统一 | 预留操作空间、避免过定位 |

**工艺性检查清单：**

```
□ 壁厚是否均匀（铸造/注塑）？
□ 是否有足够的拔模斜度？
□ 加工面是否都在同一平面上？
□ 有无难加工的深孔/斜孔？
□ 有无过小的内圆角（刀具难进入）？
□ 零件是否方便装夹和找正？
□ 能否用标准刀具完成加工？
□ 装配顺序是否合理？
□ 能否分组对配（减少总装难度）？
```

### 2.5 经济性准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 影响因素 | 降低成本的方法 | 效果 |
|:---|:---|:---:|
| 材料成本 | 选用廉价材料替代（Q235代45钢） | 直降20%~50% |
| 加工成本 | 减少加工面、合并工序 | 工时↓30%~60% |
| 模具成本 | 一模多件、标准模架 | 分摊成本 |
| 装配成本 | 减少零件数、快装结构 | 装配时间↓50% |
| 质量成本 | 公差合理放宽 | 废品率↓ |
| 物流成本 | 标准包装、托盘化 | 运输效率↑ |
| 维修成本 | 模块化、易更换件 | 售后成本↓ |

**成本对比公式（批量生产）：**

$$C_{total} = C_{material} + C_{processing} + C_{assembly} + C_{overhead}$$

**降成本"三七原则"：**
- 70%的成本在设计阶段已决定
- 30%在制造和使用阶段控制

| 设计阶段 | 影响成本比例 | 控制能力 |
|:---:|:---:|:---:|
| 概念设计 | 60%~70% | ★★★★★ |
| 详细设计 | 20%~30% | ★★★☆☆ |
| 工艺设计 | 5%~10% | ★★☆☆☆ |
| 制造 | 0%~5% | ★☆☆☆☆ |

### 2.6 标准化准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 项目 | 标准化内容 | 标准代号示例 |
|:---|:---|:---|
| 尺寸标准化 | 优先数系（R5/R10/R20/R40） | GB/T 321-2005 |
| 螺纹标准化 | 公制螺纹M、管螺纹G | GB/T 193-2003 |
| 轴承标准化 | 深沟球轴承、角接触轴承 | GB/T 276-2013 |
| 密封件标准化 | O形圈、油封、垫片 | GB/T 3452.1-2005 |
| 紧固件标准化 | 螺栓、螺母、垫圈、销 | GB/T 5780~5786 |
| 键/花键标准化 | 平键、半圆键、渐开线花键 | GB/T 1095-2003 |
| 公差标准化 | 公差等级IT01~IT18 | GB/T 1800.1-2020 |
| 材料标准化 | 钢/铸铁/有色/非金属牌号 | GB/T 700、GB/T 9439等 |

**标准化设计原则：**

| 原则 | 含义 | 收益 |
|:---|:---|:---|
| 系列化 | 同一功能产品按参数分级 | 减少品种、降低成本 |
| 通用化 | 不同产品中采用相同零部件 | 互换性好、维修方便 |
| 模块化 | 功能独立模块组合 | 快速变型设计 |
| 组合化 | 标准功能单元组合 | 缩短开发周期 |

### 2.7 人机工程准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 设计要素 | 人体数据参考 | 设计应用 |
|:---|:---|:---|
| 操作力 | 手指2~5N、手5~20N、手臂10~40N | 按钮/手柄/摇轮设计 |
| 操作高度 | 站立750~900mm、坐姿600~750mm | 工作台面高度 |
| 视野范围 | 水平±60°、垂直上30°下40° | 仪表布置、显示屏定位 |
| 抓握尺寸 | 手柄φ20~50mm、按钮φ10~30mm | 握把粗细设计 |
| 施力方向 | 推>拉、下压>上提 | 操作方向与施力匹配 |
| 反应时间 | 视觉0.2~0.3s、触觉0.15~0.2s | 安全距离设计 |

**人机交互设计方法：**

| 设计目标 | 方法 |
|:---|:---|
| 减少误操作 | 防错设计（Poka-Yoke）、不同形状/颜色区分 |
| 降低疲劳 | 自然姿势操作、减振、力不平衡 |
| 信息清晰 | 重要信息置于视野中心、颜色编码 |
| 应急操作 | 急停按钮红色、大尺寸、易触及 |
| 左右手通用 | 对称设计、可调握把 |

### 2.9 腐蚀防护准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 腐蚀类型 | 防护方法 | 适用场合 |
|:---|:---|:---|
| 电化学腐蚀 | 阴极保护、牺牲阳极 | 船体、管道 |
| 化学腐蚀 | 选用耐蚀材料（不锈钢/塑料/陶瓷） | 化工设备 |
| 缝隙腐蚀 | 避免缝隙结构、密封处理 | 法兰连接 |
| 应力腐蚀 | 消除残余应力、选择抗SCC材料 | 高压容器 |
| 点蚀 | 添加合金元素（Mo） | 不锈钢设备 |
| 晶间腐蚀 | 低碳/稳定化处理（316L/321） | 焊接件 |
| 大气腐蚀 | 涂层（油漆/喷塑/镀层） | 通用机械 |

**涂层选择：**

| 涂层类型 | 厚度 | 耐候性 | 成本 | 适用 |
|:---|:---:|:---:|:---:|:---|
| 油漆 | 30~100μm | ★★★☆☆ | 低 | 室内设备 |
| 粉末喷涂 | 60~200μm | ★★★★☆ | 中 | 室外设备 |
| 镀锌 | 5~50μm | ★★★★☆ | 中 | 标准件 |
| 镀铬 | 10~30μm | ★★★☆☆ | 高 | 装饰/耐磨 |
| 阳极氧化 | 5~30μm | ★★★★★ | 中 | 铝件 |
| 达克罗 | 5~10μm | ★★★★★ | 高 | 高强紧固件 |
| 特氟龙 | 10~50μm | ★★★★☆ | 高 | 不粘/耐化学 |

**结构防腐蚀设计要点：**

- ✓ 避免积水凹面——设置排水孔
- ✓ 避免缝隙结构——减少搭接面
- ✓ 异种金属隔离——加绝缘垫片（电位差≤0.25V）
- ✓ 焊缝位于空气侧——不在积水侧
- ✓ 考虑涂装可达性——所有面可涂覆

### 2.10 安全准则
<!-- 来源: 第5篇机械产品结构设计 第53页 -->

| 安全类型 | 设计措施 | 说明 |
|:---|:---|:---|
| 机械安全 | 防护罩、联锁、双手启动 | 防止接触运动件 |
| 过载保护 | 安全销、安全联轴器、摩擦离合器 | 按设计断开 |
| 防松脱 | 防松螺母、开口销、弹簧垫圈 | 振动不松动 |
| 限位保护 | 行程开关、机械挡块 | 防止超程 |
| 防坠落 | 安全钩、双重支撑 | 起重设备 |
| 防触电 | 接地、绝缘、漏电保护 | 电气安全 |
| 防爆 | 防爆外壳、防爆电气 | 危险环境 |
| 安全色标 | 红(禁止/急停)/黄(警告)/蓝(指令)/绿(安全) | GB/T 2893-2020 |

**安全设计"三化"原则：**

```
本质安全化 → 保护安全化 → 警示安全化
     ↑              ↑              ↑
 消除危险     加防护装置     警告标识
（最优先）    （第二方案）    （最后手段）
```

**风险评估矩阵：**

| 严重度\\概率 | 很可能(A) | 可能(B) | 偶尔(C) | 极不可能(D) |
|:---:|:---:|:---:|:---:|:---:|
| 灾难性(4) | 不可接受 | 不可接受 | 需减轻 | 评审 |
| 严重(3) | 不可接受 | 需减轻 | 需减轻 | 可接受 |
| 轻微(2) | 需减轻 | 可接受 | 可接受 | 可接受 |
| 可忽略(1) | 可接受 | 可接受 | 可接受 | 可接受 |

### 2.11 环保准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 环保方向 | 设计方法 | 评价指标 |
|:---|:---|:---|
| 材料绿色化 | 选用可再生/可回收材料 | 可回收率≥95% |
| 减量化 | 减少材料用量、中空结构 | 单位功能材料消耗 |
| 节能设计 | 低摩擦、高效传动、轻量化 | 能效等级 |
| 可拆卸设计 | 卡扣替代粘接、标准化连接 | 拆解时间 |
| 易回收设计 | 单一材料、色标标识 | 回收率 |
| 无害化 | 禁用有害物质（RoHS） | 有害物含量 |
| 低噪声 | 减振、阻尼、隔声 | 噪声≤85dB(A) |
| 低排放 | 密封、过滤、循环 | 排放物指标 |

**RoHS限制物质（含量≤0.1%）：**

| 物质 | 主要来源 | 替代方案 |
|:---|:---|:---|
| 铅(Pb) | 焊料、PVC稳定剂 | 无铅焊料SnAgCu |
| 汞(Hg) | 开关、继电器 | 电子开关 |
| 镉(Cd) | 镀层、颜料 | 锌镍合金镀层 |

| W5 | 厚薄板直接对焊 | 熔合不良、烧穿 | 薄板侧加过渡段（斜度1:5） |
| W6 | 角焊缝焊脚过大(>1.2t) | 变形大、浪费 | 焊脚K=0.7~1.0t |
| W7 | 未留坡口（厚板） | 熔深不足 | t≥6mm开坡口 |
| W8 | 拘束度过大 | 焊接裂纹 | 适当放松约束、预热 |

### 5.4 切削加工设计禁忌
<!-- 来源: 第4篇机构 第119页 -->

| 编号 | 禁忌 | 后果 | 正确做法 |
|:---:|:---|:---|:---|
| M1 | 深孔（L/D>10） | 排屑难、刀具断 | 改为通孔、阶梯孔或改用镗削 |
| M2 | 加工面在不同平面 | 多次换刀换位 | 加工面布置在同一平面/轴线上 |
| M3 | 无退刀槽（螺纹/磨削） | 刀具干涉 | 设退刀槽宽≥1.5P |
| M4 | 内直角/内尖角槽 | 无法加工 | 内圆角R≥3mm或改为直通 |
| M5 | 孔轴线不平行/不垂直 | 加工复杂 | 孔轴线尽量互相平行/垂直 |
| M6 | 薄壁件无支撑 | 振动、变形 | 增加加强筋或留有支撑余量 |
| M7 | 公差全部标注过严 | 成本飞涨 | 非必要不开高公差 |
| M8 | 无中心孔（长轴件） | 无法两顶尖定位 | 长轴两端设中心孔 |

### 5.5 装配设计禁忌
<!-- 来源: 第2篇机械制图 第155页 -->

| 编号 | 禁忌 | 后果 | 正确做法 |
|:---:|:---|:---|:---|
| A1 | 多个定位面同时接触 | 过定位、装不上 | 一面两销定位 |
| A2 | 密封槽过深/过窄 | 密封件安装难 | 按O形圈标准槽设计（GB/T 3452.3） |
| A3 | 螺纹盲孔无底孔余量 | 螺杆顶底、假拧紧 | 余量≥(3~5)P |
| A4 | 无防错设计（对称件） | 装反 | 非对称、防错销/槽 |
| A5 | 操作空间不足（扳手位） | 工具无法使用 | 空间≥工具尺寸+50mm，转角≥60° |
| A6 | 轴承无轴向定位 | 轴向窜动 | 轴肩/卡簧/螺母定位 |
| A7 | 重件无吊装孔 | 搬运困难 | 设吊环螺钉孔 |
| A8 | 过盈量过大 | 装配困难、胀裂 | 按公差配合标准选型 |

### 5.6 结构安全设计禁忌
<!-- 来源: 第4篇机构 第119页 -->

| 编号 | 禁忌 | 后果 | 正确做法 |
|:---:|:---|:---|:---|
| S1 | 运动件无防护罩 | 夹伤/卷入 | 加固定防护罩，网孔<手指 |
| S2 | 高压管无安全阀 | 爆裂伤人 | 设安全阀+定期检验 |
| S3 | 高温面裸露 | 烫伤 | 隔热层+警示标识 |
| S4 | 无急停装置 | 紧急时无法停机 | 红色急停按钮，显眼位置 |
| S5 | 悬臂结构无安全系数 | 断裂坠落 | 安全系数n≥5 |
| S6 | 尖角锐边未倒角 | 划伤 | 锐边倒角C≥0.5mm |
| S7 | 重物无二级防护 | 坠落事故 | 双保险/安全钩 |

### 5.7 防腐蚀设计禁忌
<!-- 来源: 第2篇机械制图 第155页 -->

| 编号 | 禁忌 | 后果 | 正确做法 |
|:---:|:---|:---|:---|
| E1 | 异种金属直接接触 | 电化学腐蚀 | 加绝缘垫片（电位差≤0.25V） |
| E2 | 结构积水凹面 | 缝隙腐蚀 | 设排水孔/d≥3mm |

### 5.8 维修性设计禁忌
<!-- 来源: 第2篇机械制图 第155页 -->

| 编号 | 禁忌 | 后果 | 正确做法 |
|:---:|:---|:---|:---|
| R1 | 易损件埋藏在最内部 | 更换需全部拆卸 | 易损件布置在外部可达位置 |
| R2 | 所有连接都用焊接 | 无法拆卸 | 可拆连接（螺栓/卡扣） |

| R4 | 不同规格螺钉混用 | 需多种工具 | 同一设备限2~3种规格 |
| R5 | 无备件清单 | 维修时无配件 | 提供易损件清单+供应商 |

### 5.9 其他常见禁忌
