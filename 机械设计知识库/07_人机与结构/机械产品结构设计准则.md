## 1. 符合造型要求的结构设计准则
<!-- 来源: 第1篇一般设计资料 第733页 -->

### 1.1 造型设计基本原则
<!-- 来源: 第1篇 第3页 [前言] -->

| 设计原则 | 说明 | 应用方法 |
|----------|------|----------|
| 功能优先 | 造型服务于功能 | 结构形式首先满足力学和功能要求 |
| 简洁性 | 形态简洁明了 | 减少冗余线条和装饰 |
| 统一性 | 整体风格统一 | 线型、倒角、色彩协调一致 |
| 比例协调 | 各部件尺寸比例合理 | 黄金分割比0.618、整数比、相似比 |
| 均衡稳定 | 视觉重心稳定 | 上小下大、左右对称或视觉平衡 |
| 动感流畅 | 线条流畅有方向感 | 斜线/弧线引导视觉 |
| 人机适应 | 符合人机工程学 | 操作面高度、角度、手感 |
| 工艺性 | 适应制造工艺 | 铸造/锻造/焊接/注塑工艺性 |

### 1.2 形态设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 准则 | 要求 | 示例 |
|------|------|------|
| 线型统一 | 同一产品中直线/曲线风格一致 | 全部圆角过渡或全部直角 |
| 减少间断 | 避免造型上的连续性中断 | 外观面少分型线、少接缝 |
| 合理分割 | 面与面的过渡自然 | 采用圆角R≥3~5mm过渡 |
| 均匀壁厚 | 壁厚变化平缓 | 变化梯度≤1:3~1:4 |
| 避免锐角 | 减少锐利边缘和尖角 | 边缘倒角C≥1~2mm |
| 对称与平衡 | 视觉平衡考虑 | 多数机械产品对称造型 |
| 色彩统一 | 主色彩≤3种 | 基色+辅色+点缀色 |
| 表面质感 | 表面处理一致性 | 喷塑/喷漆/拉丝/抛光统一 |

### 1.3 轮廓与形状设计准则
<!-- 来源: 第1篇一般设计资料 第720页 -->

| 设计准则 | 具体要求 | 理由 |
|----------|----------|------|
| 整体轮廓清晰 | 主轮廓简练，便于识别 | 提高辨识度和美观性 |
| 避免复杂曲面 | 优先采用平面和简单曲面 | 降低模具/加工成本 |
| 转角圆滑 | 所有外转角R≥2mm | 安全、美观、模具寿命 |
| 分型面隐藏 | 分型线设在非外观面 | 提升外观品质 |
| 筋条隐蔽 | 加强筋设在内侧 | 外观完整 |
| 按键/面板凹陷 | 面板设定位凹台 | 防止误触，提升触感 |
| 通风口造型 | 通风栅格设计规则排列 | 兼顾功能和美观 |
| 缝隙均匀 | 装配缝隙均匀≤0.5mm | 外观精致 |

### 1.4 表面处理与装饰设计
<!-- 来源: 第2篇机械制图 第155页 -->

| 表面处理方式 | 适用材料 | 效果 | 设计注意事项 |
|-------------|----------|------|-------------|
| 喷漆 | 钢/铸铁/铝 | 色彩丰富，保护 | 考虑附着力，底漆+面漆 |
| 喷塑（粉末喷涂） | 钢/铝 | 耐磨，厚度均匀 | 厚度60~120μm |
| 阳极氧化 | 铝/铝合金 | 耐磨，绝缘 | 颜色受限（银/金/黑/红） |
| 镀铬 | 钢/铜 | 高光泽，耐磨 | 需抛光基面 |
| 镀锌 | 钢 | 防腐，性价比高 | 钝化处理防白锈 |
| 发黑/发蓝 | 钢 | 黑色，防锈中等 | 低成本，配合涂油 |
| 电泳 | 钢/铸铁 | 均匀涂装，防腐好 | 颜色以黑/灰为主 |
| 拉丝 | 不锈钢/铝 | 金属质感 | 统一方向，避免划伤 |
| 抛光 | 不锈钢/铝/铜 | 镜面光泽 | 高加工成本 |
| 丝印/移印 | 塑料/金属 | 文字/LOGO | 耐磨损等级 |
| 激光雕刻 | 金属/塑料 | 永久标记 | 精细度高 |
| IMD/IML | 塑料 | 图案在涂层内 | 高耐磨，立体感 |

### 1.5 色彩设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 项目 | 推荐方案 | 说明 |
|------|----------|------|
| 工业机械主色 | 灰（RAL 7035/7042）、蓝（RAL 5010/5015） | 耐污，视觉效果舒适 |
| 警示色 | 黄黑条纹、红色（RAL 3020） | 引起注意 |
| 安全色 | 绿色（通行/安全）、红色（停止/紧急） | GB 2893安全色 |
| 操纵件色 | 红色=停止，绿色=启动，黄色=注意 | 操纵器颜色编码 |
| 塑料件色 | 浅灰、米白、工业蓝 | 注塑件配色稳定 |
| 设备涂装 | 两层底漆+两层面漆 | 防腐+装饰 |
| 配色原则 | 主色≤3种，明度/彩度协调 | 避免高纯色大面积使用 |
| 抗褪色要求 | 户外设备UV稳定等级≥5级 | 添加UV吸收剂 |

### 1.6 造型与散热设计
<!-- 来源: 第2篇机械制图 第155页 -->

| 散热结构 | 造型特点 | 设计要求 |
|----------|----------|----------|
| 散热格栅 | 水平/垂直排列长槽 | 槽宽≥5mm，间距≥3mm |
| 散热筋片 | 平行筋片 | 间距≥壁厚的2倍 |
| 散热孔 | 圆孔阵列 | 孔径≥φ4mm，圆角处理 |
| 散热鳍片 | 外露鳍片造型 | 鳍片厚度≥1.5mm |
| 风扇格栅 | 同心圆/放射状 | 开口率≥50%，防手指伸入（≤φ8mm） |

## 2. 符合载荷要求的结构设计准则
<!-- 来源: 第1篇一般设计资料 第735页 -->

### 2.1 载荷分类与设计考虑
<!-- 来源: 第1篇一般设计资料 第808页 -->

| 载荷类型 | 描述 | 设计考虑 |
|----------|------|----------|
| 静载荷 | 大小、方向、位置不变 | 强度、刚度、稳定性 |
| 动载荷 | 随时间变化 | 疲劳强度、振动分析 |
| 冲击载荷 | 瞬时大幅变化 | 动态放大系数、缓冲结构 |
| 交变载荷 | 循环变化 | 疲劳极限、应力集中 |
| 热载荷 | 温度变化引起 | 热膨胀补偿、散热 |
| 风载荷 | 风力作用 | 风压计算、高耸结构 |
| 地震载荷 | 地震作用 | 抗震设计、阻尼器 |
| 装配载荷 | 装配时作用力 | 过盈量控制、预紧力 |

### 2.2 强度设计准则
<!-- 来源: 第5篇机械产品结构设计 第24页 -->

| 准则 | 内容 | 计算公式/方法 |
|------|------|--------------|
| 静强度 | 最大应力≤许用应力 | σ_max ≤ [σ] = σ_s / n_s 或 σ_b / n_b |
| 疲劳强度 | 循环应力下不产生疲劳裂纹 | σ_max ≤ σ_(-1) / n_f |
| 冲击强度 | 冲击能量吸收能力 | E_absorb ≥ E_impact |
| 接触强度 | 接触应力边界层控制 | σ_H ≤ [σ_H] |
| 稳定强度 | 压杆不失稳 | P ≤ P_cr / n_st |
| 断裂强度 | 断裂韧性校核 | K_I ≤ K_IC / n_k |
| 蠕变强度 | 高温长期载荷 | ε ≤ [ε]，σ ≤ σ_creep |
| 热强度 | 热应力不超标 | σ_thermal ≤ [σ] |

### 2.3 常用安全系数
<!-- 来源: 第2篇机械制图 第155页 -->

| 工况 | 材料 | 安全系数 | 说明 |
|------|------|----------|------|
| 静载荷（一般） | 塑性材料 | 1.2~2.0 | 以屈服极限为基准 |
| 静载荷（重要） | 塑性材料 | 2.0~3.0 | 重要零部件 |
| 静载荷 | 脆性材料 | 2.5~4.0 | 以强度极限为基准 |
| 动载荷 | 塑性材料 | 3.0~5.0 | 考虑疲劳 |
| 冲击载荷 | 各种材料 | 2.5~5.0 | 含动态放大 |
| 交变载荷 | 各种材料 | 2.0~4.0 | 考虑应力集中 |
| 高温工况 | 耐热合金 | 2.0~3.0 | 蠕变考虑 |
| 低温工况 | 低温钢 | 2.5~4.0 | 脆断考虑 |

### 2.4 刚度设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 结构类型 | 刚度要求 | 变形控制值 | 设计方法 |
|----------|----------|------------|----------|
| 机床床身 | 高刚度 | 变形≤0.001~0.005mm | 闭式截面，加强筋 |
| 精密导轨 | 极高刚度 | 变形≤0.001mm/m | 预紧，高刚截面 |
| 一般支架 | 中等刚度 | 挠度≤L/1000~L/500 | 合理截面形状 |
| 轴类零件 | 挠度控制 | 挠度≤0.0001~0.0005L | 加大直径或改用空心轴 |
| 箱体 | 整体刚度 | 变形≤0.01~0.02mm | 加强筋，壁厚均匀 |
| 薄壁件 | 局部刚度 | 不产生可见变形 | 增加翻边、压筋 |
| 连接件 | 接触刚度 | 接触变形≤0.005mm | 增加接触面积、预压 |

### 2.5 结构设计对强度和刚度的影响
<!-- 来源: 第5篇机械产品结构设计 第24页 -->

| 结构设计参数 | 对强度影响 | 对刚度影响 | 设计建议 |
|-------------|-----------|-----------|----------|
| 截面面积A | 正比于拉压强度 | 正比于拉压刚度 | 满足功能的最小截面 |
| 截面惯性矩I | - | 正比于弯曲刚度 | 采用工字/箱形截面 |
| 截面系数W | 正比于弯曲强度 | - | 材料远离中性轴 |
| 壁厚t | 正比于局部强度 | 正比于局部刚度 | 壁厚均匀过渡 |
| 加强筋高度h | h²比例 | h³比例 | 加高筋效果显著 |
| 加强筋间距 | 负相关 | 负相关 | 间距≤3~5倍壁厚 |
| 倒角半径R | 减少应力集中 | - | R≥(0.1~0.2)t |
| 孔洞 | 应力集中系数K_t | 降低局部刚度 | 孔边加厚或翻边 |

### 2.6 加强筋设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 设计准则 | 推荐值 | 说明 |
|----------|--------|------|
| 筋高度 | ≤5倍壁厚 | 过高易出现缩痕 |
| 筋根部厚度 | 0.6~0.8倍壁厚 | 注塑件，防止缩水 |
| 筋端部厚度 | 0.4~0.6倍壁厚 | 注塑件脱模方便 |
| 筋间距 | 3~5倍壁厚 | 效果最佳 |
| 筋的倾角（脱模角） | 0.5°~2° | 注塑/压铸 |
| 筋的布置方向 | 与主载荷方向一致 | 载荷传递路径 |
| 十字交叉筋 | 中心减薄处理 | 避免缩孔 |
| 筋与壁连接 | R≥0.5mm圆角 | 减少应力集中 |
| 筋高度比 | h/b = 2~5 | 矩形截面 |

### 2.7 应力集中减缓设计
<!-- 来源: 第1篇 第3页 [前言] -->

| 措施 | 设计方法 | 减缩效果 | 应用场景 |
|------|----------|----------|----------|
| 倒圆角 | 增大过渡R | K_t降低30~50% | 轴肩、壁面过渡 |
| 椭圆缺口 | 长轴与应力方向一致 | 效果显著 | 减轻孔设计 |
| 卸荷槽 | 在应力集中区附近开槽 | K_t降低20~40% | 轴肩、退刀槽 |
| 渐过渡 | 锥度1:3~1:5 | 均匀应力分布 | 变截面过渡 |
| 局部加强 | 加厚应力区 | 降低局部应力 | 孔周边加厚 |
| 预压应力 | 表面喷丸/滚压 | 疲劳寿命提高2~5倍 | 弹簧、轴类 |
| 缓变截面 | 避免截面突变 | 应力分布在弹性体 | 连杆、支架 |
| 避免锐角 | 全部过渡R≥1mm | 防止应力集中 | 所有结构件 |

### 2.8 抗疲劳设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 准则 | 要点 | 设计措施 |
|------|------|----------|
| 降低应力水平 | 工作应力<疲劳极限 | 增大截面，改变传力路径 |
| 减少应力集中 | 过渡平缓 | 大圆角，卸荷结构 |
| 表面强化 | 表面压应力 | 喷丸、滚压、氮化 |
| 避免腐蚀疲劳 | 防护涂层 | 镀层/涂层/不锈钢 |
| 避免微动磨损 | 接触面处理 | 润滑/涂层/提高硬度 |
| 合理接头 | 疲劳载荷下的连接 | 避免单剪，采用双剪 |
| 预紧控制 | 螺栓预紧力合适 | 预应力不低于工作载荷的1.5倍 |
| 制造质量 | 避免刻痕、烧伤 | 磨削烧伤控制、加工痕迹方向 |

### 2.9 焊接结构强度设计准则
<!-- 来源: 第1篇一般设计资料 第720页 -->

| 准则 | 要求 | 设计方法 |
|------|------|----------|
| 焊缝等强 | 焊缝强度≥母材 | 焊缝尺寸≥母材厚度 |
| 避免交叉焊缝 | 防止热影响区重叠 | 焊缝间距≥50mm |
| 减少焊缝数量 | 焊缝数最少 | 整体构件代替拼焊 |
| 对称布置 | 焊缝对称于中性轴 | 减少变形 |
| 避免三板交叉 | 避免三块板交汇 | 阶梯形过渡 |
| 坡口设计 | 全熔透 | 对接焊开坡口 |
| 角焊缝 | 等脚长 | 焊脚高度K=(0.6~1.0)t |
| 焊接顺序 | 对称焊 | 减少焊接变形 |
| 焊后退火 | 消除残余应力 | 大型结构振动时效 |

## 3. 符合公差要求的结构设计准则
<!-- 来源: 第1篇一般设计资料 第740页 -->

### 3.1 公差设计基本原则
<!-- 来源: 第1篇 第3页 [前言] -->

| 原则 | 内容 | 目的 |
|------|------|------|
| 基准统一 | 设计基准/工艺基准/检测基准一致 | 避免误差累积 |
| 链长最短 | 尺寸链最短 | 减少公差累积 |
| 公差经济性 | 在满足功能前提下取大公差 | 降低制造成本 |
| 包容原则 | 孔包容轴 | 装配方便 |
| 失配设计 | 利于装配导向 | 导入倒角/导向锥 |
| 温度补偿 | 考虑工作温度与装配温度差异 | 热膨胀补偿 |

### 3.2 尺寸链设计准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 准则 | 要求 | 实例 |
|------|------|------|
| 封闭环选非重要尺寸 | 最重要尺寸不作为封闭环 | 装配间隙为封闭环 |
| 组成环数尽可能少 | 尺寸链最短原则 | 减少中间尺寸 |
| 组成环公差合理分配 | 等精度或等公差法 | 按加工难度分配 |
| 考虑修配环 | 预留可调/可修配环节 | 调整垫片/修配面 |
| 极值法 | 完全互换，100%合格 | 批量生产 |
| 概率法 | 统计公差，允许部分超差 | 大量生产 |
| 分组法 | 分组选配 | 精密配合 |

### 3.3 公差等级选用
<!-- 来源: 第2篇机械制图 第197页 -->

| 应用场合 | 推荐公差等级 | 说明 |
|----------|-------------|------|
| 精密配合（精密机床主轴） | IT4~IT5 | 极高精度 |
| 重要配合（滚动轴承配合） | IT5~IT6 | 高精度 |
| 一般配合（齿轮孔/轴） | IT6~IT7 | 中等精度 |
| 自由配合（滑动轴承） | IT7~IT8 | 较松配合 |
| 粗糙配合（不重要尺寸） | IT9~IT11 | 大公差 |
| 焊接件/铸件 | IT12~IT14 | 粗糙 |
| 锻件/型材 | IT14~IT16 | 自由公差 |
| 非配合尺寸（自由公差） | IT12~IT18 | 无配合要求 |

### 3.4 配合选择准则
<!-- 来源: 第2篇机械制图 第155页 -->

| 配合类别 | 配合特性 | 公差带（举例） | 典型应用 |
|----------|----------|---------------|----------|

### 6.3 防震缓冲包装设计
<!-- 来源: 第1篇 第3页 [前言] -->

| 产品脆值 G | 缓冲类型 | 缓冲材料 | 缓冲厚度/mm |
|------------|----------|----------|-------------|
| >100G | 低缓冲 | EPE珍珠棉 | 10~20 |
| 60~100G | 中缓冲 | 泡沫塑料EPS | 20~50 |
| 30~60G | 较高缓冲 | 泡沫塑料+衬垫 | 50~80 |
| 10~30G | 高缓冲 | 弹簧悬吊/聚氨酯 | 80~150 |
| <10G | 极高缓冲 | 多层复合+悬吊 | >150 |

> **脆值 G**：产品能承受的最大加速度的倍数。例如，G=60表示可承受6g加速度（约60m/s²）。

### 6.10 包装成本控制原则
<!-- 来源: 第1篇 第3页 [前言] -->

| 原则 | 方法 | 效果 |
|------|------|------|
| 标准化 | 统一包装规格尺寸 | 减少模具/库存成本 |
| 轻量化 | 减薄/小型化 | 降低运输成本 |
| 材料替代 | 瓦楞纸代木，EPE代EPS | 环保且成本低 |
| 模块化 | 组合包装 | 提高装箱率 |
| 可回收 | 可回收材料+重复使用 | 节省包装成本 |
| 自动化 | 适合自动包装线 | 减少人工 |
| 废弃处理 | 减少不可降解材料 | 处理成本低 |
