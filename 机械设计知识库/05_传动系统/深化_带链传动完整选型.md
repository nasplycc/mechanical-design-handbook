### 1.5 V带选型完整流程

#### 1.5.1 流程图解
<!-- 来源: 第3卷  第820页 -->

```
          ┌──────────────────────────────────────┐
          │  输入已知条件：P, n1, n2(or i), 工况  │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  计算设计功率：Pca = KA × P          │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  选择V带型号：根据Pca和n1查选型图     │
          │  (Y/Z/A/B/C/D/E或SPZ/SPA/SPB/SPC)    │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  确定小轮基准直径dd1 ≥ dmin           │
          │  并取标准直径系列值                    │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  计算大轮基准直径：dd2 = i·dd1        │
          │  取标准值并验算实际传动比误差≤±5%      │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  验算带速：v = π·dd1·n1/60000        │
          │  推荐 5 ≤ v ≤ 25 m/s                 │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  确定中心距a0和基准长度Ld             │
          │  a0在0.7(dd1+dd2)~2(dd1+dd2)之间     │
          │  Ld取标准系列值                       │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  验算小轮包角α1 ≥ 120°               │
          │  α1 = 180° - (dd2-dd1)/a × 57.3°     │
          │  α1 < 120°→增大中心距或加张紧轮      │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  计算V带根数：z = Pca/[(P0+ΔP0)KαKL] │
          │  圆整为整数，推荐z ≤ 10               │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  计算初拉力F0和轴压力FQ               │
          │  F0 = 500Pca/(zv)·(2.5/Kα -1) + qv²  │
          │  FQ = 2zF0·sin(α1/2)                  │
          └────────────────┬─────────────────────┘
                          ↓
          ┌──────────────────────────────────────┐
          │  带轮结构设计（S/H/P/R型）            │
          │  绘制零件图                            │
          └─────────────────────────────────────┘
```

### 1.6 V带传动设计算例
<!-- 来源: 第3卷  第820页 -->

**条件：** 电动机驱动鼓风机，$P=7.5\text{kW}$，$n_1=1450\text{r/min}$，$n_2=400\text{r/min}$，每天工作16h，载荷变动小。

**步骤1：设计功率**

$K_A=1.1$（载荷变动小+长工作时间修正+0.1）
$P_{ca}=1.1 \times 7.5 = 8.25\text{kW}$

**步骤2：选择型号**

$P_{ca}=8.25\text{kW}$，$n_1=1450\text{r/min}$ → 选 **B型V带**

**步骤3：确定带轮基准直径**

取 $d_{d1}=140\text{mm}$（推荐最小125mm，取稍大）

$i = n_1/n_2 = 1450/400 = 3.625$

$d_{d2} = i\cdot d_{d1} = 3.625 \times 140 = 507.5\text{mm}$

取标准 $d_{d2}=500\text{mm}$

**验算传动比：** $i_{实际} = 500/140 = 3.571$，误差 $(3.625-3.571)/3.625 = 1.49\% < 5\% \checkmark$

**步骤4：验算带速**

$v = \pi \times 140 \times 1450 / 60000 = 10.63\text{ m/s}$

$5 < 10.63 < 25 \text{ m/s} \quad \checkmark$

**步骤5：确定中心距和带长**

初选 $a_0 = 0.7(d_{d1}+d_{d2}) = 0.7 \times 640 = 448\text{mm}$

取 $a_0 = 500\text{mm}$

$L_{d0} = 2 \times 500 + \pi \times 640/2 + (500-140)^2/(4 \times 500)$

$L_{d0} = 1000 + 1005.3 + 64.8 = 2070.1\text{mm}$

取标准带长 $L_d = 2000\text{mm}$（B型带）

**实际中心距：**

$a \approx 500 + (2000 - 2070.1)/2 = 464.9\text{mm}$

取 $a = 465\text{mm}$

**调整范围：**
$a_{\min} = 465 - 0.015 \times 2000 = 435\text{mm}$
$a_{\max} = 465 + 0.03 \times 2000 = 525\text{mm}$

**步骤6：验算小轮包角**

$\alpha_1 = 180^\circ - (500-140)/465 \times 57.3^\circ = 180^\circ - 44.4^\circ = 135.6^\circ$

$\alpha_1 = 135.6^\circ > 120^\circ \quad \checkmark$

查表 $K_\alpha = 0.88$（$\alpha_1=135^\circ$）

**步骤7：计算V带根数**

查表 $P_0 = 2.32\text{kW}$（B型，$d_{d1}=140$，$n_1=1450$）

$\Delta P_0 = 0.18\text{kW}$（$i=3.625 > 1.19$）

$K_L$ 查表 $L_d=2000\text{mm}$，B型 → $K_L=1.01$

$z = 8.25 / [(2.32 + 0.18) \times 0.88 \times 1.01] = 8.25 / [2.50 \times 0.88 \times 1.01] = 8.25 / 2.224 = 3.71$

取 $z = 4$ 根

**步骤8：初拉力与轴压力**

$F_0 = 500 \times 8.25 / (4 \times 10.63) \times (2.5/0.88 - 1) + 0.17 \times 10.63^2$

$F_0 = 500 \times 8.25 / 42.52 \times (2.84 - 1) + 0.17 \times 113.0$

$F_0 = 97.0 \times 1.84 + 19.21 = 178.5 + 19.2 = 197.7\text{ N}$

$F_Q = 2 \times 4 \times 197.7 \times \sin(135.6^\circ/2) = 1581.6 \times \sin67.8^\circ = 1581.6 \times 0.926 = 1465\text{ N}$

**步骤9：带轮结构设计**

$d_{d1}=140\text{mm}$ < 300mm → 腹板轮
$d_{d2}=500\text{mm}$ → 轮辐轮（$d > 500$）

轮毂外径 $d_h = 1.7d_s$（$d_s$为轴径），轮毂长 $L_h \ge 1.5d_s$

## 6. 链传动设计计算完整公式和算例
<!-- 来源: 第3卷  第6页 -->

### 6.1 完整设计公式
<!-- 来源: 第3卷  第820页 -->

#### 6.1.1 基本运动学公式
<!-- 来源: 第3卷  第820页 -->

**平均链速：**

$$v = \frac{z_1 n_1 p}{60 \times 1000} \quad \text{(m/s)}$$

**传动比：**

$$i = \frac{n_1}{n_2} = \frac{z_2}{z_1}$$

**有效圆周力：**

$$F_e = \frac{1000P}{v} \quad \text{(N)}$$

#### 6.1.2 额定功率计算
<!-- 来源: 第3卷  第820页 -->

**单排链额定功率 $P_0$（kW）——由实验功率曲线查取。**

**修正功率：**

$$P_c = \frac{K_A P}{K_z K_m}$$

| 符号 | 含义 | 说明 |
|:----|:-----|:-----|
| $K_A$ | 工况系数 | 见表 |
| $K_z$ | 小链轮齿数系数 | 见表 |
| $K_m$ | 多排链系数 | 见表 |

#### 6.1.3 工况系数 $K_A$
<!-- 来源: 第3卷  第814页 -->

| 载荷类型 | 原动机类型 | | |
|:---------|:----------|:---:|:---:|
| | 电动机（平稳） | 内燃机（多缸） | 内燃机（单缸） |
| **载荷平稳** | 1.0 | 1.1 | 1.2 |
| **中等冲击** | 1.2 | 1.3 | 1.4 |
| **较大冲击** | 1.4 | 1.5 | 1.6 |

#### 6.1.4 小链轮齿数系数 $K_z$

<!-- 来源: 第3卷  第820页 -->

| 排数 | 1 | 2 | 3 | 4 | 5 | 6 |
|:---:|:-:|:-:|:-:|:-:|:-:|:-:|
| $K_m$ | 1.0 | 1.7 | 2.5 | 3.3 | 4.0 | 4.5 |

#### 6.1.6 最小轮齿数
<!-- 来源: 第3卷  第820页 -->

| 链速 $v$ (m/s) | 推荐 $z_1$ | 最小 $z_{1\min}$ |
|:--------------|:----------|:---------------|
| $v < 0.6$（极低速） | $\ge 13$ | 9 |
| $v = 0.6\sim 3$（低速） | $\ge 15$ | 13 |
| $v = 3\sim 8$（中速） | $\ge 19$ | 17 |
| $v > 8$（高速） | $\ge 23$ | 21 |

**大链轮最大齿数：** $z_{2\max} \le 120$（推荐 $z_2 \le 100$）

**齿数选择原则：**
1. $z_1$ 优先取奇数（与偶数节链条配合磨损均匀）
2. $z_1$ 与 $z_2$ 互质（使每个齿磨损均匀）
3. 避免使用过渡链节（$L_p$ 取偶数）

#### 6.1.7 中心距与链节数
<!-- 来源: 第3卷  第820页 -->

**推荐中心距：**

$$a_0 = (30\sim 50)p \quad (\text{最大 } a_{\max} \le 80p)$$

$p$ 为节距。

**链节数 $L_p$：**

$$L_p = \frac{2a_0}{p} + \frac{z_1+z_2}{2} + \frac{p}{a_0}\left(\frac{z_2-z_1}{2\pi}\right)^2$$

$L_p$ 应圆整为整数，**最好取偶数**（避免使用过渡链节）。

**实际中心距 $a$：**

$$a = \frac{p}{4}\left[L_p - \frac{z_1+z_2}{2} + \sqrt{\left(L_p - \frac{z_1+z_2}{2}\right)^2 - 8\left(\frac{z_2-z_1}{2\pi}\right)^2}\right]$$

实际装配中心距应比计算值小 $\Delta a = (0.002\sim 0.004)a$，以给链条留出一定的垂度。

**中心距调整量：**

| 中心距 $a$ (mm) | 调整量 $\Delta a$ (mm) |
|:---------------:|:---------------------:|
| $a \le 500$ | 2~3 |
| $500 < a \le 1000$ | 3~5 |
| $1000 < a \le 2000$ | 5~8 |
| $a > 2000$ | 8~12 |

#### 6.1.8 链速修正
<!-- 来源: 第3卷  第820页 -->

**铰链比压校核（验算单位面积压力）：**

$$p_a = \frac{K_A F_e}{A} \le [p]$$

其中 $A$ 为铰链承压面积（$A = d_2 b_1$），$[p]$ 为许用比压。

#### 6.1.9 轴压力
<!-- 来源: 第3卷  第895页 -->

$$F_Q = K_F \cdot F_e$$

| 链条方向 | $K_F$ |
|:--------|:-----:|
| 水平传动（倾角 $\le 40^\circ$） | 1.15 |
| 倾斜传动（$40^\circ < \theta \le 70^\circ$） | 1.20 |
| 垂直传动（$\theta > 70^\circ$） | 1.25 |

### 6.2 链传动设计完整流程
<!-- 来源: 第3卷  第820页 -->

```
输入：P, n1, n2(或i), 工作条件, 中心距范围
                ↓
确定传动比 i = n1/n2
                ↓
选择小链轮齿数 z1（按v和i查表）
计算 z2 = i·z1，取整并 ≤ zmax
                ↓
选择链节距 p
（根据Pc = KAP/(KzKm) 查额定功率曲线选链号）
                ↓
验算链速 v = z1·n1·p/60000
                ↓
初定中心距 a0，计算链节数 Lp
圆整 Lp 为偶数
                ↓
计算实际中心距 a
                ↓
验算链轮齿数包角（一般无需校核）
                ↓
计算有效圆周力 Fe = 1000P/v
计算轴压力 FQ = KF·Fe
                ↓
确定润滑方式（按v和链号查润滑推荐表）
                ↓
确定链轮结构，绘制图纸
```

### 6.3 链传动设计算例
<!-- 来源: 第3卷  第820页 -->

#### 算例1：通用链传动设计
<!-- 来源: 第3卷  第6页 -->

**条件：** 电动机 $P=7.5\text{kW}$，$n_1=720\text{r/min}$，$n_2=240\text{r/min}$，载荷平稳，中心距约 $a\approx 600\text{mm}$，水平传动。

**步骤1：确定传动比**

$i = 720/240 = 3$

**步骤2：选择链轮齿数**

按 $v$ 未定，初选 $z_1=19$（奇数）

$z_2 = 3 \times 19 = 57 < 120 \checkmark$

**步骤3：计算修正功率**

$K_A=1.0$（载荷平稳，电动机），$K_z=1.00$（$z_1=19$），$K_m=1.0$（单排）

$P_c = 1.0 \times 7.5/(1.00 \times 1.0) = 7.5\text{kW}$

**步骤4：选链号**

$n_1=720\text{r/min}$，$P_c=7.5\text{kW}$ → 查额定功率曲线选 **16A**（$p=25.4\text{mm}$）

**步骤5：验算链速**

$v = 19 \times 720 \times 25.4 / 60000 = 5.80\text{ m/s}$

$v=5.80\text{m/s} \in (0.6, 8)\text{m/s}$，属中速，$z_1=19 \ge 17 \checkmark$

**步骤6：链节数**

$a_0=600\text{mm}$

$L_p = 2\times600/25.4 + (19+57)/2 + 25.4/600 \times ((57-19)/(2\pi))^2$

$L_p = 47.24 + 38 + 25.4/600 \times (38/(2\pi))^2 = 47.24 + 38 + 0.0423 \times 36.56$

$L_p = 47.24 + 38 + 1.55 = 86.79$

取 $L_p = 88$（偶数，避免过渡链节）

**步骤7：实际中心距**

$a = \frac{25.4}{4}\left[88 - (19+57)/2 + \sqrt{(88 - 38)^2 - 8\left(\frac{57-19}{2\pi}\right)^2}\right]$

$a = 6.35 \times [88 - 38 + \sqrt{50^2 - 8 \times 36.56}] = 6.35 \times [50 + \sqrt{2500 - 292.5}]$

$a = 6.35 \times [50 + \sqrt{2207.5}] = 6.35 \times [50 + 46.98] = 6.35 \times 96.98 = 615.8\text{mm}$

装配中心距：$a_{装配} = 615.8 - 0.003 \times 615.8 = 613.9\text{mm}$

**步骤8：有效圆周力与轴压力**

$F_e = 1000 \times 7.5 / 5.80 = 1293\text{ N}$

$F_Q = 1.15 \times 1293 = 1487\text{ N}$（水平传动）

**步骤9：确定润滑方式**

$v=5.80\text{m/s}$，链号16A → **油浴或飞溅润滑（方式Ⅲ）**

**结果：**

**条件：** 电动机 $P=30\text{kW}$，$n_1=100\text{r/min}$，$n_2=25\text{r/min}$，载荷较大冲击，中心距 $a\approx 1200\text{mm}$。

#### 算例3：高速高功率链传动（多排链）
<!-- 来源: 第3卷  第6页 -->

**条件：** 电动机 $P=75\text{kW}$，$n_1=1200\text{r/min}$，$i=3$，载荷中等冲击，水平传动。

$v$ 预估高速，取 $z_1=23$

**步骤2：修正功率**

## 7. 润滑方式完整表
<!-- 来源: 第3卷  第820页 -->

### 7.1 V带与平带润滑
<!-- 来源: 第3卷  第820页 -->

V带和平带通常不需要润滑（润滑反而会使摩擦系数下降导致打滑）。

**注意事项：**
- 带轮轴承需润滑
- 带表面严禁油污
- 如带噪声大，可少量涂抹特殊带蜡

### 7.2 同步带润滑
<!-- 来源: 第3卷  第820页 -->

同步带通常**不需要润滑**。高负载时可使用少量专用润滑脂或干式润滑剂。

### 8.3 张紧轮设计
<!-- 来源: 第3卷  第820页 -->

#### 8.3.1 张紧轮尺寸
<!-- 来源: 第3卷  第820页 -->

| 传动类型 | 张紧轮直径 | 张紧轮宽度 |
|:---------|:----------|:----------|
| **V带传动** | $d_t \ge (0.8\sim 1.0)d_{\min}$ | 大于带顶宽 |
| **平带传动** | $d_t \ge 0.8d_{\min}$ | 大于带宽 |
| **同步带传动** | $d_t \ge d_{1\min}$（按最小齿数） | 大于带宽 |
| **链传动** | $d_t = (0.7\sim 1.0)d_1$（$d_1$为小链轮节圆） | 与链轮等宽 |

#### 8.3.2 张紧轮布置原则
<!-- 来源: 第3卷  第820页 -->

1. **V带/平带张紧轮：**
   - 布置在松边内侧（靠近大轮），使包角增大
   - 直径不小于小带轮直径的80%
   - 避免布置在紧边

2. **同步带张紧轮：**
   - 布置在松边内侧（带齿侧），防止爬齿
   - 光滑轮（无齿）压带背侧
   - 有齿轮压带齿侧

3. **链传动张紧轮：**
   - 布置在松边（链条松弛侧）
   - 可用于上下两侧
   - 张紧轮直径一般是小链轮直径的0.7~1.0倍
   - 与链条接触包角不小于60°
   - 大中心距时可用托板代替张紧轮

### 8.4 链传动垂度控制
<!-- 来源: 第3卷  第820页 -->

#### 8.4.1 垂度推荐值
<!-- 来源: 第3卷  第820页 -->

| 传动方向 | 推荐垂度 $f$ | 说明 |
|:---------|:------------|:-----|
| **水平传动**（$\theta \le 40^\circ$） | $f = (0.01\sim 0.02)a$ | 松边放松下垂 |
| **倾斜传动**（$40^\circ < \theta \le 70^\circ$） | $f = (0.015\sim 0.025)a$ | 取中间值 |
| **垂直传动**（$\theta > 70^\circ$） | $f = (0.02\sim 0.03)a$ | 需辅助张紧 |

#### 8.4.2 垂度控制方法
<!-- 来源: 第3卷  第820页 -->

| 方法 | 操作 | 特点 | 适用场合 |
|:-----|:-----|:-----|:---------|
| 调整中心距 | 移动轴承座或电机底座 | 简便易行 | 所有 |
| 去链节 | 减少2节或4节链条 | 不可微调 | 链条磨损后可去除 |
| 张紧轮 | 加装张紧链轮 | 自动补偿 | 大中心距 |
| 托板 | 托住松边 | 防止过下垂 | 垂直/大中心距 |
| 压板 | 限制链条跳动 | — | 高速传动 |

### 8.5 带传动初拉力检测
<!-- 来源: 第3卷  第17页 -->

#### 8.5.1 初拉力检测方法（V带）
<!-- 来源: 第3卷  第820页 -->

**悬重法（挠度法）：**

在带的中部施加垂直力 $W$，测量挠度 $y$。

$$W = \frac{F_0}{16} \sim \frac{F_0}{20}$$

**推荐施力与挠度：**

| V带型号 | 推荐施力 $W$ (N) | 推荐挠度 $y$ (mm/100mm跨度) |
|:-------:|:---------------:|:--------------------------:|
| Z | 5~10 | 1.6 |
| A | 10~15 | 1.6 |
| B | 15~25 | 1.6 |
| C | 25~40 | 1.6 |
| D | 40~60 | 1.6 |
| E | 60~80 | 1.6 |

#### 8.5.2 带传动初拉力 $F_0$ 与挠度关系
<!-- 来源: 第3卷  第820页 -->

**检测公式：**

$$y = \frac{W \cdot L}{4F_0} \quad \text{(mm)}$$

其中 $L$ 为两带轮中心距（mm），$W$ 为施加的垂直力（N），$F_0$ 为初拉力（N）。

**推荐检测条件：**

| 皮带跨度 $L$ (mm) | 每100mm跨度对应的挠度 (mm) |
|:-----------------:|:-------------------------:|
| $< 500$ | 1.5~2.0 |
| $500\sim 1000$ | 1.5~1.8 |
| $> 1000$ | 1.2~1.5 |

### 8.6 张紧装置设计示例
<!-- 来源: 第3卷  第12页 -->

**例：** 某输送机链传动，$a=2500\text{mm}$，链号16A，采用滑轨式张紧。

**张紧行程计算：**

链条磨损伸长量：按2%节距伸长计

$\Delta a = 0.02 \times 2500 = 50\text{mm}$

**张紧调节量：**

$S = \Delta a + 10\text{mm}$（预留量）$= 60\text{mm}$

**滑轨长度：** $L_{rail} \ge S + 30\text{mm} = 90\text{mm}$

**调节螺栓：** M16×1.5，每转调节1.5mm

**张紧力：** $F_Q \times 3\%$（约垂直传动链的垂度补偿）
