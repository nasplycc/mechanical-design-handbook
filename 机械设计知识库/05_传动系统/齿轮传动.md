## 1. 齿轮传动概述
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

### 1.1 齿轮传动分类
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

齿轮传动按轴线相对位置、齿线形状、工作条件等可分为多种类型：

| 分类方式 | 类型 | 说明 |
|:--------|:-----|:-----|
| **轴线位置** | 平行轴传动 | 直齿/斜齿/人字齿圆柱齿轮 |
| | 相交轴传动 | 直齿/斜齿/曲线齿锥齿轮（常用轴交角 Σ=90°） |
| | 交错轴传动 | 交错轴斜齿轮、蜗杆蜗轮、准双曲面齿轮 |
| **齿线形状** | 直齿 | 齿线与轴线平行，无轴向力 |
| | 斜齿 | 齿线与轴线成螺旋角 β，有轴向力 |
| | 曲线齿 | 弧齿/摆线齿锥齿轮 |
| **工作条件** | 闭式传动 | 润滑良好，用于重要传动 |
| | 开式传动 | 润滑不良，易磨损 |
| **齿面硬度** | 软齿面 | HBS ≤ 350 |
| | 硬齿面 | HBS > 350，或经渗碳/渗氮/表面淬火 |

### 1.2 各类齿轮传动特点与适用范围
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 类型 | 效率 η | 速度范围 | 功率范围 | 特点 | 适用场合 |
|:----|:-------|:---------|:---------|:-----|:---------|
| 直齿圆柱齿轮 | 0.96~0.99 | 低速~中速 | 小~大 | 无轴向力，制造简单 | 通用减速器、机床等 |
| 斜齿圆柱齿轮 | 0.96~0.99 | 中速~高速 | 中~大 | 传动平稳，承载能力高 | 高速传动、汽车变速箱 |
| 人字齿圆柱齿轮 | 0.96~0.99 | 中速~高速 | 大 | 无轴向力，承载最高 | 重载减速器、船舶 |
| 直齿锥齿轮 | 0.95~0.98 | 低速~中速 | 小~中 | 制造较简单 | 相交轴低速传动 |
| 弧齿锥齿轮 | 0.95~0.98 | 中速~高速 | 中~大 | 传动平稳，承载高 | 汽车驱动桥、直升机 |
| 交错轴斜齿轮 | 0.50~0.90 | 低速 | 小 | 滑动大，效率低 | 仅用于轻载 |

### 1.3 齿轮传动基本要求
<!-- 来源: 第3卷 第11篇润滑与密封 第3页 -->

1. **传动比准确** — 瞬时传动比恒定
2. **承载能力高** — 足够的齿面接触强度和齿根弯曲强度
3. **效率高** — 摩擦损失小
4. **可靠性高** — 长期工作不失效
5. **噪声小** — 良好的啮合性能

### 1.4 齿轮传动主要参数术语
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 术语 | 符号 | 定义 |
|:----|:----|:-----|
| 模数 | $m$ | 齿轮尺寸的基本参数，$m = p/\pi$ |
| 压力角 | $\alpha$ | 渐开线在分度圆上的压力角，标准值 20° |
| 齿数 | $z$ | 齿轮上轮齿的数量 |
| 分度圆直径 | $d$ | $d = mz$ |
| 齿顶圆直径 | $d_a$ | $d_a = d + 2h_a$ |
| 齿根圆直径 | $d_f$ | $d_f = d - 2h_f$ |
| 中心距 | $a$ | $a = (d_1 + d_2)/2 = m(z_1 + z_2)/2$ |
| 齿宽 | $b$ | 齿轮轮齿的轴向宽度 |
| 螺旋角 | $\beta$ | 斜齿轮齿线与轴线的夹角 |
| 传动比 | $i$ | $i = z_2/z_1 = n_1/n_2$ |

## 2. 渐开线圆柱齿轮设计
<!-- 来源: 第3卷 第11篇润滑与密封 第3页 -->

> 依据标准：**GB/T 10095-2008**《渐开线圆柱齿轮精度》

### 2.1 基本参数
<!-- 来源: 第3卷 第14篇带、链传动 第97页 -->

#### 2.1.1 模数（GB/T 1357-2008）
<!-- 来源: 第3卷 第14篇带、链传动 第98页 -->

标准模数 $m$ 系列（mm）：

| 系列 | 模数值 |
|:----|:-------|
| **第一系列** (优先选用) | 1, 1.25, 1.5, 2, 2.5, 3, 4, 5, 6, 8, 10, 12, 16, 20, 25, 32, 40, 50 |
| **第二系列** | 1.125, 1.375, 1.75, 2.25, 2.75, 3.5, 4.5, 5.5, (6.5), 7, 9, 11, 14, 18, 22, 28, 36, 45 |

#### 2.1.2 压力角
<!-- 来源: 第3卷 第15篇齿轮传动 第602页 -->

| 压力角 $\alpha$ | 适用范围 |
|:---------------|:---------|
| **20°** | 标准齿轮，最常用 |
| 14.5° | 英制齿轮（主要用于维修） |
| 25° | 重载齿轮，齿根强度更高 |
| 22.5° | 介于20°与25°之间 |
| 28° | 特重载齿轮 |

#### 2.1.3 齿数与变位系数
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

**最小齿数 $z_{\min}$（避免根切）：**

| 压力角 $\alpha$ | 齿顶高系数 $h_a^*=1$ | $h_a^*=0.8$ |
|:---------------|:--------------------:|:-----------:|
| 20° | 17 | 14 |
| 25° | 12 | 10 |

**变位齿轮：**
- 正变位：提高齿根强度，增大中心距
- 负变位：降低齿根强度，减小中心距
- 变位系数限制：
  - 齿顶不变尖：$s_a \ge 0.25m$
  - 不根切：$x \ge x_{\min} = \frac{h_a^*(z_{\min} - z)}{z_{\min}}$
  - 重合度：端面重合度 $\varepsilon_\alpha \ge 1.2$

### 2.2 几何计算
<!-- 来源: 第3卷 第15篇齿轮传动 第45页 -->

#### 2.2.1 外啮合直齿圆柱齿轮几何尺寸
<!-- 来源: 第3卷 第15篇齿轮传动 第653页 -->

| 参数 | 符号 | 计算公式 |
|:----|:----|:---------|
| 模数 | $m$ | 按强度或结构确定 |
| 压力角 | $\alpha$ | 标准20° |
| 齿数 | $z$ | $z_1 \ge z_{\min}$ |
| 分度圆直径 | $d$ | $d = mz$ |
| 基圆直径 | $d_b$ | $d_b = d\cos\alpha$ |
| 齿距 | $p$ | $p = \pi m$ |
| 齿顶高 | $h_a$ | $h_a = h_a^* m$ ($h_a^* = 1$) |
| 齿根高 | $h_f$ | $h_f = (h_a^* + c^*) m$ ($c^* = 0.25$) |
| 全齿高 | $h$ | $h = h_a + h_f = (2h_a^* + c^*)m$ |
| 齿顶圆直径 | $d_a$ | $d_a = d + 2h_a = m(z + 2)$ |
| 齿根圆直径 | $d_f$ | $d_f = d - 2h_f = m(z - 2.5)$ |
| 基圆齿距 | $p_b$ | $p_b = p\cos\alpha = \pi m\cos\alpha$ |
| 中心距 | $a$ | $a = \frac{d_1 + d_2}{2} = \frac{m(z_1 + z_2)}{2}$ |
| 齿宽 | $b$ | $b = \psi_d \cdot d$（$\psi_d$为齿宽系数） |

#### 2.2.2 变位齿轮几何计算
<!-- 来源: 第3卷 第15篇齿轮传动 第625页 -->

| 参数 | 符号 | 计算公式 |
|:----|:----|:---------|
| 中心距变动系数 | $y$ | $y = \frac{a' - a}{m}$ |
| 啮合角 | $\alpha'$ | $\text{inv}\,\alpha' = \frac{2(x_1 + x_2)}{z_1 + z_2}\tan\alpha + \text{inv}\,\alpha$ |
| 中心距 | $a'$ | $a' = \frac{m(z_1 + z_2)}{2} \cdot \frac{\cos\alpha}{\cos\alpha'}$ |
| 齿顶高变动系数 | $\Delta y$ | $\Delta y = x_1 + x_2 - y$ |
| 齿顶高 | $h_a$ | $h_a = (h_a^* + x - \Delta y)m$ |
| 齿根高 | $h_f$ | $h_f = (h_a^* + c^* - x)m$ |
| 齿顶圆直径 | $d_a$ | $d_a = d + 2h_a$ |
| 分度圆齿厚 | $s$ | $s = \frac{\pi m}{2} + 2xm\tan\alpha$ |
| 公法线长度 | $W_k$ | $W_k = m\cos\alpha[\pi(k-0.5) + z\text{inv}\,\alpha] + 2xm\sin\alpha$ |
| 跨齿数 | $k$ | $k = \frac{\alpha}{180^\circ}z + 0.5$（四舍五入） |

#### 2.2.3 斜齿圆柱齿轮几何计算
<!-- 来源: 第3卷 第11篇润滑与密封 第3页 -->

| 参数 | 符号 | 计算公式 |
|:----|:----|:---------|
| 法面模数 | $m_n$ | 标准值 |
| 端面模数 | $m_t$ | $m_t = m_n/\cos\beta$ |
| 法面压力角 | $\alpha_n$ | 标准20° |
| 端面压力角 | $\alpha_t$ | $\alpha_t = \arctan(\tan\alpha_n/\cos\beta)$ |
| 分度圆直径 | $d$ | $d = m_t z = m_n z/\cos\beta$ |
| 齿顶圆直径 | $d_a$ | $d_a = d + 2h_{an} = d + 2m_n$ |
| 齿根圆直径 | $d_f$ | $d_f = d - 2h_{fn} = d - 2.5m_n$ |
| 中心距 | $a$ | $a = \frac{d_1 + d_2}{2} = \frac{m_n(z_1 + z_2)}{2\cos\beta}$ |
| 当量齿数 | $z_v$ | $z_v = z/\cos^3\beta$（用于强度计算） |
| 端面重合度 | $\varepsilon_\alpha$ | $\varepsilon_\alpha = \frac{1}{2\pi}[z_1(\tan\alpha_{at1}-\tan\alpha_t) + z_2(\tan\alpha_{at2}-\tan\alpha_t)]$ |
| 轴向重合度 | $\varepsilon_\beta$ | $\varepsilon_\beta = \frac{b\sin\beta}{\pi m_n}$ |
| 总重合度 | $\varepsilon_\gamma$ | $\varepsilon_\gamma = \varepsilon_\alpha + \varepsilon_\beta$ |

### 2.3 强度计算
<!-- 来源: 第3卷 第13篇螺旋传动、摩擦轮传动 第65页 -->

> 齿轮强度计算依据 **GB/T 3480-1997**《渐开线圆柱齿轮承载能力计算方法》，等效采用 **ISO 6336**

#### 2.3.1 齿面接触强度计算
<!-- 来源: 第3卷 第15篇齿轮传动 第384页 -->

**基本公式（赫兹接触理论）：**

$$\sigma_H = Z_H Z_E Z_\varepsilon Z_\beta \sqrt{\frac{F_t}{d_1 b} \cdot \frac{u+1}{u} \cdot K_A K_V K_{H\beta} K_{H\alpha}} \le [\sigma_H]$$

**各系数含义：**

| 符号 | 名称 | 说明 |
|:----|:-----|:-----|
| $Z_H$ | 节点区域系数 | 标准齿轮 $Z_H = 2.5$ |
| $Z_E$ | 弹性系数 | 钢对钢 $Z_E = 189.8 \sqrt{\text{MPa}}$ |
| $Z_\varepsilon$ | 重合度系数 | $Z_\varepsilon = \sqrt{(4 - \varepsilon_\alpha)/3}$ |
| $Z_\beta$ | 螺旋角系数 | $Z_\beta = \sqrt{\cos\beta}$ |
| $F_t$ | 分度圆切向力 | $F_t = 2T_1/d_1$ (N) |
| $d_1$ | 小齿轮分度圆直径 | mm |
| $b$ | 齿宽 | mm |
| $u$ | 齿数比 | $u = z_2/z_1 \ge 1$ |
| $K_A$ | 使用系数 | 考虑原动机和工作机特性 |
| $K_V$ | 动载系数 | 考虑齿轮制造精度和速度 |
| $K_{H\beta}$ | 齿向载荷分布系数 | 考虑齿轮轴变形 |
| $K_{H\alpha}$ | 齿间载荷分配系数 | 考虑多齿啮合 |

**许用接触应力：**

$$[\sigma_H] = \frac{\sigma_{H\lim} Z_N Z_L Z_V Z_R Z_W Z_X}{S_H}$$

**各系数：**

| 符号 | 名称 | 说明 |
|:----|:-----|:-----|
| $\sigma_{H\lim}$ | 试验齿轮接触疲劳极限 | 由材料决定 |
| $Z_N$ | 寿命系数 | 考虑应力循环次数 |
| $Z_L$ | 润滑剂系数 | 考虑油粘度 |
| $Z_V$ | 速度系数 | 考虑圆周速度 |
| $Z_R$ | 粗糙度系数 | 考虑齿面粗糙度 |
| $Z_W$ | 工作面硬化系数 | 考虑软硬齿面搭配 |
| $Z_X$ | 尺寸系数 | 考虑模数影响 |
| $S_H$ | 安全系数 | 一般 $S_H \ge 1.0 \sim 1.3$ |

**齿面接触强度校核的简化形式：**

$$d_1 \ge \sqrt[3]{\frac{2KT_1}{\psi_d} \cdot \left(\frac{Z_H Z_E}{[\sigma_H]}\right)^2 \cdot \frac{u+1}{u}}$$

或

$$a \ge (u+1)\sqrt[3]{\left(\frac{Z_H Z_E}{[\sigma_H]}\right)^2 \cdot \frac{KT_1}{2\psi_a u}}$$

#### 2.3.2 齿根弯曲强度计算
<!-- 来源: 第3卷 第15篇齿轮传动 第80页 -->

**基本公式：**

$$\sigma_F = \frac{F_t}{b m_n} Y_{Fa} Y_{Sa} Y_\varepsilon Y_\beta K_A K_V K_{F\beta} K_{F\alpha} \le [\sigma_F]$$

**各系数含义：**

| 符号 | 名称 | 说明 |
|:----|:-----|:-----|
| $Y_{Fa}$ | 齿形系数 | 与齿数、变位系数有关 |
| $Y_{Sa}$ | 应力修正系数 | 考虑齿根圆角应力集中 |
| $Y_\varepsilon$ | 重合度系数 | $Y_\varepsilon = 0.25 + 0.75/\varepsilon_\alpha$ |
| $Y_\beta$ | 螺旋角系数 | $Y_\beta = 1 - \varepsilon_\beta \frac{\beta}{120^\circ}$ |
| $K_{F\beta}$ | 齿向载荷分布系数 | |
| $K_{F\alpha}$ | 齿间载荷分配系数 | |

**许用弯曲应力：**

$$[\sigma_F] = \frac{\sigma_{F\lim} Y_{ST} Y_N Y_{relT} Y_{RrelT} Y_X}{S_F}$$

**各系数：**

| 符号 | 名称 | 说明 |
|:----|:-----|:-----|
| $\sigma_{F\lim}$ | 试验齿轮弯曲疲劳极限 | 由材料决定 |
| $Y_{ST}$ | 应力修正系数 | $Y_{ST}=2$ |
| $Y_N$ | 弯曲寿命系数 | 考虑循环次数 |
| $Y_{relT}$ | 相对齿根圆角敏感系数 | |
| $Y_{RrelT}$ | 相对表面状况系数 | |
| $Y_X$ | 弯曲尺寸系数 | 考虑模数影响 |
| $S_F$ | 弯曲安全系数 | 一般 $S_F \ge 1.25 \sim 1.6$ |

**齿根弯曲强度校核的简化形式（模数初选）：**

$$m_n \ge \sqrt[3]{\frac{2KT_1}{\psi_d z_1^2} \cdot \frac{Y_{Fa}Y_{Sa}}{[\sigma_F]}}$$

#### 2.3.3 强度计算公式汇总
<!-- 来源: 第3卷 第15篇齿轮传动 第228页 -->

| 项目 | 接触强度 | 弯曲强度 |
|:----|:---------|:---------|
| **应力公式** | $\sigma_H = Z_H Z_E Z_\varepsilon Z_\beta \sqrt{\frac{F_t}{d_1 b} \cdot \frac{u+1}{u} \cdot K}$ | $\sigma_F = \frac{F_t}{b m_n} Y_{Fa} Y_{Sa} Y_\varepsilon Y_\beta K$ |
| **约束条件** | $\sigma_H \le [\sigma_H]$ | $\sigma_F \le [\sigma_F]$ |
| **许用应力** | $[\sigma_H] = \frac{\sigma_{H\lim}}{S_H} Z_N \cdots$ | $[\sigma_F] = \frac{\sigma_{F\lim}}{S_F} Y_N \cdots$ |
| **主要变量** | $d_1$（或 $a$） | $m_n$ |
| **设计顺序** | 先按接触强度定 $d_1$ | 再按弯曲强度定 $m_n$ |
| **安全系数** | $S_H = 1.0\sim1.3$ | $S_F = 1.25\sim1.6$ |

### 2.4 载荷系数详解
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

#### 2.4.1 使用系数 $K_A$
<!-- 来源: 第3卷 第15篇齿轮传动 第4页 -->

| 原动机 | 工作机载荷特性 | |
|:-------|:--------------|:---|
| | **均匀平稳** | **中等冲击** | **较大冲击** |
| 电动机（均匀运转） | 1.00 | 1.25 | 1.75 |
| 多缸内燃机 | 1.25 | 1.50 | 2.00 |
| 单缸内燃机 | 1.50 | 1.75 | 2.25 |

**工作机载荷特性示例：**

| 载荷特性 | 示例 |
|:--------|:-----|
| 均匀平稳 | 发电机、均匀物料输送机、通风机 |
| 中等冲击 | 机床、起重机、搅拌机 |
| 较大冲击 | 破碎机、轧钢机、挖掘机 |

#### 2.4.2 动载系数 $K_V$
<!-- 来源: 第3卷 第15篇齿轮传动 第4页 -->

$K_V$ 取决于齿轮精度等级和节圆线速度 $v$：

$$K_V = 1 + \left(\frac{K_1}{K_A F_t/b + K_2}\right) \cdot \frac{z_1 v}{100} \cdot \sqrt{\frac{u^2}{1+u^2}}$$

式中 $K_1$、$K_2$ 由精度等级确定：

| 精度等级 | 5 | 6 | 7 | 8 | 9 |
|:--------|:--|:--|:--|:--|:--|
| $K_1$ | 5.6 | 8.7 | 15.3 | 24.2 | 38.4 |
| $K_2$ | 0.043 | 0.041 | 0.039 | 0.036 | 0.032 |

#### 2.4.3 齿向载荷分布系数
<!-- 来源: 第3卷 第15篇齿轮传动 第144页 -->

$K_{H\beta}$（接触强度）和 $K_{F\beta}$（弯曲强度）：

$$K_{H\beta} = 1 + (K_{H\beta}^0 - 1)K_{Hw}K_{He}$$
$$K_{F\beta} = (K_{H\beta})^{N_F}$$

其中 $N_F = (b/h)^2 / (1 + b/h + (b/h)^2)$

#### 2.4.4 齿宽系数
<!-- 来源: 第3卷 第15篇齿轮传动 第7页 -->

| 齿轮布置 | 齿宽系数 $\psi_d = b/d_1$ |
|:---------|:--------------------------|
| 对称布置 | 0.9~1.4 |
| 非对称布置（轴刚性大） | 0.7~1.15 |
| 非对称布置（轴刚性小） | 0.55~0.9 |
| 悬臂布置 | 0.4~0.6 |

齿宽系数 $\psi_a = b/a$ 的常用范围：

| 齿轮类型 | $\psi_a$ |
|:---------|:---------|
| 通用减速器 | 0.2~0.4 |
| 轧钢机 | 0.3~0.5 |
| 高速齿轮 | 0.25~0.4 |
| 汽车变速器 | 0.25~0.35 |

### 2.5 齿轮设计步骤
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

#### 2.5.1 闭式软齿面齿轮传动设计流程
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

```
输入：功率P, 转速n1/n2, 工作条件
       |
       v
选择材料、热处理 → 确定 [σH], [σF]
       |
       v
按齿面接触强度初定 d1 (或中心距 a)
       |
       v
选择 z1 → 计算 m = d1/z1 → 取标准模数
       |
       v
计算齿轮几何尺寸 (d, da, df, a, b)
       |
       v
校核齿面接触强度 σH ≤ [σH]
       |
       v
校核齿根弯曲强度 σF ≤ [σF]
       |
       v
不满足 → 调整参数 (增加 m/d1 或换材料)
       |
       v
满足 → 确定精度等级 → 结构设计 → 绘制图纸
```

#### 2.5.2 闭式硬齿面齿轮传动设计特点
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 项目 | 软齿面 | 硬齿面 |
|:----|:-------|:-------|
| **设计准则** | 接触强度为主，弯曲强度校核 | 弯曲强度为主，接触强度校核 |
| **齿面硬度** | HBS ≤ 350 | HRC ≥ 40 |
| **热处理** | 正火、调质 | 渗碳淬火、渗氮、表面淬火 |
| **模数选择** | 较小 | 较大 |
| **设计流程** | 先按 $d_1$ 设计 | 先按 $m_n$ 设计 |

#### 2.5.3 开式齿轮传动设计特点
<!-- 来源: 第3卷 第15篇齿轮传动 第80页 -->

| 项目 | 说明 |
|:----|:-----|
| **主要失效** | 齿面磨损、轮齿折断 |

#### 2.6.3 精度等级选择原则
<!-- 来源: 第3卷 第11篇润滑与密封 第3页 -->

1. **传动精度要求**：分度传动 ≥ 5级；动力传动 6~9级
2. **圆周速度**：$v$ 越高，精度要求越高
3. **噪声要求**：要求低噪声时选高精度
4. **成本控制**：精度每提高一级，制造成本约增加40%~60%

## 3. 锥齿轮设计
<!-- 来源: 第3卷 第15篇齿轮传动 第334页 -->

### 3.1 锥齿轮传动概述
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

锥齿轮用于两相交轴之间的传动，常用轴交角 $\Sigma = 90^\circ$。

| 类型 | 齿线 | 特点 | 应用 |
|:----|:-----|:-----|:-----|
| 直齿锥齿轮 | 直线（指向锥顶） | 制造简单，轴向力小 | 低速、中小载荷 |
| 斜齿锥齿轮 | 斜直线 | 传动较平稳 | 中速传动 |
| 弧齿锥齿轮 | 圆弧线 | 传动平稳，承载高 | 高速重载（汽车后桥） |
| 零度弧齿锥齿轮 | 圆弧（螺旋角=0°） | 轴向力小 | 类似直齿但更平稳 |
| 摆线齿锥齿轮 | 延伸外摆线 | 切削效率高 | 大批量生产 |

### 3.2 直齿锥齿轮几何计算
<!-- 来源: 第3卷 第11篇润滑与密封 第3页 -->

#### 3.2.1 基本参数
<!-- 来源: 第3卷 第14篇带、链传动 第97页 -->

| 参数 | 符号 | 公式/说明 |
|:----|:----|:----------|
| 大端模数 | $m$ | 取标准模数（大端为基准） |
| 齿数比 | $u$ | $u = z_2/z_1$ |
| 分锥角 | $\delta$ | $\tan\delta_1 = z_1/z_2$, $\delta_2 = 90^\circ - \delta_1$（当 $\Sigma=90^\circ$） |
| 分度圆直径（大端） | $d$ | $d = mz$ |
| 外锥距 | $R$ | $R = \frac{d}{2\sin\delta} = \frac{m}{2}\sqrt{z_1^2 + z_2^2}$ |
| 齿宽 | $b$ | $b \le R/3$ 且 $b \le 10m$ |
| 齿宽中点模数 | $m_m$ | $m_m = m(1 - 0.5\psi_R)$，其中 $\psi_R = b/R$ |
| 齿宽中点直径 | $d_m$ | $d_m = d(1 - 0.5\psi_R)$ |
| 当量齿数 | $z_v$ | $z_v = z/\cos\delta$ |
| 当量齿轮分度圆直径 | $d_v$ | $d_v = d_m/\cos\delta$ |
| 当量齿数比 | $u_v$ | $u_v = u^2$（$\Sigma=90^\circ$） |

#### 3.2.2 几何尺寸计算（$\Sigma=90^\circ$）
<!-- 来源: 第3卷 第12篇弹簧 第72页 -->

| 参数 | 符号 | 计算公式 |
|:----|:----|:---------|
| 齿顶高 | $h_a$ | $h_a = m$（标准） |
| 齿根高 | $h_f$ | $h_f = 1.2m$（标准） |
| 全齿高 | $h$ | $h = 2.2m$ |
| 齿顶圆直径 | $d_a$ | $d_a = d + 2m\cos\delta$ |
| 齿根圆直径 | $d_f$ | $d_f = d - 2.4m\cos\delta$ |
| 齿顶角 | $\theta_a$ | $\tan\theta_a = h_a/R$ |
| 齿根角 | $\theta_f$ | $\tan\theta_f = h_f/R$ |
| 顶锥角 | $\delta_a$ | $\delta_a = \delta + \theta_a$ |
| 根锥角 | $\delta_f$ | $\delta_f = \delta - \theta_f$ |

### 3.3 锥齿轮强度计算
<!-- 来源: 第3卷 第15篇齿轮传动 第300页 -->

#### 3.3.1 齿面接触强度
<!-- 来源: 第3卷 第15篇齿轮传动 第19页 -->

**按齿宽中点当量齿轮计算：**

$$\sigma_H = Z_H Z_E Z_\varepsilon \sqrt{\frac{F_{tm}}{d_{m1} b} \cdot \frac{\sqrt{u_v^2 + 1}}{u_v} \cdot K_A K_V K_{H\beta}} \le [\sigma_H]$$

其中 $F_{tm} = 2T_1/d_{m1}$ 为齿宽中点切向力。

**简化设计公式：**

$$d_{m1} \ge \sqrt[3]{\frac{2KT_1}{\psi_R(1 - 0.5\psi_R)^2 u} \cdot \left(\frac{Z_H Z_E}{[\sigma_H]}\right)^2 \cdot \frac{\sqrt{u^2 + 1}}{u}}$$

#### 3.3.2 齿根弯曲强度
<!-- 来源: 第3卷 第15篇齿轮传动 第25页 -->

$$\sigma_F = \frac{F_{tm}}{b m_m} Y_{Fa} Y_{Sa} Y_\varepsilon K_A K_V K_{F\beta} \le [\sigma_F]$$

**简化设计公式：**

$$m \ge \sqrt[3]{\frac{4KT_1}{\psi_R(1 - 0.5\psi_R)^2 z_1^2 \sqrt{u^2 + 1}} \cdot \frac{Y_{Fa}Y_{Sa}}{[\sigma_F]}}$$

### 3.4 弧齿锥齿轮主要参数
<!-- 来源: 第3卷 第13篇螺旋传动、摩擦轮传动 第9页 -->

| 参数 | 常用范围 |
|:----|:---------|
| 螺旋角 $\beta_m$ | 35°（通用），35°~40°（航空） |
| 压力角 $\alpha$ | 20°（通用），22.5°（重载），16°（轻载低噪声） |
| 齿宽系数 $\psi_R$ | 0.25~0.35 |
| 小齿轮齿数 $z_1$ | 16~20（一般），≥12（硬齿面），≥24（软齿面） |

### 3.5 锥齿轮精度等级
<!-- 来源: 第3卷 第15篇齿轮传动 第352页 -->

| 精度等级 | 用途 | 加工方法 |
|:--------|:-----|:---------|
| 4~5级 | 精密机床分度 | 磨齿 |
| 6级 | 航空发动机、精密传动 | 磨齿、研齿 |

### 6.1 精度等级体系
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

> 依据 **GB/T 10095.1-2008**《渐开线圆柱齿轮精度 第1部分：轮齿同侧齿面偏差》

齿轮精度共 **13级**，从 **0级**（最高）到 **12级**（最低）。常用范围为 **6~9级**。

### 6.2 精度偏差项目

#### 6.2.2 齿廓总偏差 $F_\alpha$
<!-- 来源: 第3卷 第15篇齿轮传动 第95页 -->

$F_\alpha$ — 在齿廓工作部分内，实际齿廓与设计齿廓之间的法向偏差。

### 6.3 齿坯精度
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 项目 | 精度要求 |
|:----|:---------|
| 基准孔/轴直径公差 | 精密级（IT5~IT7） |
| 基准端面跳动 | 严格控制，影响载荷分布 |
| 顶圆直径公差 | 作为加工找正基准时需严格控制 |

## 7. 齿轮失效形式
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

### 7.1 失效形式分类
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 序号 | 失效形式 | 特征 | 发生部位 | 主要原因 |
|:---|:---------|:-----|:---------|:---------|
| 1 | **齿面点蚀** | 麻点状凹坑 | 齿面节线附近 | 接触应力循环作用，表面疲劳 |
| 2 | **齿面磨损** | 齿厚减薄 | 整个齿面 | 磨粒进入啮合区或润滑不良 |
| 3 | **齿面胶合** | 撕伤沟痕 | 齿顶/齿根 | 重载高速下油膜破裂 |
| 4 | **齿面塑性变形** | 齿面凹坑或凸起 | 齿面 | 软齿面过载 |
| 5 | **轮齿折断** | 齿根裂纹/断裂 | 齿根 | 弯曲疲劳或过载冲击 |

### 7.2 齿面点蚀
<!-- 来源: 第3卷 第15篇齿轮传动 第251页 -->

**分类：**

| 类型 | 特点 | 发生阶段 |
|:----|:-----|:---------|
| 初始点蚀 | 小麻点（<1mm），可自愈 | 跑合初期 |
| 扩展性点蚀 | 麻点扩大、加深、连片 | 稳定运行期 |
| 剥落（深层点蚀） | 大片金属剥落（>1mm深） | 长期过载 |

**防止措施：**
- 提高齿面硬度
- 改善润滑条件
- 降低表面粗糙度
- 适当增加齿宽或变位系数
- 选用高粘度润滑油

### 7.3 齿面磨损
<!-- 来源: 第3卷 第15篇齿轮传动 第224页 -->

**分类：**

| 磨损类型 | 成因 | 特征 |
|:--------|:-----|:-----|
| 磨粒磨损 | 杂质、硬颗粒进入啮合 | 磨损均匀，有划痕 |
| 腐蚀磨损 | 润滑油中的化学腐蚀 | 齿面失去光泽 |
| 微动磨损 | 微小相对运动 | 接触边缘磨蚀 |

**防止措施：**
- 闭式传动，加强密封
- 定期更换清洁润滑油
- 提高齿面硬度
- 开式齿轮适当加大模数

### 7.4 齿面胶合
<!-- 来源: 第3卷 第15篇齿轮传动 第251页 -->

**分类：**

| 类型 | 条件 | 特征 |
|:----|:-----|:-----|
| 热胶合 | 高速重载→高温→油膜破裂 | 齿面沿滑动方向撕裂 |
| 冷胶合 | 低速重载→油膜无法建立 | 表面金属直接接触撕脱 |

**防止措施：**
- 使用极压（EP）添加剂润滑油
- 降低表面粗糙度
- 采用不同材料配对（如钢对铜）
- 控制齿面温度
- 合理选择润滑油粘度

### 7.5 齿面塑性变形
<!-- 来源: 第3卷 第15篇齿轮传动 第165页 -->

| 形式 | 特征 | 原因 |
|:----|:-----|:-----|
| 压痕 | 齿面压出凹坑 | 硬颗粒压入软齿面 |
| 飞边 | 齿顶/齿端挤出金属毛刺 | 重载下材料流动 |
| 波纹 | 齿面出现波纹状 | 滚动+滑动复合作用 |

**防止措施：**
- 提高齿面硬度
- 降低接触应力
- 改善润滑

### 7.6 轮齿折断
<!-- 来源: 第3卷 第15篇齿轮传动 第183页 -->

**分类：**

| 折断类型 | 特征 | 原因 |
|:--------|:-----|:-----|
| **疲劳折断** | 齿根裂纹逐渐扩展 | 长期交变弯曲应力 |
| **过载折断** | 突然断裂，断面粗糙 | 短时超载或冲击 |
| **局部折断** | 齿端斜断 | 偏载或端部应力集中 |

**典型断口特征：**
- 疲劳折断：疲劳源（光滑区）+ 瞬断区（粗糙区）
- 过载折断：整体粗糙纤维状断口

**防止措施：**
- 增大齿根圆角半径
- 避免齿根刀痕
- 齿根喷丸强化
- 齿端倒角修缘
- 提高材料韧性

### 7.7 失效形式与设计对策对照
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 失效形式 | 主导设计参数 | 材料对策 | 润滑对策 |
|:--------|:------------|:---------|:---------|
| 点蚀 | $d_1$（或中心距 $a$） | 提高表面硬度 | 高粘度油 |
| 磨损 | $m$（模数放大） | 提高表面硬度+耐磨材料 | 清洁+过滤 |
| 胶合 | 降低滑差、齿顶修缘 | 表面处理、不同材料配对 | EP添加剂 |
| 塑性变形 | 降低 $\sigma_H$ | 提高硬度 | — |
| 折断 | $m_n$（加大模数） | 提高韧性+芯部硬度 | — |

### 7.8 齿轮可靠性与安全系数
<!-- 来源: 第3卷 第15篇齿轮传动 第80页 -->

| 应用领域 | 接触强度安全系数 $S_H$ | 弯曲强度安全系数 $S_F$ | 可靠性水平 |
|:--------|:---------------------:|:---------------------:|:---------|
| 通用机械 | 1.0~1.1 | 1.25~1.4 | 中等 |
| 重要工业传动 | 1.1~1.25 | 1.4~1.6 | 较高 |
| 航空、航天 | 1.25~1.5 | 1.6~2.0 | 很高 |
| 军事装备 | 1.5~2.0 | 2.0~2.5 | 极高 |
