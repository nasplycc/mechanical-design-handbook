## 1. 螺旋传动概述
<!-- 来源: 第14篇带、链传动 第19页 -->

### 1.1 螺旋传动工作原理
<!-- 来源: 第11篇润滑与密封 第3页 -->

螺旋传动通过螺杆与螺母的螺旋面相对运动，将旋转运动转换为直线运动，同时传递运动和动力。

### 1.2 螺旋传动分类
<!-- 来源: 第14篇带、链传动 第19页 -->

| 分类方式 | 类型 | 说明 |
|:--------|:-----|:-----|
| **摩擦性质** | **滑动螺旋** | 螺旋面间为滑动摩擦 |
| | **滚动螺旋** | 螺旋面间通过滚动体（滚珠/滚柱） |
| | **静压螺旋** | 螺旋面间为静压油膜 |
| **用途** | 传力螺旋 | 以传递动力为主，如千斤顶、压力机 |
| | 传导螺旋 | 以传递运动为主，如机床进给 |
| | 调整螺旋 | 以调整位置为主，如千分尺、仪器微调 |
| **螺旋副材料** | 钢-钢 | 常用 |
| | 钢-铸铁 | 耐磨性好 |
| | 钢-铜合金 | 减摩性好 |
| **螺纹牙型** | 梯形螺纹 | 传力螺旋（最常用） |
| | 锯齿形螺纹 | 单向重载传力 |
| | 矩形螺纹 | 传动效率高（加工困难） |
| | 三角形螺纹 | 紧固为主，不用于传动 |

### 1.3 常用螺纹牙型对比
<!-- 来源: 第11篇润滑与密封 第13页 -->

| 螺纹牙型 | 牙型角 | 效率 | 特点 | 适用场合 |
|:--------|:------|:----|:-----|:---------|
| **三角形** | 60° | 低 | 自锁性好，强度高 | 连接紧固 |
| **梯形 (Tr)** | 30° | 中等 ($\eta \approx 0.3\sim0.5$) | 强度高，对中性好 | 双向传力（机床丝杠） |
| **锯齿形 (B)** | 33°（工作边3°,非工作边30°） | 较高 | 单向承载大 | 单向重载（压力机） |
| **矩形** | 0° | 最高 ($\eta \approx 0.4\sim0.6$) | 效率最高，加工难 | 轻载高效率 |

**螺纹代号与标记：**

| 螺纹类型 | 示例 | 含义 |
|:--------|:-----|:-----|
| 梯形螺纹 | Tr40×7 | 公称直径40mm，螺距7mm |
| 锯齿形螺纹 | B40×7 | 公称直径40mm，螺距7mm |
| 多线梯形 | Tr40×14(P7) | 导程14mm，螺距7mm，双线 |

### 1.4 螺旋传动特点
<!-- 来源: 第14篇带、链传动 第19页 -->

| 优点 | 缺点 |
|:----|:-----|
| 减速比大（以较小力矩获得较大推力） | 效率较低（滑动螺旋） |
| 可实现准确位移 | 磨损失效（滑动螺旋） |
| 具有自锁能力（滑动螺旋） | 低速可能爬行 |
| 结构紧凑，承载能力大 |  |
| 运动平稳，无噪声 |  |

### 1.5 螺旋传动基本参数
<!-- 来源: 第11篇润滑与密封 第3页 -->

| 参数 | 符号 | 定义 |
|:----|:----|:-----|
| 公称直径 | $d$ | 螺纹大径 |
| 螺距 | $P$ | 相邻两螺纹牙对应点的轴向距离 |
| 导程 | $P_h$ | 螺杆旋转一周螺母的轴向位移，$P_h = nP$ |
| 线数 | $n$ | 螺纹头数 |

| 牙型半角 | $\beta$ | 梯形$15^\circ$，锯齿形$1.5^\circ/15^\circ$ |
| 螺母高度 | $H$ | 螺母长度 |
| 旋合圈数 | $u$ | $u = H/P$ |

### 2.3 滑动螺旋设计计算
<!-- 来源: 第11篇润滑与密封 第3页 -->

#### 2.3.1 耐磨性计算
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第13页 -->

耐磨性计算为滑动螺旋设计的首要准则。磨损是滑动螺旋的主要失效形式。

**螺纹工作面上的压强（约束条件）：**

$$p = \frac{F}{\pi d_2 h u} \le [p]$$

**按耐磨性确定螺纹中径（设计公式）：**

$$d_2 \ge \sqrt{\frac{F}{\pi \phi \xi [p]}}$$

或按梯形螺纹：

$$d_2 \ge 0.8 \sqrt{\frac{F}{\phi [p]}}$$

**各参数含义：**

| 符号 | 含义 | 说明 |
|:----|:-----|:-----|
| $F$ | 轴向载荷 (N) | |
| $d_2$ | 螺纹中径 (mm) | |
| $h$ | 螺纹牙工作高度 (mm) | 梯形 $h = 0.5P$ |
| $u$ | 旋合圈数 | $u = H/P$ |
| $\phi$ | 螺母形式系数 | $\phi = H/d_2$（整体螺母 $\phi = 1.2\sim 2.5$，剖分螺母 $\phi = 2.5\sim 3.5$） |
| $\xi$ | 载荷分布不均系数 | $\xi = 0.5\sim 0.7$ |
| $[p]$ | 许用压强 (MPa) | 见下表 |

**许用压强 $[p]$（MPa）：**

| 螺杆-螺母材料 | 滑动速度（低速→高速） |
|:------------|:-------------------:|
| | **低速 ($v \le 0.05\text{m/s}$)** | **中速 ($v \le 0.1\text{m/s}$)** | **高速 ($v > 0.1\text{m/s}$)** |
| 钢-青铜 | 11~18 | 7~11 | 4~7 |
| 钢-铸铁 | 7~13 | 4~7 | — |
| 钢-钢 | 7~11 | 4~7 | — |
| 淬火钢-青铜 | 11~13 | 7~11 | 4~7 |

#### 2.3.2 强度计算
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第65页 -->

**螺杆强度（危险截面的当量应力）：**

螺杆受轴向压力（或拉力）和扭矩的联合作用，按第四强度理论校核：

$$\sigma_{ca} = \sqrt{\sigma^2 + 3\tau^2} = \sqrt{\left(\frac{F}{A}\right)^2 + 3\left(\frac{T}{W_T}\right)^2} \le [\sigma]$$

| 符号 | 含义 | 公式 |
|:----|:-----|:-----|
| $A$ | 螺杆危险截面面积 | $A = \pi d_3^2/4$ |
| $W_T$ | 螺杆抗扭截面系数 | $W_T = \pi d_3^3/16$ |
| $T$ | 驱动转矩 | $T = F \cdot d_2 \cdot \tan(\psi + \rho')/2$ |
| $\sigma$ | 正应力 | $F/A$ |
| $\tau$ | 切应力 | $T/W_T$ |
| $[\sigma]$ | 许用应力 | $[\sigma] = \sigma_s / (3\sim 5)$ |

**螺纹牙强度：**

**螺母螺纹牙弯曲强度：**

$$\sigma_b = \frac{3Fh}{\pi d b^2 u} \le [\sigma_b]$$

**螺母螺纹牙剪切强度：**

$$\tau = \frac{F}{\pi d b u} \le [\tau]$$

其中 $b$ 为螺纹牙根部宽度（梯形 $b = 0.65P$），$d$ 为螺母螺纹大径。

**许用应力：**

| 材料 | $[\sigma_b]$ (MPa) | $[\tau]$ (MPa) |
|:----|:-----------------:|:-------------:|
| 青铜 | 40~60 | 30~40 |
| 铸铁 | 40~50 | 25~35 |
| 钢 | 50~80 | 35~50 |

#### 2.3.3 稳定性校核
<!-- 来源: 第14篇带、链传动 第19页 -->

**螺杆受压时的稳定性校核（长径比较大的螺杆）：**

$$F_c = \frac{\pi^2 EI}{(\mu l)^2} \ge (2.5\sim 4)F$$

| 符号 | 含义 |
|:----|:-----|
| $F_c$ | 临界载荷 (N) |
| $E$ | 弹性模量，钢 $E = 2.07\times 10^5$ MPa |
| $I$ | 螺杆危险截面惯性矩，$I = \pi d_3^4/64$ |
| $\mu$ | 长度系数（见表） |
| $l$ | 螺杆计算长度 |

**长度系数 $\mu$：**

| 支承方式 | 图示 | $\mu$ |
|:--------|:----|:------|
| 两端固定 | — | 0.50 |
| 一端固定、一端不完全固定 | — | 0.60 |
| 一端固定、一端铰支 | — | 0.70 |
| 两端铰支 | — | 1.00 |
| 一端固定、一端自由 | — | 2.00 |

**工作安全系数要求：**

$$S = \frac{F_c}{F} \ge S_{\min}$$

其中 $S_{\min} = 2.5\sim 4$（传力螺旋取大值，传导螺旋取小值）

#### 2.3.4 自锁条件
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第4页 -->

**自锁条件：**

$$\psi \le \rho'$$

**螺旋升角 $\psi$：**

$$\tan\psi = \frac{P_h}{\pi d_2}$$

**当量摩擦角 $\rho'$：**

$$\rho' = \arctan\left(\frac{f}{\cos\beta}\right)$$

| 螺纹类型 | $\beta$ | $\tan\rho'$ |
|:--------|:-------|:-----------|
| 梯形螺纹 | 15° | $\tan\rho' = f / \cos15^\circ \approx 1.035f$ |
| 锯齿形螺纹 | 3° | $\tan\rho' = f / \cos3^\circ \approx 1.001f$ |
| 矩形螺纹 | 0° | $\tan\rho' = f$ |

**摩擦系数 $f$ 参考值：**

| 螺旋副材料 | 润滑条件 | $f$ |
|:----------|:--------|:----|
| 钢-青铜 | 良好润滑 | 0.08~0.10 |
| 钢-铸铁 | 一般润滑 | 0.12~0.15 |
| 钢-钢 | 一般润滑 | 0.11~0.17 |
| 钢-钢 | 干摩擦 | 0.20~0.30 |

**自锁保证：** 为保证可靠自锁，需 $\psi \le \rho' - 1^\circ$

#### 2.3.5 效率计算
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第14页 -->

**螺旋副效率（正行程，旋转→直线）：**

$$\eta = \frac{\tan\psi}{\tan(\psi + \rho')}$$

**螺旋副效率（反行程，直线→旋转）：**

$$\eta' = \frac{\tan(\psi - \rho')}{\tan\psi}$$

**不同螺旋升角下的效率（$f = 0.1$）：**

| $\psi$ | 效率 $\eta$（梯形螺纹） | 是否自锁 |
|:------|:---------------------:|:--------:|
| 2° | ~0.19 | **是** (自锁) |
| 4° | ~0.34 | **是** (自锁) |
| 6° | ~0.45 | 否 |
| 8° | ~0.53 | 否 |
| 10° | ~0.59 | 否 |
| 12° | ~0.64 | 否 |
| 15° | ~0.69 | 否 |

> 可见：单线梯形螺纹（$\psi \approx 2^\circ\sim 4^\circ$）可自锁，但效率仅 20%~35%。  
> 多线螺纹可通过增大 $\psi$ 提高效率，但会丧失自锁能力。

### 2.5 滑动螺旋设计示例流程
<!-- 来源: 第14篇带、链传动 第19页 -->

```
输入：轴向载荷 F, 行程 L, 工作条件
       ↓
选择螺杆、螺母材料 ↓ 确定 [p], [σ], [τ]
       ↓
按耐磨性计算中径 d2（或公称直径 d）
       ↓
标准化 → 确定 d, P, d2, d3
       ↓
计算螺旋升角 ψ，校核自锁条件 ψ ≤ ρ' - 1°
       ↓
螺杆强度校核（当量应力计算）
       ↓
螺纹牙强度校核（弯曲、剪切）
       ↓
稳定性校核（长螺杆）
       ↓
效率计算 η
       ↓
结构设计：螺母高度 H = φd2，旋合圈数 u ≤ 10
       ↓
绘制零件图
```

## 3. 滚动螺旋传动（滚珠丝杠）
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第14页 -->

### 3.1 滚珠丝杠工作原理
<!-- 来源: 第11篇润滑与密封 第3页 -->

在螺杆和螺母的螺旋滚道之间放入滚珠，使螺旋面间的摩擦由滑动变为滚动，从而大幅降低摩擦阻力。

### 3.2 滚珠丝杠特点
<!-- 来源: 第14篇带、链传动 第19页 -->

| 优点 | 缺点 |
|:----|:-----|
| 传动效率高（$\eta = 0.85\sim 0.95$） | 不自锁（需制动装置） |
| 摩擦力小，启动力矩低 | 制造精度要求高，成本高 |
| 运动灵敏度高 | 对冲击载荷敏感 |
| 磨损小，寿命长 | 结构复杂 |
| 精度保持性好 | 抗污染能力差 |
| 可通过预紧消除间隙 |  |
| 高速性能好 |  |

### 3.3 滚珠丝杠结构形式

### 3.5 滚珠丝杠选型计算
<!-- 来源: 第11篇润滑与密封 第3页 -->

#### 3.5.1 额定动载荷与寿命计算
<!-- 来源: 第14篇带、链传动 第106页 -->

**基本额定寿命：**

$$L_{10} = \left(\frac{C_a}{F_a}\right)^3 \times 10^6\ \ (\text{转})$$

**额定寿命（小时）：**

$$L_h = \frac{10^6}{60n_m} \left(\frac{C_a}{F_a}\right)^3\ \ (\text{h})$$

| 符号 | 含义 |
|:----|:-----|
| $L_{10}$ | 基本额定寿命（可靠性90%） |
| $C_a$ | 额定动载荷 (N)，查产品样本 |
| $F_a$ | 当量动载荷 (N) |
| $n_m$ | 平均转速 (r/min) |

**当量动载荷：**

$$F_a = \frac{1}{3} (F_{\max} + 2F_{\min}) \quad\text{（变载荷简化计算）}$$

或按变工况加权：

$$F_a = \sqrt[3]{\frac{F_1^3 n_1 t_1 + F_2^3 n_2 t_2 + \cdots}{n_m (t_1 + t_2 + \cdots)}}$$

**推荐寿命值：**

| 机械类型 | 推荐寿命 $L_h$ (h) |
|:--------|:-----------------:|
| 数控机床 | 15000~20000 |
| 一般机床 | 10000~15000 |
| 工业机器人 | 20000~30000 |
| 自动生产线 | 20000~30000 |
| 汽车转向机构 | 3000~5000 |

#### 3.5.2 额定静载荷
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第51页 -->

$$C_{0a} \ge S_0 F_{a\max}$$

| 符号 | 含义 |
|:----|:-----|
| $C_{0a}$ | 额定静载荷 (N) |
| $F_{a\max}$ | 最大轴向静载荷 (N) |
| $S_0$ | 静载安全系数（一般 $S_0 = 1\sim 3$，高精度 $S_0 = 3\sim 5$） |

#### 3.5.3 转速校核
<!-- 来源: 第14篇带、链传动 第19页 -->

**临界转速 $n_c$（避免共振）：**

$$n_c = \frac{60 \lambda_1^2}{2\pi l_1^2} \sqrt{\frac{EI}{\rho A}} \times 0.8$$

简化公式（钢制丝杠）：

$$n_c = f \times \frac{10^7 d_2}{l_1^2}\ \ (\text{r/min})$$

| 符号 | 含义 |
|:----|:-----|
| $\lambda_1$ | 长度系数（与支撑方式有关） |
| $l_1$ | 丝杠支承间距 (mm) |
| $d_2$ | 丝杠螺纹底径 (mm) |
| $f$ | 与支撑方式有关的系数（见表） |

**支撑方式系数 $f$：**

| 支撑方式 | 固定-固定 | 固定-支承 | 固定-自由 |
|:--------|:--------:|:--------:|:--------:|
| $f$ | 21.0 | 15.1 | 3.4 |

**校核要求：** $n_{\max} \le 0.8 n_c$

#### 3.5.4 压杆稳定性
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第55页 -->

## 4. 静压螺旋传动
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第4页 -->

### 4.1 静压螺旋工作原理
<!-- 来源: 第11篇润滑与密封 第3页 -->

静压螺旋通过外部供油系统向螺旋副工作面之间注入高压油，形成承载油膜，使螺旋面完全脱离接触。

### 4.2 静压螺旋特点
<!-- 来源: 第14篇带、链传动 第19页 -->

| 优点 | 缺点 |
|:----|:-----|
| 摩擦极低（$\mu \approx 0.0005$） | 需要复杂的供油系统 |
| 效率极高（$\eta \approx 0.99$） | 成本高 |
| 无磨损，寿命长 | 对油液清洁度要求高 |
| 刚性好，阻尼大 | 油膜受温度影响 |
| 运动平稳、无爬行 | 不能突然断油 |
| 精度保持性好 |  |

### 4.3 静压螺旋结构与供油系统
<!-- 来源: 第14篇带、链传动 第19页 -->

**结构形式：**
- **轴向油腔式**：在螺母螺旋面上开设轴向油腔
- **周向油腔式**：在螺母螺旋面上开设周向油腔
- **油腔节流式**：每个油腔对应一个毛细管或小孔节流器

**供油系统组成：**
| 组件 | 作用 |
|:----|:-----|
| 液压泵 | 提供压力油（0.2~6MPa） |
| 节流器 | 调节各油腔压力，形成承载能力 |
| 滤油器 | 保证油液清洁度（精度要求10~20μm） |
| 溢流阀 | 控制系统压力 |
| 蓄能器 | 断电时保压 |

### 4.4 静压螺旋设计要点
<!-- 来源: 第11篇润滑与密封 第262页 -->

| 设计项目 | 要点 |
|:--------|:-----|
| 油腔面积 | 应足够大，满足承载要求 |
| 封油边宽 | 过大则泄漏小但承载力降，过小则泄漏大 |
| 油膜厚度 | 一般 $h_0 = 0.01\sim 0.05\text{mm}$ |
| 供油压力 | $p_s = 0.5\sim 6\text{MPa}$ |
| 润滑油粘度 | ISO VG32~VG68 |

## 5. 摩擦轮传动概述
<!-- 来源: 第14篇带、链传动 第19页 -->

### 5.1 摩擦轮传动工作原理
<!-- 来源: 第11篇润滑与密封 第3页 -->

摩擦轮传动依靠两轮直接接触并压紧，利用接触面间的摩擦力传递运动和动力。

### 5.2 摩擦轮传动分类
<!-- 来源: 第14篇带、链传动 第19页 -->

| 分类方式 | 类型 | 说明 |
|:--------|:-----|:-----|
| **轮形** | 圆柱摩擦轮 | 两轴线平行 |
| | 圆锥摩擦轮 | 两轴线相交 |
| | 槽形摩擦轮 | V形槽工作面 |
| | 端面摩擦轮 | 圆柱端面与圆盘 |
| **传动形式** | 定传动比 | 固定速比 |
| | 无级变速 | 连续改变传动比 |
| **工作条件** | 干摩擦 | 直接接触，无润滑 |
| | 湿摩擦 | 有润滑油或牵引油 |

### 5.3 摩擦轮传动特点
<!-- 来源: 第14篇带、链传动 第19页 -->

| 优点 | 缺点 |
|:----|:-----|
| 传动平稳，无噪声 | 承载能力低 |
| 结构简单，成本低 | 传动比不准确（有滑动） |
| 过载打滑保护 | 效率较低（$\eta = 0.85\sim 0.95$） |
| 可无级变速 | 轴压力大 |
| 无齿隙 | 摩擦材料有磨损 |

### 5.4 摩擦轮传动基本参数
<!-- 来源: 第12篇弹簧 第165页 -->

$$F_f = \mu F_n \ge F_t$$

| 参数 | 符号 | 定义 |
|:----|:----|:-----|
| 摩擦力 | $F_f$ | 接触面间的摩擦力 |
| 正压力 | $F_n$ | 两轮间的压紧力 |
| 圆周力 | $F_t$ | 传递的切向力，$F_t = 2T/d$ |
| 摩擦系数 | $\mu$ | 材料间的摩擦系数 |
| 安全系数 | $K$ | $K = F_f/F_t \ge 1.25\sim 1.5$ |

**压紧力与圆周力的关系：**

$$F_n = \frac{K F_t}{\mu}$$

可见摩擦轮所需压紧力远大于圆周力本身（$K/\mu$ 可达数倍至十数倍），这是摩擦轮传动的主要缺点。

## 6. 摩擦轮传动设计

### 6.1 摩擦轮常用材料
<!-- 来源: 第14篇带、链传动 第19页 -->

| 材料配对 | 摩擦系数 $\mu$ | 许用接触应力 $[\sigma_H]$ (MPa) | 特点 | 应用 |
|:--------|:-------------:|:---------------------------:|:-----|:-----|
| 淬火钢-淬火钢（油中） | 0.04~0.06 | 800~1200 | 承载高，需润滑 | 金属摩擦轮 |
| 钢-钢（干） | 0.10~0.20 | 500~800 | 易磨损 | 低速轻载 |
| 钢-铸铁（干） | 0.15~0.25 | 400~600 | 成本低 | 通用摩擦轮 |
| 铸铁-铸铁（干） | 0.10~0.15 | 300~500 | 耐磨性一般 | 低速传动 |
| 钢-夹布胶木 | 0.20~0.35 | 100~150 | 摩擦系数大 | 轻载减摩 |
| 钢-橡胶 | 0.30~0.50 | 20~50 | 弹性好，噪声小 | 轻载无级变速 |

**摩擦轮材料选择原则：**
1. 高摩擦系数 → 减小压紧力
2. 高弹性模量 → 提高接触强度
3. 耐磨性好 → 寿命长
4. 配对材料合理 → 避免粘着和咬合

### 6.2 圆柱摩擦轮传动
<!-- 来源: 第13篇螺旋传动、摩擦轮传动 第73页 -->

#### 6.2.1 结构形式
<!-- 来源: 第12篇弹簧 第174页 -->

| 形式 | 结构 | 应用 |
|:----|:-----|:-----|
| 外圆柱摩擦轮 | 两圆柱外切 | 平行轴同向传动 |
| 内圆柱摩擦轮 | 大轮内孔接触小轮 | 平行轴同向，紧凑 |

#### 6.2.2 设计计算
<!-- 来源: 第14篇带、链传动 第11页 -->

**接触强度条件（按赫兹接触理论）：**

$$\sigma_H = 0.418 \sqrt{\frac{F_n E}{b \rho}} \le [\sigma_H]$$

**其中当量曲率半径 $\rho$：**

$$\frac{1}{\rho} = \frac{1}{R_1} + \frac{1}{R_2}\quad\text{（外接触）}$$

$$\frac{1}{\rho} = \frac{1}{R_1} - \frac{1}{R_2}\quad\text{（内接触）}$$

**设计需要的压紧力：**

$$F_n = \frac{K F_t}{\mu} = \frac{K \cdot 2T_1}{\mu d_1}$$

| 符号 | 含义 |
|:----|:-----|
| $F_n$ | 压紧力 (N) |
| $E$ | 当量弹性模量 (MPa)，$\frac{2}{E} = \frac{1-\nu_1^2}{E_1} + \frac{1-\nu_2^2}{E_2}$ |
| $b$ | 轮宽 (mm) |
| $\rho$ | 当量曲率半径 (mm) |
| $R_1, R_2$ | 两轮半径 (mm) |
| $[\sigma_H]$ | 许用接触应力 (MPa) |
| $K$ | 安全系数，$K = 1.25\sim 1.5$ |
| $\mu$ | 摩擦系数 |
| $F_t$ | 圆周力 (N) |

**按接触强度确定轮宽 $b$：**

$$b \ge \frac{0.175 F_n E}{\rho [\sigma_H]^2}$$

### 6.3 圆锥摩擦轮传动
<!-- 来源: 第14篇带、链传动 第19页 -->

#### 6.3.1 几何关系
<!-- 来源: 第15篇齿轮传动 第499页 -->

用于两相交轴传动（常用轴交角 $\Sigma = 90^\circ$）：

| 参数 | 符号 | 公式 |
|:----|:----|:-----|
| 传动比 | $i$ | $i = n_1/n_2 = R_2/R_1$ |
| 分锥角 | $\delta$ | $\tan\delta_1 = \sin\Sigma / (i + \cos\Sigma)$ |
| | | $\delta_2 = \Sigma - \delta_1$（$\Sigma = 90^\circ$时$\tan\delta_1 = 1/i$） |
| 平均接触点处的半径 | $R_m$ | $R_m = R - b\sin\delta/2$ |

#### 6.3.2 设计计算
<!-- 来源: 第14篇带、链传动 第11页 -->

**压紧力：**

$$F_n = \frac{K F_t}{\mu \cos\alpha}$$

其中 $\alpha$ 为圆锥顶半角。

**接触强度：**

$$\sigma_H = 0.418\sqrt{\frac{F_n E}{b \rho_m}} \le [\sigma_H]$$

$$\rho_m = \frac{R_{m1} R_{m2}}{R_{m1} \cos\delta_2 + R_{m2} \cos\delta_1}$$

### 6.4 摩擦轮压紧力产生方式
<!-- 来源: 第14篇带、链传动 第19页 -->

| 方式 | 结构 | 特点 |
|:----|:-----|:-----|
| **弹簧压紧** | 弹簧施加恒压 | 压紧力恒定，结构简单 |
| **重力压紧** | 自重或配重 | 简单可靠，占空间 |
| **液压/气压压紧** | 油缸/气缸 | 压力可调，可控 |
| **离心力自动压紧** | 随转速增加压力增大 | 自动适应载荷 |
| **电磁压紧** | 电磁铁压紧 | 可快速离合 |

### 6.5 打滑条件
<!-- 来源: 第14篇带、链传动 第19页 -->

**不打滑条件：**

$$\mu F_n \ge K F_t$$

**安全系数 $K$：**

| 工作条件 | $K$ |
|:--------|:----|
| 载荷稳定 | 1.25~1.35 |
| 载荷有轻微冲击 | 1.35~1.50 |
| 载荷有较大冲击 | 1.50~2.00 |

## 7. 无级变速摩擦轮传动
<!-- 来源: 第11篇润滑与密封 第3页 -->

### 7.1 无级变速原理
<!-- 来源: 第14篇带、链传动 第19页 -->

通过改变摩擦轮的工作接触半径，实现传动比连续变化。

### 7.2 常用无级变速摩擦轮型式
<!-- 来源: 第14篇带、链传动 第19页 -->

| 类型 | 结构 | 调速范围 | 特点 |
|:----|:-----|:--------|:-----|
| **圆盘-滚轮** | 圆盘端面与滚轮接触 | $R_b = 3\sim 4$ | 简单，变速范围大 |
| **锥轮-锥轮** | 两圆锥轮相对移动 | $R_b = 3\sim 6$ | 承载较大 |
| **环柱-锥轮** | 环形与锥轮接触 | $R_b = 4\sim 8$ | 变速范围大 |
| **脉动式** | 单向离合器+偏心机构 | $R_b = 3\sim 5$ | 输出脉动 |
| **环盘式** | 锥盘与环面接触 | $R_b = 6\sim 10$ | 紧凑，承载大 |

**调速范围 $R_b$：**

$$R_b = \frac{n_{2\max}}{n_{2\min}}$$

### 7.3 圆盘-滚轮无级变速器
<!-- 来源: 第11篇润滑与密封 第51页 -->

**工作原理：**
- 主动轮为圆盘（恒速旋转）
- 从动滚轮可在圆盘径向移动
- 改变滚轮接触半径 $r$ → 改变传动比

**传动比：**

$$i = \frac{r}{R}$$

其中 $R$ 为主动圆盘半径，$r$ 为滚轮接触点到圆盘中心的距离。

**调速范围：**

$$R_b = \frac{r_{\max}}{r_{\min}}$$

### 7.4 环柱-锥轮无级变速器
<!-- 来源: 第14篇带、链传动 第19页 -->

| 参数 | 说明 |
|:----|:-----|
| 主动元件 | 圆锥轮（变速） |
| 从动元件 | 环形柱面轮 |
| 调速方式 | 调整主动锥轮与从动环柱的轴向相对位置 |
| 传动比范围 | $i = 0.25\sim 4$（双向） |
| 功率范围 | 0.1~15kW |
| 效率 | 0.80~0.92 |

### 7.5 常用无级变速器性能对比
<!-- 来源: 第14篇带、链传动 第19页 -->

| 类型 | 功率 (kW) | 调速范围 $R_b$ | 效率 | 噪声 | 成本 |
|:----|:---------|:-------------|:----|:----|:----|
| 圆盘-滚轮 | 0.1~3 | 3~4 | 0.75~0.85 | 低 | 低 |
| 锥轮-锥轮（钢-钢） | 0.5~15 | 3~6 | 0.80~0.90 | 中 | 中 |
| 环柱-锥轮 | 0.1~15 | 4~8 | 0.80~0.92 | 中 | 中 |
| 环盘式 | 0.5~30 | 6~10 | 0.85~0.93 | 低 | 高 |
| 金属带式 | 20~200 | 5~6 | 0.90~0.95 | 低 | 高 |
| 脉动式 | 0.1~7.5 | 3~5 | 0.65~0.80 | 大 | 低 |

### 7.6 无级变速器选用要点
<!-- 来源: 第14篇带、链传动 第19页 -->

| 要点 | 说明 |
|:----|:-----|
| 功率匹配 | 连续工作功率不超过额定功率的80% |
| 调速范围 | 根据工艺需求选用合适的 $R_b$ |
| 输出特性 | 恒功率输出或恒转矩输出 |
| 润滑 | 牵引油或专用CVT油 |
| 控制方式 | 手动/电动/液压/伺服控制 |
| 安装形式 | 卧式/立式，输入输出轴方向 |
