## 一、概述
<!-- 来源: 第3卷 第11篇润滑与密封 第142页 -->

| 项目 | 说明 |
|------|------|
| 定义 | 通过多个齿轮副同时啮合传递动力的传动方式 |
| 特点 | 功率分流、均载、紧凑 |
| 优势 | 单级传动比大（3~200）、承载能力强、体积小 |
| 应用 | 大型减速器、风力发电、工程机械、冶金机械 |
| 设计难点 | 均载设计、变形谐调、制造精度 |

## 二、多点啮合传动类型
<!-- 来源: 第3卷 第11篇润滑与密封 第142页 -->

### 2.1 行星齿轮传动
<!-- 来源: 第3卷 第15篇齿轮传动 第45页 -->

| 类型 | 传动比范围 | 效率 | 特点 |
|------|-----------|------|------|
| NGW型（2K-H） | 2.8~12.5（单级） | 0.97~0.99 | 最常用，结构紧凑 |
| NW型（2K-H） | 1.55~21 | 0.97~0.99 | 双联行星轮 |
| NN型（2K-H） | 8~30 | 0.95~0.97 | 内啮合+内啮合 |
| ZUWGW型 | 25~4000（多级） | 0.90~0.96 | 大减速比 |
| 3K型 | 20~500 | 0.85~0.90 | 极大减速比 |

### 2.2 NGW型行星传动（最常用）
<!-- 来源: 第3卷 第15篇齿轮传动 第454页 -->

**传动比：**

| 固定件 | 主动件 | 从动件 | 传动比i |
|-------|-------|-------|---------|
| 齿圈b固定 | 太阳轮a | 行星架g | i=1+z_b/z_a |
| 齿圈b固定 | 行星架g | 太阳轮a | i=1/(1+z_b/z_a) |
| 太阳轮a固定 | 齿圈b | 行星架g | i=z_b/(z_a+z_b) |
| 太阳轮a固定 | 行星架g | 齿圈b | i=(z_a+z_b)/z_b |

> z_a—太阳轮齿数，z_b—齿圈齿数

**行星轮数量（推荐）：**

| 传动比i | 推荐行星轮数n_p | 载荷不均匀系数K_p |
|---------|--------------|----------------|
| 2.8~4 | 3~4 | 1.1~1.25 |
| 4~6.3 | 3~4 | 1.1~1.3 |
| 6.3~10 | 3~5 | 1.15~1.35 |
| >10 | 4~6 | 1.2~1.5 |

### 2.3 齿数选择条件
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 条件 | 公式 | 说明 |
|------|------|------|
| 传动比条件 | i=1+z_b/z_a | 满足传动比要求 |
| 装配条件 | (z_a+z_b)/n_p=整数 | 行星轮能均匀装配 |
| 同心条件 | z_a+2z_x=z_b | 太阳轮+2×行星轮=齿圈 |
| 邻接条件 | 2a_w sin(π/n_p) > d_a,x | 相邻行星轮不干涉（a_w—中心距，d_a,x—行星轮齿顶圆直径） |

### 2.4 均载机构
<!-- 来源: 第3卷 第15篇齿轮传动 第490页 -->

| 均载方式 | 结构 | 效果 | 制造要求 |
|---------|------|------|---------|
| 太阳轮浮动 | 太阳轮用齿形联轴器连接 | 好 | 中等 |
| 行星架浮动 | 行星架用弹性元件 | 较好 | 中等 |
| 齿圈浮动 | 齿圈用弹性支撑 | 好 | 较高 |
| 弹性销 | 行星轮轴用弹性销支撑 | 较好 | 中等 |
| 油膜均载 | 行星轮支撑用浮动油膜 | 优 | 高 |
| 复合浮动 | 太阳轮+行星架+齿圈组合浮动 | 最优 | 高 |

## 三、摆线针轮传动
<!-- 来源: 第3卷 第15篇齿轮传动 第8页 -->

| 参数 | 说明 |
|------|------|
| 传动原理 | 摆线轮与针齿差齿啮合，1齿差实现大减速 |
| 传动比 | i=1/(1-z_p/z_b)（z_p—摆线轮齿数，z_b—针齿数） |
| 单级传动比范围 | 11~87 |
| 效率 | 0.90~0.95 |
| 特点 | 多齿同时啮合（约1/3齿），承载大，体积小 |
| 应用 | 工业机器人/精密定位/管磨机 |

## 四、谐波传动
<!-- 来源: 第1卷 第5篇机械产品结构设计 第588页 -->

| 参数 | 说明 |
|------|------|
| 原理 | 柔轮弹性变形与刚轮差齿啮合 |
| 传动比 | i=(z_g-z_r)/z_g（刚轮固定时） |
| 典型传动比 | 50~500 |
| 效率 | 0.80~0.90 |
| 特点 | 零回差（精密）、体积小、减速比极大 |
| 应用 | 机器人关节/精密仪器/卫星天线 |

## 五、功率分流与载荷分配
<!-- 来源: 第3卷 第15篇齿轮传动 第4页 -->

| 功率分流方式 | 特点 | 应用 |
|-------------|------|------|
| 平行轴双路分流 | 扭矩由两对齿轮副分担 | 大型减速器 |
| 人字齿轮分流 | 左右旋向齿轮副分流 | 矿山/水泥磨 |
| 多行星轮分流 | 每个行星轮分担1/n功率 | 行星传动 |
| 双输入/输出分流 | 多电机输入，多输出端 | 重型机械 |

## 六、变形谐调分析
<!-- 来源: 第3卷 第14篇带、链传动 第19页 -->

| 变形类型 | 影响因素 | 对传动影响 |
|---------|---------|-----------|
| 扭转变形 | 扭矩+轴长 | 引起齿面偏载 |
| 弯曲变形 | 径向力+支撑刚度 | 引起齿向偏载 |
| 接触变形 | 赫兹接触应力 | 齿面接触不均匀 |
| 热变形 | 温升+热膨胀系数 | 齿廓变化 |
| 支撑变形 | 轴承刚度+机架刚度 | 轴线平行度下降 |

**偏载系数：** K_Hβ = K_Hβ⁰ + Δ_K_Hβ
- 通常需计算齿轮轴扭转变形和弯曲变形引起的偏载
- 可通过修形（齿向修形、齿廓修形）补偿

## 七、设计要点
<!-- 来源: 第3卷 第11篇润滑与密封 第361页 -->

| 设计步骤 | 内容 |
|---------|------|
| 1. 方案选型 | 根据传动比/功率/空间选类型（行星/摆线/谐波） |
| 2. 齿数确定 | 按传动比+装配/同心/邻接条件计算齿数 |
| 3. 齿轮强度计算 | 按ISO 6336计算接触强度+弯曲强度 |
| 4. 均载结构设计 | 选用均载方案，计算不均匀系数 |
| 5. 轴承选型与校核 | 行星轮轴承（受交变载荷）、太阳轮轴承 |
| 6. 变形分析 | 扭转变形+弯曲变形+偏载校核 |
| 7. 结构设计 | 行星架/输出轴/润滑/密封 |
| 8. 热平衡校核 | 计算散热量，需满足油温≤80°C |
