## 1 液压控制概述
<!-- 来源: 第4卷 第22篇液压控制 第24页 -->

### 1.1 液压控制的分类
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

液压控制是指在液压系统中通过各类控制阀对液流的方向、压力和流量进行调节和控制，以实现对执行元件（液压缸、液压马达）的运动控制。

**按控制类别：**

| 控制类型 | 控制参数 | 实现元件 | 功能 |
|---------|---------|---------|------|
| 方向控制 | 液流通断和流向 | 单向阀、换向阀 | 启停、换向 |
| 压力控制 | 系统或局部压力 | 溢流阀、减压阀、顺序阀、压力继电器 | 限压、减压、顺序动作 |
| 流量控制 | 液体流量大小 | 节流阀、调速阀 | 调速、同步 |

**按控制方式：**

| 控制方式 | 信号类型 | 精度等级 | 响应速度 | 应用 |
|---------|---------|---------|---------|------|
| 开关控制 | 通/断信号 | 低 | — | 一般工业 |
| 电液比例控制 | 连续电信号 | 中（1%~5%） | 20~100 Hz | 工程机械、注塑机 |
| 电液伺服控制 | 连续电信号 | 高（0.1%~1%） | 100~200 Hz | 精密机床、模拟台 |
| 数字控制 | 数字脉冲 | 中高 | 中 | 特殊用途 |

### 1.2 液压控制技术发展趋势
<!-- 来源: 第5卷 第23篇气压传动 第185页 -->

| 趋势 | 内容 |
|------|------|
| **集成化** | 阀组集成、插装阀系统使结构紧凑，减少泄漏点 |
| **智能化** | 智能阀（带传感器+微控制器），自诊断 |
| **网络化** | CAN总线、EtherCAT控制，分布式控制架构 |
| **节能化** | 负载敏感+电控泵+高效阀，系统效率大幅提升 |
| **电控化** | 比例/伺服阀比例逐年提升，传统开关阀逐步替代 |

## 2 方向控制阀
<!-- 来源: 第4卷 第21篇液压传动 第779页 -->

### 2.1 单向阀
<!-- 来源: 第4卷 第21篇液压传动 第254页 -->

#### 2.1.1 普通单向阀
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

**结构原理：**

阀芯在弹簧力作用下压在阀座上，液流从A口流向B口时推开阀芯导通；反方向时阀芯紧压阀座，液流被截断。

**特性：**

| 参数 | 典型值 |
|------|--------|
| 开启压力 | 0.05~0.5 MPa |
| 额定流量下压降 | 0.1~0.8 MPa |
| 最高工作压力 | ≤35 MPa |
| 反向泄漏 | ≤0.01 mL/min |
| 响应时间 | 10~50 ms |

**分类与选择：**

| 类型 | 结构特点 | 应用 |
|------|---------|------|
| 直通式（管式） | 阀体直接接于管路 | 简单管路 |
| 直角式（板式） | 进出油口成直角 | 集成阀板安装 |
| 插装式 | 插入阀块 | 集成系统 |
| 球阀式单向阀 | 钢球作阀芯 | 高压小流量 |

#### 2.1.2 液控单向阀
<!-- 来源: 第4卷 第21篇液压传动 第796页 -->

在普通单向阀基础上增加控制活塞，当控制口K通入压力油时，可强制打开阀芯实现反向导通。

**分类：**

| 类型 | 卸荷方式 | 特点 |
|------|---------|------|
| 内泄式 | 控制油来自被控管路 | 简单，但受背压影响 |
| 外泄式 | 单独泄油口 | 高压时可靠 |
| 带卸荷阀芯 | 先导小阀芯卸荷 | 大流量时开启平稳 |

**应用场景：**

- 液压缸锁紧回路（双向液控单向阀 = 液压锁）
- 平衡回路（防止垂直缸下滑）
- 蓄能器保压回路

**液压锁（双液控单向阀）的锁紧精度：**

\[
\text{锁紧位移} \leq \frac{Q_{leak} \times t}{A}
\]

在理想密封条件下，锁紧精度可达0.01mm以内（实际受密封影响）。

### 2.2 换向阀
<!-- 来源: 第4卷 第21篇液压传动 第254页 -->

#### 2.2.1 换向阀的基本原理
<!-- 来源: 第4卷 第21篇液压传动 第3页 -->

换向阀通过改变阀芯在阀体中的位置来控制多个油口之间的通断关系，实现液流方向切换。

**基本参数：**

| 参数 | 含义 |
|------|------|
| **通径** | 阀口名义尺寸，如6、10、16、20、25、32mm等 |
| **位数** | 阀芯的工作位置数（两位、三位） |
| **通数** | 阀口数量（二通、三通、四通、五通等） |
| **公称压力** | 允许的最高工作压力 |
| **公称流量** | 额定压降下的最大流量 |
| **切换时间** | 阀芯从一端移到另一端的时间 |

#### 2.2.2 三位四通换向阀的中位机能
<!-- 来源: 第4卷 第21篇液压传动 第58页 -->

三位四通换向阀的中位（中间位置）油口连接方式决定了系统的中位特性：

**常见中位机能对比：**

| 机能型号 | 符号特征 | P口 | A/B口 | T口 | 特点 | 应用 |
|---------|---------|-----|-------|-----|------|------|
| **O型** | P/A/B/T均封闭 | 封闭 | 封闭 | 封闭 | 油缸两腔锁紧，泵不卸荷 | 锁紧要求，双向负载 |
| **H型** | P/A/B/T互通 | 卸荷 | 通T | 通 | 泵卸荷，缸浮动 | 泵卸荷+缸浮动 |
| **Y型** | A/B通T，P封闭 | 封闭 | 通T | 分别通 | 缸浮动（回油），泵不卸荷 | 需缸浮动的场合 |
| **P型** | P与A/B通，T封闭 | 通A/B | 通P | 封闭 | 差动回路，缸两腔通压 | 差动快进 |
| **M型** | P与T通，A/B封闭 | 卸荷 | 封闭 | 通P | 泵卸荷，缸锁紧 | 泵卸荷+缸锁紧 |
| **K型** | P/A/T通，B封闭 | 通A/T | A通P/T,B封 | 通P/A | 泵卸荷，A腔通P | 单侧浮动 |
| **J型** | P封闭，A/B各态 | 封闭 | A不同,B通T | 通B | — | 特殊场合 |
| **N型** | P封闭，B/T通 | 封闭 | A封,B通T | 通B | — | 精确定位 |
| **C型** | P与A通，B/T封闭 | 通A | A通P,B/T封 | 封闭 | — | — |
| **U型** | P封闭，A/B通T | 封闭 | 通T | 通A/B | 缸浮动（差于Y型） | — |

**三位四通换向阀中位选择原则：**

| 考虑因素 | 推荐机能 | 原因 |
|---------|---------|------|
| 需锁紧液压缸 | O型、M型 | A/B口封闭 |
| 需泵卸荷 | H型、M型、K型 | P-T连通 |
| 需液压缸浮动 | H型、Y型 | A/B接T |
| 单杆缸差动快进 | P型 | P-A-B连通 |
| 换向平稳性 | O型＜H型 | H型中位腔通，背压低 |
| 定位精度 | O型最佳 | 封闭腔形成液压锁 |

## 3 压力控制阀
<!-- 来源: 第4卷 第22篇液压控制 第148页 -->

### 3.1 压力控制阀概述
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

压力控制阀利用液体压力作用在阀芯上的力与弹簧力平衡的原理，通过控制阀口开度来控制系统压力。

**基本方程（力平衡）：**

\[
pA = k(x_0 + x) \quad \text{或} \quad p = \frac{k(x_0 + x)}{A}
\]

式中：
- \( p \) — 被控制的压力
- \( A \) — 阀芯有效作用面积
- \( k \) — 弹簧刚度
- \( x_0 \) — 弹簧预压缩量
- \( x \) — 阀芯位移

### 3.2 溢流阀
<!-- 来源: 第4卷 第21篇液压传动 第254页 -->

#### 3.2.1 溢流阀的功能
<!-- 来源: 第5卷 第23篇气压传动 第67页 -->

1. **定压溢流**：在定量泵系统中保持系统压力恒定，多余油液溢流回油箱
2. **安全保护**：系统过载时溢流，限制最高压力
3. **背压**：在回油路中建立背压
4. **卸荷**：通过遥控口使系统卸荷
5. **顺序动作**：与顺序阀配合实现多缸顺序动作

#### 3.2.2 直动式溢流阀
<!-- 来源: 第4卷 第21篇液压传动 第697页 -->

**结构原理：**

压力油作用在阀芯底部（锥阀或滑阀），当液压力超过弹簧设定力时，阀芯开启，油液溢回油箱。

**特点：**

| 项目 | 说明 |
|------|------|
| 结构 | 简单，零件少 |
| 响应速度 | 快（无先导级延迟） |
| 调压范围 | 3~40 MPa |
| 流量范围 | ≤40 L/min（单级） |
| 调压偏差 | 大（随流量变化压力波动大） |
| 启闭特性 | 较差 |

**调压偏差（压力-流量特性）：**

\[
\Delta p = p_{max} - p_{min} = \frac{k \cdot x_{max}}{A}
\]

直动式溢流阀的调压偏差较大，不适合大流量高精度场合。

#### 3.2.3 先导式溢流阀
<!-- 来源: 第4卷 第22篇液压控制 第186页 -->

**结构原理：**

由先导调压阀（小流量锥阀+精密弹簧）和主阀（滑阀或锥阀）组成。先导阀设定压力，主阀控制大流量溢流。

**工作过程：**
1. 系统压力低于设定值：先导阀关闭，主阀关闭
2. 系统压力达到先导阀开启压力：先导阀开启，主阀上腔压力下降
3. 主阀上下腔产生压差，主阀开启溢流

**特性：**

| 参数 | 说明 |
|------|------|
| 压力范围 | 0.5~31.5 MPa |
| 流量范围 | 40~1000 L/min |
| 调压偏差 | 小（约±0.5 MPa） |
| 启闭特性 | 好（溢流后压力波动小） |
| 遥控卸荷 | 可从先导阀遥控口接电磁换向阀卸荷 |
| 最低稳定压力 | 0.5~1.0 MPa（先导阀需一定背压） |

**遥控卸荷回路：**

先导式溢流阀的遥控口（K口）通过二位二通电磁阀接油箱，电磁阀通电时主阀上腔压力降为零，主阀全开实现卸荷。

#### 3.2.4 溢流阀的技术参数
<!-- 来源: 第4卷 第21篇液压传动 第3页 -->

| 参数 | 直动式 | 先导式 |
|------|--------|--------|
| 公称压力 | ≤40 MPa | ≤31.5 MPa |
| 公称流量 | ≤40 L/min | ≤1000 L/min |
| 调压范围 | 0.5~40 MPa | 0.5~31.5 MPa |
| 最低稳定流量 | 0.5 L/min | 2~5 L/min |
| 卸荷压力 | — | 0.3~0.6 MPa |
| 响应时间 | 10~30 ms | 30~100 ms |
| 内泄漏 | 小 | 较大（先导级泄漏） |

#### 3.2.5 溢流阀应用要点
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 应用场景 | 选型要点 |
|---------|---------|
| 安全阀（过载保护） | 选用直动式（响应快），调定压力=系统最高压力×1.1 |
| 定压溢流 | 选用先导式（压力稳定），设定值=工作压力 |
| 远程调压 | 先导式+远程调压阀 |
| 卸荷 | 先导式+电磁换向阀 |
| 背压阀 | 选开启压力合适的溢流阀 |
| 缓冲 | 直动式（响应快），设定值略高于正常压力 |

### 3.3 减压阀
<!-- 来源: 第4卷 第21篇液压传动 第254页 -->

#### 3.3.1 基本原理
<!-- 来源: 第4卷 第22篇液压控制 第25页 -->

减压阀用于降低系统中某一部分的压力并保持其恒定。出口压力低于进口压力。

**分类：**

| 类型 | 结构 | 特点 |
|------|------|------|
| 直动式减压阀 | 出口压力直接作用于主阀芯 | 小流量、低压 |
| 先导式减压阀 | 先导阀设定+主阀调节 | 大流量、高精度 |

#### 3.3.2 先导式减压阀工作原理
<!-- 来源: 第5卷 第23篇气压传动 第63页 -->

出口压力油分别经阻尼孔进入主阀下腔和先导阀，当出口压力超设定值时先导阀开启，主阀上腔压力下降，主阀芯上移关小阀口，降低出口压力。出口压力低于设定值时先导阀关闭，主阀芯下移开大阀口。

**减压阀与溢流阀的区别：**

| 项目 | 溢流阀 | 减压阀 |
|------|--------|--------|
| 控制对象 | 进口压力（P口） | 出口压力（A口） |
| 常态位置 | 常闭 | 常开 |
| 泄油方式 | 内部泄油（回T） | 外部泄油（单独泄油口L） |
| 作用 | 限制最高压力 | 获得稳定低压 |
| 连接方式 | 并联在油路中 | 串联在支路中 |

#### 3.3.3 减压阀技术参数
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 参数 | 典型值 |
|------|--------|
| 公称压力 | 3.5~31.5 MPa |
| 减压范围 | 0.3~10 MPa（常用），0.3~20 MPa（高压） |
| 流量范围 | 5~400 L/min |
| 调压偏差（稳态） | ≤±0.2 MPa |
| 响应时间 | 50~200 ms |
| 最低压差（进口-出口） | 0.5~1.0 MPa |

#### 3.3.4 减压阀的应用
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 应用场合 | 说明 |
|---------|------|
| 夹紧回路 | 夹紧油路减压至所需夹紧力（通常3~7MPa） |
| 控制油路 | 提供稳定的控制油（0.5~3MPa） |
| 润滑回路 | 减压至0.1~0.5MPa |
| 多级压力系统 | 不同执行元件不同压力等级 |

### 3.4 顺序阀
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

#### 3.4.1 基本原理
<!-- 来源: 第4卷 第22篇液压控制 第25页 -->

顺序阀利用油液自身压力来控制液压执行元件的顺序动作。

**工作原理：**

顺序阀的进口压力油作用在阀芯底部，当压力达到设定值时阀芯开启，使油液通过到下游执行元件。未达到设定值时阀关闭。

#### 3.4.2 分类
<!-- 来源: 第4卷 第21篇液压传动 第70页 -->

| 类型 | 特点 | 应用 |
|------|------|------|
| 直动式顺序阀 | 结构简单，调压偏差大 | 低压顺序动作 |
| 先导式顺序阀 | 调压精度高，可大流量 | 中高压顺序动作 |
| 内控内泄式 | 控制油来自进口，泄油回T | 通用型 |
| 内控外泄式 | 控制油来自进口，单独泄油 | 出口背压影响小 |
| 外控内泄式 | 控制油来自外部油源 | 遥控启闭 |
| 外控外泄式 | 外部控制油+单独泄油 | 灵活控制 |

#### 3.4.3 技术参数
<!-- 来源: 第4卷 第22篇液压控制 第181页 -->

| 参数 | 直动式 | 先导式 |
|------|--------|--------|
| 公称压力 | ≤21 MPa | ≤31.5 MPa |
| 公称流量 | ≤60 L/min | ≤400 L/min |
| 调压范围 | 0.5~21 MPa | 0.3~31.5 MPa |

#### 3.4.4 顺序阀与溢流阀的互换使用
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

将顺序阀出口接油箱时，顺序阀即作溢流阀使用（但泄油方式不同）。实际中：

- 溢流阀作安全阀：并联在主油路上
- 顺序阀作平衡阀：在垂直缸下行回路中做背压支撑

**平衡阀（由顺序阀+单向阀组成）：**

垂直安装的液压缸在负载作用下有下行趋势，在缸下腔出口串联平衡阀（外控内泄顺序阀+单向阀），当上腔压力达到设定值时平衡阀才能打开，防止缸因重力失速。

### 3.5 压力继电器
<!-- 来源: 第4卷 第22篇液压控制 第162页 -->

#### 3.5.1 工作原理
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

压力继电器将液压系统的压力信号转换为电信号（开关量），用于控制电磁阀动作或电机启停。

**动作过程：**

压力达到设定值时，液压油推动柱塞，柱塞推动弹簧，带动微动开关动作发出电信号。压力降低到复位值时弹簧使开关恢复。

| 参数 | 典型值 |
|------|--------|
| 调压范围 | 0.5~35 MPa |
| 重复精度 | ≤设定值的±1% |
| 动作时间 | 10~30 ms |
| 通断能力 | AC 250V/3A，DC 24V/2A |
| 防护等级 | IP65（工业标准） |
| 寿命 | ≥10⁶次 |

#### 3.5.3 应用
<!-- 来源: 第4卷 第21篇液压传动 第3页 -->

| 应用 | 说明 |
|------|------|
| 顺序控制 | 压力到达信号→下一动作 |
| 安全保护 | 过压或欠压报警 |

## 4 流量控制阀
<!-- 来源: 第4卷 第22篇液压控制 第53页 -->

### 4.1 流量控制基本原理
<!-- 来源: 第4卷 第21篇液压传动 第3页 -->

流量控制阀通过改变节流口的通流面积来控制通过阀口的流量。

**节流孔口流量公式（薄壁孔口）：**

\[
Q = C_d A \sqrt{\frac{2\Delta p}{\rho}}
\]

式中：
- \( C_d \) — 流量系数（0.6~0.82，锐边孔取0.62~0.65）
- \( A \) — 节流口通流面积（m²）
- \( \Delta p \) — 节流口前后压差（Pa）
- \( \rho \) — 油液密度（kg/m³）

**长孔（细长孔）节流特性：**

\[
Q = \frac{\pi d^4 \Delta p}{128 \mu L}
\]

式中 \( \mu \) 为动力粘度，\( L \) 为孔长。

**关键结论：** 薄壁孔流量受粘度影响小（仅与密度有关），而长孔节流流量受粘度影响大。

### 4.2 节流阀
<!-- 来源: 第4卷 第21篇液压传动 第254页 -->

#### 4.2.1 节流口形式
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 形式 | 结构 | 流量特性 | 特点 |
|------|------|---------|------|
| 针阀式 | 锥形阀芯与阀座 | \( Q \propto x \)（线性） | 调节方便，小流量不稳 |
| 偏心槽式 | 阀芯上开偏心槽 | \( Q \propto x^{0.5} \) | 调节范围大 |
| 轴向三角槽式 | 阀芯轴向三角槽 | \( Q \propto x^{1.5} \) | 最常用，流量范围大 |
| 周向缝隙式 | 阀芯旋转改变缝隙 | \( Q \propto x^{1} \) | 不易堵塞 |
| 薄壁孔式 | 孔板 | \( Q \propto \sqrt{\Delta p} \) | 粘度影响小 |

#### 4.2.2 节流阀的流量特性
<!-- 来源: 第4卷 第21篇液压传动 第70页 -->

**流量-开度特性：**

节流阀在不同压差下流量与开度关系。定义**节流阀的调节比**：

\[
R = \frac{Q_{max}}{Q_{min}} \quad \text{（一般可达50:1~100:1）}
\]

**流量-压差特性：**

\[
\frac{Q_2}{Q_1} = \sqrt{\frac{\Delta p_2}{\Delta p_1}}
\]

**节流阀的刚度：**

\[
T = \frac{d(\Delta p)}{dQ} = \frac{2\Delta p}{Q} \quad \text{（薄壁孔）}
\]

刚度越大，流量受负载变化的影响越小。

### 4.3 调速阀
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

#### 4.3.1 调速阀（带压力补偿的节流阀）
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

调速阀由节流阀和**定差减压阀**串联组成。减压阀自动调节使节流阀前后压差保持恒定，从而使通过节流阀的流量不受负载变化的影响。

**工作原理：**

减压阀阀芯受节流阀前后压力 \( p_1 \) 和 \( p_2 \) 的作用：

当负载压力 \( p_3 \) 升高 → \( p_2 \) 升高 → 减压阀芯上移（左侧力大于右侧）→ 减压阀口开大 → \( p_1 \) 升高 → \( p_2 \) 不变。

当负载压力下降时过程相反。

**最终效果：** 节流阀两端压差 \( \Delta p_j = p_1 - p_2 = \text{常数} \)，因此：

\[
Q_j = C_d A_j \sqrt{\frac{2\Delta p_j}{\rho}} = \text{常数}
\]

#### 4.3.2 调速阀与节流阀的对比
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 特性 | 节流阀 | 调速阀 |
|------|--------|--------|
| 流量稳定性 | 受负载压力影响大 | 基本不受负载影响（≤±5%） |
| 最小稳定流量 | 相对较大 | 可小至20~50 mL/min |
| 工作压差 | 任何压差 | 需最小工作压差（0.5~1.0MPa） |
| 压力损失 | 小 | 大（增加定差减压阀压降） |
| 价格 | 低 | 中高 |
| 应用 | 速度精度要求不高的场合 | 速度稳定性要求高的场合 |

#### 4.3.3 温度补偿调速阀
<!-- 来源: 第4卷 第22篇液压控制 第437页 -->

当油温变化导致粘度变化时，节流特性会受影响。温度补偿调速阀在节流阀中加入了**温度补偿杆**（使用热膨胀系数不同的材料），自动调节开度以补偿粘度变化的影响。

**流量稳定性：** 温度从20°C变化到60°C时，普通调速阀流量变化约10%~20%，温度补偿型可控制在±（2%~5%）以内。

### 4.4 分流-集流阀（同步阀）
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

#### 4.4.1 分流阀
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

分流阀用于将一股油液按比例分成两股或多股，实现多个执行元件的同步运动。

**工作原理：**

利用两个相同的节流孔，通过压力反馈自动调节，使两出口流量保持相等（或特定比例）。

#### 4.4.2 集流阀
<!-- 来源: 第4卷 第21篇液压传动 第469页 -->

集流阀将两股不同压力的回油按比例汇集为一股。分流-集流阀兼具分流和集流功能。

**同步精度：**

| 类型 | 同步精度 | 适用工况 |
|------|---------|---------|
| 机械式分流阀 | 1%~3% | 一般同步 |
| 自调式分流阀 | 2%~5% | 中精度同步 |
| 分流阀组（串联） | 0.5%~2% | 多缸高精度同步 |

### 4.5 流量控制回路
<!-- 来源: 第4卷 第22篇液压控制 第208页 -->

**节流调速回路分类：**

| 回路类型 | 特点 | 效率 | 速度刚度 |
|---------|------|------|---------|
| 进油节流 | 阀在进油腔，减活塞杆推力 | 低（30%~50%） | 较好（有杆腔回油背压） |
| 回油节流 | 阀在回油腔，产生背压 | 低（30%~50%） | 好（有背压稳定运动） |
| 旁路节流 | 阀与缸并联，多余油回油 | 中（50%~70%） | 差（负载影响大） |

**容积调速回路：**

通过改变泵的排量（变量泵）或马达的排量来调节速度，效率高但成本高。

**容积-节流复合调速（LF/TLFA系统）：**

结合变量泵和调速阀的优点，效率介于两者之间，速度稳定性好。

| 滞环 | ≤3% | ≤0.5% |
| 重复精度 | ≤1% | ≤0.1% |
| 响应时间 | 30~80 ms | 20~50 ms |

#### 5.3.1 伺服阀分类
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 类型 | 结构 | 特点 | 应用 |
|------|------|------|------|
| 单级伺服阀 | 力矩马达直接驱动滑阀 | 流量小（≤4L/min），响应快 | 先导级 |
| 双级伺服阀（喷嘴挡板式） | 力矩马达→喷嘴挡板→主滑阀 | 最常用，性能稳定 | 通用伺服系统 |
| 双级伺服阀（射流管式） | 力矩马达→射流管→接收器→主滑阀 | 抗污染能力强于喷嘴挡板 | 工业恶劣环境 |

#### 5.3.2 喷嘴挡板式双级伺服阀
<!-- 来源: 第4卷 第22篇液压控制 第267页 -->

**结构组成：**
1. **力矩马达**：永久磁铁、衔铁、控制线圈—将电信号转换为力矩
2. **喷嘴-挡板**：挡板在两个喷嘴之间摆动，形成压力差推动先导级
3. **主滑阀**：先导压差推动阀芯移动，控制主油口开口
4. **反馈**：主阀芯位移通过反馈弹簧反馈到力矩马达

**工作过程：**
- 输入电流为零时，挡板居中，两喷嘴背后压力相等，主阀芯处于中位
- 输入正向电流时力矩马达使挡板偏向一侧，产生压差推动主阀芯移动
- 主阀芯移动带动反馈弹簧产生反馈力矩，与电磁力矩平衡

**关键性能参数：**

| 参数 | 典型值 |
|------|--------|
| 额定流量（Δp=7MPa时） | 4~200 L/min |
| 额定电流 | 10~300 mA |
| 线圈电阻 | 80~1000 Ω |
| 频率响应（-3dB） | 80~200 Hz |
| 滞环 | ≤3%~0.5%（可±2%~±5%重叠） |
| 分辨率 | ≤0.1%~0.5% |
| 零漂 | ≤1%~2%（温度变化40°C时） |
| 工作压力 | 3.5~31.5 MPa |
| 油液清洁度 | ISO 4406 15/13/10（NAS 4~6级） |

#### 5.3.3 伺服阀的选用
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 考虑因素 | 说明 |
|---------|------|
| **流量匹配** | 额定流量应能满足执行元件最大速度需求，通常取1.1~1.3倍最大需求流量 |
| **压力匹配** | 伺服阀额定压降一般为7MPa（部分为3.5MPa） |
| **响应速度** | 伺服系统的频宽应大于系统指令频率的3~5倍 |
| **零位特性** | 滞环、分辨率、零漂等参数影响系统静态精度 |
| **环境条件** | 温度、振动、污染度直接影响伺服阀寿命 |

## 6 基本液压回路
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

### 6.1 压力控制回路
<!-- 来源: 第4卷 第22篇液压控制 第194页 -->

#### 6.1.1 调压回路
<!-- 来源: 第4卷 第21篇液压传动 第59页 -->

| 回路类型 | 原理 | 应用 |
|---------|------|------|
| 单级调压 | 泵出口并联一个溢流阀 | 单压力系统 |
| 多级调压 | 多个溢流阀通过换向阀切换 | 多压力工作循环 |
| 远程调压 | 先导式溢流阀+远程调压阀 | 远程控制 |
| 比例调压 | 比例溢流阀实现无级调压 | 连续变化压力 |
| 恒压回路 | 压力补偿变量泵 | 恒压大流量系统 |

**多级调压回路设计：**

使用电磁换向阀（两位四通或三位四通）选择不同的远程调压阀接入先导式溢流阀的遥控口，可获得不同的系统压力。

例如：低压=2MPa（合模），高压=14MPa（注射），各压力由对应的远程调压阀设定。

#### 6.1.2 卸荷回路
<!-- 来源: 第4卷 第21篇液压传动 第792页 -->

**目的：** 执行元件不工作时使泵在接近零压力下运转，节能降噪。

| 卸荷方式 | 回路 | 特点 |
|---------|------|------|
| 换向阀卸荷 | M/H/K型中位机能换向阀 | 简单，成本低 |
| 溢流阀遥控卸荷 | 先导溢流阀+电磁换向阀 | 适用于大流量 |
| 压力补偿变量泵 | 变量泵自动减小排量 | 理论上无溢流损耗 |
| 二位二通阀卸荷 | 卸荷阀直接通油箱 | 卸荷彻底 |

#### 6.1.3 背压回路
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

在液压缸回油腔保持一定压力（背压），使运动平稳，防止冲击和爬行。

| 背压元件 | 特点 | 背压范围 |
|---------|------|---------|
| 溢流阀（作背压阀） | 压力稳定可调 | 0.5~3 MPa |
| 节流阀 | 简单，精度低 | 不可调或手动调节 |
| 顺序阀 | 卸载时无背压 | 行程终端卸荷 |
| 平衡阀 | 可承受负负载 | 适用于垂直缸 |

#### 6.1.4 保压回路
<!-- 来源: 第4卷 第21篇液压传动 第63页 -->

| 保压方式 | 原理 | 保压精度 | 适用性 |
|---------|------|---------|--------|
| 液控单向阀锁紧 | 液压锁封油 | 好（无泄漏阀） | 短时保压 |
| 蓄能器保压 | 蓄能器补充泄漏 | 好 | 系统漏油补偿 |
| 变量泵补偿 | 泵按需供油 | 好 | 恒压系统 |
| 保压补油泵 | 小流量高压泵 | 好 | 长时间保压 |

#### 6.1.5 减压回路
<!-- 来源: 第4卷 第21篇液压传动 第61页 -->

在夹紧、润滑、控制油路等支路中，将主系统压力降低到所需值：

- 单级减压：一个减压阀，调定单一出口压力
- 多级减压：多个减压阀+换向阀切换
- 比例减压：比例减压阀连续可调

**注意：** 减压阀出口与主油路之间有至少0.5~1.0 MPa的压差才能正常工作。

#### 6.1.6 平衡回路
<!-- 来源: 第4卷 第21篇液压传动 第69页 -->

垂直安装或负重较大的液压缸，需在回油侧建立背压以平衡负载重量，防止下滑。

**平衡回路方案：**

| 方案 | 元件 | 特点 |
|------|------|------|
| 单向顺序阀（平衡阀） | 顺序阀+单向阀 | 最常用，内控外泄 |
| 液控顺序阀 | 外控型 | 控制压力可来自上腔 |
| 自锁型液压锁 | 双液控单向阀 | 密封好，但负载有冲击 |
| 比例平衡阀 | 比例溢流阀+单向阀 | 平衡压力可调 |

### 6.2 速度控制回路
<!-- 来源: 第4卷 第22篇液压控制 第231页 -->

#### 6.2.1 节流调速回路
<!-- 来源: 第4卷 第21篇液压传动 第397页 -->

**进油节流调速：**

节流阀装在液压缸进油路上，定量泵+溢流阀。

特性：\( v = \frac{Q_j}{A_1} \)，\( Q_j = C_d A_j \sqrt{\frac{2\Delta p}{\rho}} \)

速度刚度：\( T = \frac{\partial F}{\partial v} \)，进油刚度中等。

**回油节流调速：**

节流阀装在液压缸回油路上，定量泵+溢流阀。

速度刚度优于进油节流（回油腔有背压，运动平稳）。

**旁路节流调速：**

节流阀与液压缸并联。泵的大部分流量供给油缸，多余部分通过节流阀回油箱。

效率高于进油/回油节流，但速度刚度差。

#### 6.2.2 容积调速回路
<!-- 来源: 第4卷 第21篇液压传动 第97页 -->

| 回路类型 | 组成 | 效率 | 速度范围 | 应用 |
|---------|------|------|---------|------|
| 变量泵-定量马达(缸) | 变量泵+安全阀+执行元件 | 高 | 大 | 工程机械 |
| 定量泵-变量马达 | 定量泵+变量马达+换向阀 | 高 | 中等 | 车辆行走 |
| 变量泵-变量马达 | 全变量系统 | 最高 | 极大 | 大功率场合 |

#### 6.2.3 容积-节流复合调速
<!-- 来源: 第4卷 第21篇液压传动 第70页 -->

负载敏感（Load Sensing, LS）系统：

变量泵+负载敏感阀+压力补偿阀，使泵的输出流量和压力与负载需求相匹配，既实现速度调节又保证高效节能。

**负载敏感系统的效率优势：**

效率 = \( \frac{p_L Q_L}{p_p Q_p} \approx \frac{p_L}{p_L + \Delta p_{LS}} \)

其中 \( \Delta p_{LS} \) 为LS压差（通常1.5~2.5 MPa），远小于节流系统的溢流损失。

**LUDV（负载独立流量分配）系统：**

多个执行元件同时工作时，LUDV系统按各执行元件的开口面积比例分配流量，不受负载差异影响，实现抗饱和流量分配。

### 6.4 多缸动作回路
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

#### 6.4.1 顺序动作回路
<!-- 来源: 第4卷 第21篇液压传动 第93页 -->

**行程控制顺序动作：**

利用行程阀（或行程开关+电磁阀）检测执行元件的位置，当到位后触发下一动作。

典型：缸A伸出到位 → 触发行程阀 → 缸B伸出 → ...

**压力控制顺序动作：**

利用顺序阀检测缸A的进油压力，当缸A到位（压力升高）时，顺序阀开启，压力油进入缸B。

典型两个动作顺序：
1. 缸A伸出 → 到位后压力升高 → 顺序阀打开 → 缸B伸出
2. 缸B缩回 → 到位后压力升高 → 顺序阀打开 → 缸A缩回

#### 6.4.2 同步回路
<!-- 来源: 第4卷 第21篇液压传动 第787页 -->

| 同步方式 | 同步精度 | 成本 | 说明 |
|---------|---------|------|------|
| 机械刚性连接 | — | 高 | 最简单但易卡死 |
| 分流-集流阀 | 1%~5% | 中 | 最常用 |
| 串联缸 | — | 低 | 受泄漏影响 |
| 同步马达（泵） | 1%~3% | 高 | 两个同参数马达 |
| 比例阀闭环控制 | 0.1%~0.5% | 很高 | 高精度同步 |
| 伺服阀+光栅闭环 | 0.01%~0.1% | 极高 | 超高精度 |

#### 6.4.3 防干扰回路
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

多个执行元件共用一个油源时，需防止一个缸的动作影响另一个缸。

| 方案 | 原理 | 适用 |
|------|------|------|
| 进油路+调速阀 | 每个支路独立调速阀 | 一般精度 |
| 减压阀隔离 | 减压阀保持各支路压力独立 | 压力敏感场合 |
| 单向阀 | 防止压力油反流 | 简单隔离 |
| 分流阀 | 各支路独立供油 | 多缸同步 |
| 独立供油 | 各执行元件自带泵 | 高要求系统 |

### 7.1 YT4543型组合机床动力滑台液压系统
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

#### 7.1.1 工作循环
<!-- 来源: 第4卷 第22篇液压控制 第148页 -->

```
快进 → 一工进（慢速）→ 二工进（更慢速）→ 死挡铁停留 → 快退 → 停止
```

**工况参数：**

| 参数 | 快进 | 一工进 | 二工进 | 快退 |
|------|------|--------|--------|------|
| 速度(m/min) | 4.5 | 0.1 | 0.05 | 4.5 |
| 推力(kN) | 2 | 18 | 18 | 1.5 |
| 时间(s) | 1.3 | 37 | 20 | 1.3 |

#### 7.1.2 系统特点
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 特点 | 实现方式 |
|------|---------|
| 差动快进 | 三位五通电磁阀+行程阀配合，使缸两腔通压 |
| 一工进 | 回油路上的调速阀（电磁阀切换） |
| 二工进 | 另一个调速阀（串联方式，速度更慢） |
| 死挡铁停留 | 缸碰上死挡铁停止，压力升高触发压力继电器 |
| 快退 | 压力继电器信号→电磁阀换向 |
| 停止 | 行程开关发信→电磁阀回中位 |

<!-- 来源: 第4卷 第22篇液压控制 第148页 -->

<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

### 7.5 工业液压系统设计共性问题
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 问题 | 典型案例 | 解决措施 |
|------|---------|---------|
| 液压冲击 | 换向阀快速切换，高压管路 | 换向阀加阻尼、蓄能器吸收、先导卸压 |
| 气穴（空穴） | 泵吸油不足、回油无背压 | 加大吸油管、降低安装高度、加背压阀 |
| 爬行 | 滑动面润滑不良、空气混入、摩擦力变化 | 排气、加背压、改善导轨润滑 |
| 温升过高 | 系统匹配不合理、回路压差过大 | 提高效率、合理匹配、冷却器 |
| 噪音异常 | 泵吸空、阀共振、管路振动 | 检查气蚀、调整阀参数、加固管路 |
| 油液污染 | 颗粒、水分、空气、化学污染 | 精细过滤、定期换油、清洁度控制 |
| 泄漏 | 密封老化、装配不良、压力波动 | 选用合适密封、控制安装质量 |

| 比例压力阀 | 压力 | 比例电磁铁电流 | 连续调压 | 中 |
| 伺服阀 | 方向+流量 | 力矩马达电流 | 精密控制 | 高 |
| 插装阀 | 方向/压力/流量 | 先导控制 | 大流量控制 | 取决于先导级 |
