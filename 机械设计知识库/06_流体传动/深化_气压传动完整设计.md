## 1. 气压传动特点
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

### 1.1 气压与液压传动对比总表
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 对比项目 | 气压传动 | 液压传动 | 说明 |
|:---|:---|:---|:---|
| 工作压力 | **0.4 ~ 0.8 MPa**（常用0.5~0.63） | **7 ~ 21 MPa**（常用10~16） | 气压为低压系统 |
| 工作介质 | 压缩空气（可直接排放） | 矿物液压油（需回收） | 气压介质成本极低 |
| 运动速度 | **0.1 ~ 2 m/s**（高速可达10 m/s） | 0.01 ~ 0.5 m/s | 气压速度更快 |
| 速度稳定性 | 差（空气可压缩，受负载变化影响大） | 优（液压油不可压缩性） | 气压需要锁紧/平衡回路 |
| 定位精度 | ±0.5 ~ ±5 mm（开环） | ±0.01 ~ ±0.5 mm | 液压精定位明显更优 |
| 输出力密度 | 小（同直径气缸出力仅为液压缸1/20~1/50） | 大 | 液压在大力矩场合不可替代 |
| 清洁度需求 | 高（精密阀芯间隙5~10 μm，受水分/杂质影响大） | 中高（过滤精度10~25 μm） | 气压对水分极其敏感 |
| 配管要求 | 管径大、沿程损失大 | 管径小、可高压长距离 | 气压管路需注意压降 |
| 安全性 | 高（无触电/起火风险，过载自动停止） | 中（高压油泄漏、高温） | 气压在易燃易爆环境优势明显 |
| 成本 | 低（空压机便宜，阀便宜，维护成本低） | 高（泵站/阀/管路贵） | 气压系统整体成本约液压的30~50% |
| 噪音 | 大（空压机+排气噪音80~95 dB(A)） | 小（泵站60~75 dB(A)） | 气压需配置消音器 |
| 适用环境 | -20°C ~ +80°C（需考虑水分结冰） | -10°C ~ +70°C（油品粘度变化） | 气压低温需除水 |
| 响应时间 | 中（压缩空气波速约340 m/s） | 高（液压波速约1200 m/s） | 气压远距离控制有滞后 |

### 1.2 气压传动的核心优势
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 优势项 | 说明 | 典型应用场景 |
|:---|:---|:---|
| 介质易得 | 空气取之不尽，直接排放无需回收管路 | 食品加工、医药、电子 |
| 无污染 | 即使泄漏也不污染产品和环境 | 无尘车间、食品包装 |
| 安全防爆 | 无火花、无高温、无电击风险 | 喷涂、化工、煤矿 |
| 过载保护 | 气缸堵住自动停止，不会烧毁 | 夹紧、定位工装 |
| 动作快速 | 不需回油管路，换向速度快 | 包装机、自动化产线 |
| 系统简单 | 单管供气+电磁阀+气缸，结构简洁 | 拾取、分拣 |

### 1.3 气压传动的局限性
<!-- 来源: 第4卷 第22篇液压控制 第124页 -->

| 局限项 | 说明 | 对策 |
|:---|:---|:---|
| 可压缩性 | 速度/位置精度受负载波动影响大 | 加液压缓冲器/气液联动缸/伺服比例阀 |
| 输出力小 | 0.5 MPa下Φ100 mm气缸推力≈3.9 kN | 大出力优先选用液压 |
| 水分结冰 | 压缩空气含饱和水蒸气，冬季易冰堵 | 配置冷冻干燥机+后冷却器 |
| 排气噪音 | 排气冲击噪音可达100 dB(A) | 加装消音器/集气管集中排放 |
| 效率低 | 空压机能量转换效率仅10~20% | 变频空压机+管路检漏 |
| 润滑问题 | 部分元件需油雾气润滑 | 油雾器集中供油 |

### 2.1 气源系统组成框图
<!-- 来源: 第4卷 第22篇液压控制 第24页 -->

```
大气 → 空气过滤器 → 空压机 → 后冷却器 → 储气罐 →

```

### 2.2 空压机选型对比
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 类型 | 工作压力范围 | 排气量范围 | 效率 | 噪音 | 维护周期 | 推荐应用 |
|:---|:---|:---|---:|:---|:---|---:|
| **活塞式** | 0.3~1.0 MPa | 0.1~20 m³/min | 65~75% | 大(90+ dB) | 500 h换油 | 小型/移动供气 |
| **螺杆式** | 0.5~1.3 MPa | 1~100 m³/min | 80~88% | 中(75~85 dB) | 2000 h换油 | 工厂集中供气**首选** |
| **离心式** | 0.3~1.0 MPa | 20~5000 m³/min | 80~85% | 小(<75 dB) | 5000 h保养 | 大型厂矿/特大气量 |

#### 活塞式空压机选型参数
<!-- 来源: 第5卷 第23篇气压传动 第32页 -->

| 参数项 | 典型值范围 | 选型要点 |
|:---|:---|:---|
| 气缸数 | 1~4缸 | 多缸脉动小 |
| 转速 | 500~1500 r/min | 转速低则寿命长 |
| 比功率 | 6~9 kW/(m³·min⁻¹) | 越低越节能 |
| 排气温度 | 150~200°C | 需后冷却器降至<40°C |
| 润滑油消耗 | 10~50 mg/m³ | 高精油耗影响空气质量 |
| 启动方式 | 直接/星三角/变频 | 大型需降压启动 |

#### 螺杆式空压机选型参数（工厂首选）
<!-- 来源: 第5卷 第23篇气压传动 第32页 -->

| 参数项 | 典型值范围 | 选型要点 |
|:---|:---|:---|
| 排气压力 | 0.7 / 1.0 / 1.3 MPa 三级 | 系统用气压力≤额定×0.85 |
| 排气量 | 1.2~40 m³/min（常用） | 按总耗气量×1.1~1.3余量 |
| 电机功率 | 7.5~250 kW | 约对应0.12~0.15 kW/(m³·h⁻¹) |
| 比功率 | 5.5~7.5 kW/(m³·min⁻¹) | 一级能效≤6.0 |
| 冷却方式 | 风冷/水冷 | 风冷简单，水冷效率高 |
| 变频/VSD | ±1%稳压，节能20~35% | 用气波动大时必选 |
| 油气分离 | 残油量≤3 mg/m³ | 无油机可达0.01 mg/m³ |

> **选型口诀**：压力看系统最高×1.2，气量看总用气×1.15，螺杆优先小厂也适用，功率对应约0.13 kW/(m³·h)。

### 2.3 储气罐计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

储气罐作用：稳压、缓存、冷却、除水。

#### 容积计算公式
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

\[
V = Q \times t
\]

| 参数 | 含义 | 推荐取值 |
|:---|:---|:---|
| V | 储气罐容积（m³） | — |
| Q | 空压机排气量（m³/min） | 系统最大用气量×1.1~1.3 |
| t | 缓存时间（min） | **1.5 ~ 3 min**（手册推荐） |
| P_max | 罐体最高工作压力 | ≥空压机额定压力 |

#### 推荐容积速查表
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 空压机排气量 Q (m³/min) | 储气罐容积 V (m³) | 推荐直径×高度 (mm) |
|:---:|:---:|:---|
| 1 | 1.5 ~ 3.0 | Φ600×1800 |
| 2 | 3.0 ~ 6.0 | Φ800×2000 |
| 3 | 5.0 ~ 9.0 | Φ1000×2200 |
| 6 | 9.0 ~ 18.0 | Φ1200×2500 |
| 10 | 15.0 ~ 30.0 | Φ1400×3000 |
| 20 | 30.0 ~ 60.0 | Φ1600×3600 |
| 40 | 60.0 ~ 120.0 | Φ2000×4500 |

> **实用经验**：小系统取上限（3 min），大系统可适度降低比例但绝对值不宜小于0.5 m³。储气罐必须配安全阀（整定压力≤1.1倍工作压力）+ 排污阀（每天排水）。

### 2.4 过滤器选型
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 过滤级别 | 过滤精度 | 残留油分 | 应用场合 |
|:---|:---:|---:|:---|
| **主管线过滤器** | 5~40 μm | 5 mg/m³ | 一般气动工具、气缸 |
| **精密过滤器** | 1~5 μm | 1 mg/m³ | 气动阀、仪表气源 |
| **超精密过滤器** | 0.01~1 μm | 0.1 mg/m³ | 精密气动、涂装 |
| **活性炭过滤器** | 0.01 μm | 0.003 mg/m³ | 无油要求（食品/医药） |

#### 过滤器选型要点
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 要点 | 说明 |
|:---|:---|
| 流量匹配 | 过滤器额定流量 ≥ 管路最大流量×1.5 |
| 排水方式 | 自动排水器（浮球式/电动式）优先，手动排水仅用于备用 |
| 安装位置 | 储气罐出口→主管线过滤器→干燥器→精密过滤器 |
| 压降控制 | 清洁状态下初压降≤0.02 MPa，更换阈值≤0.07 MPa |
| 壳体材质 | 铝合金（常用）/ 不锈钢（食品/腐蚀环境）/ 塑料（轻型） |

### 2.5 干燥器选型对比
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 比较项目 | 冷冻式干燥器 | 吸附式干燥器 |
|:---|:---|:---|
| 工作原理 | 压缩空气冷却至2~10°C，析出液态水 | 采用分子筛/活性氧化铝吸附水蒸气 |
| 压力露点 | **2 ~ 10°C** | **-20 ~ -70°C** |
| 除水效果 | 去除约85~90%水分 | 去除99%以上水分 |
| 适用场合 | 一般工厂（室内、无低温） | 户外/低温环境/精密仪器 |
| 能耗 | 低（0.2~0.5 kW·h/m³） | 高（再生需10~15%气耗或电加热） |
| 设备成本 | 低（约吸附式的1/3~1/2） | 高 |
| 维护 | 简单（每年清洗冷凝器） | 较复杂（定期更换吸附剂） |
| 冬季适用 | ❌ 露点>0°C，户外管路仍可能结冰 | ✅ 露点-40°C以下，冬季安全 |
| 压缩空气损失 | <1% | 10~18%（再生用气） |

### 2.6 油雾器选型
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 参数 | 典型值 | 说明 |
|:---|:---|:---|
| 供油量调节范围 | 0 ~ 200 滴/min | 一般取10~30滴/min |
| 储油杯容积 | 0.1 ~ 0.5 L | 需满足8 h连续供油 |
| 适用管径 | G1/8" ~ G1" | 按气管内径选型 |
| 供油压力范围 | 0.1 ~ 1.0 MPa | 超出范围雾化不良 |
| 使用油品 | ISO VG32（推荐） | 特殊要求可用食品级VG32 |
| 安装距离 | 距用气点 ≤ 5 m | 过远油滴会沉降在管路中 |
| 压降 | <0.015 MPa（满流量下） | 压降过大需加粗管径 |

> ⚠ **重要**：油雾器必须在**过滤器-减压阀之后**安装。含油压缩空气不宜用于涂装、精密气浮轴承等场合。

### 2.7 气源处理三联件（FRL）
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 组件 | 功能 | 标准配置 |
|:---|:---|:---|
| F — 过滤器 **Filter** | 过滤固体杂质+分离液态水 | 5~40 μm滤芯 + 手动/自动排水 |
| R — 减压阀 **Regulator** | 稳压至所需压力（0.05~0.85 MPa） | 带压力表 |
| L — 油雾器 **Lubricator** | 为气动元件提供油雾润滑 | 带流量调节针阀 |

> **安装顺序**：过滤器 → 减压阀 → 油雾器（不可颠倒）
> **选型口诀**：管径对管径、流量对流量、压力范围匹配即可

## 3. 气缸分类与选型参数

### 3.1 气缸分类总表
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 类别 | 子类型 | 特点 | 标准/系列 | 典型应用 |
|:---|:---|:---|:---|:---|
| **单作用** | 弹簧复位/弹簧伸出 | 一通气口，靠弹簧力复位 | ISO 6431/ISO 6432 | 夹紧、定位、制动 |

| **摆动气缸** | 叶片式/齿轮齿条式 | 输出旋转运动（90°/180°/270°） | ISO 15407 | 转臂/翻转/阀门开关 |
| **气爪** | 平行/摆动/三点式 | 输出夹持力 | ISO 15552 | 机器人末端/拾取 |
| **薄型气缸** | 紧凑型 | 轴向短、轻量化 | ISO 21287 | 安装空间小 |
| **双杆气缸** | 双活塞杆 | 抗扭转、高导向精度 | 厂商系列 | 防转应用 |
| **滑台气缸** | 带直线导轨 | 高刚性、高导向精度 | 厂商系列 | 精密搬运 |

### 3.2 单作用气缸 vs 双作用气缸
<!-- 来源: 第5卷 第23篇气压传动 第84页 -->

| 比较项 | 单作用（弹簧复位） | 双作用 |
|:---|:---|:---|
| 气口数 | 1个 | 2个 |
| 推力计算 | F = 0.25πD²·p - F_弹簧 | F = 0.25πD²·p·η |
| 复位力 | 靠弹簧，不可调 | 靠反向进气，可调节 |
| 弹簧占行程 | 弹簧缩回时占压缩长度，有效行程减少 | 无弹簧，全行程可用 |
| 耗气量 | 约为同缸径双作用的50% | 标准耗气 |
| 行程限制 | 一般≤300 mm（弹簧长度限制） | 可达2000 mm |
| 速度控制 | 慢（受弹簧力阻碍） | 快，双向可调节 |
| 断电断气 | 自动复位到初始位置 ✅ | 停在当前位置（需加锁紧阀） |
| 典型用途 | 安全夹紧、制动、定位 | 通用自动化搬运 |

### 3.3 缓冲气缸参数
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 缓冲方式 | 结构特点 | 缓冲能力 | 适用速度范围 | 调节方式 |
|:---|:---|:---:|:---|:---|
| 不可调缓冲垫 | 末端橡胶/聚氨酯垫 | 小（约吸收10~20%动能） | 0.1~0.5 m/s | 不可调 |
| 可调缓冲（气垫式） | 柱塞+节流孔 | 中（约吸收40~60%动能） | 0.2~1.5 m/s | 调节针阀 |
| 可调缓冲（浮动式） | 浮动衬套+密封圈 | 中高（约吸收50~70%动能） | 0.3~2.0 m/s | 调节缓冲调节阀 |
| 外置液压缓冲器 | 外加液压缓冲器 | 大（吸收80~95%动能） | 0.5~5.0 m/s | 选型缓冲器规格 |

#### 缓冲吸收力计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

对于可调缓冲气缸，缓冲末期吸收的能量为：

\[
E_{缓冲} = \frac{1}{2} m v^2 - p_{背压} \cdot A_{缓冲} \cdot S_{缓冲}
\]

| 参数 | 含义 |
|:---|---:|
| m | 运动件总质量（kg） |
| v | 撞击缓冲前速度（m/s） |
| p_背压 | 缓冲腔背压（Pa） |
| A_缓冲 | 缓冲柱塞截面积（m²） |
| S_缓冲 | 缓冲行程（m） |

> **实用经验**：当负载动能 > 气缸额定吸收能量的80%，必须加装外置液压缓冲器。

### 3.4 无杆气缸对比

## 4. 气缸输出力计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

### 4.1 基本计算公式
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 项目 | 公式 | 符号说明 |
|:---|:---|---:|
| **推力（无杆腔）** | \(F_1 = \dfrac{\pi}{4} D^2 \cdot p \cdot \eta\) | D — 缸径 (m)；p — 工作压力 (Pa)；η — 效率 |
| **拉力（有杆腔）** | \(F_2 = \dfrac{\pi}{4} (D^2 - d^2) \cdot p \cdot \eta\) | d — 活塞杆径 (m) |
| **理论推力** | \(F_{th} = A \cdot p\) | A — 活塞面积 (m²)；忽略效率 |
| **实际输出力** | \(F_{act} = F_{th} \cdot \eta\) | η 包含密封摩擦 + 背压损失 |

### 4.2 效率系数 η 取值
<!-- 来源: 第4卷 第21篇液压传动 第47页 -->

| 缸径范围 | 气缸状态 | 效率 η | 说明 |
|:---:|:---|:---:|:---|
| ≤40 mm | 新气缸/无负载预紧 | 0.80~0.85 | 小缸径密封圈摩擦占比大 |
| ≤40 mm | 使用中/正常负载 | 0.70~0.80 | — |
| 50~100 mm | 新气缸 | 0.85~0.92 | 常用缸径，效率较高 |
| 50~100 mm | 使用中 | 0.78~0.88 | — |
| ≥125 mm | 新气缸 | 0.88~0.95 | 大缸径摩擦占比小 |
| ≥125 mm | 使用中 | 0.82~0.90 | — |

> **实用取值**：一般设计计算时取 **η = 0.80~0.85** 即可。对于精密低速运动（<50 mm/s），由于静摩擦影响大，η 需取低值 0.60~0.70。

### 4.4 气缸负载率计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

负载率 = 实际负载力 / 理论输出力

| 运动状态 | 推荐负载率 | 说明 |
|:---|:---:|:---|
| 静止夹紧（无运动） | **≤ 85%** | 仅需克服弹簧力+密封摩擦 |
| 水平运动（正常速度） | **≤ 50~70%** | 需克服惯性+摩擦力 |
| 水平运动（高速 >1 m/s） | **≤ 40~50%** | 需预留加速力空间 |
| 垂直提升（正方向） | **≤ 30~40%** | 重力+加速同时要求出力 |
| 垂直下降（负方向） | **≤ 50~60%** | 背压平衡+受控下降 |

#### 选缸实用示例
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

> **工况**：水平输送，负载质量 m = 30 kg，摩擦系数 μ = 0.15，要求速度 v = 0.5 m/s，系统压力 p = 0.6 MPa。
>
> **计算**：
> - 需克服摩擦力 F_f = mg·μ = 30 × 9.81 × 0.15 = 44.1 N
> - 负载率取 60%，需理论推力 F_th = 44.1 / 0.6 = 73.5 N
> - 所需缸径 D = √(4×73.5 / (π×0.6×10⁶)) = 0.0125 m → 12.5 mm
> - **选择 Φ32 mm 气缸**（留有充足余量，兼顾缓冲能力）

### 4.5 缓冲吸收力详细计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

当运动件具有较大动能时，必须校核缓冲能力。

#### 缓冲评价步骤
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

```
① 计算总动能 Ek = 0.5 × m_total × v²
② 确定允许吸收能量 E_max（气缸样本给出）
③ 若 Ek > E_max → 需外置液压缓冲器或减速回路
```

<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 缸径 (mm) | 允许吸收动能 (J) (标准缓冲) | 允许吸收动能 (J) (加强缓冲) | 最大缓冲行程 (mm) |
|:---:|:---:|:---:|:---:|
| 32 | 2.5 | 5 | 15 |
| 40 | 4 | 8 | 17 |
| 50 | 8 | 16 | 20 |
| 63 | 14 | 28 | 22 |
| 80 | 25 | 50 | 25 |
| 100 | 40 | 80 | 28 |
| 125 | 65 | 130 | 32 |
| 160 | 100 | 200 | 36 |
| 200 | 160 | 320 | 40 |

#### 缓冲力计算示例
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

**工况**：Φ63 mm 气缸，m = 50 kg，v = 1.2 m/s，标准缓冲。

1. 动能 Ek = 0.5 × 50 × 1.2² = **36 J**
2. 标准缓冲允许 14 J → **严重超限（256%）**
3. 需加装外置液压缓冲器，或降低速度至 v_limit = √(2×14/50) = **0.75 m/s**

> **结论**：要么降速到 0.75 m/s 以下使用标准缓冲，要么强制加装外置缓冲器。

### 6.1 真空发生器
<!-- 来源: 第5卷 第23篇气压传动 第19页 -->

#### 工作原理与参数
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

真空发生器利用压缩空气通过喷嘴产生高速气流形成的文丘里效应制造真空。

\[
F_{吸力} = 0.1 \times A \times (n \times p)
\]

| 参数 | 含义 | 单位 |
|:---|---:|:---:|
| F_吸力 | 理论吸力 | N |
| A | 吸盘有效面积 | cm² |
| n | 真空度系数（通常0.6~0.9） | — |
| p | 供气压力 | MPa |

#### 真空发生器选型参数
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 型式 | 最大真空度 (kPa) | 最大吸入流量 (L/min) | 供气压力 (MPa) | 耗气量 (L/min) | 喷嘴直径 (mm) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| 微型 | -88 | 5~15 | 0.4~0.6 | 15~30 | 0.5 |
| 小型 | -88 | 20~40 | 0.4~0.6 | 30~60 | 0.7 |
| 中型 | -88 | 50~100 | 0.4~0.6 | 60~120 | 1.0 |
| 大型 | -88 | 120~300 | 0.4~0.6 | 120~300 | 1.5 |
| 特大型 | -85 | 350~800 | 0.5~0.7 | 350~900 | 2.0 |

#### 真空发生器选型原则
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 因素 | 选型要点 |
|:---|:---|
| 真空度要求 | 一般取-70~-88 kPa（标准型），高真空场合选多级型 |
| 吸入流量 | ≥ 吸盘容积×3 / 响应时间（s） |
| 响应时间 | 吸力建立时间 = 容积/吸入流量，一般要求 < 0.5 s |
| 耗气量 | 发生器耗气量约占系统供气的20~40%，需纳入空压机选型 |
| 节能 | 带**节能阀**的发生器（真空OK后自动切断供气），省气50~80% |
| 过滤 | 真空发生器进气端需装5 μm过滤器 |

<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

#### 吸盘选型计算流程
<!-- 来源: 第4卷 第21篇液压传动 第14页 -->

> **已知**：工件质量 m (kg)，工况（水平/垂直）、安全系数 s

**步骤 1 — 计算所需吸力**

| 工况 | 理论吸力 (N) | 说明 |
|:---|:---|---:|
| 水平吸附（垂直提升） | F = m × g × s | 安全系数 s = 2~4 |
| 垂直吸附（水平横移） | F = m × g × s / μ | μ为摩擦系数（0.1~0.8）；s = 3~5 |

**步骤 2 — 计算所需吸盘面积**

\[
A = \frac{F}{n \cdot p_{真空}}
\]

- n — 真空度系数（取0.7~0.85，对应真空度-70~-85 kPa）
- p_真空 — 真空度 (kPa)

**步骤 3 — 选型**

\[
D_{吸盘} = \sqrt{\frac{4A}{\pi}}
\]

向上取整为标准吸盘直径。

### 7.4 配管布局设计原则
<!-- 来源: 第4卷 第21篇液压传动 第3页 -->

| 原则 | 说明 |
|:---|:---|
| **主管路环状布局** | 主管路闭合成环状，任何一点供气可从两边获得 |
| **支管路从主管顶部引出** | 防止主管内凝结水流入支管路 |
| **管路向下倾斜 1~2%** | 顺气流方向倾斜，便于凝结水排至低处排水阀 |

| **软管弯曲半径 ≥ 6倍外径** | 防止弯扁导致通流面积骤减 |
| **支管阀门** | 每个支路起点安装球阀，便于隔离维修 |
| **管材选型** | 厂区主干用镀锌钢管（≤1.0 MPa），末端用尼龙管/聚氨酯管 |
| **管径统一** | 同一层级尽量使用同一规格，减少接头种类 |
| **管色标识** | 压缩空气管 — 蓝色/绿色，真空管 — 白色/灰色，润滑管 — 黄色 |

### 8.2 第1步：确定气缸类型
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

```
工况分析：
  - 水平夹紧 → 双作用气缸（方向可控）
  - 断电断气自动松开 → 采用弹簧复位单作用气缸（NC常闭）
  - 行程25 mm → 短行程，弹簧复位可行
  - 夹紧力需求200 N → 可选小缸径

结论：选用 ▸ 单作用弹簧复位气缸
```

### 8.3 第2步：缸径计算
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 步骤 | 计算 | 结果 |
|:---|---:|:---:|
| 所需夹紧力 | F_req = m × g × μ × s = 12 × 9.81 × 0.3 × 2.0 | 70.6 N |
| 理论力（夹紧方向为推力） | F_th = F_req / η = 70.6 / 0.70（单作用弹簧复位效率低） | 100.9 N |
| 所需活塞面积 | A = F_th / p = 100.9 / (0.6 × 10⁶) | 1.68 × 10⁻⁴ m² |
| 所需缸径 | D = √(4A/π) = √(4×1.68×10⁻⁴/π) | 0.0146 m ≈ 14.6 mm |
| **选型** | 向上取整标准缸径 | **▸ Φ20 mm 气缸** |

**校核弹簧力**：
- 单作用气缸弹簧复位力通常为理论推力的10~25%
- Φ20 mm @0.6 MPa 理论推力 ≈ 188 N
- 弹簧力估算 ≈ 25 N
- 实际有效推力 = 188 × 0.7 − 25 = **106.6 N > 70.6 N ✅**

| 气缸不动 | 无气压/阀未换向/缸卡死 | ①检查总气源→②阀手动测试→③缸拆检 | 按序逐级排查 |
| 动作缓慢 | 气压不足/节流太小/管路堵塞 | ①测压力→②调节流→③清理管路 | 调压或清堵 |
| 爬行抖动 | 润滑不足/密封圈磨损/背压不稳 | ①加油雾器→②检查密封→③换缓冲阀 | 低速尤其注意润滑 |
| 内泄漏（缸内窜气） | 活塞密封圈磨损 | 保压测试（充气后断电观察压降） | 更换密封圈 |
| 外泄漏（杆处漏气） | 防尘圈/密封圈磨损 | 杆上可见油渍 | 更换杆密封 |
| 阀不换向 | 电磁铁烧毁/阀芯卡住/信号丢失 | ①听电磁铁声音→②手动测试→③测电压 | 更换线圈或清洗阀芯 |
| 泵不建压 | 进气阀卡住/安全阀泄漏/皮带打滑 | ①检查进气滤芯→②查安全阀→③查皮带 | 清洗或更换 |
| 管路结冰 | 干燥器失效/后冷不足 | ①检查露点→②查冷干机→③排水阀 | 确认干燥器正常 |
| 真空度不足 | 喷嘴堵塞/吸盘破损/真空泄漏 | ①听发生器声音→②堵吸盘测真空→③换吸盘 | 清理喷嘴或更换 |

### 9.6 气压系统设计检核清单
<!-- 来源: 第4卷 第22篇液压控制 第95页 -->

| 检核项 | 检查内容 | 完成 | 备注 |
|:---|:---|---:|:---|
| □ 1 | 供气压力是否满足所有用气点需求 | ☐ | 最远点压降核算 |
| □ 2 | 空压机排气量是否覆盖峰值用气×1.15安全系数 | ☐ | 要纳入泄漏和后期扩容 |
| □ 3 | 储气罐容积是否≥空压机Q×1.5 min | ☐ | 小系统取2~3 min |
| □ 4 | 干燥器露点是否满足最低环境温度要求 | ☐ | 户外冬季需-20°C |
| □ 5 | 过滤精度是否覆盖阀/缸要求 | ☐ | 精密阀需≤5 μm |
| □ 6 | 管径是否按流速8~15 m/s校核 | ☐ | 长管路取下限 |
| □ 7 | 气缸实际出力是否≥所需负载力×安全系数 | ☐ | 水平s=1.5~2，垂直s=2.5~4 |
| □ 8 | 缓冲能力是否满足运动件动能要求 | ☐ | Ek≤额定值80% |
| □ 9 | 电磁阀有效截面积≥气缸/管路需求 | ☐ | 阀S≥管路S×0.8 |
| □ 10 | 真空吸盘吸力是否满足工件重量×安全系数 | ☐ | 水平s≥3，垂直s≥4 |
| □ 11 | 消音器/排放是否考虑了噪音要求 | ☐ | 一般要求≤85 dB(A) |
| □ 12 | 是否有紧急停机/断气安全回路 | ☐ | 断电断气后气缸到安全位置 |
| □ 13 | 管路是否有坡度(1~2%)和最低点排水阀 | ☐ | 防止积水冬季结冰 |
| □ 14 | 是否有备用气源接口/扩容余量 | ☐ | 预留20%未来扩产 |
