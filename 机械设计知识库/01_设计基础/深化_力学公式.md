#### 算例1-1：飞轮加速计算
<!-- 来源: 第1卷  第6页 -->
**问题**：飞轮从静止开始，以 $\alpha=2\text{ rad/s²}$ 匀加速转动5秒。求：(1)5秒末的角速度；(2)5秒内的转数。
**解**：
(1) $\omega = \alpha t = 2 \times 5 = 10\text{ rad/s}$
(2) $\theta = \frac{1}{2}\alpha t^2 = \frac{1}{2} \times 2 \times 25 = 25\text{ rad}$
转数 $N = \frac{\theta}{2\pi} = \frac{25}{2\pi} \approx 3.98\text{ r}$

### 1.2 动力学基本定律

#### 算例1-2：动量矩守恒应用
<!-- 来源: 第1卷  第6页 -->
**问题**：质量为60kg的人在转台边缘(半径2m)，转台角速度 $\omega_1=0.5\text{ rad/s}$，转动惯量 $J_{\text{台}}=200\text{ kg·m²}$。人走到中心(r=0.5m)时，角速度变为多少？
**解**：
$J_1 = J_{\text{台}} + mR^2 = 200 + 60\times4 = 440\text{ kg·m²}$
$J_2 = J_{\text{台}} + mr^2 = 200 + 60\times0.25 = 215\text{ kg·m²}$
动量矩守恒：$J_1\omega_1 = J_2\omega_2$
$\omega_2 = \dfrac{440\times0.5}{215} \approx 1.023\text{ rad/s}$

| 12 | 实心球体（质量$m$，半径$R$） | 沿直径 | $J = \dfrac{2}{5}mR^2$ | 均质球 |

| 16 | 长方体（质量$m$，边$a,b,c$） | 通过质心沿$a$方向 | $J = \dfrac{1}{12}m(b^2+c^2)$ | 均质 |
| 17 | 正三棱柱（质量$m$） | 中心轴 | $J = \dfrac{1}{2}mR^2$ | $R$为外接圆半径 |

| 19 | 半圆薄板（质量$m$，半径$R$） | 通过质心垂直于板面 | $J \approx 0.3199mR^2$ | $t\ll R$ |
| 20 | 椭圆截面环（质量$m$） | 中心轴 | $J = \dfrac{1}{2}m(a^2+b^2)$ | 半轴$a,b$ |

**平行轴定理（斯坦纳定理）**：$J = J_C + md^2$
- $J_C$：通过质心轴的转动惯量
- $d$：平行轴与质心轴的距离
- 适用条件：任意刚体

**回转半径**：$i = \sqrt{J/m}$
- 意义：假想质量集中于距转轴$i$处的等效半径

#### 算例1-3：复合系统的转动惯量
<!-- 来源: 第1卷  第6页 -->
**问题**：电机轴端安装一个飞轮(实心圆柱，$D=400\text{mm}, m=50\text{kg}$)和联轴器半个(近似为圆盘，$D=200\text{mm}, m=8\text{kg}$)。求系统对轴的总转动惯量。
**解**：
飞轮：$J_1 = \frac{1}{2}m_1R_1^2 = \frac{1}{2}\times50\times0.2^2 = 1.0\text{ kg·m²}$
联轴器：$J_2 = \frac{1}{2}m_2R_2^2 = \frac{1}{2}\times8\times0.1^2 = 0.04\text{ kg·m²}$
**总转动惯量**：$J = J_1+J_2 = 1.04\text{ kg·m²}$

#### 算例1-4：电机功率计算
<!-- 来源: 第1卷  第6页 -->
**问题**：提升机构以 $v=0.5\text{m/s}$ 提升 $m=2000\text{kg}$ 的重物，传动效率 $\eta=0.85$，求电机功率。
**解**：
$P_{\text{有效}} = mgv = 2000\times9.81\times0.5 = 9810\text{ W} = 9.81\text{ kW}$
$P_{\text{电机}} = \dfrac{P_{\text{有效}}}{\eta} = \dfrac{9.81}{0.85} = 11.54\text{ kW}$
取电机标准功率 **11kW** 或 **15kW**

#### 算例1-5：飞轮储能的动能公式应用
<!-- 来源: 第1卷  第6页 -->
**问题**：飞轮 $J=10\text{kg·m²}$，转速从 $n_1=1500\text{r/min}$ 降至 $n_2=1200\text{r/min}$，释放的能量。
**解**：
$\omega_1 = \frac{2\pi\times1500}{60} = 157.08\text{ rad/s}$
$\omega_2 = \frac{2\pi\times1200}{60} = 125.66\text{ rad/s}$
$\Delta E_k = \frac{1}{2}J(\omega_1^2-\omega_2^2) = \frac{1}{2}\times10\times(157.08^2-125.66^2)$
$= 5\times(24674-15791) = 5\times8883 = 44415\text{ J} \approx 44.4\text{ kJ}$

#### 算例2-2：传动轴扭转强度和刚度校核
<!-- 来源: 第1卷  第6页 -->
**问题**：传动轴 $n=960\text{r/min}$，$P=40\text{kW}$，材料45钢，$[\tau]=40\text{MPa}$，$[\varphi']=0.5°/m$，$G=80\text{GPa}$。求实心轴最小直径。
**解**：
**扭矩计算**：
$T = 9550\dfrac{P}{n} = 9550\times\dfrac{40}{960} = 397.9\text{ N·m} = 397900\text{ N·mm}$

**按强度条件**：
$d \geq \sqrt[3]{\dfrac{16T}{\pi[\tau]}} = \sqrt[3]{\dfrac{16\times397900}{\pi\times40}} = \sqrt[3]{50670} \approx 37.0\text{ mm}$

**按刚度条件**：
$\varphi' = \dfrac{T}{GI_p}\times\dfrac{180}{\pi}\times1000 \leq [\varphi']$
$I_p \geq \dfrac{T}{G[\varphi']}\times\dfrac{180}{\pi}\times1000 = \dfrac{397900}{80000\times0.5}\times57.296 = 569.8\text{ mm⁴}$
$d \geq \sqrt[4]{\dfrac{32I_p}{\pi}} = \sqrt[4]{\dfrac{32\times569.8}{\pi}} = \sqrt[4]{5803} \approx 8.7\text{ mm}$（刚度有余）

#### 算例2-3：空心轴与实心轴重量比较
<!-- 来源: 第1卷  第6页 -->
**问题**：空心轴 $\alpha=0.8$，与实心轴传递相同扭矩，求空心轴相对节省重量。
**解**：
设实心轴 $d_0$，空心轴外径 $D$，内径 $d=\alpha D$。
强度相等：$\dfrac{T}{W_{p0}} = \dfrac{T}{W_{p1}} \Rightarrow \dfrac{\pi d_0^3}{16} = \dfrac{\pi D^3}{16}(1-\alpha^4)$
$\dfrac{d_0^3}{16} = \dfrac{D^3}{16}(1-0.8^4) \Rightarrow D = \dfrac{d_0}{\sqrt[3]{1-0.4096}} = \dfrac{d_0}{\sqrt[3]{0.5904}} = \dfrac{d_0}{0.839} = 1.192d_0$
**重量比**：$\dfrac{W_{\text{空}}}{W_{\text{实}}} = \dfrac{A_{\text{空}}}{A_{\text{实}}} = \dfrac{\pi(D^2-d^2)/4}{\pi d_0^2/4} = \dfrac{D^2(1-\alpha^2)}{d_0^2} = \dfrac{(1.192d_0)^2(1-0.64)}{d_0^2} = 1.420 \times 0.36 = 0.511$
**节省**：约 **48.9%** 重量。

### 2.5 应力状态与强度理论

#### 2.5.1 平面应力状态
<!-- 来源: 第1卷  第1009页 -->

**任意斜截面上的应力**（$\alpha$为正截面法线与$x$轴夹角）：

$$\sigma_\alpha = \dfrac{\sigma_x+\sigma_y}{2} + \dfrac{\sigma_x-\sigma_y}{2}\cos2\alpha - \tau_{xy}\sin2\alpha$$

$$\tau_\alpha = \dfrac{\sigma_x-\sigma_y}{2}\sin2\alpha + \tau_{xy}\cos2\alpha$$

**主平面方位**：

$$\tan2\alpha_0 = -\dfrac{2\tau_{xy}}{\sigma_x-\sigma_y}$$

**主应力**：

$$\sigma_{1,2} = \dfrac{\sigma_x+\sigma_y}{2} \pm \sqrt{\left(\dfrac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2}$$

**最大剪应力**：

$$\tau_{\max} = \sqrt{\left(\dfrac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2} = \dfrac{\sigma_1-\sigma_2}{2}$$

$\tau_{\max}$所在截面与主平面成45°。

#### 2.5.3 四个强度理论完整推导与适用条件
<!-- 来源: 第1卷  第421页 -->

**第一强度理论（最大拉应力理论，Rankine）**

| 项目 | 内容 |
|------|------|
| 基本假设 | 不论材料处于何种应力状态，只要最大拉应力$\sigma_1$达到材料单向拉伸断裂时的极限应力$\sigma_b$，材料即断裂 |
| 强度条件 | $\sigma_{r1} = \sigma_1 \leq [\sigma]$ |
| 推导 | 单向拉伸断裂条件：$\sigma_1 = \sigma_b$，除以安全系数$S$得$[\sigma]=\sigma_b/S$ |
| 相当应力 | $\sigma_{r1} = \sigma_1$ |
| 适用材料 | **脆性材料**（铸铁、混凝土、玻璃、石料等） |
| 限制 | 只适用于$\sigma_1 > 0$（有拉应力时） |

**第二强度理论（最大伸长线应变理论，Saint-Venant）**

| 项目 | 内容 |
|------|------|
| 基本假设 | 不论应力状态如何，只要最大伸长线应变$\varepsilon_1$达到材料单向拉伸断裂时的极限应变$\varepsilon_u$，材料即断裂 |
| 强度条件 | $\sigma_{r2} = \sigma_1 - \nu(\sigma_2+\sigma_3) \leq [\sigma]$ |
| 推导 | 广义胡克定律$\varepsilon_1=\frac{1}{E}[\sigma_1-\nu(\sigma_2+\sigma_3)]$；单向拉伸时$\varepsilon_u=\sigma_b/E$，两者相等即得 |
| 相当应力 | $\sigma_{r2} = \sigma_1 - \nu(\sigma_2+\sigma_3)$ |
| 适用材料 | **脆性材料**（在二向拉伸/拉压时比第一理论更符合试验） |
| 限制 | $\nu$取值对结果影响大；不准确预测纯剪破坏 |

**第三强度理论（最大剪应力理论，Tresca）**

| 项目 | 内容 |
|------|------|
| 基本假设 | 不论应力状态如何，只要最大剪应力$\tau_{\max}$达到材料单向拉伸屈服时的极限剪应力$\tau_s$，材料即屈服 |
| 推导 | 单向拉伸屈服：$\sigma_1=\sigma_s, \sigma_2=\sigma_3=0$，则$\tau_{\max}=\sigma_1/2 = \sigma_s/2 = \tau_s$ |
| 复杂应力状态：$\tau_{\max} = (\sigma_1-\sigma_3)/2$ | 令$(\sigma_1-\sigma_3)/2 = \sigma_s/2$ |
| 强度条件 | $\sigma_{r3} = \sigma_1 - \sigma_3 \leq [\sigma]$ |
| 相当应力 | $\sigma_{r3} = \sigma_1 - \sigma_3$ |
| 适用材料 | **塑性材料**（钢、铜、铝等） |
| 优点 | 概念清晰，公式简单，偏安全 |
| 缺点 | 未考虑中间主应力$\sigma_2$的影响（误差约15%） |

**第四强度理论（形状改变比能理论，von Mises）**

| 项目 | 内容 |
|------|------|
| 基本假设 | 不论应力状态如何，只要形状改变比能$v_d$达到材料单向拉伸屈服时的极限形状改变比能$v_{ds}$，材料即屈服 |
| 推导 | 形状改变比能$v_d = \dfrac{1+\nu}{6E}[(\sigma_1-\sigma_2)^2+(\sigma_2-\sigma_3)^2+(\sigma_3-\sigma_1)^2]$ |
| | 单向拉伸屈服：$\sigma_1=\sigma_s,\sigma_2=\sigma_3=0$，则$v_{ds}=\dfrac{1+\nu}{3E}\sigma_s^2$ |
| | 令$v_d = v_{ds}$即得强度条件 |
| 强度条件 | $\sigma_{r4} = \sqrt{\frac{1}{2}[(\sigma_1-\sigma_2)^2+(\sigma_2-\sigma_3)^2+(\sigma_3-\sigma_1)^2]} \leq [\sigma]$ |
| 相当应力 | $\sigma_{r4} = \sqrt{\sigma_1^2+\sigma_2^2+\sigma_3^2-\sigma_1\sigma_2-\sigma_2\sigma_3-\sigma_3\sigma_1}$ |
| 适用材料 | **塑性材料**（更精确，广泛用于机械设计） |
| 优点 | 考虑了$\sigma_2$影响，与试验吻合较好 |
| 对比Tresca | von Mises比Tresca约大15%（更充分利用材料强度） |

**四个理论的适用范围总结表**：

| 理论 | 相当应力 $\sigma_r$ | 适用材料类型 | 典型应用 |
|------|-------------------|-------------|---------|
| 第一（Rankine） | $\sigma_1$ | 脆性材料 | 铸铁拉伸、压力容器 |
| 第二（Saint-Venant） | $\sigma_1-\nu(\sigma_2+\sigma_3)$ | 脆性材料 | 铸铁、砖石结构 |
| 第三（Tresca） | $\sigma_1-\sigma_3$ | 塑性材料 | 轴类零件、齿轮轴 |
| 第四（von Mises） | $\sqrt{\frac{1}{2}[(\sigma_1-\sigma_2)^2+(\sigma_2-\sigma_3)^2+(\sigma_3-\sigma_1)^2]}$ | 塑性材料 | 通用机械设计、有限元 |

#### 2.5.4 圆轴弯扭组合强度
<!-- 来源: 第1卷  第10页 -->

**弯扭组合变形**（圆轴 $\sigma$ 和 $\tau$ 组合）：

$\sigma = \dfrac{M}{W}$（弯曲正应力），$\tau = \dfrac{T}{W_p}$（扭转剪应力）

**第三强度理论（Tresca）**：

$$\sigma_{r3} = \sqrt{\sigma^2 + 4\tau^2} = \dfrac{1}{W}\sqrt{M^2 + T^2}$$

推导：$\sigma_1 = \dfrac{\sigma}{2} + \sqrt{\left(\dfrac{\sigma}{2}\right)^2 + \tau^2}$，$\sigma_3 = \dfrac{\sigma}{2} - \sqrt{\left(\dfrac{\sigma}{2}\right)^2 + \tau^2}$

$\sigma_{r3} = \sigma_1 - \sigma_3 = 2\sqrt{\left(\dfrac{\sigma}{2}\right)^2 + \tau^2} = \sqrt{\sigma^2 + 4\tau^2}$

**第四强度理论（von Mises）**：

$$\sigma_{r4} = \sqrt{\sigma^2 + 3\tau^2} = \dfrac{1}{W}\sqrt{M^2 + 0.75T^2}$$

**当量弯矩**：
- 第三强度：$M_{\text{eq3}} = \sqrt{M^2 + T^2}$
- 第四强度：$M_{\text{eq4}} = \sqrt{M^2 + 0.75T^2}$

**强度条件**：
$$\dfrac{M_{\text{eq}}}{W} \leq [\sigma]$$

#### 算例2-4：转轴弯扭组合强度校核
<!-- 来源: 第1卷  第6页 -->
**问题**：转轴直径 $d=50\text{mm}$，截面处合成弯矩 $M=600\text{N·m}$，扭矩 $T=400\text{N·m}$，材料40Cr(调质)，$[\sigma]=90\text{MPa}$。按第四强度理论校核。
**解**：
$W = \dfrac{\pi d^3}{32} = \dfrac{\pi\times50^3}{32} = 12272\text{ mm³}$
$\sigma = \dfrac{M}{W} = \dfrac{600000}{12272} = 48.9\text{ MPa}$
$\tau = \dfrac{T}{W_p} = \dfrac{400000}{\pi\times50^3/16} = \dfrac{400000}{24545} = 16.3\text{ MPa}$

**第四强度**：
$\sigma_{r4} = \sqrt{48.9^2 + 3\times16.3^2} = \sqrt{2391 + 797} = \sqrt{3188} = 56.5\text{ MPa}$

**安全系数**：$S = \dfrac{90}{56.5} = 1.59$（一般$S \geq 1.3\sim1.5$，满足要求）

## 3. 接触应力（赫兹公式）

### 3.1 赫兹接触理论基本原理
<!-- 来源: 第1卷  第606页 -->

**基本假设**：
1. 两接触体为弹性体，各向同性
2. 接触面为光滑曲面（无摩擦）
3. 接触面尺寸远小于物体曲率半径
4. 小变形范围内，应变-应力关系线性
5. 接触区内仅有法向压力

### 3.2 基本参数定义
<!-- 来源: 第1卷  第1009页 -->

**综合弹性模量 $E^*$**：

$$\dfrac{1}{E^*} = \dfrac{1-\nu_1^2}{E_1} + \dfrac{1-\nu_2^2}{E_2}$$

- $E_1, E_2$：两种材料的弹性模量(MPa)
- $\nu_1, \nu_2$：泊松比

**综合曲率半径 $R^*$**：

$$\dfrac{1}{R^*} = \dfrac{1}{R_1} \pm \dfrac{1}{R_2}$$

- $R_1, R_2$：两物体在接触点处的曲率半径(mm)
- $+$：外接触（凸面接触凸面）
- $-$：内接触（凸面接触凹面，$R_2 > R_1$时取负号）

### 3.3 两平行圆柱体外接触
<!-- 来源: 第1卷  第1009页 -->

| 参数 | 公式 | 说明 |
|------|------|------|
| 接触半宽 | $b = \sqrt{\dfrac{4FR^*}{\pi L E^*}}$ | $b$: mm，$F$: N，$L$: 接触长度(mm) |
| 最大接触应力 | $\sigma_{H\max} = \sqrt{\dfrac{FE^*}{\pi L R^*}}$ | 接触带中心线上 |
| 接触应力分布 | $\sigma_H(y) = \sigma_{H\max}\sqrt{1-\left(\dfrac{y}{b}\right)^2}$ | $y$: 距中心线距离 |
| 平均接触应力 | $\bar{\sigma}_H = \dfrac{\pi}{4}\sigma_{H\max} \approx 0.785\sigma_{H\max}$ | — |

#### 算例3-1：两圆柱齿轮齿面接触应力

## 4. 压杆稳定性
<!-- 来源: 第1卷  第220页 -->

### 4.1 欧拉公式完整推导
<!-- 来源: 第1卷  第112页 -->

**基本微分方程**（小挠度情况下）：

$$EI\frac{d^2y}{dx^2} = M(x) = -Fy$$

令 $k^2 = \dfrac{F}{EI}$，则：

$$\frac{d^2y}{dx^2} + k^2y = 0$$

**通解**：$y = A\sin kx + B\cos kx$

**边界条件**（两端铰支）：$y(0)=0$，$y(L)=0$

由 $y(0)=0$：$B=0$；由 $y(L)=0$：$A\sin kL = 0$

若 $A=0$ 则为直杆（平凡解）。非平凡解要求 $\sin kL=0$，即 $kL = n\pi$（$n=1,2,3,\ldots$）。

$$k = \dfrac{n\pi}{L},\quad F = EI\left(\dfrac{n\pi}{L}\right)^2$$

**最小临界载荷**（$n=1$）：

$$F_{cr} = \dfrac{\pi^2 EI}{L^2}$$

### 4.2 各种边界条件的长度系数 $\mu$ 表
<!-- 来源: 第1卷  第216页 -->

| 序号 | 支承形式 | 示意图 | 长度系数 $\mu$ | 临界载荷公式 | 有效长度 $L_e$ |
|------|---------|-------|---------------|-------------|-------------- |
| 1 | 两端铰支（球铰） | $\bigcirc-\bigcirc$ | 1.0 | $F_{cr} = \dfrac{\pi^2 EI}{L^2}$ | $L$ |
| 2 | 一端固定、一端自由 | $\blacksquare-\bigcirc$ | 2.0 | $F_{cr} = \dfrac{\pi^2 EI}{(2L)^2}$ | $2L$ |
| 3 | 两端固定 | $\blacksquare-\blacksquare$ | 0.5 | $F_{cr} = \dfrac{\pi^2 EI}{(0.5L)^2}$ | $0.5L$ |
| 4 | 一端固定、一端铰支 | $\blacksquare-\bigcirc$ | 0.7 | $F_{cr} = \dfrac{\pi^2 EI}{(0.7L)^2}$ | $0.7L$ |
| 5 | 一端固定、一端平移约束 | $\blacksquare-\triangle$ | 1.0 | $F_{cr} = \dfrac{\pi^2 EI}{L^2}$ | $L$ |
| 6 | 两端固定（可横向移动） | $\blacksquare-\blacksquare$ | 1.0 | $F_{cr} = \dfrac{\pi^2 EI}{L^2}$ | $L$ |

**通用欧拉公式**：

$$F_{cr} = \dfrac{\pi^2 EI}{(\mu L)^2}$$

**临界应力**：

$$\sigma_{cr} = \dfrac{F_{cr}}{A} = \dfrac{\pi^2 Ei^2}{(\mu L)^2} = \dfrac{\pi^2 E}{(\mu L/i)^2} = \dfrac{\pi^2 E}{\lambda^2}$$

**柔度（长细比）**：

$$\lambda = \dfrac{\mu L}{i},\quad i = \sqrt{\dfrac{I}{A}}$$

### 4.3 欧拉公式适用范围
<!-- 来源: 第1卷  第10页 -->

| 材料 | 比例极限 $\sigma_p$ (MPa) | 极限柔度 $\lambda_p$ | 适用条件 |
|------|--------------------------|---------------------|---------|

<!-- 来源: 第1卷  第220页 -->

| 工况 | $n_{st}$（安全系数） | 说明 |
|------|-------------------|------|
| 一般机械构件 | 1.8~3.0 | 常用结构件 |
| 精密机械构件 | 2.5~5.0 | 丝杠、精密导轨 |

| 飞机结构 | 1.5~2.0 | 减重要求高 |
| 矿山机械 | 3.0~6.0 | 冲击载荷大 |

**稳定校核公式**：

$$\dfrac{F_{cr}}{F} \geq n_{st} \quad\text{或}\quad \dfrac{\sigma_{cr}}{\sigma} \geq n_{st}$$

#### 算例4-1：压杆稳定校核
<!-- 来源: 第1卷  第6页 -->
**问题**：Q235钢圆截面压杆，$d=40\text{mm}$，$L=1.5\text{m}$，两端铰支，轴向压力 $F=100\text{kN}$，$n_{st}=2.5$。校核稳定性。
**解**：
$i = \dfrac{d}{4} = 10\text{mm}$，$\mu=1.0$
$\lambda = \dfrac{\mu L}{i} = \dfrac{1500}{10} = 150$

$\lambda > \lambda_p=100$，属于大柔度杆，用欧拉公式：

$\sigma_{cr} = \dfrac{\pi^2\times206000}{150^2} = \dfrac{2033400}{22500} = 90.4\text{ MPa}$

$A = \dfrac{\pi d^2}{4} = \dfrac{\pi\times40^2}{4} = 1256.6\text{ mm²}$

$F_{cr} = \sigma_{cr} A = 90.4\times1256.6 = 113598\text{ N} \approx 113.6\text{ kN}$

$n = \dfrac{F_{cr}}{F} = \dfrac{113.6}{100} = 1.136 < n_{st}=2.5$❌ **不满足要求！**

**改进措施**：增大直径或改变支承形式。

若改为两端固定：$\mu=0.5$，$\lambda=0.5\times1500/10=75$
$\lambda_s=61.4 < \lambda=75 < \lambda_p=100$，属于中柔度杆：

$\sigma_{cr} = 304 - 1.12\times75 = 304 - 84 = 220\text{ MPa}$
$F_{cr} = 220\times1256.6 = 276.5\text{ kN}$
$n = 276.5/100 = 2.765 > 2.5$ ✓ **满足要求**

#### 算例4-2：千斤顶丝杠稳定校核
<!-- 来源: 第1卷  第6页 -->
**问题**：螺旋千斤顶，丝杠为45钢，$d=40\text{mm}$（螺纹小径$d_3=33\text{mm}$），最大起重高度$L=500\text{mm}$（简化计算用螺纹小径截面），最大轴向力 $F=60\text{kN}$。丝杠下端固定，上端自由。
**解**：
$d = d_3 = 33\text{mm}$，$i = d/4 = 8.25\text{mm}$
$\mu=2.0$（一端固定、一端自由）
$\lambda = \dfrac{2\times500}{8.25} = 121.2$

$\lambda > \lambda_p=81$，大柔度杆，欧拉公式：

$\sigma_{cr} = \dfrac{\pi^2\times206000}{121.2^2} = \dfrac{2033400}{14690} = 138.4\text{ MPa}$

$A = \dfrac{\pi\times33^2}{4} = 855.3\text{ mm²}$
$F_{cr} = 138.4\times855.3 = 118.4\text{ kN}$

$n = \dfrac{118.4}{60} = 1.97$（千斤顶一般要求 $n_{st}=2.5\sim4.0$，接近下限，可接受或适当加大直径）

### 5.1 疲劳基本概念
<!-- 来源: 第1卷  第1009页 -->

**疲劳极限 $\sigma_{-1}$**：材料在对称循环应力下，能承受无限次循环而不破坏的最大应力。

**S-N曲线（Wöhler曲线）公式**：

$$\sigma^m N = C$$

取对数形式：$\lg N = \lg C - m\lg\sigma$

- $N$：疲劳寿命（循环次数）
- $\sigma$：应力幅值
- $m$：材料指数
- $C$：材料常数

|------|-----------|------|
| 钢（正火/调质） | 9~12 | 常用 $m\approx9$ |
| 钢（淬火/渗碳） | 6~10 | 高强度钢指数偏小 |
| 铸铁 | 6~8 | — |
| 铝合金 | 5~7 | — |
| 铜合金 | 5~8 | — |
| 钛合金 | 7~10 | — |

### 5.8 疲劳安全系数完整计算流程
<!-- 来源: 第1卷  第1009页 -->

#### 5.8.1 对称循环应力下的安全系数
<!-- 来源: 第1卷  第1946页 -->

**弯曲/拉压**：

$$n_\sigma = \dfrac{\sigma_{-1}}{\dfrac{K_\sigma}{\varepsilon_\sigma\beta}\sigma_a + \psi_\sigma\sigma_m}$$

**扭转**：

$$n_\tau = \dfrac{\tau_{-1}}{\dfrac{K_\tau}{\varepsilon_\tau\beta}\tau_a + \psi_\tau\tau_m}$$

**弯扭复合**：

$$n = \dfrac{n_\sigma n_\tau}{\sqrt{n_\sigma^2 + n_\tau^2}}$$

#### 5.8.2 非对称循环应力下的安全系数
<!-- 来源: 第1卷  第1009页 -->

当应力为非对称循环时，采用**Goodman公式**或**Soderberg公式**修正：

**Goodman修正**（常用）：

$$\dfrac{\sigma_a}{\sigma_{-1}} + \dfrac{\sigma_m}{\sigma_b} = 1$$

**Soderberg修正**（更保守）：

$$\dfrac{\sigma_a}{\sigma_{-1}} + \dfrac{\sigma_m}{\sigma_s} = 1$$

**安全系数表达式**：

$$n = \dfrac{\sigma_{-1}}{\dfrac{K_\sigma}{\varepsilon_\sigma\beta}\sigma_a + \psi_\sigma\sigma_m}$$

#### 5.8.3 有限寿命疲劳强度
<!-- 来源: 第1卷  第10页 -->

当 $N < N_0$（$N_0$为循环基数，通常 $10^7$）时：

$$\sigma_{-1N} = \sigma_{-1}\sqrt[m]{\dfrac{N_0}{N}}$$

**安全系数**：

$$n = \dfrac{\sigma_{-1N}}{\dfrac{K_\sigma}{\varepsilon_\sigma\beta}\sigma_a + \psi_\sigma\sigma_m}$$

#### 5.8.4 许用安全系数
<!-- 来源: 第1卷  第1009页 -->

| 工况 | 推荐 $[n]$ | 说明 |
|------|-----------|------|
| 材质均匀、载荷精确 | 1.3~1.5 | 精密机械 |
| 一般机械 | 1.5~2.0 | 通用设计 |

#### 算例5-1：转轴疲劳强度校核（完整流程）
<!-- 来源: 第1卷  第6页 -->
**问题**：一传动轴危险截面已知：$d=50\text{mm}$，材料40Cr调质($\sigma_b=800\text{MPa},\sigma_s=650\text{MPa},\sigma_{-1}=350\text{MPa},\tau_{-1}=195\text{MPa}$)，弯矩 $M=550\text{N·m}$，扭矩 $T=300\text{N·m}$（脉动），有键槽、轴肩圆角 $r=1.5\text{mm}$，表面精车，要求安全系数 $[n]=1.8$。

**解**：

**步骤1：计算应力**
弯曲应力：
$\sigma = \dfrac{M}{W} = \dfrac{550000}{\pi\times50^3/32} = \dfrac{550000}{12272} = 44.8\text{ MPa}$
（对称循环）$\sigma_a = 44.8\text{ MPa}$，$\sigma_m = 0$

扭转应力：
$W_p = \dfrac{\pi\times50^3}{16} = 24545\text{ mm³}$
$\tau = \dfrac{T}{W_p} = \dfrac{300000}{24545} = 12.22\text{ MPa}$
（脉动循环）$\tau_a = \tau_m = \dfrac{12.22}{2} = 6.11\text{ MPa}$

**步骤2：确定各系数**
$K_\sigma$（轴肩圆角，$r/d=0.03$，$\sigma_b=800$）：查表得 $K_\sigma \approx 1.80$（插值）
$K_\tau$（键槽+轴肩，取 $K_\tau \approx 1.50$）
$\varepsilon_\sigma$（$d=50\text{mm}$，合金钢）：0.73
$\varepsilon_\tau$（$d=50\text{mm}$，合金钢）：0.78
$\beta$（精车，$\sigma_b=800$）：0.83
$\psi_\sigma$：0.25，$\psi_\tau$：0.12

**步骤3：计算弯曲安全系数**
$n_\sigma = \dfrac{350}{\dfrac{1.80}{0.73\times0.83}\times44.8 + 0.25\times0} = \dfrac{350}{2.97\times44.8} = \dfrac{350}{133.1} = 2.63$

**步骤4：计算扭转安全系数**
$n_\tau = \dfrac{195}{\dfrac{1.50}{0.78\times0.83}\times6.11 + 0.12\times6.11} = \dfrac{195}{2.32\times6.11 + 0.73} = \dfrac{195}{14.18+0.73} = \dfrac{195}{14.91} = 13.08$

**步骤5：计算弯扭复合安全系数**
$n = \dfrac{2.63\times13.08}{\sqrt{2.63^2+13.08^2}} = \dfrac{34.4}{13.34} = 2.58$

**步骤6：校核**
$n = 2.58 > [n] = 1.8$ ✓ **安全**

#### 算例6-1：单自由度系统振动参数
<!-- 来源: 第1卷  第6页 -->
**问题**：质量 $m=50\text{kg}$ 安装于弹簧上，$k=20\text{kN/m}$，阻尼比 $\zeta=0.1$，受简谐力 $F(t)=200\sin(30t)\text{N}$。求系统固有频率、稳态振幅。

### 6.2 多自由度系统基础
<!-- 来源: 第1卷  第3页 -->

#### 6.2.1 二自由度系统
<!-- 来源: 第1卷  第1009页 -->

**运动方程（矩阵形式）**：

$$\begin{bmatrix} m_1 & 0 \\ 0 & m_2 \end{bmatrix}\begin{Bmatrix} \ddot{x}_1 \\ \ddot{x}_2 \end{Bmatrix} + \begin{bmatrix} k_1+k_2 & -k_2 \\ -k_2 & k_2 \end{bmatrix}\begin{Bmatrix} x_1 \\ x_2 \end{Bmatrix} = \begin{Bmatrix} F_1(t) \\ F_2(t) \end{Bmatrix}$$

**特征方程**（无阻尼自由振动，令 $F_1=F_2=0$，$x_i=A_i\sin\omega t$）：

$$\det\begin{bmatrix} k_1+k_2-m_1\omega^2 & -k_2 \\ -k_2 & k_2-m_2\omega^2 \end{bmatrix} = 0$$

**固有频率**：

$$\omega_{1,2}^2 = \dfrac{1}{2}\left[\dfrac{k_1+k_2}{m_1}+\dfrac{k_2}{m_2} \pm \sqrt{\left(\dfrac{k_1+k_2}{m_1}-\dfrac{k_2}{m_2}\right)^2 + \dfrac{4k_2^2}{m_1m_2}}\right]$$

**振型比**（第一振型 $\omega_1$ 和第二振型 $\omega_2$）：

$$\dfrac{A_2}{A_1}^{(i)} = \dfrac{k_2}{k_2-m_2\omega_i^2}, \quad i=1,2$$

#### 6.2.2 多自由度系统一般形式
<!-- 来源: 第1卷  第93页 -->

**矩阵运动方程**：

$$[M]\{\ddot{x}\} + [C]\{\dot{x}\} + [K]\{x\} = \{F(t)\}$$

**特征值问题**（求解固有频率和振型）：

$$([K] - \omega^2[M])\{\phi\} = \{0\}$$

**模态正交性**：

$$\{\phi_i\}^T[M]\{\phi_j\} = \begin{cases} 0, & i\neq j \\ M_i, & i=j \end{cases}$$

$$\{\phi_i\}^T[K]\{\phi_j\} = \begin{cases} 0, & i\neq j \\ K_i, & i=j \end{cases}$$

**模态质量 $M_i$ 和模态刚度 $K_i$**：

$$M_i = \{\phi_i\}^T[M]\{\phi_i\},\quad K_i = \{\phi_i\}^T[K]\{\phi_i\}$$

$$\omega_i^2 = \dfrac{K_i}{M_i}$$

**振型叠加法**：令 $\{x\} = [\Phi]\{q\}$，将物理坐标转换为模态坐标，解耦成 $n$ 个单自由度方程。

### 6.3 轴的临界转速完整计算方法
<!-- 来源: 第1卷  第1009页 -->

#### 6.3.1 单盘转子
<!-- 来源: 第1卷  第1009页 -->

**模型**：无质量弹性轴，中点有集中质量 $m$（盘），忽略轴本身质量。

$$\omega_{cr} = \sqrt{\dfrac{k}{m}} = \sqrt{\dfrac{48EI}{mL^3}} \quad (\text{简支})$$

$$n_{cr} = \dfrac{60}{2\pi}\omega_{cr} = \dfrac{30}{\pi}\sqrt{\dfrac{k}{m}} \quad (\text{r/min})$$

**计入轴自重**（假设轴质量分布均匀）：

利用瑞利(Rayleigh)法，将轴质量的一部分计入集中质量：

$$m_{\text{等效}} = m + \alpha m_s$$

其中 $m_s$ 为轴总质量，$\alpha$ 为修正系数：

| 支承形式 | 修正系数 $\alpha$ |
|---------|-----------------|
| 简支梁，中点集中质量 | 0.486 |
| 简支梁，两端集中质量 | 0.5 |
| 悬臂梁，端部集中质量 | 0.235 |

#### 6.3.2 多盘转子
<!-- 来源: 第1卷  第1009页 -->

**瑞利法（Rayleigh）**——近似计算第一阶临界转速：

$$\omega_{cr}^2 = \dfrac{g\sum_{i=1}^{n} m_i y_i}{\sum_{i=1}^{n} m_i y_i^2}$$

$$n_{cr} = \dfrac{30}{\pi}\omega_{cr}$$

- $m_i$：第 $i$ 个盘的质量(kg)
- $y_i$：第 $i$ 个盘在自重作用下的静挠度(mm)
- $g$：重力加速度(9800mm/s²)

#### 6.3.3 等直径光轴临界转速
<!-- 来源: 第1卷  第821页 -->

**两端简支**：

$$n_{cr} = \dfrac{\pi}{2L^2}\sqrt{\dfrac{EI}{\rho A}} \times \dfrac{60}{\pi} = \dfrac{30\pi}{L^2}\sqrt{\dfrac{EI}{\rho A}}$$

简化（钢轴，$E=2.06\times10^{11}\text{Pa}, \rho=7850\text{kg/m³}$）：

$$n_{cr} \approx 1.2\times10^8\dfrac{d}{L^2}\sqrt{\dfrac{1}{L}} \quad (\text{不精确})$$

更精确的**实用公式**（钢轴简支）：

$$n_{cr} \approx \dfrac{3.14\times10^7 d}{L^2} \quad (\text{r/min})$$

- $d$：轴直径(m)
- $L$：轴长(m)

#### 6.3.5 阶梯轴临界转速
<!-- 来源: 第1卷  第1009页 -->

**邓克莱(Dunkerley)公式**——适用于多质量系统的第一阶临界转速近似：

$$\dfrac{1}{n_{cr}^2} \approx \sum_{i=1}^{n} \dfrac{1}{n_{cri}^2}$$

- $n_{cri}$：第 $i$ 个质量单独作用时的临界转速
- 适用于各质量影响可叠加的情况

**传递矩阵法（Prohl法）**——精确计算：

将轴分段，每段视为等截面，建立各截面状态向量传递关系：

$$\{Z\}_{i+1} = [T]_i \{Z\}_i$$

其中 $\{Z\} = \{y, \theta, M, V\}^T$（挠度、转角、弯矩、剪力），$[T]$ 为传递矩阵。通过边界条件求解特征值。

#### 6.3.6 轴的临界转速常用判断

|-------------|------|---------|

| $n \approx n_{cr}$ | 共振 | 必须避免 |

#### 算例6-2：轴的临界转速计算
<!-- 来源: 第1卷  第6页 -->
**问题**：简支钢轴直径 $d=60\text{mm}$，跨度 $L=2\text{m}$，轴中央安装 $m=200\text{kg}$ 的转子。求系统临界转速。若轴的工作转速 $n=1500\text{r/min}$，是否安全？

**解**：

**方法1：单盘模型**
$I = \dfrac{\pi d^4}{64} = \dfrac{\pi\times60^4}{64} = 636173\text{ mm⁴} = 6.362\times10^{-7}\text{ m⁴}$
$k = \dfrac{48EI}{L^3} = \dfrac{48\times2.06\times10^{11}\times6.362\times10^{-7}}{2^3} = \dfrac{6.291\times10^6}{8} = 7.864\times10^5\text{ N/m}$

$\omega_{cr} = \sqrt{\dfrac{k}{m}} = \sqrt{\dfrac{7.864\times10^5}{200}} = \sqrt{3932} = 62.72\text{ rad/s}$
$n_{cr} = \dfrac{60}{2\pi}\times62.72 = 598.6\text{ r/min}$

**方法2：计入轴自重**
轴质量 $m_s = \rho AL = 7850\times\dfrac{\pi\times0.06^2}{4}\times2 = 7850\times0.002827\times2 = 44.4\text{ kg}$
$\alpha = 0.486$（简支中点集中质量时的等效系数）

$m_{\text{等效}} = 200 + 0.486\times44.4 = 200 + 21.6 = 221.6\text{ kg}$

$n_{cr} = 598.6\times\sqrt{\dfrac{200}{221.6}} = 598.6\times0.95 = 568.7\text{ r/min}$

**评估**：工作转速 $n=1500\text{r/min}$ 大于 $1.4n_{cr}=796\text{r/min}$。
需要检查是否低于第二阶临界转速（约为 $4n_{cr1}\approx2275\text{r/min}$）。

$n=1500 > 0.7n_{cr2}=1593$，不安全！（在0.7~1.4倍区间内）

**结论**：需修改设计——增大轴径或改变支承位置，提高临界转速至合理范围。

#### 算例6-3：刚性轴判断
<!-- 来源: 第1卷  第6页 -->
**问题**：简支轴 $d=80\text{mm}$，$L=1.5\text{m}$，工作转速 $n=3000\text{r/min}$。
**解**：
$n_{cr} \approx \dfrac{3.14\times10^7\times0.08}{1.5^2} = \dfrac{2512000}{2.25} = 1116444\text{ r/min}$（不计集中质量）

计入 $m=100\text{kg}$ 转子后，先算静挠度：
$\delta_{st} = \dfrac{mgL^3}{48EI} = \dfrac{100\times9.81\times1.5^3}{48\times2.06\times10^{11}\times\dfrac{\pi\times0.08^4}{64}} = \dfrac{3308}{48\times2.06\times10^{11}\times2.01\times10^{-6}}$
$= \dfrac{3308}{1.988\times10^7} = 1.66\times10^{-4}\text{ m} = 0.166\text{ mm}$

$n_{cr} \approx \dfrac{30}{\pi}\sqrt{\dfrac{g}{\delta_{st}}} = \dfrac{30}{\pi}\sqrt{\dfrac{9.81}{0.000166}} = 9.55\sqrt{59096} = 9.55\times243 = 2321\text{ r/min}$

$0.75n_{cr}=1741 > 3000$？不，检查：$n=3000 > 1.4n_{cr}=3249$，实际 $n=3000 < 3249$，仍在危险区。

**结论**：需要更大轴径或减短跨度。
