## 二、比例（GB/T 14690-1993）

## 三、图线（GB/T 4457.4-2002）
<!-- 来源: 第1卷  第13页 -->

| 图线名称 | 线型 | 线宽/mm | 主要用途 |
|---------|------|---------|---------|
| 粗实线 | 连续 | 0.5~0.7（常用0.7） | 可见轮廓线/螺纹牙顶线 |
| 细实线 | 连续 | 0.25~0.35 | 尺寸线/剖面线/引出线 |
| 波浪线 | 连续波浪 | 细 | 断裂边界/视图分界 |
| 双折线 | 连续双折 | 细 | 断裂边界 |
| 虚线 | 间距约3mm | 细（0.35） | 不可见轮廓线 |
| 细点画线 | 长≈15mm+短≈2mm+点 | 细 | 轴线/对称中心线 |
| 粗点画线 | 同细点画线 | 粗（0.7） | 有特殊要求的线 |
| 双点画线 | 长≈15mm+两短≈2mm+点 | 细 | 假想轮廓/运动极限位置/相邻零件轮廓 |

## 四、字体（GB/T 14691-1993）
<!-- 来源: 第1卷  第12页 -->

| 项目 | 要求 |
|------|------|
| 汉字 | 长仿宋体，高度≥3.5mm |
| 字母/数字 | A型（斜体向右75°）或B型（直体），高度≥2.5mm |
| 字宽 | h为字高，汉字宽≈0.7h，字母/数字宽≈0.6h |
| 字高系列 | 1.8/2.5/3.5/5/7/10/14/20(mm) |
| 字间距 | 汉字约0.5h，字母约0.2h |
| 行距 | 汉字约h/3，字母约h/2 |

## 五、尺寸标注（GB/T 4458.4-2003）
<!-- 来源: 第1卷  第925页 -->

### 5.1 尺寸组成
<!-- 来源: 第1卷  第1009页 -->

| 要素 | 要求 |
|------|------|
| 尺寸线 | 细实线，与标注方向平行，两端箭头/斜线 |
| 尺寸界线 | 细实线，自轮廓线/轴线/中心线引出约2~3mm |
| 箭头 | 长度≈4b(b=线宽)，宽≈(2~3)b，夹角≈15° |
| 尺寸数字 | 在尺寸线上方或中断处，水平放置 |

### 5.2 标注规则
<!-- 来源: 第1卷  第1009页 -->

| 规则 | 说明 |
|------|------|
| 线性尺寸数字方向 | 水平方向字头朝上，垂直方向字头朝左 |
| 角度尺寸数字 | 一律水平书写 |
| 直径符号Φ | 在数字前加Φ |
| 半径符号R | 在数字前加R |
| 球面符号SΦ/SR | 球面前加S |
| 正方形符号□ | 正方形截面前加□ |
| 倒角标注 | C×45°格式 |

### 5.3 公差标注
<!-- 来源: 第1卷  第1028页 -->

| 标注方式 | 格式 | 示例 |
|---------|------|------|
| 偏差标注 | 基本尺寸±偏差 | φ50±0.02 |
| 公差带标注 | 基本尺寸+公差等级 | φ50H7 |
| 综合标注 | 基本尺寸+公差带+偏差 | φ50H7(+0.025/0) |
| 形位公差标注 | 框格+特征符+公差值+基准 | ⌭0.01|A|

### 5.4 表面粗糙度标注
<!-- 来源: 第1卷  第1009页 -->

| 标注符号 | 说明 | 示例 |
|---------|------|------|
| ✓(尖头) | 基本符号 | 加工面 |
| ✓(圆头) | 去除材料获得 | Ra 1.6 |
| ✓(横线圆头) | 不去除材料 | 铸/锻/冲表面 |
| 标注位置 | 在轮廓线/延长线/尺寸线上 | — |

## 六、视图表达（GB/T 17451-1998、GB/T 4458.1-2002）
<!-- 来源: 第1卷  第1009页 -->

### 6.1 视图分类
<!-- 来源: 第1卷  第1009页 -->

| 视图类型 | 定义 | 应用 |
|---------|------|------|
| 基本视图 | 正/俯/左/右/仰/后6个投影方向 | 完整表达外形 |
| 向视图 | 可自由配置的视图 | 布置合理需要 |
| 局部视图 | 局部投影 | 省略整体投影 |
| 斜视图 | 倾斜面投影 | 非水平/垂直面 |
| 剖视图 | 假想剖切后投影 | 表达内部结构 |
| 断面图 | 仅断面投影 | 表达截面形状 |

### 6.2 剖视图分类
<!-- 来源: 第1卷  第1009页 -->

| 类型 | 特点 | 应用 |
|------|------|------|
| 全剖视图 | 一个剖切面完全剖开 | 不对称外形/内部复杂 |
| 半剖视图 | 一半剖一半不剖 | 对称外形零件 |
| 局部剖视图 | 局部剖切 | 需表达局部内部 |
| 阶梯剖 | 平行平面剖切 | 孔/槽不在同一平面 |
| 旋转剖 | 相交平面剖切剖后旋转 | 呈角度分布结构 |
| 复合剖 | 阶梯+旋转组合 | 复杂形状 |

### 6.3 剖面符号（常用材料）
<!-- 来源: 第1卷  第929页 -->

| 材料 | 剖面符号 |
|------|---------|
| 金属材料 | 45°平行等距细实线 |
| 非金属（塑料/橡胶） | 45°交叉细实线 |
| 木材/泡沫 | 不规则细小点 |
| 玻璃 | 小十字线 |
| 混凝土 | 小三角形+细水纹 |
| 液体 | 水平直线 |

## 七、装配图要求
<!-- 来源: 第1卷  第1009页 -->

| 要素 | 要求 |
|------|------|
| 零件序号 | 从1开始升序，按顺时针/逆时针排列 |
| 指引线 | 细实线，末端圆点或箭头，不交叉 |
| 明细栏 | 从下往上填序号/代号/名称/数量/材料/备注 |
| 技术要求 | 装配要求/检验/调试/涂装/包装说明 |
| 标题栏 | 名称/图号/比例/重量/设计/审核/材料 |
| 配合标注 | 在尺寸上标注配合代号（如H7/g6） |

## 八、标准件画法（简化画法）
<!-- 来源: 第1卷  第1009页 -->

| 标准件 | 简化画法要点 |
|--------|-------------|
| 螺纹（外） | 大径粗实线、小径细实线、螺纹终止线粗实线 |
| 螺纹（内） | 大径细实线、小径粗实线、倒角不画 |
| 螺纹连接 | 内外螺纹旋合段画外螺纹 |
| 弹簧 | 仅画端部1~2圈+中心线穿过其余 |
| 轴承 | 半剖画法或示意画法 |
| 齿轮 | 节圆用点画线、齿根圆用细实线或省略 |
| 花键 | 大径粗实线、小径细实线 |
| 焊缝 | 焊缝符号标注在指引线上 |
