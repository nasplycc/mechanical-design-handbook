## 1. 公差与配合基础
<!-- 来源: 第1卷  第1009页 -->

### 1.1 基本术语
<!-- 来源: 第1卷  第1094页 -->
| 术语 | 定义 | 公式/说明 |
|------|------|-----------|
| 基本尺寸 | 设计给定的尺寸 | 如 φ50 |
| 实际尺寸 | 测量得到的尺寸 | 有误差 |
| 极限尺寸 | 允许尺寸变化的两个界限 | 最大/最小极限尺寸 |
| 上偏差(ES/es) | 最大极限尺寸 - 基本尺寸 | ES(孔), es(轴) |
| 下偏差(EI/ei) | 最小极限尺寸 - 基本尺寸 | EI(孔), ei(轴) |
| 公差(T) | 允许尺寸的变动量 | T = |ES-EI| = |es-ei| |
| 配合 | 基本尺寸相同、相互结合的孔轴公差带关系 | 间隙/过盈/过渡 |

### 1.2 公差带组成
<!-- 来源: 第1卷  第1009页 -->
```
        最大极限尺寸 ───┐
                        │ ← 上偏差(ES/es)
                        │
        基本尺寸    ────┤
                        │
                        │ ← 下偏差(EI/ei)
        最小极限尺寸 ───┘
                        │
                        │ ← 公差带 T
```

## 4. 基准制
<!-- 来源: 第1卷  第13页 -->

### 4.1 基孔制 vs 基轴制
<!-- 来源: 第1卷  第1009页 -->
| 基准制 | 基准件 | 公差带 | 特点 |
|--------|--------|--------|------|
| 基孔制 | 孔 | H (EI=0) | 优先选用，减少刀具/量具规格 |
| 基轴制 | 轴 | h (es=0) | 用于冷拉轴、一轴多孔等场合 |

| 配合类型 | 配合代号 | 名称 | 应用举例 |

| 间隙 | H7/h6 | 间隙定位 | 一般滑动配合 |
| 间隙 | H8/f7 | 中等间隙 | 一般轴在轴承中转动 |

## 5. 配合选择
<!-- 来源: 第1卷  第1009页 -->

### 5.1 选用原则
<!-- 来源: 第1卷  第1111页 -->
1. **确定配合性质**：间隙/过渡/过盈
2. **选择基准制**：优先基孔制
3. **选择公差等级**：配合面IT5~IT7，非配合面IT11~IT13
4. **查表确定偏差**：根据基本尺寸和代号查表

### 5.2 间隙配合选择
<!-- 来源: 第1卷  第1009页 -->
| 工况 | 推荐配合 | 说明 |
|------|----------|------|
| 精密滑动 | H7/g6 | 高精度导向 |
| 一般转动 | H8/f7 | 轴在轴承中转动 |
| 低速重载 | H9/d9 | 大间隙、润滑好 |
| 粗糙环境 | H11/b11 | 防尘、易拆装 |

### 5.3 过渡配合选择
<!-- 来源: 第1卷  第1009页 -->
| 工况 | 推荐配合 | 说明 |
|------|----------|------|
| 精确定心 | H7/k6 | 可拆、对中好 |
| 一般定心 | H7/n6 | 需压入工具 |
| 不常拆 | H7/m6 | 中等过盈量 |

### 5.4 过盈配合选择
<!-- 来源: 第1卷  第1009页 -->
| 工况 | 推荐配合 | 说明 |
|------|----------|------|
| 轻载固定 | H7/p6 | 可拆、不用键 |
| 中等载荷 | H7/s6 | 最常用过盈配合 |
| 重载冲击 | H7/u7 | 需热装/冷装 |

### 5.5 过盈配合装配力估算
<!-- 来源: 第1卷  第1009页 -->
```
F = π · d · L · μ · p

其中：
d - 配合直径(mm)
L - 配合长度(mm)
μ - 摩擦系数(钢-钢: 0.1~0.15)
p - 接触压力(MPa)
p = δ / [d·(1/E₁·(d²+d₁²)/(d²-d₁²)+ν₁) + 1/E₂·(d₂²+d²)/(d₂²-d²)-ν₂)]
δ - 实际过盈量(mm)
```

### 7.3 几何公差标注原则
<!-- 来源: 第1卷  第1135页 -->
- 公差带形状：⌀ 表示圆柱形，否则为两平行平面/圆之间
- 基准：A、B、C 表示基准要素
- 最大实体要求：Ⓜ
- 最小实体要求：Ⓛ
- 可逆要求：Ⓟ
