### 2.6 组合变形强度理论

## 3. 接触应力（赫兹公式）
<!-- 来源: 第1卷 第1篇一般设计资料 第647页 -->

### 3.1 两圆柱体接触
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->
```
σ_H = √[ F·E* / (π·L·R*) ]

其中：
1/E* = (1-ν₁²)/(2E₁) + (1-ν₂²)/(2E₂)
1/R* = 1/R₁ + 1/R₂ (外接触取+)，1/R₁ - 1/R₂ (内接触取-)
F - 法向载荷(N)
L - 接触长度(mm)
R₁, R₂ - 圆柱体半径(mm)
```

**最大接触应力简化公式（钢对钢）：**
```
σ_H = 0.418·√(F·E·(R₁+R₂)/(L·R₁·R₂))
```

### 3.2 球与平面接触
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->
```
接触半径 a = (3FR*/4E*)^(1/3)
最大接触应力 σ_Hmax = (6F·E*²/(π³R*²))^(1/3)
```

### 3.3 齿轮接触强度
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->
```
σ_H = Z_H·Z_E·Z_ε·√[ F_t/(b·d₁) · (u+1)/u ]

其中：
Z_H - 区域系数（标准齿轮≈2.5）
Z_E - 弹性系数（钢对钢≈189.8√MPa）
Z_ε - 重合度系数
F_t - 分度圆切向力(N)
b - 齿宽(mm)
d₁ - 小齿轮分度圆直径(mm)
u - 齿数比(z₂/z₁)
```

### 3.4 齿轮弯曲强度
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->
```
σ_F = F_t/(b·m) · Y_Fa·Y_Sa·Y_ε

其中：
m - 模数(mm)
Y_Fa - 齿形系数
Y_Sa - 应力修正系数
Y_ε - 重合度系数
```

## 4. 压杆稳定性
<!-- 来源: 第1卷 第1篇一般设计资料 第195页 -->

### 4.1 欧拉公式（细长杆）
<!-- 来源: 第1卷 第1篇一般设计资料 第195页 -->
```
F_cr = π²EI / (μL)²
σ_cr = π²E / λ²

其中：
μ - 长度系数：
  两端铰支 μ=1
  一端固定一端自由 μ=2
  两端固定 μ=0.5
  一端固定一端铰支 μ=0.7
L - 杆长(m)
i = √(I/A) - 惯性半径
λ = μL/i - 柔度(长细比)
```

### 4.2 临界应力分类
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->
| 杆类型 | 柔度范围 | 临界应力公式 |
|--------|----------|-------------|
| 大柔度杆（细长） | λ ≥ λ_p | σ_cr = π²E/λ² (欧拉) |
| 中柔度杆 | λ_s < λ < λ_p | σ_cr = a - bλ (直线公式) |
| 小柔度杆（粗短） | λ ≤ λ_s | σ_cr = σ_s (屈服) |

**Q235钢：** λ_p ≈ 100, λ_s ≈ 60, a = 304 MPa, b = 1.12 MPa

### 5.2 对称循环疲劳强度
<!-- 来源: 第1卷 第1篇一般设计资料 第625页 -->
```
n = σ₋₁ / (K_σ·σ_a/ε_σ·β + ψ_σ·σ_m)

其中：
σ₋₁ - 材料对称循环疲劳极限
K_σ - 有效应力集中系数
ε_σ - 尺寸系数
β - 表面质量系数
ψ_σ - 材料敏感系数
σ_a - 应力幅
σ_m - 平均应力
```

```
其中：
m - 材料指数（钢 m≈9）
N - 循环次数
C - 常数
```
