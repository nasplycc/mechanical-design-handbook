## 1. 机加工方法总论
<!-- 来源: 第1卷  第1009页 -->

### 1.1 机加工基本分类
<!-- 来源: 第1卷  第1009页 -->

机械加工是利用切削工具从工件上切除多余材料以获得所需形状、尺寸和表面质量的加工方法。

| 分类方式 | 类型 | 说明 |
|---------|------|------|
| 按切削运动 | 主运动旋转类 | 车削、铣削、磨削、钻削 |
| | 主运动直线类 | 刨削、拉削、插削 |
| 按刀具类型 | 单刃刀具 | 车刀、刨刀 |
| | 多刃刀具 | 铣刀、钻头、铰刀、丝锥 |
| | 磨料工具 | 砂轮、砂带、油石 |
| 按加工精度 | 粗加工 | IT11~IT13 |
| | 半精加工 | IT8~IT10 |
| | 精加工 | IT6~IT7 |
| | 精密加工 | IT5级以上 |
| 按自动化程度 | 手动 | 普通机床 |
| | 半自动 | 数控单机 |
| | 自动 | 加工中心、柔性制造 |

### 1.3 加工方法选择原则
<!-- 来源: 第1卷  第1075页 -->

1. **按精度要求选择**：经济精度能满足要求时，不选更精密的加工
2. **按批量选择**：大批量优先选高生产率方法（拉削/滚齿/多刀车），单件小批选通用设备
3. **按工件材料选择**：淬硬钢只能用磨削，有色金属不宜磨削
4. **按工件形状选择**：回转体选车削，平面选铣/刨，内孔选钻/镗/拉
5. **按设备条件选择**：现有设备优先

<!-- 来源: 第1卷  第1009页 -->

| 项目 | 说明 |
|-----|------|
| 定义 | 铣刀旋转产生主运动，工件（或铣刀）作进给运动的切削加工 |

### 3.2 铣削方式
<!-- 来源: 第1卷  第1009页 -->

| 分类方式 | 类型 | 特点 | 对比 |
|---------|------|------|------|
| 按铣刀与工件相对位置 | 周铣 | 圆周刃切削，端刃少用 | 效率较低 |
| | 端铣 | 端面刃切削，周刃少用 | 效率高，表面质量好 |
| | 周端复合铣 | 同时使用 | 加工台阶面 |
| 按铣刀转向与进给方向 | 逆铣 | 切削厚度从0→最大 | 无冲击，表面质量好 |
| | 顺铣 | 切削厚度从最大→0 | 效率高，需消除间隙 |
| | 对称铣 | 逆顺交替 | 精度好 |
| 按铣刀类型 | 圆柱铣刀铣削 | 加工平面 | 中小平面 |
| | 端铣刀铣削 | 加工大平面 | 大平面高效 |
| | 立铣刀铣削 | 加工台阶/槽/曲面 | 数控应用广 |
| | 三面刃铣刀铣削 | 加工直角沟槽 | 批量槽 |
| | 成型铣刀铣削 | 加工成型面 | 批量专用 |

### 3.3 顺铣与逆铣对比
<!-- 来源: 第1卷  第1009页 -->

| 对比项 | 逆铣 | 顺铣 |
|-------|------|------|
| 切削厚度变化 | 从0逐渐增大 | 从最大逐渐减小 |
| 切入方式 | 挤压刮擦→切削 | 直接切削 |
| 切削力方向 | 向上抬工件 | 向下压工件 |
| 工件固定 | 需较大夹紧力 | 夹紧力较小 |
| 机床要求 | 无需间隙消除 | 需消除丝杠间隙 |
| 表面粗糙度 | 较好（切削平稳） | 差（有冲击） |
| 刀具磨损 | 后刀面磨损 | 前刀面磨损 |
| 加工余量均匀 | 不均匀 | 较均匀 |
| 薄壁件 | 易变形 | 较好 |
| 热处理件（硬皮） | 可加工（从下切） | 不易切入 |

推荐：精铣用逆铣，粗铣用顺铣；数控铣床通常用顺铣。

### 4.4 刨削与铣削对比
<!-- 来源: 第1卷  第1009页 -->

| 对比项 | 刨削 | 铣削 |
|-------|------|------|
| 主运动 | 直线往复 | 旋转 |

| 窄长平面/导轨 | 优势（加工精度高） | 不如刨削 |
| 刀具成本 | 低（单刃刨刀） | 较高（铣刀较贵） |
| 表面质量 | 较低（有冲击） | 较高 |
| 操作难度 | 较易 | 较复杂 |

## 5. 磨削加工
<!-- 来源: 第1卷  第1108页 -->

### 5.1 磨削加工特点
<!-- 来源: 第1卷  第1009页 -->

| 项目 | 说明 |
|-----|------|
| 定义 | 用磨料磨具（砂轮/砂带/油石）去除工件材料的加工方法 |
| 工件材料 | 淬硬钢、硬质合金、陶瓷、玻璃、铸铁、有色金属 |

## 12. 切削力计算公式
<!-- 来源: 第1卷  第1009页 -->

### 12.1 车削力计算公式（主切削力）
<!-- 来源: 第1卷  第1009页 -->

#### 12.1.1 基本公式
<!-- 来源: 第1卷  第413页 -->

```
F_c = C_F · a_p^xF · f^yF · v_c^nF · K_F
```

其中：
- F_c — 主切削力（N）
- C_F — 工件材料系数
- a_p — 背吃刀量（mm）
- f — 进给量（mm/r）
- v_c — 切削速度（m/min）
- xF — a_p的指数
- yF — f的指数
- nF — v_c的指数
- K_F — 修正系数综合乘积

#### 12.1.2 各材料车削力系数与指数
<!-- 来源: 第1卷  第1009页 -->

| 工件材料 | C_F | xF | yF | nF |
|---------|-----|----|----|----|
| 碳钢（σ_b=650MPa） | 2650 | 1.0 | 0.75 | -0.15 |
| 合金钢（σ_b=750MPa） | 2950 | 1.0 | 0.75 | -0.15 |
| 铸铁（HB=190） | 900 | 1.0 | 0.75 | -0.15 |
| 铝合金（HB=120） | 400 | 1.0 | 0.72 | -0.15 |
| 不锈钢（1Cr18Ni9） | 3200 | 1.0 | 0.75 | -0.15 |
| 铜合金（H62） | 450 | 1.0 | 0.72 | -0.15 |

#### 12.1.3 修正系数K_F
<!-- 来源: 第1卷  第1742页 -->

K_F由以下各项修正系数的乘积组成：

```
K_F = K_mF · K_rF · K_γF · K_λF · K_hF
```

**材料修正系数K_mF：**

| 碳钢 | σ_b(MPa) | 400 | 500 | 600 | 650 | 700 | 800 | 900 | 1000 |
|------|---------|-----|-----|-----|-----|-----|-----|-----|------|
| | K_mF | 0.70 | 0.82 | 0.93 | 1.00 | 1.06 | 1.18 | 1.30 | 1.42 |

| 铸铁 | HB | 140 | 160 | 180 | 190 | 200 | 220 | 240 |
|------|-----|-----|-----|-----|-----|-----|-----|-----|
| | K_mF | 0.73 | 0.82 | 0.92 | 1.00 | 1.05 | 1.17 | 1.30 |

**主偏角修正系数K_rF：**

| 主偏角κ_r(°) | 30 | 45 | 60 | 75 | 90 |
|-------------|----|----|----|----|----|

#### 12.1.4 三种分力关系
<!-- 来源: 第1卷  第1009页 -->

车削切削力的三个分力：

| 分力 | 符号 | 方向 | 与F_c的近似比例 |
|-----|------|------|--------------|
| 主切削力 | F_c | 切线方向（主运动方向） | 1.0（基准） |
| 进给抗力 | F_f | 进给方向（轴向） | 0.3~0.5 |
| 径向抗力 | F_p | 垂直于进给方向（径向） | 0.3~0.6 |

经验估算（主偏角45~90°时）：
- F_f = (0.3~0.5) × F_c
- F_p = (0.3~0.6) × F_c

### 12.2 铣削力计算公式
<!-- 来源: 第1卷  第1009页 -->

#### 12.2.1 端铣时主切削力
<!-- 来源: 第1卷  第1009页 -->

```
F_c = C_F · a_p^xF · f_z^yF · a_e^uF · z^wF · K_F
```

其中：
- F_c — 铣削主切削力（N）
- a_p — 背吃刀量（mm）
- f_z — 每齿进给量（mm/齿）
- a_e — 铣削宽度（mm）
- z — 铣刀齿数

| 工件材料 | C_F | xF | yF | uF | wF |
|---------|-----|----|----|----|----|
| 碳钢 | 665 | 0.90 | 0.72 | 0.80 | 0.80 |
| 合金钢 | 790 | 0.90 | 0.72 | 0.80 | 0.80 |
| 铸铁 | 300 | 0.95 | 0.70 | 0.75 | 0.75 |
| 铝合金 | 200 | 0.85 | 0.70 | 0.75 | 0.80 |

#### 12.2.2 立铣周铣时圆周铣削力
<!-- 来源: 第1卷  第1009页 -->

```
F_c = 9.81 · C_F · a_p^0.86 · f_z^0.72 · d^-0.86 · a_e^1.0 · z · K_F
```

其中d为铣刀直径（mm），其他符号同上。

### 12.3 切削功率计算
<!-- 来源: 第1卷  第1009页 -->

#### 12.3.1 车削功率
<!-- 来源: 第1卷  第1009页 -->

```
P_c = (F_c · v_c) / (60 × 1000)   (kW)
```

其中F_c为N，v_c为m/min。

#### 12.3.2 铣削功率
<!-- 来源: 第1卷  第1009页 -->

```
P_c = (F_c · v_c) / (60 × 1000)   (kW)
```

#### 12.3.3 机床电机功率
<!-- 来源: 第1卷  第1009页 -->

```
P_motor = P_c / η
```

其中η为机床传动效率（0.70~0.85，视机床新旧程度）。

### 12.4 切削功率估算速查
<!-- 来源: 第1卷  第1009页 -->

| 加工方式 | 功率估算经验公式 | 说明 |
|---------|----------------|------|
| 粗车（碳钢） | P_c = a_p · f · v_c / (60×300) kW | a_p(mm), f(mm/r), v_c(m/min) |
| 精车（碳钢） | P_c = a_p · f · v_c / (60×400) kW | 同上 |

<!-- 来源: 第1卷  第1009页 -->

#### 15.1.1 夹具的定义
<!-- 来源: 第1卷  第1009页 -->

夹具是用于在机床上定位和夹紧工件，使工件相对于刀具具有正确位置的工艺装备。

#### 15.1.2 夹具的组成
<!-- 来源: 第1卷  第1009页 -->

| 组成部分 | 功能 | 举例 |
|---------|------|------|
| 定位元件 | 使工件在夹具中占正确定位 | V形块/定位销/支撑板/定位套 |
| 夹紧装置 | 固定工件位置，防止切削力下移动 | 螺旋夹紧/偏心夹紧/液压夹紧 |
| 导向元件 | 引导刀具方向 | 钻套/镗套 |
| 对刀元件 | 确定刀具与夹具的相对位置 | 对刀块 |
| 夹具体 | 连接各元件的基体 | 底座/壳体/支架 |
| 分度装置 | 分度定位 | 分度盘/定位销 |
| 辅助装置 | 其他辅助功能 | 平衡块/冷却管/排屑槽 |

#### 15.1.3 夹具分类
<!-- 来源: 第1卷  第1009页 -->

| 分类方式 | 类型 | 特点 |
|---------|------|------|
| 按使用机床 | 车床夹具/铣床夹具/钻床夹具/磨床夹具/镗床夹具 | 专用 |
| 按通用程度 | 通用夹具/专用夹具/成组夹具/组合夹具 | 见下表 |
| 按动力源 | 手动/气动/液压/电动/磁力/真空 | 自动化程度不同 |
| 按加工工序 | 钻孔夹具/铣平面夹具/车外圆夹具 | 专用化 |

| 夹具类型 | 特点 | 适用批量 | 成本 |
|---------|------|---------|------|
| 通用夹具 | 标准件，三爪卡盘/虎钳/分度头 | 单件/小批 | 低 |
| 专用夹具 | 为特定工件设计 | 中批/大批 | 中高 |
| 成组夹具 | 一组相似工件共用 | 中小批（成组技术） | 中 |
| 组合夹具 | 标准元件拼装，可拆 | 单件/小批（新品试制） | 中低 |
| 拼装夹具 | 专用+标准组合 | 中小批 | 中 |
| 数控夹具 | 柔性化，自动定位夹紧 | 中小批（FMS） | 高 |
| 随行夹具 | 随工件进入加工系统 | 自动线 | 高 |

### 15.2 定位基准选择原则
<!-- 来源: 第1卷  第7页 -->

#### 15.2.1 基准分类
<!-- 来源: 第1卷  第1009页 -->

| 基准类型 | 定义 | 示例 |
|---------|------|------|
| 设计基准 | 零件图上标注尺寸的起始点 | 轴线/端面/底面 |
| 工艺基准 | 制造过程中使用的基准 | |
| ├ 工序基准 | 工序图上标注尺寸的基准 | 加工表面直接参考 |
| ├ 定位基准 | 定位时使用的基准（夹具中） | 外圆柱面（V形块） |
| ├ 测量基准 | 测量时使用的基准 | 端面/中心孔 |
| └ 装配基准 | 装配时确定零件位置的基准 | 配合面/安装面 |

#### 15.2.2 粗基准选择原则（第一道工序使用毛坯面定位）
<!-- 来源: 第1卷  第1009页 -->

| 序号 | 原则 | 说明 |
|-----|------|------|
| 1 | 选取重要加工面余量均匀 | 应选择余量最小的表面作为粗基准 |
| 2 | 保证不加工面与加工面位置正确 | 选不加工面为粗基准 |
| 3 | 粗基准表面应平整光洁 | 无飞边/冒口/浇口/分模面 |
| 4 | 粗基准只使用一次 | 避免重复使用导致精度下降 |
| 5 | 便于装夹和加工 | 便于设置定位基准 |

粗基准选择优先顺序：
```
重要余量均匀面 → 不加工面 → 较大平整面 → 便于装夹面
```

#### 15.2.3 精基准选择原则（后续工序使用加工面定位）
<!-- 来源: 第1卷  第1009页 -->

| 序号 | 原则 | 说明 |
|-----|------|------|
| 1 | 基准重合原则 | 定位基准与设计基准重合（消除基准不重合误差） |
| 2 | 基准统一原则 | 尽可能使用同一基准加工各面（减少装夹误差） |
| 3 | 自为基准原则 | 加工面本身作为基准（如珩磨/无心磨） |
| 4 | 互为基准原则 | 两面互为基准反复加工（保证同轴度） |
| 5 | 便于装夹原则 | 基准面足够大/刚性好/变形小 |

**基准重合原则示例：**

```
设计要求尺寸 A（以底面为设计基准）
工序定位基准应选底面（基准重合）
若选其他面为定位基准 → 产生基准不重合误差
```

**基准统一原则典型应用：**

| 典型零件 | 统一基准 | 加工内容 |
|---------|---------|---------|
| 轴类零件 | 两中心孔 | 各段外圆/螺纹/端面 |
| 箱体类 | 一面两销（一个大平面+两个定位销孔） | 各轴承孔/端面/螺纹 |
| 盘类 | 大端面+内孔 | 外圆/端面/齿形 |
| 连杆 | 大端孔+端面 | 小端孔/结合面 |

**自为基准典型应用：**

| 加工方式 | 基准形式 | 说明 |
|---------|---------|------|
| 珩磨 | 孔本身（导向） | 浮动珩磨头以孔自导向 |
| 无心磨 | 外圆本身 | 导轮+托板定位 |
| 拉孔 | 预制孔本身 | 拉刀引导部分导向 |
| 铰孔 | 预制孔本身 | 铰刀引导部分导向 |

**互为基准典型应用：**

| 零件 | 往复加工内容 | 精度要求 |
|------|------------|---------|
| 精密主轴 | 外圆↔中心孔（修正中心孔磨外圆） | 同轴度0.005~0.01mm |
| 齿轮 | 齿面↔内孔（磨齿前磨内孔） | 精度IT5~IT6 |

#### 15.3.3 V形块定位
<!-- 来源: 第1卷  第1009页 -->

V形块是最常用的外圆柱面定位元件。

**V形块结构参数：**

| 符号 | 含义 | 常用值 |
|-----|------|-------|
| α | V形槽夹角 | 60°/90°/120°（最常用90°） |
| H | V形块高度 | 见下表 |
| d | 定位直径范围 | 见下表 |

**V形块尺寸系列（GB/T2208-2013）：**

| 型号 | 定位直径范围d(mm) | H(mm) | B(mm) | L(mm) | α |
|------|------------------|-------|-------|-------|----|
| V12 | 5~12 | 16 | 22 | 40 | 90° |
| V20 | 8~20 | 20 | 28 | 50 | 90° |
| V30 | 12~30 | 25 | 35 | 60 | 90° |
| V45 | 16~45 | 30 | 42 | 80 | 90° |
| V60 | 20~60 | 35 | 50 | 100 | 90° |
| V80 | 25~80 | 42 | 60 | 120 | 90° |
| V100 | 32~100 | 50 | 70 | 150 | 90° |
| V130 | 40~130 | 60 | 85 | 180 | 90° |
| V160 | 50~160 | 72 | 100 | 220 | 90° |

**V形块定位误差分析：**

| V形块夹角α | 定位误差ε（直径方向） | 说明 |
|-----------|-------------------|------|
| 60° | ε = 1.5δd | δd为工件外圆公差 |
| 90° | ε = 1.2071δd | 最常用 |
| 120° | ε = 1.1547δd | 垂直方向误差小 |

注：ε为定位基准（工件轴线）在垂直方向的位置误差。

**V形块定位压紧设计原则：**

1. 长圆柱用双V形块（一固定一浮动）
2. V形块高度H应保证工件中心高于V形块上平面
3. 短圆柱用一个宽V形块即可
4. 90°V形块最常用，兼顾定位精度和稳定性
5. 粗基准用60°V形块，精基准用90°V形块

#### 15.3.4 一面两销定位
<!-- 来源: 第1卷  第1009页 -->

箱体/壳体类零件最常用的定位方式。

| 定位元件 | 限制自由度 | 说明 |
|---------|-----------|------|
| 大平面（三个支撑钉/支撑板） | X轴旋转/Y轴旋转/Z轴平移 | 3个自由度 |
| 圆柱销 | X轴平移/Y轴平移 | 2个自由度 |
| 菱形销 | Z轴旋转 | 1个自由度 |

4. 菱形销的长轴应垂直于两销连线方向

### 15.4 夹紧装置
<!-- 来源: 第1卷  第1887页 -->

#### 15.4.1 夹紧力设计原则
<!-- 来源: 第1卷  第1009页 -->

| 原则 | 说明 |
|-----|------|
| 夹紧力方向应垂直主定位面 | 保证定位稳定 |
| 夹紧力应指向支撑点 | 防止工件翻转 |
| 夹紧力作用点应在刚性最大处 | 减少工件变形 |
| 夹紧力应靠近加工区域 | 减少振动 |
| 夹紧力大小要适当 | 足够但不引起变形 |

**夹紧力计算：**

```
W_k = K · W
```

其中：
- W_k — 所需夹紧力（N）
- W — 理论夹紧力（由切削力平衡计算）（N）
- K — 安全系数（粗加工K=2.5~3.0，精加工K=1.5~2.0）

```
K = K_0 · K_1 · K_2 · K_3 · K_4 · K_5 · K_6
```

| 系数 | 含义 | 取值范围 |
|-----|------|---------|
| K_0 | 基本安全系数 | 1.2~1.5 |
| K_1 | 加工性质系数 | 粗加工1.2，精加工1.0 |
| K_2 | 刀具钝化系数 | 1.0~1.9（依刀具材料） |
| K_3 | 切削特点系数 | 连续切削1.0，断续切削1.2 |
| K_4 | 夹紧力稳定性系数 | 手动1.3，气动1.0，液压1.0 |
| K_5 | 手柄位置系数 | 方便1.0，不便1.2 |
| K_6 | 工件支撑面系数 | 平面支撑1.0，V形块2.0 |

#### 15.4.2 常用夹紧机构
<!-- 来源: 第1卷  第1009页 -->

| 夹紧机构 | 特点 | 夹紧力 | 自锁性 | 应用 |
|---------|------|--------|-------|------|
| 螺旋夹紧 | 结构简单，行程大，自锁好 | 大 | 好 | 通用性最广 |
| 偏心夹紧 | 快速夹紧/松开 | 中 | 中等 | 小件/批量 |
| 楔形夹紧 | 增力比大 | 大 | 好 | 气动/液压 |
| 螺旋压板 | 组合式，使用最广 | 大 | 好 | 铣/刨/车 |
| 铰链夹紧 | 快速联动 | 中 | 差 | 自动化 |
| 气动夹紧 | 快速，可控 | 中 | 需保压 | 自动线 |
| 液压夹紧 | 力大，平稳 | 大 | 好（保压） | 强力切削 |
| 电磁夹紧 | 不损伤表面 | 中 | 安全 | 磨床 |
| 真空夹紧 | 薄板/非导磁 | 小 | 安全 | 薄壁件 |

**螺旋夹紧力计算：**

```
W = Q · L / (r · tan(φ+α))
```

其中：
- W — 夹紧力（N）
- Q — 手柄施力（N）（工人一般施力100~150N）
- L — 手柄长度（mm）
- r — 螺纹中径（mm）
- φ — 螺纹摩擦角（约8.5°~10°）
- α — 螺纹螺旋角（约2.5°~4.5°）

**常用螺旋夹紧力速查：**

| 螺纹规格 | 扳手长度L(mm) | 施力Q(N) | 夹紧力W(N) |
|---------|-------------|---------|----------|
| M8 | 80 | 100 | ~1500 |
| M10 | 100 | 100 | ~2000 |
| M12 | 125 | 100 | ~2800 |
| M16 | 160 | 100 | ~4000 |
| M20 | 200 | 100 | ~5500 |
| M24 | 250 | 120 | ~7500 |

**偏心夹紧设计参数：**

| 偏心轮直径D(mm) | 偏心距e(mm) | 夹紧行程s(mm) | 夹紧力W(N)（Q=100N时） |
|----------------|-----------|------------|---------------------|
| 32 | 1.5 | 1.5 | ~1500 |
| 40 | 2.0 | 2.0 | ~1800 |
| 50 | 2.5 | 2.5 | ~2200 |
| 60 | 3.0 | 3.0 | ~2600 |
| 70 | 3.5 | 3.5 | ~3000 |
| 80 | 4.0 | 4.0 | ~3500 |

偏心夹紧自锁条件：
```
e / R ≤ f    （f为摩擦系数，钢对钢f=0.10~0.15）
```
一般取e/R ≤ 0.14，即偏心轮转角在30°~45°范围内自锁。

### 15.6 典型定位方案

### 15.7 夹具公差与精度
<!-- 来源: 第1卷  第1009页 -->

**夹具公差选择：**

| 工件公差IT | 定位件公差 | 导向件/刀具引导件公差 | 夹具体形位公差 |
|-----------|----------|--------------------|------------|
| IT6 | IT5 | IT5 | 0.01~0.015mm |
| IT7 | IT6 | IT5~IT6 | 0.015~0.025mm |
| IT8 | IT6~IT7 | IT6 | 0.02~0.03mm |
| IT9 | IT7 | IT6~IT7 | 0.03~0.05mm |
| IT10 | IT7~IT8 | IT7 | 0.05~0.08mm |
| IT11 | IT8 | IT7~IT8 | 0.08~0.12mm |

**关键原则：**
1. 夹具定位元件公差 = (1/3~1/5) × 工件相应公差
2. 夹具导向元件公差 = (1/3~1/5) × 加工精度
3. 夹具体基准面平面度 ≤ 0.02/300mm
4. 定位销中心距公差 = (1/3~1/5) × 孔距公差
5. V形块对称度 ≤ 0.02mm
