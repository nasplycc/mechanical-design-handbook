## 1. 焊接技术概论
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

### 1.1 焊接的定义与分类
<!-- 来源: 第1卷 第1篇一般设计资料 第262页 -->

焊接是通过加热或加压（或两者并用），使两分离的金属件在原子间形成结合力而连接成一体的加工方法。焊接是机械制造中最重要的连接工艺之一。

**焊接方法分类：**

| 大类 | 亚类 | 典型方法 | 热源类型 | 特点 |
|------|------|---------|---------|------|
| 熔焊 | 电弧焊 | 手工电弧焊、气保焊、埋弧焊 | 电弧热 | 通用性强 |
| 熔焊 | 高能束焊 | 激光焊、电子束焊 | 高能束流 | 高精度、高热密度 |
| 熔焊 | 气焊 | 氧乙炔焊 | 燃气火焰 | 薄板用 |
| 熔焊 | 电渣焊 | 电渣焊 | 电阻热 | 厚板用 |
| 压焊 | 电阻焊 | 点焊、缝焊、对焊 | 电阻热+压力 | 薄板、大批量 |
| 压焊 | 摩擦焊 | 旋转摩擦焊、搅拌摩擦焊 | 摩擦热+压力 | 同种/异种金属 |
| 压焊 | 扩散焊 | 真空扩散焊 | 加热+压力 | 精密异种金属 |
| 钎焊 | 硬钎焊 | 炉中钎焊、感应钎焊 | 加热至钎料熔化 | 异种金属 |
| 钎焊 | 软钎焊 | 电烙铁锡焊 | 加热至焊料熔化 | 电子行业 |

### 1.2 焊接结构的优势与局限性
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 优势 | 局限性 |
|------|--------|
| 重量轻，比铆接轻15%~20% | 产生焊接残余应力 |
| 密封性好，无缝连接 | 热影响区存在组织变化 |
| 结构设计灵活，可制造复杂形状 | 变形难以完全消除 |
| 节省材料，不需钻孔等 | 可焊性受材料限制 |
| 生产效率高 | 需专用设备和工艺评定 |
| 适应性强，现场施工方便 | 焊缝需无损检测 |

## 2. 金属的可焊性
<!-- 来源: 第1卷 第1篇一般设计资料 第274页 -->

### 2.1 可焊性定义
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

可焊性（焊接性）是指金属材料在一定的焊接工艺条件下，获得优质焊接接头的难易程度。包括工艺可焊性和使用可焊性两个方面。

| 类别 | 定义 | 评价指标 |
|------|------|---------|
| 工艺可焊性 | 材料对焊接工艺的适应性 | 裂纹敏感性、气孔倾向、成形性 |
| 使用可焊性 | 焊接接头满足使用性能的能力 | 强度、韧性、耐蚀性、疲劳寿命 |

### 2.2 碳当量法评价钢的可焊性
<!-- 来源: 第1卷 第1篇一般设计资料 第272页 -->

**国际焊接学会（IIW）推荐的碳当量公式：**

```
CE(IIW) = C + Mn/6 + (Cr + Mo + V)/5 + (Ni + Cu)/15
```

**碳当量与可焊性判断：**

| CE值 | 可焊性等级 | 焊接注意事项 |
|------|-----------|------------|
| ≤0.40% | 优良 | 一般不需预热，普通工艺即可 |
| 0.40%~0.50% | 良好 | 厚度≥25mm需适当预热（50~100°C） |
| 0.50%~0.60% | 中等 | 需预热100~200°C，控制热输入 |
| 0.60%~0.70% | 较差 | 必须预热150~250°C，严格控制工艺 |
| ≥0.70% | 差 | 预热200~350°C，需特殊工艺措施 |

| 钢种 | 代表牌号 | 碳当量CE(%) | 可焊性评价 | 是否需要预热 |

#### 2.4.3 不锈钢的焊接性
<!-- 来源: 第1卷 第1篇一般设计资料 第293页 -->

**奥氏体不锈钢（18-8型）：**

| 焊接问题 | 产生原因 | 防止措施 |
|---------|---------|---------|
| 热裂纹 | 低熔点共晶在晶界 | 控制S、P含量，采用低S焊丝 |
| 晶间腐蚀 | 敏化温度区Cr₂₃C₆析出 | 低碳焊材（≤0.03%C），或加Ti、Nb稳定化 |
| 应力腐蚀 | 残余拉伸应力+腐蚀介质 | 消除应力热处理 |
| 变形 | 热膨胀系数大 | 合理焊接顺序，刚性固定 |

**常见不锈钢焊接特点比较：**

| 不锈钢类型 | 焊接性 | 焊材选择 | 特殊要求 |
|-----------|--------|---------|---------|
| 奥氏体（304, 316） | 好 | 同类焊丝 | 控制热输入 |
| 奥氏体（321, 347） | 好 | 含稳定化元素焊材 | 不适于高温 |
| 铁素体（430） | 中 | 奥氏体或铁素体焊材 | 预热100~200°C |
| 马氏体（410, 420） | 差 | 同类焊材或奥氏体焊材 | 预热200~300°C+后热 |
| 双相不锈钢（2205） | 好 | 双相或超双相焊材 | 控制热输入和层间温度 |

## 4. 焊接材料及其选择

### 4.1 焊条分类及选用
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

**焊条分类（GB/T 5117-2012）：**

| 焊条型号 | 药皮类型 | 焊接位置 | 特点 | 典型用途 |
|---------|---------|---------|------|---------|
| E43系列（低碳钢） | 钛钙型 | 全位置 | 工艺性好 | 一般钢结构 |
| E43系列（低碳钢） | 低氢型 | 全位置 | 抗裂性好 | 重要结构 |
| E50系列（490MPa级） | 钛钙型 | 全位置 | 工艺性好 | 低合金钢 |
| E50系列（490MPa级） | 低氢型 | 全位置 | 抗裂性好 | 重要低合金钢结构 |
| E55系列（540MPa级） | 低氢型 | 平、角焊 | 高强度 | Q390, Q420 |
| E60系列（590MPa级） | 低氢型 | 平、角焊 | 高强度 | Q460, Q500 |

**焊条选用原则：**

| 选用原则 | 说明 |
|---------|------|
| 等强度原则 | 焊条强度≥母材强度（结构钢） |

## 6. 焊接结构设计注意事项
<!-- 来源: 第1卷 第1篇一般设计资料 第409页 -->

### 6.1 焊接结构设计总则
<!-- 来源: 第1卷 第1篇一般设计资料 第262页 -->

| 序号 | 设计原则 | 详细说明 |
|-----|---------|---------|
| 1 | 合理选材 | 优先选用可焊性良好的材料（CE≤0.45%） |
| 2 | 减少焊缝数量 | 尽量减少焊缝总长度，降低焊接应力和变形 |
| 3 | 避免焊缝集中 | 焊缝间距≥150mm或≥3倍板厚 |
| 4 | 避免三向交叉焊缝 | 三向交叉产生复杂残余应力 |
| 5 | 焊缝布置避开应力集中区 | 焊缝远离截面急剧变化处 |
| 6 | 厚薄件过渡 | 厚度差>4mm时需斜过渡（1:2.5~1:5） |
| 7 | 便于施焊操作 | 保证焊条/焊枪可达性 |
| 8 | 考虑焊缝无损检测空间 | 为探伤预留通道和空间 |
| 9 | 考虑变形控制 | 对称设计或采用反变形 |
| 10 | 留出热影响区过渡 | 相邻焊缝热影响区不重叠 |

### 6.2 焊接结构设计的具体要求
<!-- 来源: 第1卷 第1篇一般设计资料 第262页 -->

#### 6.2.1 避免焊缝过于集中
<!-- 来源: 第1卷 第5篇机械产品结构设计 第3页 -->

## 9. 常见焊接缺陷及质量控制

### 9.1 焊接缺陷分类
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 缺陷类别 | 缺陷名称 | 产生原因 | 允许程度(Ⅱ级焊缝) |

### 9.2 焊接缺陷的防止措施
<!-- 来源: 第1卷 第1篇一般设计资料 第760页 -->

| 缺陷类型 | 防止措施 |
|---------|---------|
| 热裂纹 | 控制焊缝形状（宽深比>1）、限制有害杂质（S,P）、选用抗裂性好的焊材 |
| 冷裂纹 | 焊前预热、控制氢含量（使用低氢焊条）、控制冷却速度、后热消氢 |
| 气孔 | 焊材烘烤（使用前350~400°C烘烤1h）、保护气体流量合适、坡口清理干净 |
| 未熔合 | 适当加大焊接电流、减小焊接速度、坡口角度足够、清理坡口两侧 |
| 未焊透 | 减小钝边、增大坡口角度或间隙、调整焊枪角度 |
| 咬边 | 减小焊接电流、调整电弧长度、焊条摆动幅度控制 |
| 夹渣 | 多层焊层间清渣、调整焊条角度、提高焊接速度 |
| 变形 | 反变形法、刚性固定法、对称焊接、合理焊接顺序 |

### 10.4 钢框架节点焊接设计
<!-- 来源: 第1卷 第1篇一般设计资料 第262页 -->

**H型钢梁柱节点：**

| 节点类型 | 受力特征 | 焊缝设计要求 |
|---------|---------|-------------|
| 梁翼缘-柱焊接 | 弯矩→翼缘轴向力 | 全熔透对接焊缝，100%UT |
| 梁腹板-柱焊接 | 剪力 | 角焊缝或熔透焊缝 |
| 梁-柱全焊接节点 | 弯矩+剪力 | 翼缘熔透+腹板角焊缝 |
| 栓焊混合节点 | 弯矩+剪力 | 翼缘熔透焊+腹板高强螺栓 |

**焊缝尺寸计算示例：**

```
已知：H500×200×10×16工字钢
弯矩 M = 300kN·m
剪力 V = 200kN

翼缘焊缝受力：
N = M / h₀ = 300 / 0.468 = 641kN
翼缘面积 A_f = 200×16 = 3200mm²
翼缘应力 σ = 641000 / 3200 = 200MPa < fᵥ^w = 215MPa ✓

腹板角焊缝高度：
k_f ≥ V / (2 × 0.7 × l_w × fᶠ^w)
= 200000 / (2 × 0.7 × 460 × 160) ≈ 1.94mm → 取k_f = 6mm
```

> **本章小结：** 焊接是机械制造中最重要的连接工艺之一，适用于制造复杂的结构件和大型构件。焊接结构设计应充分考虑材料的可焊性（碳当量≤0.45%优先）、合理选择焊接方法和焊接材料，优化焊缝布置以减少变形和残余应力。GB/T 19804-2005规定了焊接结构的尺寸公差和形位公差，JB/T 5000.3-2007规定了焊接件的通用技术条件。设计时应牢记"减少焊缝数量、避免焊缝集中、保证施焊可达、控制焊接变形"的核心准则。
