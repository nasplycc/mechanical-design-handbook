# 上篇：锻造

## 1. 锻造总论
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

### 1.1 锻造概述
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

锻造是利用冲击力或静压力使金属坯料产生塑性变形，获得规定形状和尺寸锻件的加工方法。锻造能消除金属的铸态组织（如气孔、疏松），使组织致密、纤维流线合理分布。

**锻造方法分类：**

| 分类方式 | 类别 | 特点 | 适用范围 |
|---------|------|------|---------|
| 按温度 | 热锻（再结晶温度以上） | 变形抗力小、塑性好 | 大中型锻件 |
| 按温度 | 温锻（200~800°C） | 表面氧化少、精度较高 | 中碳钢、合金钢 |
| 按温度 | 冷锻（室温） | 精度高、表面光洁、强度提高 | 小件、标准件 |
| 按成形方式 | 自由锻 | 通用性强、模具简单 | 单件小批、大型件 |
| 按成形方式 | 模锻 | 精度高、效率高 | 批量生产 |
| 按成形方式 | 胎模锻 | 介于自由锻和模锻之间 | 中批生产 |
| 按成形方式 | 轧锻 | 连续成形 | 轴类、长条形件 |

| 工艺 | 尺寸精度 | 表面粗糙度Ra(μm) | 单件成本(大批量) | 模具寿命(万件) |
|------|---------|----------------|----------------|--------------|

## 2. 金属材料的可锻性
<!-- 来源: 第1卷 第1篇一般设计资料 第232页 -->

### 2.1 可锻性定义与评价指标
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

可锻性是指金属材料进行塑性成形时的难易程度，是衡量材料锻造工艺性的综合指标。

| 评价指标 | 定义 | 影响因素 |
|---------|------|---------|
| 塑性 | 金属产生永久变形而不破坏的能力 | 化学成分、组织状态、温度、变形速度 |
| 变形抗力 | 金属抵抗变形的力 | 屈服强度、加工硬化、温度、变形速度 |
| 锻造温度范围 | 可进行锻造的最低~最高温度区间 | 熔化温度、再结晶温度、氧化温度 |
| 临界变形量 | 引起晶粒粗化的变形量范围 | 材料种类、原始晶粒度 |

### 2.2 常用金属的可锻性比较
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 金属材料 | 可锻性评价 | 锻造温度范围(°C) | 特点与注意事项 |
|---------|-----------|-----------------|---------------|
| 低碳钢（含碳量<0.25%） | 优良 | 800~1250 | 塑性好，变形抗力小 |
| 中碳钢（含碳量0.25~0.60%） | 良好 | 800~1220 | 塑性较好，变形抗力中等 |
| 高碳钢（含碳量>0.60%） | 中等 | 800~1150 | 塑性降低，注意控制冷却 |
| 合金结构钢（40Cr、35CrMo） | 良好 | 800~1200 | 变形抗力高于碳钢 |
| 不锈钢（1Cr18Ni9Ti） | 中等 | 850~1180 | 变形抗力大，塑性中等 |
| 高速钢（W18Cr4V） | 较差 | 950~1150 | 温度范围窄，需严格控制 |
| 铝合金（LY12、LF21） | 良好 | 350~500 | 温度范围窄，需预热模具 |
| 镁合金 | 中等 | 280~420 | 对低应变率敏感 |
| 铜合金（H62、H68） | 优良 | 650~900 | 塑性好，温度范围宽 |
| 钛合金（TC4） | 较差 | 800~1150 | 变形抗力大，β转变点以上须注意 |
| 高温合金（GH4169） | 差 | 950~1180 | 变形抗力极大，塑性差 |

### 2.4 锻造温度范围的确定
<!-- 来源: 第1卷 第1篇一般设计资料 第262页 -->

**锻粗温度（始锻温度）的确定原则：**
- 低于固相线100~200°C（防止过热和过烧）
- 对于高碳钢和高合金钢，留更大裕量
- 考虑偏析引起的局部低熔点区域

**终锻温度（终止锻造温度）的确定原则：**
- 高于再结晶温度50~100°C（保证再结晶充分进行）
- 保证有足够的塑性
- 防止晶粒粗化

**典型锻造温度范围：**

| 钢种 | 始锻温度(°C) | 终锻温度(°C) | 温度范围(°C) |
|------|------------|------------|------------|
| Q235(A3) | 1280 | 750 | 530 |
| 20钢 | 1260 | 750 | 510 |
| 45钢 | 1220 | 780 | 440 |
| 40Cr | 1200 | 800 | 400 |
| 35CrMo | 1200 | 800 | 400 |
| 60Si2Mn | 1180 | 800 | 380 |
| W18Cr4V | 1150 | 950 | 200 |
| 1Cr18Ni9Ti | 1180 | 850 | 330 |
| GC15 | 1100 | 800 | 300 |

### 2.5 合金元素对可锻性的影响
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 合金元素 | 对锻造的影响 | 注意事项 |
|---------|------------|---------|
| C | 提高变形抗力，降低塑性 | 高碳钢需延长加热时间 |
| Mn | 增加变形抗力 | 含量>1.5%时塑性下降明显 |
| Si | 大幅增加变形抗力 | 锻造困难 |
| Cr | 提高强度和变形抗力 | 中Cr含量影响不大 |
| Ni | 增加变形抗力，降低塑性 | 需较高锻造温度 |
| Mo | 提高高温强度 | 变形抗力明显增加 |
| W | 显著增加变形抗力 | 需窄温度范围锻造 |
| V | 细化晶粒 | 对可锻性影响较小 |
| Ti | 形成碳化物 | 表面易产生裂纹 |

## 3. 锻造零件结构要素
<!-- 来源: 第1篇 第3页 [前言] -->

### 3.1 模锻件的结构要素（GB/T 12361-2003）
<!-- 来源: 第1篇 第3页 [前言] -->

**GB/T 12361-2003《钢质模锻件 通用技术条件》规定的结构要素：**

#### 3.1.1 拔模斜度
<!-- 来源: 第1卷 第2篇机械制图 第38页 -->

| 锻造设备 | 外壁斜度α | 内壁斜度β | 说明 |
|---------|-----------|-----------|------|
| 锤上模锻 | 5°~7° | 7°~10° | α内壁>α外壁2°~3° |
| 热模锻压力机 | 3°~5° | 5°~7° | 可带顶出器时更小 |
| 螺旋压力机 | 3°~5° | 5°~7° | 同压力机 |
| 平锻机 | 1°~3° | 3°~5° | 水平分模 |
| 精密模锻 | 1°~3° | 1°~3° | 需喷润滑剂+顶出器 |

**拔模斜度标注：**

```
   ┌────────────┐
   │   α=5°    │  ─── 外壁斜度（较小）
   │   ╱        │
   │  ╱         │
   │ │   β=7°│  ─── 内壁斜度（较大）
   │ │        │
   └─┘────────┘
   原则：外壁斜度<内壁斜度
```

#### 3.1.2 模锻斜度与R角的对应关系
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 锻件深度(mm) | 外壁斜度α | 内壁斜度β | 圆角R外 | 圆角R内 |
|-------------|----------|----------|--------|--------|
| ≤25 | 3° | 5° | 1.5~4 | 3~8 |
| 26~50 | 5° | 7° | 2~5 | 4~10 |
| 51~100 | 5°~7° | 7°~10° | 3~6 | 5~12 |
| 101~150 | 7° | 10° | 4~8 | 6~14 |
| >150 | 7°~10° | 10°~12° | 5~10 | 8~16 |

#### 3.1.3 模锻件圆角半径（GB/T 12361-2003）
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

**外圆角半径r（mm）：**

| 锻件质量(kg) | 普通精度 | 较高精度 |
|-------------|---------|---------|
| ≤0.4 | 1.5 | 1.0 |
| 0.4~2.5 | 2.0 | 1.5 |
| 2.5~6.3 | 3.0 | 2.0 |
| 6.3~16.0 | 4.0 | 3.0 |
| 16.0~40.0 | 5.0 | 4.0 |
| >40.0 | 6.0 | 5.0 |

**内圆角半径R（mm）：** R = (2~3) × r

#### 3.1.4 模锻件的肋和凸起
<!-- 来源: 第1卷 第1篇一般设计资料 第241页 -->

| 肋厚b(mm) | 肋高h(mm) | 圆角R | 斜度α |
|----------|----------|------|-------|
| 8~12 | ≤30 | 2~4 | 5°~7° |
| 12~20 | ≤50 | 3~6 | 5°~7° |
| 20~30 | ≤75 | 4~8 | 3°~5° |
| 30~45 | ≤100 | 6~12 | 3°~5° |

**肋的纵横比（高/厚）限制：**

| 材料 | h/b_max(敞模) | h/b_max(闭模) |
|------|-------------|-------------|
| 碳钢 | 6:1 | 8:1 |
| 合金钢 | 4:1 | 6:1 |
| 不锈钢 | 3:1 | 4:1 |
| 铝合金 | 8:1 | 12:1 |

#### 3.1.5 过渡形状设计
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 过渡类型 | 设计要求 | 示意图 |
|---------|---------|--------|
| 壁厚变化 | 锥度过渡，锥角≤30° | 厚→薄渐变 |
| 截面变化 | 圆角过渡，R≥5mm | 圆滑过渡 |
| 凸台到基体 | 圆角R≥3mm，斜度≥5° | 圆角+斜度 |

## 4. 锻件设计注意事项
<!-- 来源: 第1卷 第1篇一般设计资料 第239页 -->

### 4.1 锻件结构设计总则
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 序号 | 设计原则 | 详细说明 |
|-----|---------|---------|
| 1 | 分模面简化 | 分模面应选择在锻件最大截面上，尽量为平面 |
| 2 | 合理分模 | 分模面数目尽量少（通常1个分模面） |
| 3 | 形状简化 | 锻件形状应有利于金属流动，避免急剧截面变化 |
| 4 | 均匀变形 | 避免过大的截面差异，保证金属均匀流动 |
| 5 | 避免薄壁高筋 | 筋的高度与厚度比不宜过大 |
| 6 | 拔模斜度充足 | 足够的拔模斜度保证模具寿命和锻件质量 |
| 7 | 圆角半径足够 | 圆角太小导致应力集中和模具早期疲劳 |
| 8 | 避免内侧凹陷 | 内侧凹陷导致无法脱模 |
| 9 | 考虑收缩补偿 | 热锻件需考虑冷却收缩（钢1.2%~1.5%） |
| 10 | 加工余量合理 | 加工面预留合理的加工余量 |

### 4.2 模锻件设计注意事项
<!-- 来源: 第1篇 第3页 [前言] -->

| 注意事项 | 优化设计 | 不良设计 |
|---------|---------|---------|
| 避免过大的截面变化 | 逐级过渡，锥度≤30° | 突然变径产生折叠 |
| 避免锐角 | 外角r≥1mm，内角R≥3mm | 尖角导致应力集中 |
| 筋的高度限制 | h/b≤6（碳钢） | 筋过高难以充满 |
| 避免薄壁（<4mm） | ≥4mm | 冷却快，不易充满 |
| 凸台间距合理 | ≥8mm | 间距过小模具强度不足 |
| 冲孔连皮最小厚度 | 6~15mm（依孔径） | 太薄冲穿困难 |
| 锻件重量范围 | 经济重量≤2.5kg（锤上模锻） | 大件需自由锻 |
| 复杂锻件的分模 | 可采用多向模锻或预锻+精锻 | 一次成形不可行 |

### 4.3 自由锻件设计注意事项
<!-- 来源: 第1篇 第3页 [前言] -->

| 注意事项 | 说明 |
|---------|------|
| 避免锥面和楔形 | 自由锻难实现锥面成形 |
| 避免加强筋 | 自由锻无法锻出小筋 |
| 避免复杂曲面 | 自由锻以圆柱面、平面为主 |
| 避免台阶过多 | 台阶数≤3~4个 |
| 合理选用锻造比 | 碳钢锻造比≥2.5，合金钢≥3 |
| 注意纤维流向 | 主要受力方向与纤维方向一致 |
| 避免过大的法兰 | 法兰直径与杆径比≤2.5:1 |
| 尽量采用回转体型 | 自由锻适合轴、盘、环类 |

### 4.5 锻件的热处理
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 热处理类型 | 适用材料 | 目的 | 典型工艺 |
|-----------|---------|------|---------|
| 退火 | 中高碳钢、合金钢 | 降低硬度、消除应力 | 加热到Ac1+20~30°C，缓冷 |

## 6. 冷冲压总论
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

### 6.1 冷冲压概述
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

冷冲压是在室温下利用冲压设备和模具，使金属板料产生塑性变形或分离，获得所需工件的一种工艺方法。

**冷冲压基本工序分类：**

| 工序分类 | 工序名称 | 特点 | 应用 |
|---------|---------|------|------|
| 分离工序 | 冲裁（冲孔、落料） | 板料完全分离 | 各种冲裁件 |
| 分离工序 | 切边 | 切去多余的边缘 | 拉深件切边 |
| 分离工序 | 切口 | 部分切开，部分不切开 | 通风孔、卡扣 |
| 分离工序 | 修边 | 提高切口质量 | 精冲件 |
| 成形工序 | 弯曲 | 平面→空间折弯 | 支架、折边件 |
| 成形工序 | 拉深 | 平板→空间空心体 | 壳体、盖类 |
| 成形工序 | 翻边 | 孔或边缘翻起 | 螺纹底孔、法兰 |
| 成形工序 | 胀形 | 局部扩大成形 | 加强筋、装饰纹 |
| 成形工序 | 缩口 | 缩小开口尺寸 | 管件收口 |
| 成形工序 | 整形 | 进一步提高精度 | 精整工序 |
| 复合工序 | 冲裁拉深复合 | 在拉深过程中冲裁 | 复合模一次成型 |
| 连续工序 | 多工位级进 | 多工序连续完成 | 大批量冲压件 |

### 6.2 冲压工艺特点
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 比较项目 | 冷冲压 | 热冲压 |
|---------|--------|--------|
| 工艺温度 | 室温 | 加热至再结晶温度以上 |
| 表面质量 | 好（无氧化皮） | 较差（有氧化皮） |
| 精度等级 | IT8~IT11 | IT11~IT14 |
| 材料强度 | 受加工硬化影响 | 可充分利用强度 |
| 成形力 | 较大 | 较小（变形抗力降低） |
| 回弹 | 较大，难控制 | 较小，易控制 |
| 模具寿命 | 长（50~200万次） | 较短（2~10万次） |
| 典型材料 | 低碳钢、铝、铜 | 热成形钢（22MnB5） |

## 11. 冷挤压件结构要素

### 11.1 冷挤压件特点
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

冷挤压是在室温下利用模具的冲压力使金属产生塑性变形，形成所需形状的成形工艺。

| 冷挤压类型 | 成形方式 | 典型零件 | 尺寸精度 |
|-----------|---------|---------|---------|
| 正挤压 | 金属流动方向与冲头运动方向相同 | 轴类、管类、铆钉 | IT7~IT9 |
| 反挤压 | 金属流动方向与冲头运动方向相反 | 杯形件、壳体 | IT8~IT10 |
| 复合挤压 | 正反挤压同时进行 | 复杂形体 | IT8~IT9 |
| 径向挤压 | 金属向径向流动 | 齿轮、花键 | IT7~IT9 |

### 11.2 冷挤压件的结构参数
<!-- 来源: 第1卷 第2篇机械制图 第165页 -->

| 正挤压（空心） | ≥2 | ≤10 | 需润滑 |
| 反挤压（杯形） | ≥(0.05~0.1)d | ≤2.5 | 钢材 |
| 反挤压（杯形） | ≥(0.02~0.05)d | ≤3.5 | 有色金属 |
| 复合挤压 | ≥(0.08~0.12)d | ≤3 | 钢材 |

## 13. 拉深工艺设计
<!-- 来源: 第1卷 第1篇一般设计资料 第264页 -->

### 13.1 拉深概述
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

拉深（拉延）是利用拉深模将平板毛坯冲压成开口空心件的工序。拉深是冲压工艺中最复杂的成形方式之一，涉及材料的塑性流动、壁厚变化、起皱控制等复杂问题。

**拉深件分类：**

| 分类 | 特点 | 示例 |
|------|------|------|
| 圆筒形件 | 轴对称，壁面为圆柱面 | 杯、壳、罐 |
| 直壁类 | 非圆截面，直壁 | 方盒、矩形件 |
| 曲面类 | 曲面形状 | 汽车门板、翼子板 |
| 阶梯形 | 直径多次变化 | 阶梯筒 |
| 锥形件 | 壁面为锥面 | 锥形罩、喇叭壳 |
| 球形件 | 半球或部分球面 | 半球壳 |

### 13.2 拉深工艺参数
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 参数 | 定义 | 影响因素 |
|------|------|---------|
| 拉深系数 m | d/D（d为工件直径，D为毛坯直径） | 材料塑性、料厚、润滑、模具参数 |
| 拉深比 K | 1/m = D/d | 与拉深系数互为倒数 |
| 极限拉深系数 mₘᵢₙ | 一次拉深不破裂的最大变形量 | 材料、相对厚度、模具圆角 |
| 相对厚度 δ/D×100 | 料厚与毛坯直径比 | 影响起皱和拉深极限 |
| 凸缘宽度 b | (D-d)/2 | 影响压边力 |
| 拉深力 F | 所需冲压力 | 材料强度、拉深比、摩擦力 |

## 18. 压边拉深的条件
<!-- 来源: 第1卷 第1篇一般设计资料 第265页 -->

### 18.1 压边拉深的判定条件
<!-- 来源: 第1卷 第3篇常用机械工程材料 第500页 -->

**判断是否需要压边：**

根据毛坯的相对厚度δ/D×100和拉深系数m决定是否需要压边装置。

**需要压边的条件（首次拉深）：**

| 材料 | 相对厚度δ/D×100 | 拉深系数m | 结论 |
|------|----------------|---------|------|
| 低碳钢 | <1.5 | 任何m | 必须压边 |
| 低碳钢 | 1.5~2.0 | <0.60 | 必须压边 |
| 低碳钢 | >2.0 | <0.58 | 推荐压边 |
| 有色金属 | <1.2 | 任何m | 必须压边 |
| 有色金属 | 1.2~1.8 | <0.55 | 必须压边 |

**压边判断经验公式：**

```
是否需要压边 = δ/D × 100 < 1.5  → 必须压边
               δ/D × 100 = 1.5~2.0  → 结合m判断
               δ/D × 100 > 2.0  → 可不压边
```

### 18.2 后续拉深的压边条件
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 相对厚度δ/dₙ₋₁×100 | 拉深系数mₙ | 压边判断 |
|-------------------|-----------|---------|
| >1.0 | >0.80 | 可不压边 |
| >1.0 | 0.70~0.80 | 推荐压边 |
| 0.5~1.0 | >0.75 | 可不压边 |
| 0.5~1.0 | ≤0.75 | 必须压边 |
| <0.5 | 任何mₙ | 必须压边 |

## 19. 压边载荷计算
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

### 19.1 压边力计算公式
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

**压边力的一般公式：**

```
Fᵧ = A × p
```

其中：
- Fᵧ — 压边力（N）
- A — 压边面积（mm²）
- p — 单位压边力（MPa）

**首次拉深压边面积：**

| 拉深类型 | 压边面积A(mm²) |
|---------|--------------|
| 无凸缘筒形件 | A = π/4[D² - (d₁+2r)²] |
| 有凸缘筒形件 | A = π/4[D² - (dᵧ)²] |

### 19.2 单位压边力推荐值
<!-- 来源: 第1卷 第1篇一般设计资料 第118页 -->

| 材料 | 单位压边力p(MPa) | 说明 |
|------|----------------|------|
| 低碳钢（08F, 08Al） | 1.5~2.5 | δ≥0.5mm |
| 中碳钢 | 2.0~3.0 | 较高强度 |
| 不锈钢 | 2.5~3.5 | 变形抗力大 |
| 铝合金（LF21） | 1.0~1.8 | 塑性好 |
| 铝合金（LY12） | 2.0~2.5 | 强度高 |
| 黄铜（H62） | 1.5~2.0 | 塑性好 |
| 紫铜（T2） | 1.0~1.8 | 塑性极好 |
| 镁合金 | 1.0~1.5 | 需预热 |
| 钛合金 | 3.0~4.0 | 变形抗力大 |

### 19.3 压边力经验选取
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

| 拉深情况 | 压边力Fᵧ | 说明 |
|---------|---------|------|
| 浅拉深 | Fᵧ = 0.25~0.35F | F为拉深力 |
| 中深拉深 | Fᵧ = 0.35~0.50F | 需平衡起皱和破裂 |

## 21. 压机能力选择
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

### 21.1 压机吨位计算
<!-- 来源: 第1卷 第2篇机械制图 第155页 -->

**总压力计算：**

```
F总 = F拉 + F压边
```

其中：
- F总 — 压机所需总压力（N）
- F拉 — 拉深力（N）
- F压边 — 压边力（N）

**拉深力近似计算：**

```
F拉 = K × π × d × δ × σb
```

| 参数 | 含义 | 说明 |
|------|------|------|
| K | 系数（0.6~1.2） | 首次拉深取大值，后续取小值 |
| d | 工件直径(mm) | 首次拉深为d₁ |
| δ | 板料厚度(mm) | 原始厚度 |
| σb | 材料抗拉强度(MPa) | 查材料手册 |

### 21.2 系数K的选取
<!-- 来源: 第1卷 第1篇一般设计资料 第259页 -->

| 拉深情况 | K值 | 说明 |
|---------|------|------|
| 首次拉深（无凸缘） | 0.9~1.1 | m值较小时取大值 |
| 后续拉深 | 0.7~0.9 | 变形量减小 |
| 有凸缘拉深 | 1.0~1.2 | 凸缘约束增加 |
| 宽凸缘拉深 | 1.1~1.3 | 压边力大 |
| 拉深力精确计算 | 按公式计算 | 详见冲压手册 |

|---------|-------------|-------------|-------------|---------------|

> **本章小结：** 锻造和冲压是机械制造中最重要的塑性成形工艺。锻造通过改变金属内部组织获得优良力学性能，适用于承受重载的关键零件；冲压以高效率和低成本实现大批量薄壁件的生产。拉深作为最复杂的冲压工序之一，需要综合考虑材料特性、工艺参数和模具设计。结构设计时应遵循"合理选材、均匀变形、控制缺陷、简化模具"的核心准则。
