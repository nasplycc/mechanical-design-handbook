## 3. 标准件选型流程

### 3.1 螺栓连接选型

> **流程：** 确定载荷 → 选择强度等级 → 计算直径 → 确定预紧力 → 选型

| 步骤 | 内容 | 方法 |
|------|------|------|
| 1. 确定载荷 | 轴向力 F_a / 横向力 F_t | 工作载荷分析 |
| 2. 选择强度等级 | 根据强度要求：4.6~12.9级 | GB/T 3098.1 |
| 3. 计算最小直径 | d ≥ √(4F_s / (π[σ])) | 强度计算 |
| 4. 确定预紧力 | F_v = (0.5~0.7)σ_s × A_s | 重要连接：按扭矩法 |
| 5. 确定拧紧扭矩 | T = K × F_v × d | K=0.18~0.22（摩擦系数） |

### 3.2 滚动轴承选型

> **流程：** 载荷方向/大小 → 转速 → 精度等级 → 寿命计算 → 尺寸确定

| 步骤 | 内容 | 方法/公式 |
|------|------|----------|
| 1. 确定载荷 | 径向力 F_r、轴向力 F_a | 力分析 |
| 2. 确定转速 n | 工作转速 r/min | 实际工况 |
| 3. 选择轴承类型 | 见下表 | 按载荷方向、转速 |
| 4. 计算当量动载荷 P | P = X·F_r + Y·F_a | 查轴承手册 |
| 5. 计算所需额定动载荷 C | C = P × (L_h·60n/10⁶)^(1/ε) | 寿命公式L_h=(C/P)^ε×10⁶/60n |
| 6. 确定轴承尺寸 | 按C值选 | 查轴承手册 |
| 7. 校核静强度 | C_0 ≥ S_0 × P_0 | S_0为静强度安全系数 |
| 8. 确定精度等级 | P0（普通）~P2（精密） | GB/T 307.3 |

### 8.5 安全系数选用原则速查

| 条件 | 建议 | 理由 |
|------|------|------|
| 材料韧性好 | 取小 | 断裂前有变形预警 |
| 材料脆性 | 取大 | 无塑性变形预警 |
| 零件重要性高 | 取大 | 失效后果严重 |
| 计算依据充分 | 取小 | 载荷、应力计算准确 |
| 计算依据不确切 | 取大 | 安全余量补偿 |
| 工作环境恶劣 | 取大 | 腐蚀、高温、磨损 |
| 制造质量好（探伤合格） | 取小 | 无内部缺陷 |
| 批量生产（质量控制好） | 取小 | 一致性高 |
| 定期无损检测 | 可取小 | 可监控缺陷扩展 |
| 考虑重量控制（航空） | 取小 | 减重需求 |
